# 31.7 日期时间 API：为什么 Date 被废弃？

## 一、先问一个问题

**为什么 Java 8 要引入新的日期时间 API？**

### 生活例子：老式钟表 vs 智能手表

```
老式钟表（Date/Calendar）
├── 难用
├── 易错
└── 线程不安全

智能手表（Java 8 Date Time）
├── 好用
├── 直观
└── 线程安全
```

**旧 API 的问题**：
- 设计糟糕（月份从 0 开始）
- 可变且线程不安全
- 时区处理复杂

## 二、核心类

### 1. LocalDate（日期）

```java
// 创建日期
LocalDate today = LocalDate.now();
LocalDate date = LocalDate.of(2024, 1, 15);
LocalDate date2 = LocalDate.parse("2024-01-15");

// 获取信息
int year = date.getYear();           // 2024
int month = date.getMonthValue();    // 1
int day = date.getDayOfMonth();      // 15
DayOfWeek dow = date.getDayOfWeek(); // MONDAY

// 日期计算
LocalDate tomorrow = today.plusDays(1);
LocalDate nextWeek = today.plusWeeks(1);
LocalDate nextMonth = today.plusMonths(1);
LocalDate nextYear = today.plusYears(1);

LocalDate yesterday = today.minusDays(1);

// 日期比较
boolean isAfter = date1.isAfter(date2);
boolean isBefore = date1.isBefore(date2);
boolean isEqual = date1.isEqual(date2);

// 计算间隔
Period period = Period.between(date1, date2);
int days = period.getDays();
int months = period.getMonths();
int years = period.getYears();
```

### 2. LocalTime（时间）

```java
// 创建时间
LocalTime now = LocalTime.now();
LocalTime time = LocalTime.of(14, 30, 0);
LocalTime time2 = LocalTime.parse("14:30:00");

// 获取信息
int hour = time.getHour();      // 14
int minute = time.getMinute();  // 30
int second = time.getSecond();  // 0

// 时间计算
LocalTime later = time.plusHours(2);
LocalTime earlier = time.minusMinutes(30);

// 时间比较
boolean isAfter = time1.isAfter(time2);
```

### 3. LocalDateTime（日期时间）

```java
// 创建日期时间
LocalDateTime now = LocalDateTime.now();
LocalDateTime dt = LocalDateTime.of(2024, 1, 15, 14, 30, 0);
LocalDateTime dt2 = LocalDateTime.parse("2024-01-15T14:30:00");

// 组合创建
LocalDate date = LocalDate.of(2024, 1, 15);
LocalTime time = LocalTime.of(14, 30);
LocalDateTime dt3 = date.atTime(time);

// 获取信息
LocalDate datePart = dt.toLocalDate();
LocalTime timePart = dt.toLocalTime();

// 日期时间计算
LocalDateTime later = dt.plusDays(1).plusHours(2);
LocalDateTime earlier = dt.minusWeeks(1);

// 格式化
DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss");
String formatted = dt.format(formatter);
LocalDateTime parsed = LocalDateTime.parse("2024-01-15 14:30:00", formatter);
```

### 4. Instant（时间戳）

```java
// 获取当前时间戳
Instant now = Instant.now();

// 创建时间戳
Instant instant = Instant.parse("2024-01-15T14:30:00Z");

// 时间戳转换
long epochSecond = now.getEpochSecond();  // 秒级时间戳
long epochMilli = now.toEpochMilli();     // 毫秒级时间戳

// 从时间戳创建
Instant instant2 = Instant.ofEpochSecond(epochSecond);
Instant instant3 = Instant.ofEpochMilli(epochMilli);

// 与 LocalDateTime 转换
LocalDateTime ldt = LocalDateTime.now();
Instant instant4 = ldt.atZone(ZoneId.systemDefault()).toInstant();

LocalDateTime ldt2 = instant4.atZone(ZoneId.systemDefault()).toLocalDateTime();
```

## 三、时区处理

### ZoneId（时区）

```java
// 获取所有时区
Set<String> zoneIds = ZoneId.getAvailableZoneIds();

// 创建时区
ZoneId shanghai = ZoneId.of("Asia/Shanghai");
ZoneId tokyo = ZoneId.of("Asia/Tokyo");
ZoneId ny = ZoneId.of("America/New_York");

// 系统默认时区
ZoneId systemDefault = ZoneId.systemDefault();
```

### ZonedDateTime（带时区的日期时间）

```java
// 创建带时区的日期时间
ZonedDateTime zdt = ZonedDateTime.now();
ZonedDateTime zdt2 = ZonedDateTime.now(ZoneId.of("Asia/Shanghai"));

// LocalDateTime 转 ZonedDateTime
LocalDateTime ldt = LocalDateTime.now();
ZonedDateTime zdt3 = ldt.atZone(ZoneId.of("Asia/Shanghai"));

// Instant 转 ZonedDateTime
Instant instant = Instant.now();
ZonedDateTime zdt4 = instant.atZone(ZoneId.of("Asia/Shanghai"));

// 时区转换
ZonedDateTime tokyoTime = zdt4.withZoneSameInstant(ZoneId.of("Asia/Tokyo"));
ZonedDateTime nyTime = zdt4.withZoneSameInstant(ZoneId.of("America/New_York"));

System.out.println("上海：" + zdt4);
System.out.println("东京：" + tokyoTime);
System.out.println("纽约：" + nyTime);
```

### OffsetDateTime（带偏移量的日期时间）

```java
// 创建带偏移量的日期时间
OffsetDateTime odt = OffsetDateTime.now();
OffsetDateTime odt2 = OffsetDateTime.now(ZoneOffset.of("+8"));

// LocalDateTime 转 OffsetDateTime
LocalDateTime ldt = LocalDateTime.now();
OffsetDateTime odt3 = ldt.atOffset(ZoneOffset.of("+8"));
```

## 四、格式化和解析

### DateTimeFormatter（格式化器）

```java
// 内置格式化器
DateTimeFormatter isoDate = DateTimeFormatter.ISO_DATE;
DateTimeFormatter isoDateTime = DateTimeFormatter.ISO_DATE_TIME;
DateTimeFormatter ofPattern = DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss");

// 格式化
LocalDateTime now = LocalDateTime.now();
String formatted = now.format(ofPattern);

// 解析
LocalDateTime parsed = LocalDateTime.parse("2024-01-15 14:30:00", ofPattern);

// 自定义格式化器
DateTimeFormatter customFormatter = new DateTimeFormatterBuilder()
    .appendPattern("yyyy")
    .appendLiteral("年")
    .appendPattern("MM")
    .appendLiteral("月")
    .appendPattern("dd")
    .appendLiteral("日")
    .toFormatter();

String chineseDate = now.format(customFormatter);  // 2024 年 01 月 15 日
```

### 格式化模式

| 符号 | 含义 | 示例 |
|------|------|------|
| y | 年 | 2024 |
| M | 月 | 01 |
| d | 日 | 15 |
| H | 时（24 小时） | 14 |
| h | 时（12 小时） | 2 |
| m | 分 | 30 |
| s | 秒 | 45 |
| S | 毫秒 | 123 |
| E | 星期 | 周一 |
| z | 时区 | CST |

## 五、实用工具类

### 创建工具类

```java
public class DateUtils {
    
    // 格式化日期
    public static String formatDate(LocalDate date) {
        return date.format(DateTimeFormatter.ofPattern("yyyy-MM-dd"));
    }
    
    // 格式化日期时间
    public static String formatDateTime(LocalDateTime dateTime) {
        return dateTime.format(DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss"));
    }
    
    // 解析日期
    public static LocalDate parseDate(String dateStr) {
        return LocalDate.parse(dateStr, DateTimeFormatter.ofPattern("yyyy-MM-dd"));
    }
    
    // 解析日期时间
    public static LocalDateTime parseDateTime(String dateTimeStr) {
        return LocalDateTime.parse(dateTimeStr, DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss"));
    }
    
    // 计算年龄
    public static int calculateAge(LocalDate birthday) {
        return Period.between(birthday, LocalDate.now()).getYears();
    }
    
    // 判断是否周末
    public static boolean isWeekend(LocalDate date) {
        DayOfWeek dow = date.getDayOfWeek();
        return dow == DayOfWeek.SATURDAY || dow == DayOfWeek.SUNDAY;
    }
    
    // 获取本月第一天
    public static LocalDate getFirstDayOfMonth(LocalDate date) {
        return date.with(TemporalAdjusters.firstDayOfMonth());
    }
    
    // 获取本月最后一天
    public static LocalDate getLastDayOfMonth(LocalDate date) {
        return date.with(TemporalAdjusters.lastDayOfMonth());
    }
    
    // 获取下个工作日
    public static LocalDate getNextWorkday(LocalDate date) {
        return date.with(TemporalAdjusters.next(DayOfWeek.MONDAY));
    }
}
```

## 六、旧 API 转换

### Date 转 LocalDate

```java
// Date → LocalDate
Date date = new Date();
LocalDate localDate = date.toInstant()
    .atZone(ZoneId.systemDefault())
    .toLocalDate();

// LocalDate → Date
LocalDate localDate = LocalDate.now();
Date date = Date.from(localDate.atStartOfDay(ZoneId.systemDefault()).toInstant());
```

### Calendar 转 ZonedDateTime

```java
// Calendar → ZonedDateTime
Calendar calendar = Calendar.getInstance();
ZonedDateTime zdt = calendar.toInstant()
    .atZone(ZoneId.systemDefault());

// ZonedDateTime → Calendar
ZonedDateTime zdt = ZonedDateTime.now();
Calendar calendar = GregorianCalendar.from(zdt);
```

## 七、小结

| 类 | 用途 | 示例 |
|------|------|------|
| LocalDate | 日期 | 2024-01-15 |
| LocalTime | 时间 | 14:30:00 |
| LocalDateTime | 日期时间 | 2024-01-15 14:30:00 |
| Instant | 时间戳 | 毫秒级时间戳 |
| ZonedDateTime | 带时区日期时间 | 2024-01-15 14:30:00+08:00[Asia/Shanghai] |
| DateTimeFormatter | 格式化器 | yyyy-MM-dd HH:mm:ss |

**核心要点**：
- 不可变、线程安全
- API 设计直观
- 时区处理简单
- 格式化灵活

## 八、常见问题

### Q1：为什么 LocalDate 的月份从 1 开始？

**改进**：
```java
// 旧 API（月份从 0 开始，反人类）
Calendar calendar = Calendar.getInstance();
calendar.set(2024, 0, 15);  // 0 表示 1 月，容易错

// 新 API（月份从 1 开始，直观）
LocalDate date = LocalDate.of(2024, 1, 15);  // 1 就是 1 月
```

### Q2：如何处理时区转换？

```java
// 上海时间的会议
ZonedDateTime shanghaiMeeting = ZonedDateTime.of(
    2024, 1, 15, 14, 30, 0, 0,
    ZoneId.of("Asia/Shanghai")
);

// 转换为东京时间
ZonedDateTime tokyoMeeting = shanghaiMeeting.withZoneSameInstant(ZoneId.of("Asia/Tokyo"));

// 转换为纽约时间
ZonedDateTime nyMeeting = shanghaiMeeting.withZoneSameInstant(ZoneId.of("America/New_York"));
```

### Q3：如何计算两个日期之间的工作日？

```java
public long countWorkdays(LocalDate start, LocalDate end) {
    long workdays = 0;
    LocalDate current = start;
    
    while (!current.isAfter(end)) {
        DayOfWeek dow = current.getDayOfWeek();
        if (dow != DayOfWeek.SATURDAY && dow != DayOfWeek.SUNDAY) {
            workdays++;
        }
        current = current.plusDays(1);
    }
    
    return workdays;
}
```
