# 7.4 其他新特性

## 一、接口默认方法和静态方法

### 1.1 默认方法（Java 8）

```java
interface Animal {
    // 抽象方法
    void makeSound();
    
    // 默认方法
    default void breathe() {
        System.out.println("呼吸");
    }
    
    // 默认方法可以调用其他方法
    default void live() {
        breathe();
        makeSound();
    }
}

class Dog implements Animal {
    @Override
    public void makeSound() {
        System.out.println("汪汪叫");
    }
    
    // 可以重写默认方法
    @Override
    public void breathe() {
        System.out.println("狗用肺呼吸");
    }
}
```

### 1.2 静态方法（Java 8）

```java
interface MathUtils {
    // 静态方法
    static int max(int a, int b) {
        return a > b ? a : b;
    }
    
    static int min(int a, int b) {
        return a < b ? a : b;
    }
}

// 使用
int max = MathUtils.max(10, 20);
```

### 1.3 多重继承冲突解决

```java
interface A {
    default void show() {
        System.out.println("A");
    }
}

interface B {
    default void show() {
        System.out.println("B");
    }
}

// 实现多个有相同默认方法的接口
class C implements A, B {
    // 必须重写解决冲突
    @Override
    public void show() {
        // 可以选择调用其中一个
        A.super.show();  // 调用 A 的默认方法
        // B.super.show();  // 或调用 B 的
        System.out.println("C");
    }
}
```

## 二、方法引用

### 2.1 静态方法引用

```java
// Lambda
Function<String, Integer> f1 = s -> Integer.parseInt(s);

// 方法引用
Function<String, Integer> f2 = Integer::parseInt;
```

### 2.2 实例方法引用

```java
// Lambda
Function<String, Integer> f1 = s -> s.length();

// 方法引用
Function<String, Integer> f2 = String::length;

// 使用
Integer len = f2.apply("Hello");  // 5
```

### 2.3 对象方法引用

```java
String prefix = "Hello ";

// Lambda
Function<String, String> f1 = s -> prefix.concat(s);

// 对象方法引用
Function<String, String> f2 = prefix::concat;
```

### 2.4 构造方法引用

```java
// Lambda
Supplier<StringBuilder> s1 = () -> new StringBuilder();

// 构造方法引用
Supplier<StringBuilder> s2 = StringBuilder::new;

// 带参数的构造方法引用
Function<String, StringBuilder> f = StringBuilder::new;
StringBuilder sb = f.apply("Hello");
```

### 2.5 数组构造方法引用

```java
Function<Integer, String[]> f = String[]::new;
String[] arr = f.apply(10);  // 创建长度为 10 的数组
```

## 三、重复注解（Java 8）

### 3.1 定义

```java
// 容器注解
@interface Hints {
    Hint[] value();
}

// 可重复注解
@Repeatable(Hints.class)
@interface Hint {
    String value();
}
```

### 3.2 使用

```java
@Hint("hint1")
@Hint("hint2")
class Person {
}

// 获取注解
Hints hints = Person.class.getAnnotation(Hints.class);
for (Hint hint : hints.value()) {
    System.out.println(hint.value());
}
```

## 四、类型注解（Java 8）

```java
// 类型注解可以应用于任何类型使用的位置
List<@NonNull String> list = new ArrayList<>();

public void process(@NotNull String input) {
    // ...
}

class MyClass {
    @NonNull String name;
}
```

## 五、Java 9 新特性

### 5.1 接口私有方法

```java
interface Calculator {
    // 私有方法（Java 9+）
    private int validate(int a, int b) {
        if (a < 0 || b < 0) {
            throw new IllegalArgumentException("负数");
        }
        return a + b;
    }
    
    // 私有静态方法（Java 9+）
    private static int square(int x) {
        return x * x;
    }
    
    default int add(int a, int b) {
        validate(a, b);
        return a + b;
    }
}
```

### 5.2 try-with-resources 改进

```java
// Java 7+ 需要声明变量
try (FileInputStream fis = new FileInputStream("file.txt")) {
    // ...
}

// Java 9+ 可以使用已存在的变量
FileInputStream fis = new FileInputStream("file.txt");
try (fis) {
    // ...
}
```

### 5.3 Diamond 运算符改进

```java
// Java 7-8
interface MyInterface {
    void method();
}

MyInterface myInterface = new MyInterface() {};

// Java 9+ 可以在匿名类中使用 diamond
List<String> list = new ArrayList<>();
```

## 六、Java 10 新特性

### 6.1 var 局部变量类型推断

```java
// Java 9 及以前
String name = "张三";
List<String> list = new ArrayList<>();

// Java 10+
var name = "张三";
var list = new ArrayList<String>();

// 注意
// var x;  // 错误：必须初始化
// var x = null;  // 错误：无法推断类型
// var x = new int[]{1, 2, 3};  // 正确
```

### 6.2 不可变集合

```java
// Java 9+
List<String> list = List.of("A", "B", "C");
Set<Integer> set = Set.of(1, 2, 3);
Map<String, Integer> map = Map.of("A", 1, "B", 2);

// 不可变，修改抛异常
// list.add("D");  // UnsupportedOperationException
```

## 七、Java 11 新特性

### 7.1 String 新方法

```java
// isBlank
"   ".isBlank();  // true

// lines
"Hello\nWorld".lines().count();  // 2

// strip（比 trim 支持 Unicode）
"  Hello  ".strip();  // "Hello"
"  Hello  ".stripLeading();  // "Hello  "
"  Hello  ".stripTrailing();  // "  Hello"

// repeat
"Ha".repeat(3);  // "HaHaHa"
```

### 7.2 Optional 新方法

```java
// isEmpty（Java 11）
Optional<String> opt = Optional.empty();
opt.isEmpty();  // true

// ifPresentOrElse（Java 9）
opt.ifPresentOrElse(
    System.out::println,
    () -> System.out.println("空")
);
```

### 7.3 Predicate 新方法

```java
Predicate<String> notNull = Predicate.not(Objects::isNull);
notNull.test("Hello");  // true
```

## 八、Java 12-17 新特性

### 8.1 switch 表达式（Java 14）

```java
// 传统 switch
int result;
switch (day) {
    case MONDAY:
    case FRIDAY:
        result = 1;
        break;
    default:
        result = 0;
}

// switch 表达式（Java 14+）
int result = switch (day) {
    case MONDAY, FRIDAY -> 1;
    default -> 0;
};

// 返回值
String type = switch (day) {
    case SATURDAY, SUNDAY -> "周末";
    default -> "工作日";
};
```

### 8.2 文本块（Java 15）

```java
// Java 11 及以前
String json = "{\n" +
    "  \"name\": \"张三\",\n" +
    "  \"age\": 18\n" +
    "}";

// Java 15+ 文本块
String json = """
    {
      "name": "张三",
      "age": 18
    }
    """;

// SQL
String sql = """
    SELECT * FROM users
    WHERE age > 18
    ORDER BY name
    """;
```

### 8.3 record 类（Java 16）

```java
// Java 16+ record
public record Point(int x, int y) {}

// 使用
Point p = new Point(10, 20);
int x = p.x();  // 10
int y = p.y();  // 20

// 自动生成
// - 构造方法
// - getter 方法（无 get 前缀）
// - equals
// - hashCode
// - toString
```

### 8.4 pattern matching for instanceof（Java 16）

```java
// Java 11 及以前
if (obj instanceof String) {
    String s = (String) obj;
    System.out.println(s.length());
}

// Java 16+
if (obj instanceof String s) {
    System.out.println(s.length());
}
```

### 8.5 sealed 类（Java 17）

```java
// 密封类：限制哪些类可以继承
public sealed class Shape permits Circle, Rectangle, Triangle {}

public final class Circle extends Shape {}
public final class Rectangle extends Shape {}
public final class Triangle extends Shape {}

// 非密封类不能继承 Shape
// class Square extends Shape {}  // 编译错误
```

## 九、综合示例

### 9.1 函数式编程

```java
List<String> names = Arrays.asList("张三", "李四", "王五");

// 链式调用
names.stream()
    .filter(name -> name.length() > 2)
    .map(String::toUpperCase)
    .forEach(System.out::println);
```

### 9.2 集合工厂方法

```java
// Java 9+
List<String> list = List.of("A", "B", "C");
Set<Integer> set = Set.of(1, 2, 3);
Map<String, Integer> map = Map.of("A", 1, "B", 2);

// 可变副本
List<String> mutable = new ArrayList<>(list);
```

### 9.3 新 API 组合使用

```java
// Java 11+
String text = """
    Hello
    World
    Java
    """;

// 处理文本块
List<String> upperLines = text.lines()
    .map(String::strip)
    .map(String::toUpperCase)
    .collect(Collectors.toList());
```

## 十、小结

1. **接口新特性**：默认方法、静态方法、私有方法（Java 9）
2. **方法引用**：:: 语法简化 Lambda
3. **重复注解**：@Repeatable
4. **类型注解**：@NonNull 等
5. **Java 9-17**：
   - var 类型推断（Java 10）
   - 不可变集合（Java 9）
   - switch 表达式（Java 14）
   - 文本块（Java 15）
   - record 类（Java 16）
   - pattern matching（Java 16）
   - sealed 类（Java 17）
