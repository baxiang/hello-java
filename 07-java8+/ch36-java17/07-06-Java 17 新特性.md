## 二、Switch 表达式增强

### 传统 Switch（Java 8-16）

```java
// 啰嗦，容易忘记 break
int day = 3;
String name;
switch (day) {
    case 1:
        name = "Monday";
        break;
    case 2:
        name = "Tuesday";
        break;
    case 3:
        name = "Wednesday";
        break;
    default:
        name = "Unknown";
}
```

### Switch 表达式（Java 14+，Java 17 正式）

```java
// 简洁，不需要 break
int day = 3;
String name = switch (day) {
    case 1 -> "Monday";
    case 2 -> "Tuesday";
    case 3 -> "Wednesday";
    default -> "Unknown";
};
```

### 多值匹配

```java
// 多个值匹配同一分支
String monthName = switch (month) {
    case 1, 3, 5, 7, 8, 10, 12 -> "31 天";
    case 4, 6, 9, 11 -> "30 天";
    case 2 -> "28 或 29 天";
    default -> "未知";
};
```

### 代码块形式

```java
// 复杂逻辑用代码块
String result = switch (code) {
    case 200 -> {
        System.out.println("成功");
        yield "OK";  // yield 返回值
    }
    case 404 -> {
        System.out.println("未找到");
        yield "Not Found";
    }
    default -> {
        System.out.println("其他");
        yield "Unknown";
    }
};
```

**yield vs return**：
- `yield`：返回 switch 表达式的值
- `return`：返回整个方法的值

## 三、Record（记录类）

### 传统 JavaBean

```java
// 几十行代码
public class Point {
    private final int x;
    private final int y;
    
    public Point(int x, int y) {
        this.x = x;
        this.y = y;
    }
    
    public int getX() { return x; }
    public int getY() { return y; }
    
    @Override
    public boolean equals(Object o) {
        if (this == o) return true;
        if (o == null || getClass() != o.getClass()) return false;
        Point point = (Point) o;
        return x == point.x && y == point.y;
    }
    
    @Override
    public int hashCode() {
        return Objects.hash(x, y);
    }
    
    @Override
    public String toString() {
        return "Point{x=" + x + ", y=" + y + "}";
    }
}
```

### Record（Java 14+，Java 17 正式）

```java
// 一行代码搞定！
public record Point(int x, int y) { }
```

**自动生成**：
- 私有 final 字段
- 构造方法
- getter 方法（无 get 前缀）
- equals() 方法
- hashCode() 方法
- toString() 方法

### Record 使用示例

```java
// 创建对象
Point p = new Point(10, 20);

// 访问字段
int x = p.x();  // 10（注意：是 x() 不是 getX()）
int y = p.y();  // 20

// 自动实现 equals
Point p1 = new Point(10, 20);
Point p2 = new Point(10, 20);
System.out.println(p1.equals(p2));  // true

// 自动实现 toString
System.out.println(p);  // Point[x=10, y=20]
```

### Record 的自定义

```java
// 添加自定义方法
public record Rectangle(Point topLeft, Point bottomRight) {
    
    // 自定义方法
    public int area() {
        int width = bottomRight.x() - topLeft.x();
        int height = bottomRight.y() - topLeft.y();
        return width * height;
    }
}

// 紧凑构造方法（验证参数）
public record Person(String name, int age) {
    public Person {
        // 自动添加验证
        if (name == null || name.isBlank()) {
            throw new IllegalArgumentException("姓名不能为空");
        }
        if (age < 0 || age > 150) {
            throw new IllegalArgumentException("年龄不合法");
        }
    }
}
```

### Record 的使用场景

```java
// ✅ 适合：数据传输对象（DTO）
public record UserDTO(Long id, String username, String email) { }

// ✅ 适合：键值对
public record Pair<K, V>(K key, V value) { }

// ✅ 适合：多返回值
public record Result<T>(int code, String message, T data) { }

// ❌ 不适合：需要继承的类（Record 不能继承）
// ❌ 不适合：需要可变状态（Record 字段是 final）
```

## 四、密封类（Sealed Classes）

### 什么是密封类？

**生活例子：公司职级**

```
员工（密封类）
├── 普通员工（ permitted subclasses）
├── 经理（permitted subclasses）
└── 总监（permitted subclasses）

不能有其他子类！
```

### 密封类语法

```java
// 密封类：限制哪些类可以继承
public sealed class Shape 
    permits Circle, Rectangle, Triangle {
    
    public abstract double area();
}

// 子类：必须声明为 final、sealed 或 non-sealed
public final class Circle extends Shape {
    private final double radius;
    
    public Circle(double radius) {
        this.radius = radius;
    }
    
    @Override
    public double area() {
        return Math.PI * radius * radius;
    }
}

public final class Rectangle extends Shape {
    private final double width;
    private final double height;
    
    public Rectangle(double width, double height) {
        this.width = width;
        this.height = height;
    }
    
    @Override
    public double area() {
        return width * height;
    }
}
```

### 密封类 + Switch 模式匹配

```java
// 编译时检查所有情况
public double calculateArea(Shape shape) {
    return switch (shape) {
        case Circle c -> Math.PI * c.radius() * c.radius();
        case Rectangle r -> r.width() * r.height();
        case Triangle t -> 0.5 * t.base() * t.height();
        // 不需要 default，编译器知道所有子类
    };
}
```

### 修饰符说明

| 修饰符 | 说明 |
|--------|------|
| sealed | 密封类，只能被 permits 指定的类继承 |
| non-sealed | 非密封类，可以被任何类继承 |
| final | 最终类，不能被继承 |

## 五、模式匹配增强

### instanceof 模式匹配（Java 16+，Java 17 正式）

**之前**：
```java
if (obj instanceof String) {
    String str = (String) obj;  // 重复转换
    System.out.println(str.length());
}
```

**现在**：
```java
if (obj instanceof String str) {  // 自动转换
    System.out.println(str.length());
}
```

### Switch 模式匹配（Java 17 预览）

```java
// 根据类型匹配
String format(Object obj) {
    return switch (obj) {
        case Integer i -> "整数：" + i;
        case String s -> "字符串：" + s;
        case Double d -> "小数：" + d;
        case null -> "空值";
        default -> "其他：" + obj.toString();
    };
}
```

## 六、其他改进

### 1. 随机数生成器

```java
// 新的随机数 API
Random random = new Random();

// 生成随机整数流
IntStream stream = random.ints(5, 1, 100);  // 5 个 1-100 的随机数

// 生成随机小数流
DoubleStream dStream = random.doubles(3, 0.0, 1.0);

// 生成随机布尔流
BooleanStream bStream = random.booleans(10);
```

### 2. Math 新增方法

```java
// 精确数学运算（溢出时抛异常）
Math.addExact(100, 200);  // 300
Math.multiplyExact(10, 20);  // 200

// 溢出示例
int max = Integer.MAX_VALUE;
Math.addExact(max, 1);  // ArithmeticException
```

### 3. Stream 新增方法

```java
// takeWhile：取到不满足条件为止
List<Integer> list = Arrays.asList(1, 2, 3, 4, 5);
list.stream()
    .takeWhile(n -> n < 4)
    .toList();  // [1, 2, 3]

// dropWhile：丢弃到不满足条件为止
list.stream()
    .dropWhile(n -> n < 3)
    .toList();  // [3, 4, 5]

// Stream.ofNullable：处理可能为 null 的值
Stream.ofNullable(getValue())
    .forEach(System.out::println);
```

## 七、小结

| 特性 | 说明 | 示例 |
|------|------|------|
| Switch 表达式 | 箭头语法 | case 1 -> "值" |
| Record | 数据类 | record Point(int x, int y) |
| 密封类 | 限制继承 | sealed class Shape permits Circle |

**核心要点**：
- Switch 表达式更简洁

- 模式匹配增强

## 八、常见问题

### Q1：Record 和 Lombok @Data 有什么区别？

**Record**：
```java
public record User(String name, int age) { }
// 不可变（字段是 final）
// 不能继承
```

**Lombok @Data**：
```java
@Data
public class User {
    private String name;
    private int age;
}
// 可变
// 可以继承
```

**选择**：
- 不可变数据：用 Record
- 需要可变：用 Lombok

### Q2：密封类有什么用？

**好处**：
- 类型安全（编译器检查所有情况）
- 代码清晰（明确哪些类可以继承）
- 易于维护（添加子类时编译器会提示）

### Q3：什么时候升级 Java 17？

**建议**：
- 新项目：直接用 Java 17
- 老项目：逐步迁移
- 依赖检查：确保依赖支持 Java 17
