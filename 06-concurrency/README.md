# 第六部分：多线程与并发

本模块深入学习 Java 多线程与并发编程，从线程基础到线程安全、线程通信、线程池，再到并发编程进阶知识，是构建高并发应用的核心技能。

## 章节列表

| 章节 | 目录 | 主题 |
|------|------|------|
| 第 26 章 | ch26-thread-base | 线程基础：进程与线程概念、线程创建方式、生命周期、常用方法、线程调度 |
| 第 27 章 | ch27-thread-safe | 线程安全：线程安全问题、synchronized 关键字、同步代码块/方法、Lock 接口、volatile 关键字 |
| 第 28 章 | ch28-thread-communication | 线程通信：wait()/notify()、生产者-消费者模式、线程间数据传递 |
| 第 29 章 | ch29-thread-pool | 线程池：线程池概述、Executors 工厂类、ThreadPoolExecutor 详解、参数配置、自定义线程池 |
| 第 30 章 | ch30-advanced | 并发编程进阶：Callable/Future、并发工具类、ConcurrentHashMap、原子类、ThreadLocal、CompletableFuture |

## 学习目标

- 理解线程的概念与创建方式
- 掌握线程安全的解决方案（synchronized、Lock、volatile）
- 理解线程通信机制与生产者-消费者模式
- 掌握线程池的配置与使用
- 了解并发工具类、原子类、ThreadLocal 等进阶知识
