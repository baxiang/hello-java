# 21.1 排序与查找（生活类比：图书馆整理员）

## 一、先问一个问题

**问题**：如何对 List 进行排序、查找最大最小值？

### 生活例子：图书馆整理员

图书馆整理员的工作：把散乱的书排好序（sort），在已排序的书架上快速找书（binarySearch），找出最厚/最薄的书（max/min）。Collections 工具类就是 Java 的"图书馆整理员"。

## 二、排序操作

### 2.1 sort 排序

```java
List<Integer> list = new ArrayList<>(Arrays.asList(5, 2, 8, 1, 9));

// 自然排序（升序）
Collections.sort(list);
System.out.println(list);  // [1, 2, 5, 8, 9]

// 降序
Collections.sort(list, Comparator.reverseOrder());
System.out.println(list);  // [9, 8, 5, 2, 1]

// 自定义比较器
List<String> names = Arrays.asList("Bob", "Alice", "Charlie");
Collections.sort(names, Comparator.comparingInt(String::length));
```

### 2.2 其他排序操作

```java
List<Integer> list = new ArrayList<>(Arrays.asList(1, 2, 3, 4, 5));

// 随机打乱
Collections.shuffle(list);

// 反转
Collections.reverse(list);    // [5, 4, 3, 2, 1]

// 旋转（向右移 2 位）
Collections.rotate(list, 2);  // [2, 1, 5, 4, 3]

// 交换
Collections.swap(list, 0, 4); // 交换索引 0 和 4
```

## 三、查找操作

### 3.1 binarySearch 二分查找

```java
// 列表必须已排序！
List<Integer> list = Arrays.asList(1, 2, 3, 4, 5);

int index = Collections.binarySearch(list, 3);   // 2
int notFound = Collections.binarySearch(list, 6); // -6（负数 = 插入点）

// 自定义比较器
List<String> names = Arrays.asList("Alice", "Bob", "Charlie");
int idx = Collections.binarySearch(names, "Bob", String::compareTo);
```

### 3.2 max 和 min

```java
List<Integer> list = Arrays.asList(5, 2, 8, 1, 9);

Integer max = Collections.max(list);  // 9
Integer min = Collections.min(list);  // 1

// 自定义比较器
List<String> names = Arrays.asList("Alice", "Bob", "Charlie");
String longest = Collections.max(names, Comparator.comparingInt(String::length));
```

### 3.3 frequency 和 disjoint

```java
// 统计元素出现次数
List<String> list = Arrays.asList("A", "B", "A", "C", "A");
int count = Collections.frequency(list, "A");  // 3

// 判断两个集合是否不相交
List<Integer> a = Arrays.asList(1, 2, 3);
List<Integer> b = Arrays.asList(4, 5, 6);
boolean noOverlap = Collections.disjoint(a, b);  // true
```

## 四、填充与替换

```java
// fill：全部填充
List<Integer> list = new ArrayList<>(Arrays.asList(1, 2, 3, 4, 5));
Collections.fill(list, 0);  // [0, 0, 0, 0, 0]

// copy：复制
List<Integer> src = Arrays.asList(1, 2, 3);
List<Integer> dest = new ArrayList<>(Arrays.asList(0, 0, 0, 0, 0));
Collections.copy(dest, src);  // [1, 2, 3, 0, 0]

// replaceAll：替换所有匹配元素
List<String> names = new ArrayList<>(Arrays.asList("A", "B", "A", "C"));
Collections.replaceAll(names, "A", "X");  // [X, B, X, C]

// addAll：批量添加
List<String> list2 = new ArrayList<>();
Collections.addAll(list2, "A", "B", "C");
```

## 五、常见疑问

**Q1：Collections.sort 和 List.sort 有什么区别？**
A：功能相同，`Collections.sort(list)` 等价于 `list.sort(null)`。Java 8+ 推荐用 `list.sort(comparator)`。

**Q2：binarySearch 返回负数是什么意思？**
A：`-(insertionPoint + 1)`，即如果元素不存在，返回它应该插入的位置的负数减一。

**Q3：为什么 shuffle 后每次结果不同？**
A：默认使用随机数源。测试时可用 `Collections.shuffle(list, new Random(42))` 固定种子。

## 六、小结

| 操作 | 方法 | 说明 |
|------|------|------|
| 排序 | sort | 自然排序或自定义 |
| 打乱 | shuffle | 随机排列 |
| 反转 | reverse | 倒序 |
| 查找 | binarySearch | 二分查找，需先排序 |
| 最值 | max/min | 最大/最小值 |
| 频率 | frequency | 统计出现次数 |
