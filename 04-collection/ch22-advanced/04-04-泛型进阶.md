# 4.4 泛型进阶

## 一、泛型类

### 1.1 定义泛型类

```java
// 泛型类
public class Box<T> {
    private T content;
    
    public void set(T content) {
        this.content = content;
    }
    
    public T get() {
        return content;
    }
}

// 使用
Box<String> stringBox = new Box<>();
stringBox.set("Hello");
String value = stringBox.get();

Box<Integer> intBox = new Box<>();
intBox.set(100);
Integer num = intBox.get();
```

### 1.2 多个类型参数

```java
public class Pair<K, V> {
    private K key;
    private V value;
    
    public Pair(K key, V value) {
        this.key = key;
        this.value = value;
    }
    
    public K getKey() { return key; }
    public V getValue() { return value; }
}

// 使用
Pair<String, Integer> pair = new Pair<>("age", 18);
String key = pair.getKey();
Integer value = pair.getValue();
```

## 二、泛型方法

### 2.1 定义泛型方法

```java
public class Utils {
    
    // 泛型方法（泛型声明在返回值前）
    public static <T> T getFirst(T[] array) {
        if (array == null || array.length == 0) {
            return null;
        }
        return array[0];
    }
    
    // 使用
    String first = Utils.getFirst(new String[]{"A", "B", "C"});
    Integer num = Utils.getFirst(new Integer[]{1, 2, 3});
}
```

### 2.2 泛型方法与泛型类

```java
public class Container<T> {
    private T value;
    
    // 可以使用类的泛型 T
    public T getValue() {
        return value;
    }
    
    // 也可以定义自己的泛型方法
    public <U> U convert(Function<T, U> func) {
        return func.apply(value);
    }
}
```

## 三、类型通配符

### 3.1 无界通配符<?>

```java
// 可以接受任何类型的 List
public void printList(List<?> list) {
    for (Object obj : list) {
        System.out.println(obj);
    }
}

// 使用
printList(new ArrayList<String>());
printList(new ArrayList<Integer>());

// 注意：不能向 List<?> 添加元素（除了 null）
// list.add("A");  // 编译错误
```

### 3.2 上界通配符<? extends T>

```java
// 可以接受 Number 或其子类的 List
public double sum(List<? extends Number> list) {
    double sum = 0;
    for (Number num : list) {
        sum += num.doubleValue();
    }
    return sum;
}

// 使用
List<Integer> ints = Arrays.asList(1, 2, 3);
List<Double> doubles = Arrays.asList(1.1, 2.2, 3.3);
System.out.println(sum(ints));    // 6.0
System.out.println(sum(doubles)); // 6.6

// 注意：不能添加元素（除了 null）
// list.add(1);  // 编译错误
```

### 3.3 下界通配符<? super T>

```java
// 可以接受 Integer 或其父类的 List
public void addNumbers(List<? super Integer> list) {
    list.add(1);
    list.add(2);
    list.add(3);
}

// 使用
List<Integer> ints = new ArrayList<>();
List<Number> numbers = new ArrayList<>();
List<Object> objects = new ArrayList<>();

addNumbers(ints);      // OK
addNumbers(numbers);   // OK
addNumbers(objects);   // OK

// 注意：读取时只能得到 Object
Object obj = list.get(0);
```

### 3.4 PECS 原则

**Producer Extends, Consumer Super**

- 生产者（只读）用 `<? extends T>`
- 消费者（只写）用 `<? super T>`

```java
// 拷贝列表（源是生产者，目标是消费者）
public static <T> void copy(List<? super T> dest, List<? extends T> src) {
    for (int i = 0; i < src.size(); i++) {
        dest.set(i, src.get(i));
    }
}
```

## 四、类型擦除

### 4.1 什么是类型擦除

泛型只在编译时存在，运行时泛型信息会被擦除。

```java
Box<String> stringBox = new Box<>();
Box<Integer> intBox = new Box<>();

// 运行时类型相同
System.out.println(stringBox.getClass() == intBox.getClass());  // true
```

### 4.2 类型擦除的限制

```java
// ❌ 不能使用泛型创建数组
// T[] array = new T[10];  // 编译错误

// ✅ 可以使用 Object 数组
public class GenericArray<T> {
    private Object[] array;
    
    public GenericArray(int size) {
        array = new Object[size];
    }
    
    @SuppressWarnings("unchecked")
    public T get(int index) {
        return (T) array[index];
    }
}

// ❌ 不能使用 instanceof 检查泛型类型
// if (obj instanceof List<String>) {}  // 编译错误

// ✅ 可以使用无界通配符
if (obj instanceof List<?>) {}
```

### 4.3 桥接方法

```java
class Box<T> {
    public void set(T value) {}
}

class StringBox extends Box<String> {
    // 编译器会生成桥接方法
    // public void set(Object value) { set((String) value); }
    
    @Override
    public void set(String value) {
        System.out.println("StringBox.set");
    }
}
```

## 五、泛型约束

### 5.1 单边界

```java
// T 必须是 Number 或其子类
public class NumberBox<T extends Number> {
    private T value;
    
    public double getDoubleValue() {
        return value.doubleValue();  // 可以调用 Number 的方法
    }
}
```

### 5.2 多边界

```java
// T 必须同时实现 Comparable 和 Serializable
public class ComparableBox<T extends Comparable<T> & Serializable> {
    private T value;
    
    public int compareTo(ComparableBox<T> other) {
        return value.compareTo(other.value);
    }
}
```

## 六、综合示例

### 6.1 泛型工具类

```java
public class CollectionsUtils {
    
    // 交换列表中的两个元素
    public static <T> void swap(List<T> list, int i, int j) {
        T temp = list.get(i);
        list.set(i, list.get(j));
        list.set(j, temp);
    }
    
    // 查找最大值
    public static <T extends Comparable<T>> T max(List<T> list) {
        if (list.isEmpty()) {
            throw new IllegalArgumentException("列表为空");
        }
        T max = list.get(0);
        for (T item : list) {
            if (item.compareTo(max) > 0) {
                max = item;
            }
        }
        return max;
    }
    
    // 填充列表
    public static <T> void fill(List<? super T> list, T value) {
        for (int i = 0; i < list.size(); i++) {
            list.set(i, value);
        }
    }
    
    // 转换列表
    public static <T, U> List<U> map(List<T> list, Function<T, U> func) {
        List<U> result = new ArrayList<>();
        for (T item : list) {
            result.add(func.apply(item));
        }
        return result;
    }
}

// 使用
List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);
Integer max = CollectionsUtils.max(numbers);
List<String> strings = CollectionsUtils.map(numbers, String::valueOf);
```

### 6.2 泛型缓存

```java
public class Cache<K, V> {
    private final Map<K, V> map = new HashMap<>();
    
    public void put(K key, V value) {
        map.put(key, value);
    }
    
    public V get(K key) {
        return map.get(key);
    }
    
    public V getOrDefault(K key, V defaultValue) {
        return map.getOrDefault(key, defaultValue);
    }
    
    public void clear() {
        map.clear();
    }
    
    public int size() {
        return map.size();
    }
}

// 使用
Cache<String, Integer> cache = new Cache<>();
cache.put("age", 18);
Integer age = cache.get("age");
```

### 6.3 泛型响应类

```java
@Data
@AllArgsConstructor
@NoArgsConstructor
public class Result<T> {
    private Integer code;
    private String message;
    private T data;
    
    public static <T> Result<T> success(T data) {
        return new Result<>(200, "success", data);
    }
    
    public static <T> Result<T> error(String message) {
        return new Result<>(500, message, null);
    }
    
    public static <T> Result<T> error(Integer code, String message) {
        return new Result<>(code, message, null);
    }
}

// 使用
Result<User> userResult = Result.success(new User("张三", 18));
Result<Void> errorResult = Result.error("操作失败");
```

## 七、小结

1. **泛型类**：`class Box<T> {}`
2. **泛型方法**：`<T> T method(T t) {}`
3. **通配符**：
   - `<?>` 无界通配符
   - `<? extends T>` 上界（只读）
   - `<? super T>` 下界（只写）
4. **PECS 原则**：生产者 extends，消费者 super
5. **类型擦除**：运行时泛型信息被擦除
6. **边界**：`extends` 单边界，`&` 多边界
