# Java 全栈学习路线

> 从零基础到 Spring Cloud 微服务架构的完整教程

## 📖 教程简介

本教程涵盖 Java 开发的完整知识体系，从基础语法到 Spring Cloud 微服务架构，包含 8 个实战项目，帮助你系统性地掌握 Java 开发技能。

## 📚 学习路线

```
Java 基础 → 面向对象 → 集合框架 → 多线程并发 → Web 开发 → Spring → Spring Boot → Spring Cloud → 项目实战
```

## 🎯 学习建议

1. **循序渐进**：按照顺序学习，不要跳跃
2. **多敲代码**：每个知识点都要动手实践
3. **做笔记**：记录重点知识和踩坑经验
4. **做项目**：学完框架后开始实战项目
5. **持续学习**：技术不断更新，保持学习状态

## 📝 如何开始

1. 从 `01-java-base` 开始学习
2. 每个章节包含：
   - 知识点讲解
   - 代码示例
   - 练习题
3. 完成每个部分后进行总结复习
