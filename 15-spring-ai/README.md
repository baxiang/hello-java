## 学习建议

1. **理解 AI 基础**：了解 LLM 的工作原理和限制
2. **掌握 API**：熟练使用 ChatClient 和 Prompt API
3. **实践 RAG**：构建基于知识库的问答系统
4. **安全合规**：注意数据隐私和 API 使用规范

## 前置知识

- Spring Boot 3.x 基础
- Java 17+（推荐 Java 21）
- 基本的 REST API 概念
