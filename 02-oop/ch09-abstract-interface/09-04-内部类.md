# 9.4 内部类

## 一、内部类概述

内部类（Inner Class）是定义在另一个类内部的类。

**好处**：
- 更好的封装
- 可以访问外部类的私有成员
- 逻辑分组

## 二、内部类的分类

```
内部类
├── 成员内部类
├── 静态内部类
├── 局部内部类
└── 匿名内部类
```

## 三、成员内部类

### 3.1 定义

```java
class Outer {
    private int outerField = 10;
    
    // 成员内部类
    class Inner {
        private int innerField = 20;
        
        public void show() {
            // 可以访问外部类的私有成员
            System.out.println("外部类字段：" + outerField);
            System.out.println("内部类字段：" + innerField);
        }
    }
}
```

### 3.2 创建对象

```java
public class Main {
    public static void main(String[] args) {
        // 先创建外部类对象
        Outer outer = new Outer();
        
        // 再创建内部类对象
        Outer.Inner inner = outer.new Inner();
        
        inner.show();
    }
}
```

### 3.3 访问外部类

```java
class Outer {
    private int num = 10;
    
    class Inner {
        private int num = 20;
        
        public void show() {
            int num = 30;
            
            System.out.println(num);          // 30 - 局部变量
            System.out.println(this.num);     // 20 - 内部类成员
            System.out.println(Outer.this.num); // 10 - 外部类成员
        }
    }
}
```

## 四、静态内部类

### 4.1 定义

```java
class Outer {
    private static int outerStatic = 100;
    private int outerField = 10;
    
    // 静态内部类
    static class StaticInner {
        private static int innerStatic = 200;
        private int innerField = 20;
        
        public void show() {
            // 可以访问外部类的静态成员
            System.out.println("外部类静态：" + outerStatic);
            
            // 不能直接访问外部类的实例成员
            // System.out.println(outerField);  // ❌ 错误
            
            System.out.println("内部类静态：" + innerStatic);
            System.out.println("内部类成员：" + innerField);
        }
        
        // 静态内部类可以有静态成员
        public static void staticShow() {
            System.out.println("静态方法");
        }
    }
}
```

### 4.2 创建对象

```java
public class Main {
    public static void main(String[] args) {
        // 静态内部类可以直接创建对象
        Outer.StaticInner inner = new Outer.StaticInner();
        inner.show();
        
        // 调用静态方法
        Outer.StaticInner.staticShow();
    }
}
```

## 五、局部内部类

### 5.1 定义

```java
class Outer {
    private int outerField = 10;
    
    public void method() {
        int localVar = 30;
        
        // 局部内部类（定义在方法内）
        class LocalInner {
            public void show() {
                System.out.println("外部类字段：" + outerField);
                System.out.println("局部变量：" + localVar);
            }
        }
        
        // 使用局部内部类
        LocalInner inner = new LocalInner();
        inner.show();
    }
}
```

### 5.2 访问局部变量

```java
class Outer {
    public void method() {
        // Java 8+：局部变量可以是 final 或 effectively final
        int num = 10;
        
        class Inner {
            public void show() {
                System.out.println(num);  // ✅ 可以访问
                // num = 20;  // ❌ 不能修改
            }
        }
    }
}
```

## 六、匿名内部类

### 6.1 定义

```java
// 接口
interface Flyable {
    void fly();
}

// 使用匿名内部类实现
public class Main {
    public static void main(String[] args) {
        // 匿名内部类
        Flyable flyable = new Flyable() {
            @Override
            public void fly() {
                System.out.println("匿名类在飞");
            }
        };
        
        flyable.fly();
    }
}
```

### 6.2 继承抽象类

```java
abstract class Animal {
    public abstract void makeSound();
}

public class Main {
    public static void main(String[] args) {
        Animal animal = new Animal() {
            @Override
            public void makeSound() {
                System.out.println("汪汪叫");
            }
        };
        
        animal.makeSound();
    }
}
```

### 6.3 作为方法参数

```java
interface Runnable {
    void run();
}

public class Main {
    public static void main(String[] args) {
        // 作为参数传递
        execute(new Runnable() {
            @Override
            public void run() {
                System.out.println("执行任务");
            }
        });
        
        // Lambda 表达式（Java 8+）
        execute(() -> System.out.println("执行任务"));
    }
    
    public static void execute(Runnable runnable) {
        runnable.run();
    }
}
```

### 6.4 访问外部变量

```java
public class Main {
    private int outerField = 10;
    
    public void method() {
        int localVar = 20;
        
        Runnable runnable = new Runnable() {
            @Override
            public void run() {
                System.out.println(outerField);  // ✅ 可以访问
                System.out.println(localVar);    // ✅ 可以访问
                // localVar = 30;  // ❌ 不能修改
            }
        };
    }
}
```

## 七、综合示例

### 7.1 事件监听

```java
// 按钮类
class Button {
    private OnClickListener listener;
    
    public void setOnClickListener(OnClickListener listener) {
        this.listener = listener;
    }
    
    public void click() {
        if (listener != null) {
            listener.onClick();
        }
    }
    
    // 监听器接口
    interface OnClickListener {
        void onClick();
    }
}

// 使用
public class Main {
    public static void main(String[] args) {
        Button button = new Button();
        
        // 匿名内部类实现监听
        button.setOnClickListener(new Button.OnClickListener() {
            @Override
            public void onClick() {
                System.out.println("按钮被点击了");
            }
        });
        
        button.click();
    }
}
```

### 7.2 迭代器实现

```java
class MyList {
    private String[] data = {"A", "B", "C"};
    
    // 返回迭代器
    public Iterator iterator() {
        return new Iterator() {
            private int index = 0;
            
            @Override
            public boolean hasNext() {
                return index < data.length;
            }
            
            @Override
            public String next() {
                return data[index++];
            }
        };
    }
    
    interface Iterator {
        boolean hasNext();
        String next();
    }
}

// 使用
public class Main {
    public static void main(String[] args) {
        MyList list = new MyList();
        MyList.Iterator it = list.iterator();
        
        while (it.hasNext()) {
            System.out.println(it.next());
        }
    }
}
```

### 7.3 比较器

```java
class Person {
    String name;
    int age;
    
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    @Override
    public String toString() {
        return name + "(" + age + ")";
    }
}

public class Main {
    public static void main(String[] args) {
        List<Person> people = Arrays.asList(
            new Person("张三", 25),
            new Person("李四", 20),
            new Person("王五", 30)
        );
        
        // 匿名内部类实现比较器
        Collections.sort(people, new Comparator<Person>() {
            @Override
            public int compare(Person p1, Person p2) {
                return Integer.compare(p1.age, p2.age);
            }
        });
        
        System.out.println(people);
    }
}

interface Comparator<T> {
    int compare(T o1, T o2);
}
```

## 八、注意事项

### 8.1 静态内部类 vs 非静态

```java
// 需要访问外部类实例 → 用非静态内部类
// 不需要访问外部类实例 → 用静态内部类（推荐）

// 静态内部类更常用，因为它：
// 1. 不持有外部类引用
// 2. 内存泄漏风险小
// 3. 创建更简单
```

### 8.2 内存泄漏

```java
class Outer {
    private byte[] data = new byte[1024 * 1024];  // 1MB
    
    // ❌ 可能导致内存泄漏
    public Runnable getRunnable() {
        return new Runnable() {
            @Override
            public void run() {
                // 匿名类持有外部类引用
            }
        };
    }
}
```

### 8.3 命名规范

```java
// 内部类名通常使用描述性名称
class Outer {
    class InnerHelper { }  // ✅ 好
    class Inner { }        // ❌ 不够描述性
}
```

## 九、小结

本节要点：
1. **成员内部类**：作为外部类成员，可访问外部类所有成员
2. **静态内部类**：静态修饰，不能访问外部类实例成员
3. **局部内部类**：定义在方法内，可访问 final 局部变量
4. **匿名内部类**：无名的内部类，常用于实现接口或继承类
5. **应用**：事件监听、迭代器、比较器等
