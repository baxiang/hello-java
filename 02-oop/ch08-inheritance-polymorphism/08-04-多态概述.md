# 8.4 多态概述

## 一、多态的概念

多态（Polymorphism）是面向对象的三大特性之一，指同一个行为具有多种不同的表现形式。

**通俗理解**：
- 同一行为，不同对象执行时有不同表现
- 一个接口，多种实现

**比喻**：
```
"叫"这个行为：
- 狗 → 汪汪叫
- 猫 → 喵喵叫
- 鸭 → 嘎嘎叫

"吃"这个行为：
- 人 → 用筷子吃
- 狗 → 用嘴啃
- 鸟 → 用嘴啄
```

## 二、多态的前提

### 2.1 三个必要条件

1. **继承关系**：子类继承父类
2. **方法重写**：子类重写父类方法
3. **父类引用指向子类对象**：`父类 变量 = new 子类 ();`

### 2.2 示例

```java
// 1. 继承关系
class Animal {
    public void eat() {
        System.out.println("动物吃东西");
    }
}

class Dog extends Animal {
    // 2. 方法重写
    @Override
    public void eat() {
        System.out.println("狗吃骨头");
    }
}

class Cat extends Animal {
    @Override
    public void eat() {
        System.out.println("猫吃鱼");
    }
}

// 使用
public class Main {
    public static void main(String[] args) {
        // 3. 父类引用指向子类对象
        Animal a1 = new Dog();  // 多态
        Animal a2 = new Cat();  // 多态
        
        a1.eat();  // 输出：狗吃骨头
        a2.eat();  // 输出：猫吃鱼
    }
}
```

## 三、多态的表现形式

### 3.1 方法的多态

```java
class Animal {
    public void eat() {
        System.out.println("动物吃东西");
    }
}

class Dog extends Animal {
    @Override
    public void eat() {
        System.out.println("狗吃骨头");
    }
}

class Cat extends Animal {
    @Override
    public void eat() {
        System.out.println("猫吃鱼");
    }
}

public class Main {
    public static void main(String[] args) {
        // 同一个方法，不同对象有不同表现
        Animal a1 = new Dog();
        Animal a2 = new Cat();
        
        a1.eat();  // 狗吃骨头
        a2.eat();  // 猫吃鱼
    }
}
```

### 3.2 数组的多态

```java
public class Main {
    public static void main(String[] args) {
        // 父类数组存储子类对象
        Animal[] animals = {
            new Dog(),
            new Cat(),
            new Dog()
        };
        
        for (Animal animal : animals) {
            animal.eat();  // 多态调用
        }
    }
}
```

### 3.3 方法参数的多态

```java
public class Main {
    public static void main(String[] args) {
        Dog dog = new Dog();
        Cat cat = new Cat();
        
        feed(dog);  // 传入 Dog
        feed(cat);  // 传入 Cat
    }
    
    // 方法参数是父类类型
    public static void feed(Animal animal) {
        animal.eat();
    }
}
```

## 四、多态的好处

### 4.1 提高代码扩展性

```java
// 父类引用作为参数
public void feed(Animal animal) {
    animal.eat();
}

// 以后添加新动物，不需要修改 feed 方法
class Pig extends Animal {
    @Override
    public void eat() {
        System.out.println("猪吃剩饭");
    }
}

// 直接使用
feed(new Pig());
```

### 4.2 降低耦合度

```java
// 使用父类类型，不依赖具体子类
List<String> list = new ArrayList<>();
// 而不是
// ArrayList<String> list = new ArrayList<>();

// 以后可以方便更换实现
list = new LinkedList<>();
```

### 4.3 便于统一处理

```java
public class Zoo {
    public static void main(String[] args) {
        Animal[] animals = {
            new Dog(),
            new Cat(),
            new Pig()
        };
        
        // 统一处理所有动物
        for (Animal animal : animals) {
            animal.eat();
            animal.sleep();
        }
    }
}
```

## 五、多态的局限性

### 5.1 不能访问子类特有方法

```java
Animal a = new Dog();
a.eat();  // ✅ 可以调用继承的方法
// a.bark();  // ❌ 错误：不能调用子类特有方法
```

### 5.2 不能访问子类特有属性

```java
class Dog extends Animal {
    String breed = "哈士奇";
}

Animal a = new Dog();
// System.out.println(a.breed);  // ❌ 错误
```

## 六、多态的应用

### 6.1 USB 接口

```java
// USB 接口（父类）
interface USB {
    void connect();
    void transfer();
}

// 鼠标
class Mouse implements USB {
    @Override
    public void connect() {
        System.out.println("鼠标连接");
    }
    
    @Override
    public void transfer() {
        System.out.println("鼠标数据传输");
    }
}

// 键盘
class Keyboard implements USB {
    @Override
    public void connect() {
        System.out.println("键盘连接");
    }
    
    @Override
    public void transfer() {
        System.out.println("键盘数据传输");
    }
}

// 电脑
class Computer {
    public void useUSB(USB usb) {
        usb.connect();
        usb.transfer();
    }
}

// 使用
public class Main {
    public static void main(String[] args) {
        Computer computer = new Computer();
        computer.useUSB(new Mouse());    // 多态
        computer.useUSB(new Keyboard()); // 多态
    }
}
```

### 6.2 支付方式

```java
// 支付接口
interface Payment {
    void pay(double amount);
}

// 支付宝
class Alipay implements Payment {
    @Override
    public void pay(double amount) {
        System.out.println("使用支付宝支付：" + amount);
    }
}

// 微信
class WechatPay implements Payment {
    @Override
    public void pay(double amount) {
        System.out.println("使用微信支付：" + amount);
    }
}

// 银行卡
class BankCard implements Payment {
    @Override
    public void pay(double amount) {
        System.out.println("使用银行卡支付：" + amount);
    }
}

// 订单
class Order {
    public void checkout(Payment payment, double amount) {
        payment.pay(amount);
    }
}

// 使用
public class Main {
    public static void main(String[] args) {
        Order order = new Order();
        order.checkout(new Alipay(), 100);
        order.checkout(new WechatPay(), 200);
        order.checkout(new BankCard(), 300);
    }
}
```

### 6.3 图形计算

```java
abstract class Shape {
    public abstract double getArea();
}

class Circle extends Shape {
    private double radius;
    
    public Circle(double radius) {
        this.radius = radius;
    }
    
    @Override
    public double getArea() {
        return Math.PI * radius * radius;
    }
}

class Rectangle extends Shape {
    private double width;
    private double height;
    
    public Rectangle(double width, double height) {
        this.width = width;
        this.height = height;
    }
    
    @Override
    public double getArea() {
        return width * height;
    }
}

class Triangle extends Shape {
    private double base;
    private double height;
    
    public Triangle(double base, double height) {
        this.base = base;
        this.height = height;
    }
    
    @Override
    public double getArea() {
        return 0.5 * base * height;
    }
}

// 使用
public class Main {
    public static void main(String[] args) {
        Shape[] shapes = {
            new Circle(5),
            new Rectangle(4, 6),
            new Triangle(3, 4)
        };
        
        double totalArea = 0;
        for (Shape shape : shapes) {
            totalArea += shape.getArea();  // 多态调用
        }
        
        System.out.println("总面积：" + totalArea);
    }
}
```

## 七、综合示例

```java
// 父类：员工
class Employee {
    protected String name;
    
    public Employee(String name) {
        this.name = name;
    }
    
    public double getSalary() {
        return 0;
    }
    
    public void work() {
        System.out.println(name + " 在工作");
    }
}

// 子类：经理
class Manager extends Employee {
    private double bonus;
    
    public Manager(String name, double bonus) {
        super(name);
        this.bonus = bonus;
    }
    
    @Override
    public double getSalary() {
        return 50000 + bonus;
    }
    
    @Override
    public void work() {
        System.out.println(name + " 在管理团队");
    }
}

// 子类：程序员
class Programmer extends Employee {
    private String language;
    
    public Programmer(String name, String language) {
        super(name);
        this.language = language;
    }
    
    @Override
    public double getSalary() {
        return 20000;
    }
    
    @Override
    public void work() {
        System.out.println(name + " 在用" + language + "写代码");
    }
}

// 子类：销售
class Salesman extends Employee {
    private double commission;
    
    public Salesman(String name, double commission) {
        super(name);
        this.commission = commission;
    }
    
    @Override
    public double getSalary() {
        return 10000 + commission;
    }
    
    @Override
    public void work() {
        System.out.println(name + " 在拜访客户");
    }
}

// 使用
public class Company {
    public static void main(String[] args) {
        // 多态数组
        Employee[] employees = {
            new Manager("张三", 10000),
            new Programmer("李四", "Java"),
            new Programmer("王五", "Python"),
            new Salesman("赵六", 5000)
        };
        
        // 统一处理
        double totalSalary = 0;
        for (Employee emp : employees) {
            emp.work();
            totalSalary += emp.getSalary();
        }
        
        System.out.println("总工资：" + totalSalary);
    }
}
```

## 八、小结

本节要点：
1. **多态前提**：继承、重写、父类引用指向子类对象
2. **表现形式**：方法多态、数组多态、参数多态
3. **好处**：扩展性、低耦合、统一处理
4. **局限**：不能访问子类特有成员
5. **应用**：接口、支付、图形计算等
