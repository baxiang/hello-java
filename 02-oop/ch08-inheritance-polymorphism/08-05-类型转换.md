# 8.5 向上转型与向下转型

## 一、类型转换概述

在继承关系中，子类和父类之间可以进行类型转换。

**转换类型**：
1. 向上转型（自动）
2. 向下转型（强制）

## 二、向上转型

### 2.1 什么是向上转型

将子类对象赋值给父类引用，自动完成。

```java
// 子类 → 父类（向上转型，自动）
Animal animal = new Dog();
```

### 2.2 示例

```java
class Animal {
    public void eat() {
        System.out.println("动物吃东西");
    }
}

class Dog extends Animal {
    @Override
    public void eat() {
        System.out.println("狗吃骨头");
    }
    
    public void bark() {
        System.out.println("汪汪叫");
    }
}

public class Main {
    public static void main(String[] args) {
        // 向上转型
        Animal animal = new Dog();
        
        animal.eat();  // ✅ 输出：狗吃骨头（多态）
        // animal.bark();  // ❌ 错误：不能调用子类特有方法
    }
}
```

### 2.3 特点

| 特点 | 说明 |
|------|------|
| 自动转换 | 不需要强制转换符 |
| 安全 | 不会抛出异常 |
| 限制 | 只能调用父类中有的方法 |

### 2.4 应用场景

```java
// 1. 方法参数
public void feed(Animal animal) {
    animal.eat();
}

feed(new Dog());  // 传入子类对象

// 2. 集合存储
List<Animal> animals = new ArrayList<>();
animals.add(new Dog());
animals.add(new Cat());

// 3. 数组存储
Animal[] animals = {new Dog(), new Cat()};
```

## 三、向下转型

### 3.1 什么是向下转型

将父类引用转换为子类类型，需要强制转换。

```java
// 父类 → 子类（向下转型，强制）
Animal animal = new Dog();
Dog dog = (Dog) animal;
```

### 3.2 示例

```java
public class Main {
    public static void main(String[] args) {
        // 先向上转型
        Animal animal = new Dog();
        
        // 向下转型
        Dog dog = (Dog) animal;
        dog.eat();   // ✅ 狗吃骨头
        dog.bark();  // ✅ 汪汪叫（可以调用子类特有方法）
    }
}
```

### 3.3 前提条件

向下转型前，对象必须是该子类的实例。

```java
Animal animal = new Dog();

// ✅ 正确：animal 实际是 Dog 类型
Dog dog = (Dog) animal;

// ❌ 错误：animal 实际不是 Cat 类型
// Cat cat = (Cat) animal;  // ClassCastException
```

## 四、instanceof 关键字

### 4.1 判断对象类型

```java
// 语法：对象 instanceof 类
Animal animal = new Dog();

boolean result = animal instanceof Dog;  // true
result = animal instanceof Cat;          // false
result = animal instanceof Animal;       // true
```

### 4.2 安全向下转型

```java
public void makeSound(Animal animal) {
    // 先判断类型，再转型
    if (animal instanceof Dog) {
        Dog dog = (Dog) animal;
        dog.bark();
    } else if (animal instanceof Cat) {
        Cat cat = (Cat) animal;
        cat.meow();
    }
}
```

### 4.3 模式匹配（Java 16+）

```java
// Java 16+ 新特性
public void makeSound(Animal animal) {
    if (animal instanceof Dog dog) {
        dog.bark();  // 自动转型
    } else if (animal instanceof Cat cat) {
        cat.meow();
    }
}
```

## 五、综合示例

### 5.1 动物园

```java
class Animal {
    String name;
    
    public Animal(String name) {
        this.name = name;
    }
    
    public void eat() {
        System.out.println(name + " 吃东西");
    }
}

class Dog extends Animal {
    public Dog(String name) {
        super(name);
    }
    
    @Override
    public void eat() {
        System.out.println(name + " 吃骨头");
    }
    
    public void bark() {
        System.out.println(name + " 汪汪叫");
    }
    
    public void watchHouse() {
        System.out.println(name + " 看家");
    }
}

class Cat extends Animal {
    public Cat(String name) {
        super(name);
    }
    
    @Override
    public void eat() {
        System.out.println(name + " 吃鱼");
    }
    
    public void meow() {
        System.out.println(name + " 喵喵叫");
    }
    
    public void catchMouse() {
        System.out.println(name + " 抓老鼠");
    }
}

public class Zoo {
    public static void main(String[] args) {
        // 多态数组
        Animal[] animals = {
            new Dog("旺财"),
            new Cat("咪咪"),
            new Dog("大黄")
        };
        
        // 统一处理
        for (Animal animal : animals) {
            animal.eat();  // 多态
            
            // 向下转型调用特有方法
            if (animal instanceof Dog) {
                Dog dog = (Dog) animal;
                dog.bark();
                dog.watchHouse();
            } else if (animal instanceof Cat) {
                Cat cat = (Cat) animal;
                cat.meow();
                cat.catchMouse();
            }
        }
    }
}
```

### 5.2 员工系统

```java
class Employee {
    protected String name;
    
    public Employee(String name) {
        this.name = name;
    }
    
    public double getSalary() {
        return 0;
    }
}

class Manager extends Employee {
    private double bonus;
    
    public Manager(String name, double bonus) {
        super(name);
        this.bonus = bonus;
    }
    
    @Override
    public double getSalary() {
        return 50000 + bonus;
    }
    
    public void manage() {
        System.out.println(name + " 在管理");
    }
}

class Programmer extends Employee {
    private String language;
    
    public Programmer(String name, String language) {
        super(name);
        this.language = language;
    }
    
    @Override
    public double getSalary() {
        return 20000;
    }
    
    public void code() {
        System.out.println(name + " 在用" + language + "写代码");
    }
}

public class Company {
    public static void main(String[] args) {
        Employee[] employees = {
            new Manager("张三", 10000),
            new Programmer("李四", "Java"),
            new Programmer("王五", "Python")
        };
        
        // 计算总工资
        double total = 0;
        for (Employee emp : employees) {
            total += emp.getSalary();
        }
        System.out.println("总工资：" + total);
        
        // 调用特有方法
        for (Employee emp : employees) {
            if (emp instanceof Manager) {
                Manager m = (Manager) emp;
                m.manage();
            } else if (emp instanceof Programmer) {
                Programmer p = (Programmer) emp;
                p.code();
            }
        }
    }
}
```

## 六、注意事项

### 6.1 ClassCastException

```java
Animal animal = new Dog();

// ❌ 错误：会抛出 ClassCastException
// Cat cat = (Cat) animal;

// ✅ 正确：先判断再转型
if (animal instanceof Cat) {
    Cat cat = (Cat) animal;
}
```

### 6.2 转换规则

```java
// 继承关系才能转换
class A { }
class B extends A { }
class C { }

A a = new B();
B b = (B) a;  // ✅ 可以

// ❌ 错误：没有继承关系
// C c = (C) a;
```

### 6.3 null 的处理

```java
Animal animal = null;

// instanceof 对 null 返回 false
System.out.println(animal instanceof Dog);  // false

// 转型 null 会抛出 NullPointerException
// Dog dog = (Dog) animal;
```

## 七、小结

本节要点：
1. **向上转型**：子类→父类，自动，安全
2. **向下转型**：父类→子类，强制，需判断
3. **instanceof**：判断对象类型
4. **应用**：多态数组、方法参数
5. **注意**：ClassCastException、先判断再转型
