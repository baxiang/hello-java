# 第二部分：面向对象编程（OOP）

本模块深入学习面向对象编程思想，从类与对象出发，逐步掌握继承、多态、抽象类、接口等核心概念，是 Java 编程的核心基石。

## 章节列表

| 章节 | 目录 | 主题 |
|------|------|------|
| 第 7 章 | ch07-class-object | 类与对象：面向对象思想、类定义、构造方法、this 关键字、封装与 Getter/Setter |
| 第 8 章 | ch08-inheritance-polymorphism | 继承与多态：继承实现、方法重写、super 关键字、多态、类型转换、instanceof |
| 第 9 章 | ch09-abstract-interface | 抽象类与接口：抽象类定义、接口实现与多实现、默认方法与静态方法（Java 8+） |
| 第 10 章 | ch10-keywords | 常用关键字：static 静态变量/方法/代码块、final 关键字、单例设计模式 |
| 第 11 章 | ch11-object-class | Object 类：toString()、equals()、hashCode() 方法重写 |

## 学习目标

- 理解面向对象编程的核心思想
- 掌握类与对象的创建及使用
- 理解继承与多态的应用场景
- 区分抽象类与接口，合理选择使用
- 理解 static、final 关键字的作用与使用场景
- 掌握 Object 类常用方法的重写
