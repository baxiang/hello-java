# 7.6 标准类设计

## 一、标准类的组成

一个标准的 Java 类通常包含以下部分：

```java
[修饰符] class 类名 {
    // 1. 成员变量（私有）
    private 数据类型 变量名;
    
    // 2. 构造方法
    public 类名 () { }
    public 类名 (参数列表) { }
    
    // 3. Getter 和 Setter 方法
    public 数据类型 get 变量名 () { }
    public void set 变量名 (数据类型 变量名) { }
    
    // 4. 成员方法
    public 返回值类型 方法名 (参数列表) { }
    
    // 5. 其他方法（toString、equals 等）
    @Override
    public String toString() { }
}
```

## 二、完整示例：学生类

```java
/**
 * 学生类 - 用于封装学生信息
 * @author Teacher
 * @version 1.0
 */
public class Student {
    // ==================== 成员变量 ====================
    // 使用 private 封装
    private String id;        // 学号
    private String name;      // 姓名
    private int age;          // 年龄
    private String gender;    // 性别
    private double score;     // 成绩
    
    // ==================== 构造方法 ====================
    
    /**
     * 无参构造方法
     */
    public Student() {
    }
    
    /**
     * 带参数的构造方法
     * @param id 学号
     * @param name 姓名
     * @param age 年龄
     * @param gender 性别
     * @param score 成绩
     */
    public Student(String id, String name, int age, String gender, double score) {
        this.id = id;
        this.name = name;
        setAge(age);
        this.gender = gender;
        setScore(score);
    }
    
    // ==================== Getter 和 Setter ====================
    
    public String getId() {
        return id;
    }
    
    public void setId(String id) {
        this.id = id;
    }
    
    public String getName() {
        return name;
    }
    
    public void setName(String name) {
        if (name != null && !name.trim().isEmpty()) {
            this.name = name.trim();
        }
    }
    
    public int getAge() {
        return age;
    }
    
    public void setAge(int age) {
        if (age >= 0 && age <= 150) {
            this.age = age;
        } else {
            throw new IllegalArgumentException("年龄必须在 0-150 之间");
        }
    }
    
    public String getGender() {
        return gender;
    }
    
    public void setGender(String gender) {
        if ("男".equals(gender) || "女".equals(gender)) {
            this.gender = gender;
        }
    }
    
    public double getScore() {
        return score;
    }
    
    public void setScore(double score) {
        if (score >= 0 && score <= 100) {
            this.score = score;
        } else {
            throw new IllegalArgumentException("成绩必须在 0-100 之间");
        }
    }
    
    // ==================== 成员方法 ====================
    
    /**
     * 学习
     */
    public void study() {
        System.out.println(name + " 正在学习");
    }
    
    /**
     * 自我介绍
     */
    public void introduce() {
        System.out.println("学号：" + id);
        System.out.println("姓名：" + name);
        System.out.println("年龄：" + age);
        System.out.println("性别：" + gender);
        System.out.println("成绩：" + score);
    }
    
    /**
     * 判断是否及格
     * @return true 及格，false 不及格
     */
    public boolean isPassed() {
        return score >= 60;
    }
    
    /**
     * 获取成绩等级
     * @return 等级（优秀/良好/及格/不及格）
     */
    public String getGrade() {
        if (score >= 90) {
            return "优秀";
        } else if (score >= 80) {
            return "良好";
        } else if (score >= 60) {
            return "及格";
        } else {
            return "不及格";
        }
    }
    
    // ==================== 重写方法 ====================
    
    /**
     * 重写 toString 方法
     * @return 学生信息的字符串表示
     */
    @Override
    public String toString() {
        return "Student{" +
                "id='" + id + '\'' +
                ", name='" + name + '\'' +
                ", age=" + age +
                ", gender='" + gender + '\'' +
                ", score=" + score +
                '}';
    }
    
    /**
     * 重写 equals 方法
     * @param obj 要比较的对象
     * @return 是否相等
     */
    @Override
    public boolean equals(Object obj) {
        // 检查是否为 null
        if (obj == null) {
            return false;
        }
        // 检查是否为同一对象
        if (this == obj) {
            return true;
        }
        // 检查类型是否相同
        if (!(obj instanceof Student)) {
            return false;
        }
        // 类型转换
        Student other = (Student) obj;
        // 比较学号
        return this.id != null && this.id.equals(other.id);
    }
}
```

## 三、使用示例

```java
public class StudentDemo {
    public static void main(String[] args) {
        // 1. 使用无参构造
        Student s1 = new Student();
        s1.setId("S001");
        s1.setName("张三");
        s1.setAge(18);
        s1.setGender("男");
        s1.setScore(85.5);
        
        // 2. 使用有参构造
        Student s2 = new Student("S002", "李四", 19, "女", 92.0);
        
        // 3. 调用方法
        s1.introduce();
        System.out.println("是否及格：" + s1.isPassed());
        System.out.println("成绩等级：" + s1.getGrade());
        
        // 4. 使用 toString
        System.out.println(s1);
        System.out.println(s2);
        
        // 5. 使用 equals
        Student s3 = new Student("S001", "王五", 20, "男", 75.0);
        System.out.println("s1 和 s3 是否相等：" + s1.equals(s3));  // true（学号相同）
        
        // 6. 数据验证
        try {
            s1.setAge(200);  // 抛出异常
        } catch (IllegalArgumentException e) {
            System.out.println("错误：" + e.getMessage());
        }
    }
}
```

## 四、其他标准类示例

### 4.1 商品类

```java
/**
 * 商品类
 */
public class Product {
    // 成员变量
    private String id;
    private String name;
    private double price;
    private int stock;
    private String description;
    
    // 构造方法
    public Product() {
    }
    
    public Product(String id, String name, double price, int stock) {
        this.id = id;
        this.name = name;
        setPrice(price);
        setStock(stock);
    }
    
    // Getter 和 Setter
    public String getId() {
        return id;
    }
    
    public void setId(String id) {
        this.id = id;
    }
    
    public String getName() {
        return name;
    }
    
    public void setName(String name) {
        this.name = name;
    }
    
    public double getPrice() {
        return price;
    }
    
    public void setPrice(double price) {
        if (price >= 0) {
            this.price = price;
        }
    }
    
    public int getStock() {
        return stock;
    }
    
    public void setStock(int stock) {
        if (stock >= 0) {
            this.stock = stock;
        }
    }
    
    public String getDescription() {
        return description;
    }
    
    public void setDescription(String description) {
        this.description = description;
    }
    
    // 业务方法
    public boolean purchase(int quantity) {
        if (quantity <= 0) {
            return false;
        }
        if (quantity > stock) {
            return false;
        }
        stock -= quantity;
        return true;
    }
    
    public void restock(int quantity) {
        if (quantity > 0) {
            stock += quantity;
        }
    }
    
    public double getTotalValue() {
        return price * stock;
    }
    
    // 重写方法
    @Override
    public String toString() {
        return "Product{" +
                "id='" + id + '\'' +
                ", name='" + name + '\'' +
                ", price=" + price +
                ", stock=" + stock +
                '}';
    }
    
    @Override
    public boolean equals(Object obj) {
        if (this == obj) return true;
        if (obj == null || getClass() != obj.getClass()) return false;
        Product product = (Product) obj;
        return id != null && id.equals(product.id);
    }
}
```

### 4.2 银行账户类

```java
/**
 * 银行账户类
 */
public class BankAccount {
    // 成员变量
    private String accountNumber;
    private String accountHolder;
    private double balance;
    private String password;
    
    // 构造方法
    public BankAccount(String accountNumber, String accountHolder, double initialBalance, String password) {
        this.accountNumber = accountNumber;
        this.accountHolder = accountHolder;
        setBalance(initialBalance);
        setPassword(password);
    }
    
    // Getter 和 Setter
    public String getAccountNumber() {
        return accountNumber;
    }
    
    public String getAccountHolder() {
        return accountHolder;
    }
    
    public void setAccountHolder(String accountHolder) {
        this.accountHolder = accountHolder;
    }
    
    public double getBalance() {
        return balance;
    }
    
    private void setBalance(double balance) {
        if (balance >= 0) {
            this.balance = balance;
        }
    }
    
    private void setPassword(String password) {
        if (password != null && password.length() >= 6) {
            this.password = password;
        }
    }
    
    // 业务方法
    public boolean deposit(double amount) {
        if (amount > 0) {
            balance += amount;
            return true;
        }
        return false;
    }
    
    public boolean withdraw(double amount, String pwd) {
        if (!password.equals(pwd)) {
            System.out.println("密码错误");
            return false;
        }
        if (amount <= 0) {
            System.out.println("取款金额必须大于 0");
            return false;
        }
        if (amount > balance) {
            System.out.println("余额不足");
            return false;
        }
        balance -= amount;
        return true;
    }
    
    public boolean transfer(BankAccount target, double amount, String pwd) {
        if (withdraw(amount, pwd)) {
            target.deposit(amount);
            return true;
        }
        return false;
    }
    
    // 重写方法
    @Override
    public String toString() {
        return "BankAccount{" +
                "accountNumber='" + accountNumber + '\'' +
                ", accountHolder='" + accountHolder + '\'' +
                ", balance=" + balance +
                '}';
    }
}
```

## 五、类设计原则

### 5.1 单一职责原则

一个类只负责一项职责。

```java
// ❌ 不好：一个类做了太多事
class Student {
    void study() { }
    void eat() { }
    void sleep() { }
    void saveToDatabase() { }  // 不应该在这里
    void sendEmail() { }       // 不应该在这里
}

// ✅ 好：职责分离
class Student {
    void study() { }
    void eat() { }
    void sleep() { }
}

class StudentDAO {
    void save(Student s) { }
}

class EmailService {
    void send(Student s) { }
}
```

### 5.2 高内聚低耦合

```java
// 高内聚：相关功能放在一起
class Calculator {
    double add(double a, double b) { }
    double subtract(double a, double b) { }
    double multiply(double a, double b) { }
    double divide(double a, double b) { }
}

// 低耦合：减少依赖
class Order {
    private Product product;
    private int quantity;
    
    double getTotal() {
        return product.getPrice() * quantity;
    }
}
```

## 六、小结

本节要点：
1. **标准类组成**：成员变量、构造方法、Getter/Setter、成员方法、重写方法
2. **封装**：成员变量私有，通过公共方法访问
3. **数据验证**：在 Setter 方法中进行
4. **重写方法**：toString、equals 等
5. **设计原则**：单一职责、高内聚低耦合
