# 7.1 面向对象概述

## 一、编程思想对比

### 1.1 面向过程编程

面向过程（Procedure Oriented）关注**解决问题的步骤**。

```java
// 面向过程：把大象放进冰箱
public class PutElephant {
    public static void main(String[] args) {
        // 步骤 1：打开冰箱门
        openDoor();
        
        // 步骤 2：把大象放进去
        putElephant();
        
        // 步骤 3：关上冰箱门
        closeDoor();
    }
    
    public static void openDoor() {
        System.out.println("打开冰箱门");
    }
    
    public static void putElephant() {
        System.out.println("把大象放进去");
    }
    
    public static void closeDoor() {
        System.out.println("关上冰箱门");
    }
}
```

**特点**：
- 关注过程
- 按步骤执行
- 适合简单问题

### 1.2 面向对象编程

面向对象（Object Oriented）关注**参与的对象**。

```java
// 面向对象：找对象完成任务
public class PutElephantOO {
    public static void main(String[] args) {
        // 创建对象
        Refrigerator fridge = new Refrigerator();
        Elephant elephant = new Elephant();
        
        // 对象完成任务
        fridge.open();
        fridge.put(elephant);
        fridge.close();
    }
}

class Refrigerator {
    public void open() {
        System.out.println("冰箱：门开了");
    }
    
    public void put(Elephant e) {
        System.out.println("把大象放进冰箱");
    }
    
    public void close() {
        System.out.println("冰箱：门关了");
    }
}

class Elephant {
    // 大象的属性和行为
}
```

**特点**：
- 关注对象
- 对象交互
- 适合复杂系统

## 二、面向对象三大特性

### 2.1 封装（Encapsulation）

隐藏对象的内部细节，只暴露必要的接口。

```java
public class Person {
    // 私有属性，外部不能直接访问
    private String name;
    private int age;
    
    // 公共方法，提供访问接口
    public String getName() {
        return name;
    }
    
    public void setName(String name) {
        this.name = name;
    }
    
    public int getAge() {
        return age;
    }
    
    public void setAge(int age) {
        if (age > 0 && age < 150) {
            this.age = age;
        }
    }
}
```

**好处**：
- 提高安全性
- 便于修改实现
- 降低耦合

### 2.2 继承（Inheritance）

子类继承父类的属性和方法，实现代码复用。

```java
// 父类
class Animal {
    String name;
    
    public void eat() {
        System.out.println("吃东西");
    }
}

// 子类
class Dog extends Animal {
    public void bark() {
        System.out.println("汪汪叫");
    }
}

// 使用
Dog dog = new Dog();
dog.name = "旺财";  // 继承的属性
dog.eat();          // 继承的方法
dog.bark();         // 自己的方法
```

**好处**：
- 代码复用
- 扩展性强
- 建立类之间的关系

### 2.3 多态（Polymorphism）

同一个行为具有多种表现形式。

```java
// 父类引用指向子类对象
Animal animal = new Dog();
animal.eat();  // 执行 Dog 的 eat 方法（如果重写了）

// 方法重载
class Calculator {
    public int add(int a, int b) { return a + b; }
    public double add(double a, double b) { return a + b; }
}

// 方法重写
class Dog extends Animal {
    @Override
    public void eat() {
        System.out.println("狗吃骨头");
    }
}
```

**好处**：
- 提高灵活性
- 便于扩展
- 降低耦合

## 三、类与对象的关系

### 3.1 概念对比

| 类（Class） | 对象（Object） |
|------------|---------------|
| 抽象的概念 | 具体的实例 |
| 模板/蓝图 | 根据模板创建的实物 |
| 只有一份 | 可以有多个 |
| 静态概念 | 动态存在 |

### 3.2 比喻

```
类：汽车设计图纸
对象：根据图纸造出的具体汽车

类：月饼模具
对象：用模具做出的月饼

类：学生
对象：张三、李四（具体的学生）
```

### 3.3 代码示例

```java
// 类：定义学生的特征
class Student {
    // 属性（成员变量）
    String name;
    int age;
    String school;
    
    // 行为（成员方法）
    public void study() {
        System.out.println(name + " 在学习");
    }
    
    public void introduce() {
        System.out.println("我叫" + name + "，今年" + age + "岁");
    }
}

public class Main {
    public static void main(String[] args) {
        // 创建对象（实例化）
        Student s1 = new Student();
        s1.name = "张三";
        s1.age = 18;
        s1.school = "清华大学";
        s1.introduce();
        s1.study();
        
        Student s2 = new Student();
        s2.name = "李四";
        s2.age = 19;
        s2.school = "北京大学";
        s2.introduce();
        s2.study();
    }
}
```

## 四、面向对象的优势

### 4.1 易维护

```java
// 修改类的内部实现，不影响使用
class Student {
    private String name;
    
    // 修改实现，添加验证
    public void setName(String name) {
        if (name != null && !name.isEmpty()) {
            this.name = name;
        }
    }
}
```

### 4.2 易复用

```java
// 继承实现代码复用
class GraduateStudent extends Student {
    String导师;
    
    public void doResearch() {
        System.out.println(name + " 在做研究");
    }
}
```

### 4.3 易扩展

```java
// 多态便于扩展
interface Shape {
    double getArea();
}

class Circle implements Shape {
    public double getArea() {
        return Math.PI * r * r;
    }
}

class Rectangle implements Shape {
    public double getArea() {
        return width * height;
    }
}

// 使用
void printArea(Shape shape) {
    System.out.println(shape.getArea());
}
```

## 六、小结

本节要点：
1. **面向过程**：关注步骤，适合简单问题
2. **面向对象**：关注对象，适合复杂系统
3. **三大特性**：封装、继承、多态
4. **类与对象**：类是模板，对象是实例
5. **优势**：易维护、易复用、易扩展
