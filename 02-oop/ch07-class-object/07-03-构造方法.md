# 7.3 构造方法

## 一、构造方法概述

构造方法（Constructor）用于创建对象时初始化对象的属性。

**特点**：
- 方法名与类名相同
- 没有返回值类型（连 void 都没有）
- 创建对象时自动调用
- 可以重载

## 二、构造方法的定义

### 2.1 基本语法

```java
[修饰符] 类名 (参数列表) {
    // 初始化代码
}
```

### 2.2 示例

```java
public class Student {
    String name;
    int age;
    
    // 无参构造方法
    public Student() {
        System.out.println("创建了 Student 对象");
    }
    
    // 有参构造方法
    public Student(String name, int age) {
        this.name = name;
        this.age = age;
    }
}
```

## 三、构造方法的使用

### 3.1 无参构造

```java
public class Main {
    public static void main(String[] args) {
        // 调用无参构造
        Student s1 = new Student();
        s1.name = "张三";
        s1.age = 18;
        
        System.out.println(s1.name + ", " + s1.age);
    }
}
```

### 3.2 有参构造

```java
public class Main {
    public static void main(String[] args) {
        // 调用有参构造
        Student s1 = new Student("张三", 18);
        Student s2 = new Student("李四", 19);
        
        System.out.println(s1.name + ", " + s1.age);
        System.out.println(s2.name + ", " + s2.age);
    }
}
```

## 四、默认构造方法

### 4.1 什么是默认构造

如果类中没有定义任何构造方法，Java 会自动提供一个无参构造方法。

```java
// 没有定义构造方法
public class Student {
    String name;
    int age;
}

// 等价于
public class Student {
    String name;
    int age;
    
    // 默认构造方法
    public Student() {
    }
}
```

### 4.2 注意事项

```java
public class Student {
    String name;
    int age;
    
    // 定义了有参构造
    public Student(String name, int age) {
        this.name = name;
        this.age = age;
    }
}

public class Main {
    public static void main(String[] args) {
        // ❌ 错误：定义了有参构造后，默认无参构造不再提供
        // Student s1 = new Student();
        
        // ✅ 正确
        Student s1 = new Student("张三", 18);
    }
}
```

**解决**：手动添加无参构造

```java
public class Student {
    String name;
    int age;
    
    // 无参构造
    public Student() {
    }
    
    // 有参构造
    public Student(String name, int age) {
        this.name = name;
        this.age = age;
    }
}
```

## 五、构造方法重载

### 5.1 多个构造方法

```java
public class Student {
    String name;
    int age;
    String school;
    
    // 无参构造
    public Student() {
    }
    
    // 一个参数
    public Student(String name) {
        this.name = name;
    }
    
 // 两个参数
    public Student(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    // 三个参数
    public Student(String name, int age, String school) {
        this.name = name;
        this.age = age;
        this.school = school;
    }
}
```

### 5.2 使用

```java
public class Main {
    public static void main(String[] args) {
        Student s1 = new Student();
        Student s2 = new Student("张三");
        Student s3 = new Student("李四", 18);
        Student s4 = new Student("王五", 19, "清华大学");
    }
}
```

## 六、this 关键字

### 6.1 this 的含义

`this` 代表当前对象的引用。

### 6.2 区分成员变量和局部变量

```java
public class Student {
    String name;  // 成员变量
    int age;
    
    public Student(String name, int age) {
        // name = name;  // ❌ 错误：都是局部变量
        this.name = name;  // ✅ this.name 是成员变量
        this.age = age;
    }
}
```

### 6.3 this 调用其他构造方法

```java
public class Student {
    String name;
    int age;
    String school;
    
    public Student() {
        this("未知", 0);  // 调用其他构造方法
    }
    
    public Student(String name) {
        this(name, 0);
    }
    
    public Student(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    public Student(String name, int age, String school) {
        this(name, age);  // 必须放在第一行
        this.school = school;
    }
}
```

**注意**：`this()` 必须放在构造方法的第一行。

## 七、构造方法的应用

### 7.1 初始化集合

```java
import java.util.ArrayList;
import java.util.List;

public class Student {
    String name;
    List<String> courses = new ArrayList<>();
    
    public Student(String name) {
        this.name = name;
        // 初始化时添加默认课程
        courses.add("语文");
        courses.add("数学");
        courses.add("英语");
    }
}
```

### 7.2 参数验证

```java
public class Person {
    String name;
    int age;
    
    public Person(String name, int age) {
        if (name == null || name.isEmpty()) {
            throw new IllegalArgumentException("姓名不能为空");
        }
        if (age < 0 || age > 150) {
            throw new IllegalArgumentException("年龄不合法");
        }
        this.name = name;
        this.age = age;
    }
}
```

### 7.3 不可变对象

```java
public final class Person {
    private final String name;
    private final int age;
    
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    // 只有 getter，没有 setter
    public String getName() {
        return name;
    }
    
    public int getAge() {
        return age;
    }
}
```

## 八、综合示例

```java
public class Book {
    // 成员变量
    String title;
    String author;
    double price;
    int stock;
    
    // 无参构造
    public Book() {
        this("未知", "未知", 0.0, 0);
    }
    
    // 两个参数
    public Book(String title, String author) {
        this(title, author, 0.0, 0);
    }
    
    // 三个参数
    public Book(String title, String author, double price) {
        this(title, author, price, 0);
    }
    
    // 完整构造
    public Book(String title, String author, double price, int stock) {
        if (title == null || title.isEmpty()) {
            throw new IllegalArgumentException("书名不能为空");
        }
        if (price < 0) {
            throw new IllegalArgumentException("价格不能为负");
        }
        if (stock < 0) {
            throw new IllegalArgumentException("库存不能为负");
        }
        
        this.title = title;
        this.author = author;
        this.price = price;
        this.stock = stock;
    }
    
    // 显示信息
    public void showInfo() {
        System.out.println("《" + title + "》by " + author 
                         + ", ￥" + price + ", 库存：" + stock);
    }
    
    // 购买
    public boolean buy(int count) {
        if (count > stock) {
            System.out.println("库存不足");
            return false;
        }
        stock -= count;
        System.out.println("购买成功，剩余库存：" + stock);
        return true;
    }
}

public class BookDemo {
    public static void main(String[] args) {
        // 使用不同构造方法
        Book b1 = new Book();
        Book b2 = new Book("Java 编程思想", "Bruce Eckel");
        Book b3 = new Book("Effective Java", "Joshua Bloch", 59.0);
        Book b4 = new Book("深入理解 JVM", "周志明", 79.0, 100);
        
        b1.showInfo();
        b2.showInfo();
        b3.showInfo();
        b4.showInfo();
        
        // 购买
        b4.buy(5);
    }
}
```

## 九、注意事项

### 9.1 构造方法不能被继承

```java
class Parent {
    public Parent() {
        System.out.println("Parent 构造");
    }
}

class Child extends Parent {
    public Child() {
        // 自动调用 super()
        System.out.println("Child 构造");
    }
}
```

### 9.2 构造方法不能手动调用

```java
public Student(String name, int age) {
    this.name = name;
    this.age = age;
    
    // ❌ 错误：不能像普通方法一样调用
    // Student("张三", 18);
}
```

### 9.3 构造方法没有返回值

```java
// ❌ 错误：这不是构造方法，是普通方法
public void Student() {
}

// ✅ 正确
public Student() {
}
```

## 十、小结

本节要点：
1. **特点**：方法名与类名相同，无返回值
2. **作用**：创建对象时初始化属性
3. **默认构造**：未定义时自动提供无参构造
4. **重载**：可以有多个构造方法
5. **this**：代表当前对象，可调用其他构造
6. **注意**：不能被继承，不能手动调用
