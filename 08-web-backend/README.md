# 数据库与 Web 后端

本模块涵盖数据库基础与 Web 后端开发，从 MySQL 数据库和 JDBC 数据访问，到前端基础、Servlet 编程和 Tomcat 服务器，帮助你掌握后端开发的完整技术栈。

## 文件列表

| 文件 / 目录 | 主题 |
|------|------|
| 01-MySQL 基础.md | MySQL 数据库：安装配置、SQL 语法、数据类型、表操作、查询、索引、事务 |
| 02-JDBC.md | JDBC 数据访问：JDBC API、CRUD 操作、PreparedStatement、连接池、事务管理 |
| 03-前端基础.md | 前端基础：HTML/CSS/JavaScript 入门、AJAX、前端框架简介 |
| 04-servlet/ | Servlet 编程（5 篇）：Servlet 概述、请求与响应、Session 与 Cookie、Filter 与 Listener、文件上传 |
| 04-Tomcat 服务器.md | Tomcat 服务器：安装配置、项目部署、虚拟主机、性能调优 |

## 学习目标

- 掌握 MySQL 数据库的基本操作与 SQL 语法
- 理解 JDBC 的工作原理与数据库连接池
- 了解前端基础技术，能与前端开发协作
- 掌握 Servlet 的开发与部署
- 理解 Tomcat 服务器的配置与优化
