# 3.1 字符串处理

## 一、String 类

### 1.1 String 特点

- **不可变性**：String 对象创建后不能修改
- **线程安全**：不可变所以线程安全
- **字符串常量池**：相同字符串共享同一对象

### 1.2 创建字符串

```java
// 方式 1：字面量（推荐）
String s1 = "hello";

// 方式 2：new 关键字
String s2 = new String("hello");

// 方式 3：字符数组
char[] chars = {'h', 'e', 'l', 'l', 'o'};
String s3 = new String(chars);
```

### 1.3 常用方法

```java
String str = "Hello, World!";

// 长度
int len = str.length();  // 13

// 获取字符
char c = str.charAt(0);  // 'H'

// 截取子串
String sub1 = str.substring(0, 5);  // "Hello"
String sub2 = str.substring(7);     // "World!"

// 查找
int index = str.indexOf("World");   // 7
boolean contains = str.contains("World");  // true

// 替换
String replaced = str.replace("World", "Java");  // "Hello, Java!"

// 分割
String[] parts = str.split(", ");  // ["Hello", "World!"]

// 大小写转换
String upper = str.toUpperCase();  // "HELLO, WORLD!"
String lower = str.toLowerCase();  // "hello, world!"

// 去除空格
String trimmed = "  hello  ".trim();  // "hello"

// 比较
boolean equals = str.equals("Hello, World!");  // true
boolean equalsIgnoreCase = str.equalsIgnoreCase("hello, world!");  // true
```

## 二、StringBuilder 和 StringBuffer

### 2.1 可变字符串

```java
// StringBuilder（非线程安全，性能高）
StringBuilder sb1 = new StringBuilder();
sb1.append("Hello");
sb1.append(" ");
sb1.append("World");
String result1 = sb1.toString();  // "Hello World"

// StringBuffer（线程安全，性能稍低）
StringBuffer sb2 = new StringBuffer();
sb2.append("Hello");
String result2 = sb2.toString();
```

### 2.2 常用方法

```java
StringBuilder sb = new StringBuilder();

// 追加
sb.append("Hello");
sb.append(123);
sb.append(true);

// 插入
sb.insert(5, " World");  // "Hello World123true"

// 删除
sb.delete(5, 11);  // "Hello123true"

// 替换
sb.replace(5, 8, "Java");  // "HelloJavatrue"

// 反转
sb.reverse();  // "eurtJavaOlleh"

// 长度
int len = sb.length();

// 容量
int cap = sb.capacity();
```

### 2.3 性能对比

```java
// ❌ 低效：每次 + 都创建新对象
String result = "";
for (int i = 0; i < 1000; i++) {
    result += i;  // 创建 1000 个 String 对象
}

// ✅ 高效：只创建一个 StringBuilder
StringBuilder sb = new StringBuilder();
for (int i = 0; i < 1000; i++) {
    sb.append(i);
}
String result = sb.toString();
```

## 三、小结

1. **String**：不可变，适合少量字符串操作
2. **StringBuilder**：可变，非线程安全，性能高
3. **StringBuffer**：可变，线程安全，性能稍低
