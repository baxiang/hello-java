# 3.3 日期时间 API

## 一、旧版日期 API（Java 7 及以前）

### 1.1 Date 类

```java
import java.util.Date;

// 创建 Date 对象
Date now = new Date();  // 当前时间

// 获取时间戳（毫秒）
long timestamp = now.getTime();

// 从时间戳创建
Date date = new Date(1609459200000L);  // 2021-01-01

// 比较
Date date1 = new Date();
Date date2 = new Date();
boolean isAfter = date1.after(date2);
boolean isBefore = date1.before(date2);
boolean isEqual = date1.equals(date2);
```

### 1.2 SimpleDateFormat

```java
import java.text.SimpleDateFormat;
import java.util.Date;

// 格式化
SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss");
String str = sdf.format(new Date());
System.out.println(str);  // 2024-01-01 12:30:45

// 解析
Date date = sdf.parse("2024-01-01 12:30:45");

// 常用格式
"yyyy-MM-dd"           // 2024-01-01
"yyyy/MM/dd"           // 2024/01/01
"yyyy 年 MM 月 dd 日"     // 2024 年 01 月 01 日
"HH:mm:ss"             // 12:30:45
"yyyy-MM-dd HH:mm:ss"  // 2024-01-01 12:30:45
```

### 1.3 Calendar 类

```java
import java.util.Calendar;
import java.util.GregorianCalendar;

// 创建 Calendar 对象
Calendar cal = Calendar.getInstance();

// 获取字段
int year = cal.get(Calendar.YEAR);
int month = cal.get(Calendar.MONTH) + 1;  // 0-11，需要 +1
int day = cal.get(Calendar.DAY_OF_MONTH);
int hour = cal.get(Calendar.HOUR_OF_DAY);
int minute = cal.get(Calendar.MINUTE);

// 设置字段
cal.set(2024, 0, 1, 12, 0, 0);  // 年、月（0-11）、日、时、分、秒

// 加减
cal.add(Calendar.DAY_OF_MONTH, 7);   // 加 7 天
cal.add(Calendar.MONTH, -1);         // 减 1 个月

// 获取 Date
Date date = cal.getTime();
```

### 1.4 旧版 API 的问题

- **可变性**：Date 和 SimpleDateFormat 都是可变的，线程不安全
- **设计差**：月份从 0 开始，年份从 1900 开始
- **时区处理复杂**：需要额外处理时区

## 二、Java 8 新日期时间 API

### 2.1 LocalDate（日期）

```java
import java.time.LocalDate;

// 创建
LocalDate today = LocalDate.now();                    // 今天
LocalDate date = LocalDate.of(2024, 1, 1);           // 指定日期
LocalDate parsed = LocalDate.parse("2024-01-01");    // 解析

// 获取字段
int year = today.getYear();              // 2024
int month = today.getMonthValue();       // 1-12
int day = today.getDayOfMonth();         // 1-31
DayOfWeek dow = today.getDayOfWeek();    // MONDAY-SUNDAY

// 加减
LocalDate nextWeek = today.plusWeeks(1);
LocalDate nextMonth = today.plusMonths(1);
LocalDate nextYear = today.plusYears(1);
LocalDate lastWeek = today.minusWeeks(1);

// 比较
boolean isAfter = today.isAfter(date);
boolean isBefore = today.isBefore(date);
boolean isEqual = today.isEqual(date);

// 计算间隔
Period period = Period.between(date, today);
int days = period.getDays();
int months = period.getMonths();
int years = period.getYears();
```

### 2.2 LocalTime（时间）

```java
import java.time.LocalTime;

// 创建
LocalTime now = LocalTime.now();
LocalTime time = LocalTime.of(12, 30, 45);
LocalTime parsed = LocalTime.parse("12:30:45");

// 获取字段
int hour = now.getHour();           // 0-23
int minute = now.getMinute();       // 0-59
int second = now.getSecond();       // 0-59
int nano = now.getNano();           // 0-999999999

// 加减
LocalTime later = now.plusHours(2);
LocalTime earlier = now.minusMinutes(30);

// 比较
boolean isAfter = now.isAfter(time);
```

### 2.3 LocalDateTime（日期时间）

```java
import java.time.LocalDateTime;

// 创建
LocalDateTime now = LocalDateTime.now();
LocalDateTime dt = LocalDateTime.of(2024, 1, 1, 12, 30, 45);
LocalDateTime parsed = LocalDateTime.parse("2024-01-01T12:30:45");

// 组合
LocalDate date = LocalDate.now();
LocalTime time = LocalTime.now();
LocalDateTime dt2 = LocalDateTime.of(date, time);

// 获取
LocalDate datePart = dt.toLocalDate();
LocalTime timePart = dt.toLocalTime();

// 加减
LocalDateTime nextWeek = dt.plusWeeks(1);
LocalDateTime lastMonth = dt.minusMonths(1);

// 格式化
DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss");
String str = dt.format(formatter);
LocalDateTime parsed2 = LocalDateTime.parse("2024-01-01 12:30:45", formatter);
```

### 2.4 DateTimeFormatter（格式化）

```java
import java.time.format.DateTimeFormatter;

// 内置格式
DateTimeFormatter ISO_DATE = DateTimeFormatter.ISO_DATE;
DateTimeFormatter ISO_LOCAL_DATE_TIME = DateTimeFormatter.ISO_LOCAL_DATE_TIME;

// 自定义格式
DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss");

// 格式化
LocalDateTime now = LocalDateTime.now();
String str = now.format(formatter);

// 解析
LocalDateTime dt = LocalDateTime.parse("2024-01-01 12:30:45", formatter);

// 常用格式
"yyyy-MM-dd"              // 2024-01-01
"yyyy/MM/dd"              // 2024/01/01
"yyyy 年 MM 月 dd 日"        // 2024 年 01 月 01 日
"HH:mm:ss"                // 12:30:45
"yyyy-MM-dd HH:mm:ss"     // 2024-01-01 12:30:45
"yyyy-MM-dd HH:mm:ss.SSS" // 2024-01-01 12:30:45.123
```

### 2.5 Instant（时间戳）

```java
import java.time.Instant;

// 当前时间戳
Instant now = Instant.now();

// 获取毫秒
long millis = now.toEpochMilli();

// 从毫秒创建
Instant instant = Instant.ofEpochMilli(1609459200000L);

// 与 LocalDateTime 转换
LocalDateTime dt = LocalDateTime.ofInstant(now, ZoneId.systemDefault());
```

### 2.6 ZoneId 和 ZonedDateTime（时区）

```java
import java.time.ZoneId;
import java.time.ZonedDateTime;

// 获取所有时区
Set<String> zoneIds = ZoneId.getAvailableZoneIds();

// 创建时区
ZoneId shanghai = ZoneId.of("Asia/Shanghai");
ZoneId newyork = ZoneId.of("America/New_York");

// 创建带时区的时间
ZonedDateTime zdt = ZonedDateTime.now(shanghai);

// 转换
ZonedDateTime nyTime = zdt.withZoneSameInstant(newyork);

// LocalDateTime 与 ZonedDateTime 转换
LocalDateTime ldt = LocalDateTime.now();
ZonedDateTime zdt2 = ldt.atZone(shanghai);
LocalDateTime ldt2 = zdt2.toLocalDateTime();
```

### 2.7 Duration 和 Period

```java
import java.time.Duration;
import java.time.Period;
import java.time.LocalDateTime;

// Duration：时间间隔（秒、纳秒级别）
LocalDateTime start = LocalDateTime.now();
LocalDateTime end = start.plusHours(2);
Duration duration = Duration.between(start, end);
long hours = duration.toHours();
long minutes = duration.toMinutes();

// Period：日期间隔（年、月、日级别）
LocalDate date1 = LocalDate.of(2024, 1, 1);
LocalDate date2 = LocalDate.of(2024, 3, 15);
Period period = Period.between(date1, date2);
int years = period.getYears();
int months = period.getMonths();
int days = period.getDays();
int totalDays = period.getDays() + period.getMonths() * 30 + period.getYears() * 365;
```

## 三、新旧 API 对比

| 特性 | 旧 API | 新 API |
|------|--------|--------|
| 可变性 | 可变（线程不安全） | 不可变（线程安全） |
| 设计 | 月份从 0 开始 | 月份从 1 开始 |
| 时区 | 复杂 | 内置支持 |
| 格式化 | SimpleDateFormat | DateTimeFormatter |
| 计算 | Calendar 繁琐 | 链式调用简洁 |

## 四、综合示例

```java
import java.time.*;
import java.time.format.DateTimeFormatter;
import java.time.temporal.ChronoUnit;

public class DateTimeExample {
    public static void main(String[] args) {
        // 1. 当前时间
        LocalDate today = LocalDate.now();
        LocalTime now = LocalTime.now();
        LocalDateTime current = LocalDateTime.now();
        
        System.out.println("今天：" + today);
        System.out.println("现在：" + now);
        System.out.println("当前：" + current);
        
        // 2. 格式化
        DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy 年 MM 月 dd 日 HH 时 mm 分");
        String formatted = current.format(formatter);
        System.out.println("格式化：" + formatted);
        
        // 3. 计算
        LocalDate nextWeek = today.plusWeeks(1);
        LocalDate lastMonth = today.minusMonths(1);
        System.out.println("下周：" + nextWeek);
        System.out.println("上月：" + lastMonth);
        
        // 4. 计算年龄
        LocalDate birthday = LocalDate.of(2000, 1, 1);
        Period age = Period.between(birthday, today);
        System.out.println("年龄：" + age.getYears() + "岁");
        
        // 5. 判断闰年
        boolean isLeap = today.isLeapYear();
        System.out.println("今年是否闰年：" + isLeap);
        
        // 6. 时区转换
        ZonedDateTime beijing = ZonedDateTime.now(ZoneId.of("Asia/Shanghai"));
        ZonedDateTime tokyo = beijing.withZoneSameInstant(ZoneId.of("Asia/Tokyo"));
        System.out.println("北京时间：" + beijing);
        System.out.println("东京时间：" + tokyo);
        
        // 7. 计算间隔
        LocalDateTime start = LocalDateTime.of(2024, 1, 1, 9, 0);
        LocalDateTime end = LocalDateTime.of(2024, 1, 1, 18, 0);
        Duration duration = Duration.between(start, end);
        System.out.println("工作时长：" + duration.toHours() + "小时");
        
        // 8. 时间戳
        Instant instant = Instant.now();
        long millis = instant.toEpochMilli();
        System.out.println("时间戳：" + millis);
    }
}
```

## 五、小结

1. **旧 API**：Date、Calendar、SimpleDateFormat（不推荐）
2. **新 API**：LocalDate、LocalTime、LocalDateTime（推荐）
3. **格式化**：DateTimeFormatter（线程安全）
4. **时区**：ZoneId、ZonedDateTime
5. **间隔**：Duration（时间）、Period（日期）
