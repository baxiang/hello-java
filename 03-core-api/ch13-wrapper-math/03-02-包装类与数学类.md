# 3.2 包装类与数学类

## 一、包装类概述

### 1.1 什么是包装类

Java 为 8 种基本数据类型提供了对应的引用类型（包装类）。

| 基本类型 | 包装类 |
|----------|--------|
| byte | Byte |
| short | Short |
| int | Integer |
| long | Long |
| float | Float |
| double | Double |
| char | Character |
| boolean | Boolean |

### 1.2 为什么需要包装类

- 集合只能存储对象，不能存储基本类型
- 需要提供更多实用方法
- 支持 null 值表示"无值"

## 二、自动装箱与拆箱

### 2.1 自动装箱

```java
// 基本类型 → 包装类
int i = 10;
Integer integer = i;  // 自动装箱

// 等价于
Integer integer = Integer.valueOf(i);
```

### 2.2 自动拆箱

```java
// 包装类 → 基本类型
Integer integer = 10;
int i = integer;  // 自动拆箱

// 等价于
int i = integer.intValue();
```

### 2.3 注意事项

```java
// ❌ 可能空指针
Integer num = null;
int value = num;  // NullPointerException

// ✅ 安全做法
Integer num = 10;
int value = (num != null) ? num : 0;

// 缓存池（-128 到 127）
Integer a = 127;
Integer b = 127;
System.out.println(a == b);  // true（缓存）

Integer c = 128;
Integer d = 128;
System.out.println(c == d);  // false（新对象）
```

## 三、常用包装类方法

### 3.1 Integer

```java
// 字符串转 int
int num = Integer.parseInt("123");
int num2 = Integer.parseInt("FF", 16);  // 16 进制

// int 转字符串
String str = String.valueOf(123);
String str2 = Integer.toString(123);

// 最大值最小值
int max = Integer.MAX_VALUE;  // 2147483647
int min = Integer.MIN_VALUE;  // -2147483648

// 比较
int result = Integer.compare(10, 20);  // -1

// 判断
boolean isNum = Integer.valueOf("123").equals(123);
```

### 3.2 Double

```java
// 字符串转 double
double d = Double.parseDouble("3.14");

// double 转字符串
String str = Double.toString(3.14);

// 特殊值
double inf = Double.POSITIVE_INFINITY;  // 正无穷
double nan = Double.NaN;  // 非数字

// 判断
boolean isNan = Double.isNaN(0.0 / 0.0);  // true
boolean isInfinite = Double.isInfinite(1.0 / 0.0);  // true
```

### 3.3 Boolean

```java
// 字符串转 boolean
boolean b1 = Boolean.parseBoolean("true");   // true
boolean b2 = Boolean.parseBoolean("false");  // false
boolean b3 = Boolean.parseBoolean("yes");    // false（只有"true"才是 true）

// 常量
Boolean.TRUE
Boolean.FALSE
```

### 3.4 Character

```java
// 判断
char c = 'A';
boolean isUpper = Character.isUpperCase(c);      // true
boolean isLower = Character.isLowerCase(c);      // false
boolean isDigit = Character.isDigit('5');        // true
boolean isLetter = Character.isLetter(c);        // true
boolean isLetterOrDigit = Character.isLetterOrDigit('5');  // true

// 转换
char upper = Character.toUpperCase('a');  // 'A'
char lower = Character.toLowerCase('A');  // 'a'
```

## 四、数学类

### 4.1 Math 类

```java
// 绝对值
Math.abs(-10);  // 10

// 最大最小
Math.max(10, 20);  // 20
Math.min(10, 20);  // 10

// 幂运算
Math.pow(2, 3);  // 8.0

// 平方根
Math.sqrt(16);  // 4.0

// 四舍五入
Math.round(3.5);  // 4
Math.round(3.4);  // 3

// 向上取整
Math.ceil(3.1);  // 4.0

// 向下取整
Math.floor(3.9);  // 3.0

// 随机数（0.0-1.0）
double random = Math.random();

// 三角函数
Math.sin(Math.PI / 2);  // 1.0
Math.cos(0);            // 1.0
Math.tan(Math.PI / 4);  // 1.0

// 对数
Math.log(Math.E);   // 1.0（自然对数）
Math.log10(100);    // 2.0（以 10 为底）

// 常量
double pi = Math.PI;   // 3.141592653589793
double e = Math.E;     // 2.718281828459045
```

### 4.2 Random 类

```java
import java.util.Random;

Random random = new Random();

// 随机整数
int i = random.nextInt();       // 任意整数
int i2 = random.nextInt(100);   // 0-99

// 随机小数
double d = random.nextDouble(); // 0.0-1.0
float f = random.nextFloat();   // 0.0f-1.0f

// 随机布尔
boolean b = random.nextBoolean();

// 随机高斯分布
double g = random.nextGaussian();

// 指定范围随机数
int min = 10, max = 20;
int randomNum = random.nextInt(max - min + 1) + min;  // 10-20
```

### 4.3 ThreadLocalRandom（Java 8+）

```java
import java.util.concurrent.ThreadLocalRandom;

// 多线程环境下性能更好
int i = ThreadLocalRandom.current().nextInt(1, 101);  // 1-100
double d = ThreadLocalRandom.current().nextDouble(0.0, 1.0);
```

## 五、BigDecimal 精确计算

### 5.1 为什么需要 BigDecimal

```java
// ❌ double 精度问题
double a = 0.1;
double b = 0.2;
System.out.println(a + b);  // 0.30000000000000004

// ✅ BigDecimal 精确计算
BigDecimal bd1 = new BigDecimal("0.1");
BigDecimal bd2 = new BigDecimal("0.2");
System.out.println(bd1.add(bd2));  // 0.3
```

### 5.2 基本使用

```java
import java.math.BigDecimal;

// 创建（推荐用字符串）
BigDecimal bd1 = new BigDecimal("10.5");
BigDecimal bd2 = new BigDecimal("3");

// 加减乘除
BigDecimal sum = bd1.add(bd2);         // 13.5
BigDecimal diff = bd1.subtract(bd2);   // 7.5
BigDecimal product = bd1.multiply(bd2); // 31.5
BigDecimal quotient = bd1.divide(bd2, 2, RoundingMode.HALF_UP); // 3.50

// 比较
int result = bd1.compareTo(bd2);  // 1（大于）

// 最大值最小值
BigDecimal max = bd1.max(bd2);
BigDecimal min = bd1.min(bd2);

// 设置精度
BigDecimal rounded = bd1.setScale(2, RoundingMode.HALF_UP);  // 10.50
```

### 5.3 常用舍入模式

```java
RoundingMode.HALF_UP      // 四舍五入
RoundingMode.HALF_DOWN    // 五舍六入
RoundingMode.CEILING      // 向上取整
RoundingMode.FLOOR        // 向下取整
RoundingMode.TRUNCATE     // 截断
```

## 六、综合示例

```java
public class MathExample {
    public static void main(String[] args) {
        // 1. 包装类转换
        String str = "123";
        int num = Integer.parseInt(str);
        Integer integer = num;  // 装箱
        
        // 2. 数学计算
        double result = Math.pow(2, 10);  // 1024.0
        
        // 3. 随机数
        int randomNum = (int) (Math.random() * 100) + 1;  // 1-100
        
        // 4. 精确计算（金额）
        BigDecimal price = new BigDecimal("99.99");
        BigDecimal quantity = new BigDecimal("3");
        BigDecimal total = price.multiply(quantity);
        System.out.println("总价：" + total);  // 299.97
        
        // 5. 折扣计算
        BigDecimal discount = new BigDecimal("0.8");
        BigDecimal finalPrice = total.multiply(discount)
                                     .setScale(2, RoundingMode.HALF_UP);
        System.out.println("折后价：" + finalPrice);  // 239.98
    }
}
```

## 七、小结

1. **包装类**：基本类型的引用类型版本
2. **自动装箱/拆箱**：自动转换，注意空指针
3. **Math 类**：常用数学运算
4. **Random**：生成随机数
5. **BigDecimal**：精确计算，适合金额
