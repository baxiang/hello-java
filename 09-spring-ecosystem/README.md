# Spring 生态（Framework + Boot）

本模块整合 Spring Framework 与 Spring Boot 两大核心框架，从 IoC/AOP 基础到 Spring MVC，再从 Spring Boot 快速入门到 Web 开发、数据访问、Redis、消息队列、安全、测试与部署，全面掌握 Spring 生态技术栈。

## 章节列表

| 目录 | 主题 |
|------|------|
| 01-spring-framework | Spring Framework：IoC 容器、Bean 管理、依赖注入、AOP 面向切面编程 |
| 02-spring-mvc | Spring MVC：MVC 架构、请求映射、参数绑定、拦截器、异常处理 |
| 03-source-code | 源码解读：Spring IOC 源码解读、Spring AOP 源码解读 |
| 04-spring-boot-intro | Spring Boot 入门：自动配置原理、起步依赖、快速搭建项目 |
| 05-config | 配置文件：application.yml/properties、多环境配置、配置注入 |
| 06-web | Web 开发：RESTful API、参数校验、统一响应、全局异常处理 |
| 07-data-access | 数据访问：MyBatis/MyBatis-Plus 整合、JPA、多数据源 |
| 08-redis | Redis：Redis 简介与安装、RedisTemplate 使用、缓存注解、分布式锁、实战示例 |
| 09-mq | 消息队列：RabbitMQ 基础、Spring AMQP、Kafka 基础、消息可靠性、实战示例 |
| 10-schedule | 定时任务：@Scheduled、Cron 表达式、分布式定时任务 |
| 11-security | 安全：Spring Security 基础、认证与授权、JWT 令牌、OAuth2、实战示例 |
| 12-test | 测试：单元测试、集成测试、Mock 测试 |
| 13-deploy | 部署与监控：打包部署、Actuator 监控、日志管理 |
| 14-native | Spring Boot Native：GraalVM Native Image 编译 |

## 学习目标

- 理解 Spring IoC 与 AOP 核心原理
- 掌握 Spring MVC 的请求处理流程
- 熟练使用 Spring Boot 快速搭建应用
- 掌握数据访问、缓存、消息队列等中间件整合
- 理解 Spring Security 安全认证与授权机制
- 掌握 Spring Boot 应用的测试与部署
