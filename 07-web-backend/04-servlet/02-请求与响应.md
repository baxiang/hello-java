# 41.2 请求与响应：数据的双向传递（生活类比：快递包裹）

## 一、先问一个问题
**问题**：表单提交的数据是如何传递到服务器的？

### 生活例子
快递包裹：
- 寄件人填写包裹单（请求参数）
- 快递员收件（服务器接收）
- 派送到目的地（处理业务）
- 收件人签收（返回响应）

HttpServletRequest 和 HttpServletResponse 就是"包裹单"！

## 二、HttpServletRequest 获取请求参数

### 2.1 获取单个参数

```java
import javax.servlet.*;
import javax.servlet.http.*;
import java.io.IOException;

@WebServlet("/param")
public class ParamServlet extends HttpServlet {
    
    @Override
    protected void doGet(HttpServletRequest request, HttpServletResponse response) 
            throws ServletException, IOException {
        
        // 获取单个参数
        String username = request.getParameter("username");
        String age = request.getParameter("age");
        
        response.setContentType("text/html;charset=UTF-8");
        response.getWriter().println("用户名：" + username);
        response.getWriter().println("年龄：" + age);
    }
}
```

### 2.2 获取多个参数和请求头

```java
import javax.servlet.*;
import javax.servlet.http.*;
import java.io.IOException;

@WebServlet("/params")
public class ParamsServlet extends HttpServlet {
    
    @Override
    protected void doGet(HttpServletRequest request, HttpServletResponse response) 
            throws ServletException, IOException {
        
        // 获取多个参数（复选框）
        String[] hobbies = request.getParameterValues("hobbies");
        
        // 获取请求头
        String userAgent = request.getHeader("User-Agent");
        String referer = request.getHeader("Referer");
        
        // 获取请求方法
        String method = request.getMethod();
        
        response.setContentType("text/html;charset=UTF-8");
        response.getWriter().println("请求方法：" + method);
        response.getWriter().println("User-Agent：" + userAgent);
    }
}
```

## 三、HttpServletResponse 设置响应

### 3.1 设置响应类型和内容

```java
import javax.servlet.*;
import javax.servlet.http.*;
import java.io.IOException;
import java.io.PrintWriter;
import java.util.Date;

@WebServlet("/response")
public class ResponseServlet extends HttpServlet {
    
    @Override
    protected void doGet(HttpServletRequest request, HttpServletResponse response) 
            throws ServletException, IOException {
        
        // 设置响应类型
        response.setContentType("text/html;charset=UTF-8");
        
        // 设置响应头
        response.setHeader("Cache-Control", "no-cache");
        
        // 输出内容
        PrintWriter out = response.getWriter();
        out.println("<h1>Hello, Response!</h1>");
        out.println("<p>当前时间：" + new Date() + "</p>");
    }
}
```

### 3.2 输出 JSON 数据

```java
import javax.servlet.*;
import javax.servlet.http.*;
import java.io.IOException;

@WebServlet("/api/user")
public class UserServlet extends HttpServlet {
    
    @Override
    protected void doGet(HttpServletRequest request, HttpServletResponse response)

        // 模拟用户数据
        String json = "{\"id\":1,\"name\":\"张三\",\"email\":\"zhang@example.com\"}";
        response.getWriter().println(json);
    }
}
```

## 四、转发与重定向

### 4.1 请求转发（服务器内部跳转）

```java
import javax.servlet.*;
import javax.servlet.http.*;
import java.io.IOException;

@WebServlet("/forward")
public class ForwardServlet extends HttpServlet {
    
    @Override
    protected void doGet(HttpServletRequest request, HttpServletResponse response) 
            throws ServletException, IOException {
        
        // 设置请求属性
        request.setAttribute("message", "Hello from ForwardServlet");
        
        // 转发到另一个 Servlet
        request.getRequestDispatcher("/target").forward(request, response);
    }
}
```

### 4.2 重定向（客户端跳转）

```java
import javax.servlet.*;
import javax.servlet.http.*;
import java.io.IOException;

@WebServlet("/redirect")
public class RedirectServlet extends HttpServlet {
    
    @Override
    protected void doGet(HttpServletRequest request, HttpServletResponse response) 
            throws ServletException, IOException {
        
        // 重定向到另一个 URL
        response.sendRedirect("/login.html");
    }
}
```

### 4.3 转发 vs 重定向

| 特性 | 转发 (forward) | 重定向 (redirect) |
|------|---------------|------------------|
| 跳转位置 | 服务器内部 | 客户端重新请求 |
| URL 变化 | 不变 | 改变 |
| 请求数据 | 共享 | 丢失 |
| 速度 | 快 | 慢 |

## 五、常见疑问

**Q1：getParameter 和 getAttribute 有什么区别？**
A：getParameter 获取请求参数（表单提交），getAttribute 获取服务器端设置的属性（转发共享）。

**Q2：转发和重定向什么时候用？**
A：转发用于服务器内部跳转（如 Servlet → JSP），重定向用于跳转到外部 URL 或防止表单重复提交。

## 六、小结

| 对象 | 用途 | 常用方法 |
|------|------|----------|
| HttpServletRequest | 获取请求数据 | getParameter(), getHeader() |
| HttpServletResponse | 设置响应数据 | getWriter(), sendRedirect() |
| 转发 | 服务器内部跳转 | request.getRequestDispatcher().forward() |
| 重定向 | 客户端重新请求 | response.sendRedirect() |
