# 开发工具与构建

本模块介绍 Java 开发中必备的工程化工具，包括 Maven 项目构建与管理、Git 版本控制、IDEA 和 VSCode 高级使用技巧，以及 Spring Boot 开发环境配置，帮助你提升开发效率和团队协作能力。

## 文件列表

| 文件 | 主题 |
|------|------|
| 01-Maven.md | Maven：安装配置、POM 文件详解、依赖管理、生命周期与插件、多模块项目、依赖冲突解决 |
| 02-Git 版本控制.md | Git 版本控制：基础概念、常用命令、分支管理、远程仓库、代码合并与冲突解决 |
| 03-IDEA 高级技巧.md | IDEA 高级技巧：常用快捷键、代码模板与重构、调试技巧、插件推荐 |
| 04-VSCode 开发 Java.md | VSCode 开发 Java：扩展安装、项目创建、代码编辑、调试测试、远程开发 |
| 05-Spring Boot 开发环境.md | Spring Boot 开发环境：IDE 配置、DevTools 热部署、多环境配置、调试技巧、接口测试 |

## 学习目标

- 掌握 Maven 的项目构建、依赖管理与多模块配置
- 熟练使用 Git 进行版本控制与团队协作
- 掌握 IDEA 和 VSCode 的 Java 开发技巧
- 配置 Spring Boot 开发环境，实现高效热部署
- 掌握接口调试与测试方法
