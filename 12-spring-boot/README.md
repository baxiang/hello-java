# 第十二部分：Spring Boot

本模块学习 Spring Boot 框架，从快速入门到 Web 开发、数据访问、Redis 整合、消息队列、安全框架、测试与部署，掌握现代化 Spring 应用的开发方式。

## 章节列表

| 章节 | 目录 | 主题 |
|------|------|------|
| 第 49 章 | ch49-spring-boot-intro | Spring Boot 入门：简介、快速搭建、项目结构、起步依赖与自动配置原理 |
| 第 50 章 | ch50-config | 配置文件：application.properties/yml、自定义配置、@Value/@ConfigurationProperties、多环境配置、配置加载顺序 |
| 第 51 章 | ch51-web | Web 开发：整合 Spring MVC、统一响应格式、全局异常处理、参数校验、拦截器与过滤器、CORS 跨域 |
| 第 52 章 | ch52-data-access | 数据访问：整合 MyBatis/MyBatis-Plus、整合 JPA、多数据源、分页插件、动态 SQL |
| 第 53 章 | ch53-redis | Redis 整合：Redis 简介、Spring Data Redis、RedisTemplate、缓存注解、分布式锁 |
| 第 54 章 | ch54-mq | 消息队列：RabbitMQ 基础、Spring AMQP、Kafka 基础与整合 |
| 第 55 章 | ch55-schedule | 定时任务：@Scheduled 注解、Cron 表达式、整合 Quartz |
| 第 56 章 | ch56-security | 安全框架：Spring Security 基础、认证与授权、JWT 令牌、OAuth2 简介 |
| 第 57 章 | ch57-test | 测试：JUnit 5 单元测试、MockMvc、Mockito 模拟对象 |
| 第 58 章 | ch58-deploy | 部署与监控：打包部署、Actuator 监控、日志配置、Docker 容器化部署 |
| 第 59 章 | ch59-native | Spring Boot Native 编译：GraalVM 基础、AOT 编译、Native Image 构建、限制与最佳实践 |

## 学习目标

- 理解 Spring Boot 自动配置原理
- 掌握配置文件管理与多环境配置
- 熟练使用 Spring Boot 开发 RESTful API
- 掌握 Redis、消息队列等中间件整合
- 了解安全框架、定时任务与测试
- 掌握应用部署与监控方案
