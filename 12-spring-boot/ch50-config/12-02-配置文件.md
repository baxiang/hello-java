# 12.2 配置文件

## 一、配置文件概述

Spring Boot 使用配置文件来管理应用的配置信息，支持 properties 和 yml 两种格式。

**配置文件位置**：
```
src/main/resources/
├── application.properties
├── application.yml
├── application-dev.yml      # 开发环境
├── application-test.yml     # 测试环境
└── application-prod.yml     # 生产环境
```

## 二、properties 配置

### 2.1 基本语法

```properties
# 注释使用#
# 键值对使用=或:
key=value
key:value

# 字符串
app.name=MyApp

# 数字
app.port=8080

# 布尔值
app.enabled=true

# 日期
app.date=2024-01-01
```

### 2.2 常用配置

```properties
# 服务器配置
server.port=8080
server.servlet.context-path=/api
server.servlet.encoding.charset=UTF-8

# Spring 配置
spring.application.name=demo
spring.profiles.active=dev

# 日志配置
logging.level.root=INFO
logging.level.com.example=DEBUG
logging.file.name=app.log

# 数据库配置
spring.datasource.url=jdbc:mysql://localhost:3306/demo
spring.datasource.username=root
spring.datasource.password=123456
spring.datasource.driver-class-name=com.mysql.cj.jdbc.Driver
```

## 三、YAML 配置

### 3.1 YAML 语法

```yaml
# 注释使用#
# 键值对使用:（冒号后有空格）
key: value

# 层级使用缩进（2 个空格，不能用 Tab）
app:
  name: MyApp
  port: 8080

# 数组
app:
  servers:
    - server1
    - server2

# 行内数组
app:
  servers: [server1, server2]
```

### 3.2 完整示例

```yaml
server:
  port: 8080
  servlet:
    context-path: /api
    encoding:
      charset: UTF-8
      force: true

spring:
  application:
    name: demo
  profiles:
    active: dev
  datasource:
    url: jdbc:mysql://localhost:3306/demo
    username: root
    password: 123456
    driver-class-name: com.mysql.cj.jdbc.Driver

logging:
  level:
    root: INFO
    com.example: DEBUG
  file:
    name: app.log
  pattern:
    console: "%d{yyyy-MM-dd HH:mm:ss} - %msg%n"
```

### 3.3 对象和数组

```yaml
# 对象
person:
  name: 张三
  age: 18
  address:
    city: 北京
    district: 朝阳

# 数组
students:
  - name: 张三
    age: 18
  - name: 李四
    age: 19

# 行内对象
person: {name: 张三，age: 18}

# 行内数组
numbers: [1, 2, 3]
```

## 四、@Value 注入配置

### 4.1 基本用法

```java
@Component
public class MyComponent {
    
    // 注入单个值
    @Value("${app.name}")
    private String appName;
    
    @Value("${app.port}")
    private int port;
    
    @Value("${app.enabled:true}")  // 默认值
    private boolean enabled;
}
```

### 4.2 配置文件

```yaml
app:
  name: MyApp
  port: 8080
  enabled: true
```

### 4.3 注入数组

```java
@Component
public class MyComponent {
    
    // 注入数组
    @Value("${app.servers}")
    private String[] servers;
    
    // 注入 List
    @Value("#{'${app.names}'.split(',')}")
    private List<String> names;
}
```

```yaml
app:
  servers:
    - server1
    - server2
    - server3
  names: 张三，李四，王五
```

## 五、@ConfigurationProperties

### 5.1 基本用法

```java
@Component
@ConfigurationProperties(prefix = "app")
public class AppProperties {
    
    private String name;
    private int port;
    private boolean enabled;
    private List<String> servers;
    
    // Getter 和 Setter
    public String getName() { return name; }
    public void setName(String name) { this.name = name; }
    public int getPort() { return port; }
    public void setPort(int port) { this.port = port; }
    public boolean isEnabled() { return enabled; }
    public void setEnabled(boolean enabled) { this.enabled = enabled; }
    public List<String> getServers() { return servers; }
    public void setServers(List<String> servers) { this.servers = servers; }
}
```

### 5.2 配置文件

```yaml
app:
  name: MyApp
  port: 8080
  enabled: true
  servers:
    - server1
    - server2
```

### 5.3 使用

```java
@RestController
public class ConfigController {
    
    @Autowired
    private AppProperties appProperties;
    
    @GetMapping("/config")
    public AppProperties getConfig() {
        return appProperties;
    }
}
```

### 5.4 开启配置处理

```java
@SpringBootApplication
@EnableConfigurationProperties(AppProperties.class)
public class DemoApplication {
    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
}
```

## 六、多环境配置

### 6.1 创建多环境配置文件

```
resources/
├── application.yml           # 公共配置
├── application-dev.yml       # 开发环境
├── application-test.yml      # 测试环境
└── application-prod.yml      # 生产环境
```

### 6.2 公共配置

```yaml
# application.yml
spring:
  profiles:
    active: dev  # 默认激活开发环境

app:
  name: MyApp
```

### 6.3 开发环境

```yaml
# application-dev.yml
server:
  port: 8080

spring:
  datasource:
    url: jdbc:mysql://localhost:3306/dev_db
    username: dev_user
    password: dev_pwd

logging:
  level:
    root: DEBUG
```

### 6.4 生产环境

```yaml
# application-prod.yml
server:
  port: 80

spring:
  datasource:
    url: jdbc:mysql://prod-server:3306/prod_db
    username: prod_user
    password: prod_pwd

logging:
  level:
    root: WARN
```

### 6.5 激活环境

```bash
# 方式 1：配置文件中激活
spring.profiles.active=dev

# 方式 2：命令行参数
java -jar demo.jar --spring.profiles.active=prod

# 方式 3：环境变量
export SPRING_PROFILES_ACTIVE=prod

# 方式 4：IDE 配置
# VM Options: -Dspring.profiles.active=dev
```

## 七、配置加载顺序

### 7.1 优先级顺序

```
1. 命令行参数
2. SPRING_APPLICATION_JSON 中的属性
3. ServletConfig 参数
4. ServletContext 参数
5. JNDI 属性
6. Java 系统属性（System.getProperties()）
7. 操作系统环境变量
8. RandomValuePropertySource
9. jar 包外的 application-{profile}.properties
10. jar 包内的 application-{profile}.properties
11. jar 包外的 application.properties
12. jar 包内的 application.properties
13. @PropertySource 注解
14. 默认属性
```

### 7.2 外部配置

```bash
# 命令行参数（优先级最高）
java -jar demo.jar --server.port=9000

# 外部配置文件
java -jar demo.jar --spring.config.location=file:/path/to/config.yml
```

## 八、综合示例

### 8.1 配置类

```java
@Component
@ConfigurationProperties(prefix = "mail")
public class MailProperties {
    
    private String host;
    private int port;
    private String username;
    private String password;
    private boolean enabled;
    
    // Getter 和 Setter
    public String getHost() { return host; }
    public void setHost(String host) { this.host = host; }
    public int getPort() { return port; }
    public void setPort(int port) { this.port = port; }
    public String getUsername() { return username; }
    public void setUsername(String username) { this.username = username; }
    public String getPassword() { return password; }
    public void setPassword(String password) { this.password = password; }
    public boolean isEnabled() { return enabled; }
    public void setEnabled(boolean enabled) { this.enabled = enabled; }
}
```

### 8.2 配置文件

```yaml
# application.yml
spring:
  profiles:
    active: dev

mail:
  enabled: true

---
# application-dev.yml
mail:
  host: smtp.dev.example.com
  port: 587
  username: dev@example.com
  password: dev123

---
# application-prod.yml
mail:
  host: smtp.example.com
  port: 465
  username: admin@example.com
  password: prod123
```

### 8.3 使用配置

```java
@Service
public class MailService {
    
    @Autowired
    private MailProperties mailProperties;
    
    @Value("${app.name:DefaultApp}")
    private String appName;
    
    public void sendMail(String to, String subject, String content) {
        if (!mailProperties.isEnabled()) {
            return;
        }
        
        System.out.println("发送邮件到：" + to);
        System.out.println("SMTP 服务器：" + mailProperties.getHost());
        System.out.println("应用名称：" + appName);
    }
}
```

## 九、小结

本节要点：
1. **配置文件格式**：properties 和 yml
2. **@Value**：注入单个配置项
3. **@ConfigurationProperties**：批量注入配置
4. **多环境**：dev/test/prod 配置
5. **加载顺序**：命令行参数优先级最高
