# 12.7 定时任务

## 一、@Scheduled 注解

### 1.1 开启定时任务

```java
@SpringBootApplication
@EnableScheduling
public class DemoApplication {
    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
}
```

### 1.2 基本用法

```java
@Component
public class ScheduledTasks {
    
    // 固定延迟执行（上次执行完成后 5 秒再执行）
    @Scheduled(fixedDelay = 5000)
    public void task1() {
        System.out.println("task1: " + LocalDateTime.now());
    }
    
    // 固定频率执行（不管是否执行完，每 5 秒执行一次）
    @Scheduled(fixedRate = 5000)
    public void task2() {
        System.out.println("task2: " + LocalDateTime.now());
    }
    
    // 初始延迟 1 秒后开始，然后每 5 秒执行一次
    @Scheduled(initialDelay = 1000, fixedRate = 5000)
    public void task3() {
        System.out.println("task3: " + LocalDateTime.now());
    }
}
```

## 二、Cron 表达式

### 2.1 Cron 语法

```
秒 分 时 日 月 周
```

| 字段 | 允许值 | 特殊字符 |
|------|--------|----------|
| 秒 | 0-59 | , - * / |
| 分 | 0-59 | , - * / |
| 时 | 0-23 | , - * / |
| 日 | 1-31 | , - * ? / L W |
| 月 | 1-12 或 JAN-DEC | , - * / |
| 周 | 1-7 或 SUN-SAT | , - * ? / L # |

### 2.2 特殊字符

| 字符 | 说明 | 示例 |
|------|------|------|
| * | 任意值 | * 表示每秒 |
| ? | 不指定值 | 用于日和周字段 |
| - | 范围 | 1-5 表示 1 到 5 |
| , | 枚举 | 1,3,5 表示 1、3、5 |
| / | 增量 | 0/5 表示从 0 开始每 5 |
| L | 最后 | L 表示每月最后一天 |
| W | 工作日 | 15W 表示离 15 号最近的工作日 |
| # | 第几个周几 | 5#3 表示每月第 3 个周四 |

### 2.3 常用示例

```java
@Component
public class CronTasks {
    
    // 每分钟执行一次
    @Scheduled(cron = "0 * * * * *")
    public void everyMinute() {
        System.out.println("每分钟执行");
    }
    
    // 每小时执行一次
    @Scheduled(cron = "0 0 * * * *")
    public void everyHour() {
        System.out.println("每小时执行");
    }
    
    // 每天凌晨 1 点执行
    @Scheduled(cron = "0 0 1 * * *")
    public void everyDay() {
        System.out.println("每天凌晨 1 点执行");
    }
    
    // 每周一上午 9 点执行
    @Scheduled(cron = "0 0 9 ? * MON")
    public void everyMonday() {
        System.out.println("每周一上午 9 点执行");
    }
    
    // 每月 1 号凌晨执行
    @Scheduled(cron = "0 0 0 1 * *")
    public void everyMonth() {
        System.out.println("每月 1 号执行");
    }
    
    // 工作日早上 8 点执行
    @Scheduled(cron = "0 0 8 ? * MON-FRI")
    public void workDay() {
        System.out.println("工作日早上 8 点执行");
    }
}
```

## 三、配置文件中使用 Cron

### 3.1 配置文件

```yaml
task:
  cron:
    report: "0 0 2 * * *"
    cleanup: "0 0 3 * * *"
```

### 3.2 使用配置

```java
@Component
public class ConfigTasks {
    
    @Value("${task.cron.report}")
    private String reportCron;
    
    @Value("${task.cron.cleanup}")
    private String cleanupCron;
    
    @Scheduled(cron = "${task.cron.report}")
    public void generateReport() {
        System.out.println("生成报表");
    }
    
    @Scheduled(cron = "${task.cron.cleanup}")
    public void cleanup() {
        System.out.println("清理数据");
    }
}
```

## 四、整合 Quartz

### 4.1 添加依赖

```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-quartz</artifactId>
</dependency>
```

### 4.2 创建 Job

```java
public class DataSyncJob implements Job {
    
    @Override
    public void execute(JobExecutionContext context) {
        System.out.println("数据同步任务执行：" + LocalDateTime.now());
        
        // 业务逻辑
        JobDataMap dataMap = context.getMergedJobDataMap();
        String param = dataMap.getString("param");
        System.out.println("参数：" + param);
    }
}
```

### 4.3 配置 Job

```java
@Configuration
public class QuartzConfig {
    
    @Bean
    public JobDetail dataSyncJob() {
        return JobBuilder.newJob(DataSyncJob.class)
                .withIdentity("dataSyncJob", "dataSyncGroup")
                .withDescription("数据同步任务")
                .storeDurably()
                .build();
    }
    
    @Bean
    public Trigger dataSyncTrigger() {
        CronScheduleBuilder scheduleBuilder = 
            CronScheduleBuilder.cronSchedule("0/10 * * * * ?");
        
        return TriggerBuilder.newTrigger()
                .forJob(dataSyncJob())
                .withIdentity("dataSyncTrigger", "dataSyncGroup")
                .withSchedule(scheduleBuilder)
                .build();
    }
}
```

### 4.4 动态管理任务

```java
@Service
public class QuartzService {
    
    @Autowired
    private Scheduler scheduler;
    
    // 添加任务
    public void addJob(String jobName, String jobGroup, 
                       String cron, Class<? extends Job> jobClass) 
                       throws SchedulerException {
        JobDetail job = JobBuilder.newJob(jobClass)
                .withIdentity(jobName, jobGroup)
                .build();
        
        CronTrigger trigger = TriggerBuilder.newTrigger()
                .withIdentity(jobName + "Trigger", jobGroup)
                .withSchedule(CronScheduleBuilder.cronSchedule(cron))
                .build();
        
        scheduler.scheduleJob(job, trigger);
    }
    
    // 暂停任务
    public void pauseJob(String jobName, String jobGroup) 
            throws SchedulerException {
        scheduler.pauseJob(JobKey.jobKey(jobName, jobGroup));
    }
    
    // 恢复任务
    public void resumeJob(String jobName, String jobGroup) 
            throws SchedulerException {
        scheduler.resumeJob(JobKey.jobKey(jobName, jobGroup));
    }
    
    // 删除任务
    public void deleteJob(String jobName, String jobGroup) 
            throws SchedulerException {
        scheduler.deleteJob(JobKey.jobKey(jobName, jobGroup));
    }
    
    // 修改任务执行时间
    public void modifyCron(String jobName, String jobGroup, 
                          String cron) throws SchedulerException {
        TriggerKey triggerKey = TriggerKey.triggerKey(jobName + "Trigger", jobGroup);
        CronTrigger trigger = (CronTrigger) scheduler.getTrigger(triggerKey);
        
        CronScheduleBuilder scheduleBuilder = 
            CronScheduleBuilder.cronSchedule(cron);
        trigger = trigger.getTriggerBuilder()
                .withIdentity(triggerKey)
                .withSchedule(scheduleBuilder)
                .build();
        
        scheduler.rescheduleJob(triggerKey, trigger);
    }
}
```

## 五、应用场景

### 5.1 数据备份

```java
@Component
public class BackupTask {
    
    @Autowired
    private BackupService backupService;
    
    // 每天凌晨 2 点备份数据
    @Scheduled(cron = "0 0 2 * * *")
    public void backup() {
        System.out.println("开始备份数据...");
        backupService.backupDatabase();
        System.out.println("数据备份完成");
    }
}
```

### 5.2 订单超时取消

```java
@Component
public class OrderTimeoutTask {
    
    @Autowired
    private OrderService orderService;
    
    // 每 5 分钟检查一次超时订单
    @Scheduled(cron = "0 */5 * * * *")
    public void cancelTimeoutOrders() {
        System.out.println("检查超时订单...");
        List<Order> timeoutOrders = orderService.findTimeoutOrders();
        
        for (Order order : timeoutOrders) {
            orderService.cancelOrder(order.getId());
            System.out.println("取消订单：" + order.getId());
        }
    }
}
```

### 5.3 报表生成

```java
@Component
public class ReportTask {
    
    @Autowired
    private ReportService reportService;
    
    // 每天早上 6 点生成昨日报表
    @Scheduled(cron = "0 0 6 * * *")
    public void generateDailyReport() {
        System.out.println("生成日报表...");
        LocalDate yesterday = LocalDate.now().minusDays(1);
        reportService.generateDailyReport(yesterday);
    }
    
    // 每周一早上 7 点生成上周报表
    @Scheduled(cron = "0 0 7 ? * MON")
    public void generateWeeklyReport() {
        System.out.println("生成周报表...");
        reportService.generateWeeklyReport();
    }
    
    // 每月 1 号早上 8 点生成上月报表
    @Scheduled(cron = "0 0 8 1 * *")
    public void generateMonthlyReport() {
        System.out.println("生成月报表...");
        reportService.generateMonthlyReport();
    }
}
```

### 5.4 缓存清理

```java
@Component
public class CacheCleanupTask {
    
    @Autowired
    private RedisTemplate<String, Object> redisTemplate;
    
    // 每小时清理一次过期缓存
    @Scheduled(cron = "0 0 * * * *")
    public void cleanupCache() {
        System.out.println("清理缓存...");
        
        // 清理特定前缀的缓存
        Set<String> keys = redisTemplate.keys("cache:*");
        if (keys != null && !keys.isEmpty()) {
            redisTemplate.delete(keys);
        }
        
        System.out.println("缓存清理完成");
    }
}
```

### 5.5 数据同步

```java
@Component
public class DataSyncTask {
    
    @Autowired
    private RestTemplate restTemplate;
    
    @Autowired
    private DataMapper dataMapper;
    
    // 每 10 分钟同步一次数据
    @Scheduled(cron = "0 */10 * * * *")
    public void syncData() {
        System.out.println("开始同步数据...");
        
        // 调用外部 API 获取数据
        ResponseEntity<List<Data>> response = restTemplate.exchange(
            "http://api.example.com/data",
            HttpMethod.GET,
            null,
            new ParameterizedTypeReference<List<Data>>() {}
        );
        
        List<Data> dataList = response.getBody();
        
        // 保存到数据库
        for (Data data : dataList) {
            dataMapper.save(data);
        }
        
        System.out.println("数据同步完成，共 " + dataList.size() + " 条");
    }
}
```

## 六、注意事项

### 6.1 默认单线程

```java
// 默认情况下，定时任务是单线程的
// 一个任务阻塞会影响其他任务

// 解决方案：配置线程池
@Configuration
public class ScheduledConfig implements SchedulingConfigurer {
    
    @Override
    public void configureTasks(ScheduledTaskRegistrar taskRegistrar) {
        ThreadPoolTaskScheduler scheduler = new ThreadPoolTaskScheduler();
        scheduler.setPoolSize(10);
        scheduler.setThreadNamePrefix("scheduled-task-");
        scheduler.initialize();
        taskRegistrar.setTaskScheduler(scheduler);
    }
}
```

### 6.2 避免长任务

```java
// ❌ 不好：长任务阻塞线程
@Scheduled(fixedRate = 5000)
public void longTask() {
    // 执行 10 秒
    try { Thread.sleep(10000); } catch (Exception e) {}
}

// ✅ 好：异步执行
@Scheduled(fixedRate = 5000)
@Async
public void asyncTask() {
    // 异步执行，不阻塞定时任务线程
}
```

### 6.3 集群环境

```java
// 集群环境下，需要避免重复执行
// 方案 1：使用分布式锁
@Scheduled(cron = "0 0 2 * * *")
public void backup() {
    RLock lock = redissonClient.getLock("backup:lock");
    
    if (lock.tryLock()) {
        try {
            // 执行备份
        } finally {
            lock.unlock();
        }
    }
}
```
