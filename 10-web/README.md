# 第十部分：Web 开发基础

本模块学习 Java Web 开发的基础知识，包括前端基础了解、Servlet 技术和 Tomcat 服务器配置，是理解 Web 应用运行原理的重要基础。

## 章节列表

| 章节 | 目录 | 主题 |
|------|------|------|
| 第 40 章 | ch40-frontend | 前端基础（了解）：HTML/CSS/JavaScript 基础、AJAX 与 JSON |
| 第 41 章 | ch41-servlet | Servlet：Servlet 概述与生命周期、HttpServletRequest/Response、请求转发与重定向、Session 与 Cookie、Filter 与 Listener |
| 第 42 章 | ch42-tomcat | Tomcat 服务器：安装配置、部署 Web 应用、虚拟目录配置 |

## 学习目标

- 了解前端三件套基础与 AJAX 通信
- 理解 Servlet 的工作原理与生命周期
- 掌握 HttpServletRequest 与 HttpServletResponse 的使用
- 理解 Session 与 Cookie 的区别与应用
- 掌握 Tomcat 服务器的配置与部署
