# 56.5 实战示例：完整登录流程（生活类比：酒店入住）

## 一、先问一个问题
**问题**：如何构建完整的 JWT 登录系统？

### 酒店入住类比
酒店入住流程：
- 前台登记（登录接口）
- 验证身份证（认证）
- 发放房卡（生成 JWT）
- 凭房卡进入房间（携带 Token 访问）
- 退房注销（登出）

JWT 就是系统的"电子房卡"！

## 三、认证 Controller

```java
@RestController
@RequestMapping("/api/auth")
public class AuthController {
    
    @Autowired
    private AuthenticationManager authenticationManager;
    
    @Autowired
    private JwtUtil jwtUtil;
    
    @Autowired
    private UserDetailsService userDetailsService;
    
    // 登录
    @PostMapping("/login")
    public Result<LoginResponse> login(@RequestBody LoginRequest request) {
        // 1. 认证
        authenticationManager.authenticate(
            new UsernamePasswordAuthenticationToken(
                request.getUsername(), 
                request.getPassword()
            )
        );
        
        // 2. 加载用户信息
        UserDetails userDetails = userDetailsService.loadUserByUsername(
            request.getUsername());
        
        // 3. 生成 Token
        String token = jwtUtil.generateToken(userDetails);
        
        // 4. 获取角色
        List<String> roles = userDetails.getAuthorities().stream()
            .map(GrantedAuthority::getAuthority)
            .collect(Collectors.toList());
        
        return Result.success(new LoginResponse(
            token, request.getUsername(), roles));
    }
    
    // 注册
    @PostMapping("/register")
    public Result<Void> register(@RequestBody RegisterRequest request) {

        // ...
        return Result.success();
    }
}
```

## 四、获取用户信息

```java
@RestController
@RequestMapping("/api/user")
public class UserController {
    
    @Autowired
    private UserDetailsService userDetailsService;
    
    @GetMapping("/info")
    public Result<UserInfo> getUserInfo() {
        Authentication authentication = 
            SecurityContextHolder.getContext().getAuthentication();
        String username = authentication.getName();
        
        UserDetails userDetails = userDetailsService.loadUserByUsername(username);
        
        UserInfo info = new UserInfo();
        info.setUsername(username);
        info.setRoles(userDetails.getAuthorities().stream()
            .map(GrantedAuthority::getAuthority)
            .collect(Collectors.toList()));
        
        return Result.success(info);
    }
}
```

## 五、常见疑问

**Q1：AuthenticationManager 在哪里配置？**
A：在 SecurityConfig 中配置 authenticationManager Bean，或使用 AuthenticationConfiguration。

```java
@Bean
public AuthenticationManager authenticationManager(
        AuthenticationConfiguration config) throws Exception {
    return config.getAuthenticationManager();
}
```

**Q2：Result 是什么？**
A：统一响应封装类，包含 code、message、data 字段。

## 六、小结

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/auth/login | POST | 登录获取 Token |
| /api/auth/register | POST | 用户注册 |
| /api/user/info | GET | 获取当前用户信息 |
| Authorization Header | Bearer {token} | 携带 Token 访问 |
