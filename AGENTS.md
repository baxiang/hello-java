# AGENTS.md — hello-java

> Java 全栈学习路线教程仓库（纯 Markdown 文档，无可执行代码）

## 仓库性质

- **中文 Java 教程文档集合**，不含任何可编译/运行的源代码
- 所有内容为 `.md` 文件，按 12 个学习模块组织在编号目录中（`01-java-base/` ~ `12-spring-ai/`）

## 目录结构约定

```
NN-topic/                    ← 模块目录，NN 为两位序号
├── MM-name/                 ← 子目录（多文件主题），MM 为模块内部序号
│   └── XX-标题.md           ← 课程文件，XX 为子目录内序号
├── XX-标题.md               ← 课程文件（扁平化模块直接存放）
```

- 目录和文件名使用模块内部序号，不再使用全局章节号
- 单文件子目录已扁平化，`.md` 文件直接存放在模块目录下
- 多文件子目录保留目录层级，内部文件从 `01` 起编号

## 写作规范（来自 TASKS.md）

新增或修改课程文件时遵循：

1. **每节 150-250 行**，按功能模块拆分
2. **结构**：问题引入 → 生活类比 → 核心概念 → 代码对比 → 实现方式 → Q&A → 小结表格 → 动手练习
3. 使用生活类比解释抽象概念（类比库见 `TASKS.md`）
4. 代码示例只保留核心部分，保持精简
