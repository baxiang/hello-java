# 第十一部分：Spring Framework

本模块深入学习 Spring 框架核心内容，包括 IoC 容器、AOP 面向切面编程、事务管理、数据访问以及 Spring MVC，是 Java 企业级开发的核心框架。

## 章节列表

| 章节 | 目录 | 主题 |
|------|------|------|
| 第 43 章 | ch43-spring-overview | Spring 概述：Spring 简介与优势、体系结构、第一个 Spring 程序 |
| 第 44 章 | ch44-ioc | IoC 容器：IoC 与 DI 概念、Bean 配置（XML/注解）、作用域、生命周期、依赖注入、常用注解 |
| 第 45 章 | ch45-aop | AOP 面向切面编程：AOP 概念、动态代理（JDK/CGLIB）、基于注解/XML 的 AOP、通知类型、切点表达式 |
| 第 46 章 | ch46-transaction | Spring 事务管理：事务概述、编程式事务、声明式事务（@Transactional）、传播行为、隔离级别 |
| 第 47 章 | ch47-data-access | Spring 数据访问：JdbcTemplate、整合 MyBatis、Spring DAO 异常体系 |
| 第 48 章 | ch48-spring-mvc | Spring MVC：MVC 设计模式、工作流程、常用注解、参数绑定、RESTful 风格、文件上传、拦截器、全局异常处理 |
| 第 49 章 | ch49-source-code | Spring 源码阅读：核心源码分析与理解 |

## 学习目标

- 理解 IoC 和 DI 的核心思想
- 掌握 Spring Bean 的配置与管理
- 理解 AOP 原理并能应用于实际场景
- 掌握 Spring 声明式事务管理
- 熟练使用 Spring MVC 开发 Web 应用
