# 48.3 请求映射：URL 如何对应到方法？

## 一、先问一个问题

**Spring 如何知道哪个 URL 调用哪个方法？**

### 生活例子：餐厅菜单

```
菜单（请求映射）
├── 鱼香肉丝 → 厨房 A
├── 宫保鸡丁 → 厨房 B
└── 麻婆豆腐 → 厨房 C
```

**@RequestMapping 就是菜单**：
- 告诉 Spring：这个 URL 调用这个方法

## 二、@RequestMapping

### 基本用法

```java
@Controller
@RequestMapping("/user")  // 类级别：基础路径
public class UserController {
    
    @RequestMapping("/get")  // 方法级别：完整路径 /user/get
    public String get() {
        return "get";
    }
}
```

### 简化注解

```java
@Controller
@RequestMapping("/user")
public class UserController {
    
    @GetMapping("/get")     // = @RequestMapping(method = GET)
    public String get() {}
    
    @PostMapping("/save")   // = @RequestMapping(method = POST)
    public String save() {}
    
    @PutMapping("/update")  // = @RequestMapping(method = PUT)
    public String update() {}
    
    @DeleteMapping("/delete") // = @RequestMapping(method = DELETE)
    public String delete() {}
}
```

## 三、路径变量

### @PathVariable

```java
@GetMapping("/user/{id}")
public String getUser(@PathVariable Long id) {
    // /user/123 → id = 123
}
```

### 多个路径变量

```java
@GetMapping("/user/{userId}/order/{orderId}")
public String getOrder(
    @PathVariable Long userId,
    @PathVariable Long orderId
) {
    // /user/1/order/100
}
```

## 四、查询参数

### @RequestParam

```java
@GetMapping("/search")
public String search(
    @RequestParam String keyword,
    @RequestParam(defaultValue = "1") int page,
    @RequestParam(defaultValue = "10") int size
) {
    // /search?keyword=手机&page=1&size=10
}
```

### 可选参数

```java
@GetMapping("/search")
public String search(
    @RequestParam(required = false) String keyword
) {
    // keyword 可以不传
}
```

## 五、请求体

### @RequestBody

```java
@PostMapping("/user")
public String createUser(@RequestBody User user) {
    // JSON → User 对象
    // {"username": "张三", "age": 18}
}
```

## 六、小结

| 注解 | 用途 | 示例 |
|------|------|------|
| @RequestMapping | 映射 URL | /user |
| @GetMapping | GET 请求 | /user/{id} |
| @PostMapping | POST 请求 | /user |
| @PathVariable | 路径变量 | {id} |
| @RequestParam | 查询参数 | ?keyword= |
| @RequestBody | 请求体 | JSON |
