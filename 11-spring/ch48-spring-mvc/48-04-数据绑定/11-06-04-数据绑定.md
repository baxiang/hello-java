# 48.4 数据绑定：参数如何传递？

## 一、先问一个问题

**前端传来的参数，后端如何接收？**

### 生活例子：快递单

```
快递单（请求）
├── 收件人（参数 1）
├── 电话（参数 2）
└── 地址（参数 3）
```

**数据绑定**：
- 快递单上的信息 → 快递员记录
- 请求参数 → Java 对象

## 二、简单类型绑定

### 基本类型

```java
@GetMapping("/user")
public String getUser(
    Long id,           // ?id=123
    String name,       // ?name=张三
    Integer age        // ?age=18
) {
    // Spring 自动转换类型
}
```

### 日期类型

```java
@GetMapping("/order")
public String getOrder(
    @DateTimeFormat(pattern = "yyyy-MM-dd") LocalDate date
) {
    // ?date=2024-01-01
}
```

## 三、对象绑定

### 自动绑定

```java
public class User {
    private Long id;
    private String name;
    private Integer age;
    // getter/setter
}

@PostMapping("/user")
public String createUser(User user) {
    // ?id=1&name=张三&age=18
    // 或 JSON: {"id":1,"name":"张三","age":18}
    // Spring 自动绑定到 User 对象
}
```

## 四、注解绑定

### @RequestParam

```java
@GetMapping("/search")
public String search(
    @RequestParam("kw") String keyword  // ?kw=手机
) {
    // 参数名可以不同
}
```

### @PathVariable

```java
@GetMapping("/user/{id}")
public String getUser(@PathVariable Long id) {
    // /user/123 → id=123
}
```

### @RequestBody

```java
@PostMapping("/user")
public String createUser(@RequestBody User user) {
    // JSON → User
}
```

## 五、小结

| 方式 | 用途 | 示例 |
|------|------|------|
| 简单类型 | 基本参数 | Long id |
| 对象绑定 | 多个参数 | User user |
| @RequestParam | 查询参数 | ?kw= |
| @PathVariable | 路径参数 | {id} |
| @RequestBody | 请求体 | JSON |
