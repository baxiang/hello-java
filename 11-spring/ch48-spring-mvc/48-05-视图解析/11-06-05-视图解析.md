# 48.5 视图解析：如何返回页面？

## 一、先问一个问题

**Controller 返回的字符串，如何变成 HTML 页面？**

### 生活例子：菜单编号

```
顾客点餐：鱼香肉丝（逻辑视图名）
  ↓
厨师理解：菜品编号 001（视图解析）
  ↓
端上菜：一盘鱼香肉丝（真实页面）
```

## 二、ViewResolver

### 配置

```properties
# application.properties
spring.thymeleaf.prefix=classpath:/templates/
spring.thymeleaf.suffix=.html
```

### 工作流程

```
Controller 返回："success"
  ↓
ViewResolver 解析
  ↓
真实路径：classpath:/templates/success.html
```

## 三、返回页面

### 基本用法

```java
@Controller
public class PageController {
    
    @GetMapping("/hello")
    public String hello() {
        return "hello";  // hello.html
    }
}
```

### 传递数据

```java
@GetMapping("/user/{id}")
public String userDetail(@PathVariable Long id, Model model) {
    User user = userService.getUserById(id);
    model.addAttribute("user", user);
    return "user-detail";
}
```

### Thymeleaf 模板

```html
<!-- templates/user-detail.html -->
<html xmlns:th="http://www.thymeleaf.org">
<body>
    <h1 th:text="${user.name}">用户名</h1>
    <p th:text="${user.age}">年龄</p>
</body>
</html>
```

## 四、重定向

### redirect:

```java
@GetMapping("/redirect")
public String redirect() {
    return "redirect:/hello";  // 重定向到/hello
}
```

### forward:

```java
@GetMapping("/forward")
public String forward() {
    return "forward:/hello";  // 转发到/hello
}
```

## 五、小结

| 返回 | 作用 | 示例 |
|------|------|------|
| 字符串 | 逻辑视图名 | "success" |
| redirect: | 重定向 | "redirect:/hello" |
| forward: | 转发 | "forward:/hello" |
| Model | 传递数据 | model.addAttribute() |
