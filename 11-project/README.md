# 项目实战

本模块通过多个实战项目综合运用所学知识，从单体架构到微服务架构，涵盖博客系统、商城系统、OA 系统、秒杀系统、外卖平台等多个场景，帮助你将理论转化为实际开发能力。

## 项目列表

| 目录 | 主题 |
|------|------|
| 01-blog | 博客系统（单体架构）：项目搭建、用户认证、文章 CRUD、分类标签、评论、文件上传、部署上线 |
| 02-mall | 在线商城系统（前后端分离）：RESTful API、商品 SKU/SPU、购物车（Redis）、订单支付、Elasticsearch 搜索 |
| 03-oa | 企业 OA 系统（权限管理）：RBAC 权限模型、Spring Security + JWT、考勤审批、数据权限、导出导入 |
| 04-seckill | 秒杀系统（高并发场景）：Redis 预减库存、RabbitMQ 异步下单、Sentinel 限流、分布式锁、防超卖 |
| 05-takeout | 分布式外卖平台（Spring Cloud）：微服务拆分、Nacos、OpenFeign、Gateway、Sentinel、Seata、SkyWalking |
| 06-dashboard | 数据可视化大屏系统：数据采集清洗、统计分析、WebSocket 实时推送、缓存策略 |
| 07-exam | 在线考试系统：题库管理、自动组卷、在线答题、防作弊、自动阅卷、错题本 |
| 08-cms | 内容管理系统（CMS）：栏目管理、内容管理、模板引擎、静态页面生成、SEO 优化 |
| 09-common-skills | 项目通用技能：代码规范、Git Flow、接口文档、单元测试、日志规范 |
| 10-devops | DevOps 与部署：Linux 服务器、MySQL 主从、Redis 集群、Nginx、Docker、CI/CD、监控告警 |
| 11-minispring | MiniSpring 手写框架：手写简易 Spring 框架，深入理解核心原理 |

## 学习目标

- 能够独立完成从需求分析到部署上线的完整项目
- 掌握不同架构模式（单体/前后端分离/微服务）的开发方式
- 理解高并发场景下的性能优化方案
- 掌握 DevOps 流程与自动化部署
- 通过手写 MiniSpring 深入理解框架原理
