# 5.2 File 类

## 一、File 类概述

File 类用于表示文件或目录的路径，可以进行创建、删除、查询等操作。

**重要说明**：
- File 对象不代表文件内容，只代表路径
- 可以表示存在的或不存在的文件

## 二、创建 File 对象

```java
import java.io.File;

// 方式 1：路径字符串
File file1 = new File("/path/to/file.txt");

// 方式 2：父路径 + 子路径
File file2 = new File("/path/to", "file.txt");

// 方式 3：父 File + 子路径
File parent = new File("/path/to");
File file3 = new File(parent, "file.txt");

// 方式 4：URI
File file4 = new File(new URI("file:///path/to/file.txt"));
```

## 三、常用方法

### 3.1 文件信息

```java
File file = new File("test.txt");

// 文件名
String name = file.getName();  // "test.txt"

// 绝对路径
String absPath = file.getAbsolutePath();

// 父路径
String parent = file.getParent();
File parentFile = file.getParentFile();

// 路径
String path = file.getPath();

// 是否绝对路径
boolean isAbsolute = file.isAbsolute();
```

### 3.2 文件判断

```java
File file = new File("test.txt");

// 是否存在
boolean exists = file.exists();

// 是否是文件
boolean isFile = file.isFile();

// 是否是目录
boolean isDir = file.isDirectory();

// 是否可读
boolean canRead = file.canRead();

// 是否可写
boolean canWrite = file.canWrite();

// 是否隐藏
boolean isHidden = file.isHidden();
```

### 3.3 文件大小和修改时间

```java
File file = new File("test.txt");

// 文件大小（字节）
long size = file.length();

// 最后修改时间（时间戳）
long lastModified = file.lastModified();

// 设置最后修改时间
boolean success = file.setLastModified(System.currentTimeMillis());
```

### 3.4 创建和删除

```java
File file = new File("test.txt");

// 创建文件
boolean created = file.createNewFile();  // 存在则返回 false

// 创建目录
File dir = new File("mydir");
boolean dirCreated = dir.mkdir();  // 创建单级目录

// 创建多级目录
File multiDir = new File("a/b/c");
boolean multiCreated = multiDir.mkdirs();  // 创建多级目录

// 删除文件
boolean deleted = file.delete();

// 删除目录（必须为空）
boolean dirDeleted = dir.delete();

// 删除时自动递归（Java 8+）
Files.deleteIfExists(file.toPath());
```

### 3.5 重命名和移动

```java
File oldFile = new File("old.txt");
File newFile = new File("new.txt");

// 重命名
boolean renamed = oldFile.renameTo(newFile);

// 移动（跨目录也是 renameTo）
File dest = new File("/path/to/new.txt");
boolean moved = oldFile.renameTo(dest);
```

## 四、目录操作

### 4.1 列出文件

```java
File dir = new File("/path/to/dir");

// 列出所有文件和目录
String[] names = dir.list();
File[] files = dir.listFiles();

// 使用过滤器
File[] txtFiles = dir.listFiles((d, name) -> name.endsWith(".txt"));

// 使用文件过滤器
File[] javaFiles = dir.listFiles(new FilenameFilter() {
    @Override
    public boolean accept(File dir, String name) {
        return name.endsWith(".java");
    }
});
```

### 4.2 遍历目录

```java
// 递归遍历
public void listAllFiles(File dir, int level) {
    String indent = "  ".repeat(level);
    System.out.println(indent + dir.getName());
    
    if (dir.isDirectory()) {
        File[] files = dir.listFiles();
        if (files != null) {
            for (File file : files) {
                listAllFiles(file, level + 1);
            }
        }
    }
}

// 使用
listAllFiles(new File("/path"), 0);
```

### 4.3 创建临时文件

```java
// 创建临时文件
File tempFile = File.createTempFile("prefix", ".txt");

// 在指定目录创建临时文件
File tempDir = new File("/tmp");
File tempFile2 = File.createTempFile("prefix", ".txt", tempDir);

// 退出时删除
tempFile.deleteOnExit();
```

## 五、路径操作

### 5.1 规范化路径

```java
File file = new File("/path/./to/../file.txt");
String canonicalPath = file.getCanonicalPath();  // "/path/file.txt"
```

### 5.2 分隔符

```java
// 路径分隔符（跨平台）
String separator = File.separator;      // Linux: /  Windows: \
String pathSeparator = File.pathSeparator;  // Linux: :  Windows: ;

// 使用
File file = new File("path" + File.separator + "to" + File.separator + "file.txt");
```

## 六、综合示例

### 6.1 文件工具类

```java
public class FileUtil {
    
    // 创建文件（包括父目录）
    public static File createFile(String path) throws IOException {
        File file = new File(path);
        File parent = file.getParentFile();
        if (parent != null && !parent.exists()) {
            parent.mkdirs();
        }
        if (!file.exists()) {
            file.createNewFile();
        }
        return file;
    }
    
    // 删除目录（递归）
    public static boolean deleteDirectory(File dir) {
        if (!dir.exists()) {
            return false;
        }
        
        if (dir.isFile()) {
            return dir.delete();
        }
        
        File[] files = dir.listFiles();
        if (files != null) {
            for (File file : files) {
                deleteDirectory(file);
            }
        }
        
        return dir.delete();
    }
    
    // 获取文件大小（格式化）
    public static String formatFileSize(long size) {
        if (size < 1024) {
            return size + " B";
        } else if (size < 1024 * 1024) {
            return String.format("%.2f KB", size / 1024.0);
        } else if (size < 1024 * 1024 * 1024) {
            return String.format("%.2f MB", size / (1024.0 * 1024.0));
        } else {
            return String.format("%.2f GB", size / (1024.0 * 1024.0 * 1024.0));
        }
    }
    
    // 统计目录大小
    public static long getDirectorySize(File dir) {
        long size = 0;
        File[] files = dir.listFiles();
        if (files != null) {
            for (File file : files) {
                if (file.isFile()) {
                    size += file.length();
                } else if (file.isDirectory()) {
                    size += getDirectorySize(file);
                }
            }
        }
        return size;
    }
}
```

### 6.2 文件搜索

```java
public class FileSearch {
    
    // 按扩展名搜索
    public static List<File> searchByExtension(File dir, String extension) {
        List<File> result = new ArrayList<>();
        searchRecursive(dir, extension, result);
        return result;
    }
    
    private static void searchRecursive(File dir, String ext, List<File> result) {
        File[] files = dir.listFiles();
        if (files == null) {
            return;
        }
        
        for (File file : files) {
            if (file.isDirectory()) {
                searchRecursive(file, ext, result);
            } else if (file.getName().endsWith(ext)) {
                result.add(file);
            }
        }
    }
    
    // 按名称搜索（模糊匹配）
    public static List<File> searchByName(File dir, String keyword) {
        List<File> result = new ArrayList<>();
        File[] files = dir.listFiles((d, name) -> name.contains(keyword));
        if (files != null) {
            result.addAll(Arrays.asList(files));
        }
        return result;
    }
}
```

### 6.3 文件信息展示

```java
public class FileInfo {
    public static void printInfo(File file) {
        System.out.println("=== 文件信息 ===");
        System.out.println("名称：" + file.getName());
        System.out.println("路径：" + file.getPath());
        System.out.println("绝对路径：" + file.getAbsolutePath());
        System.out.println("是否存在：" + file.exists());
        System.out.println("是否是文件：" + file.isFile());
        System.out.println("是否是目录：" + file.isDirectory());
        
        if (file.exists()) {
            System.out.println("大小：" + FileUtil.formatFileSize(file.length()));
            System.out.println("最后修改：" + new Date(file.lastModified()));
            System.out.println("是否可读：" + file.canRead());
            System.out.println("是否可写：" + file.canWrite());
            System.out.println("是否隐藏：" + file.isHidden());
        }
    }
}

// 使用
FileInfo.printInfo(new File("test.txt"));
```

## 七、注意事项

### 7.1 路径问题

```java
// Windows 路径
File winFile = new File("C:\\path\\to\\file.txt");
File winFile2 = new File("C:/path/to/file.txt");  // 也可以

// Linux/Mac 路径
File unixFile = new File("/path/to/file.txt");

// 跨平台写法
File crossPlatform = new File("path", "file.txt");
```

### 7.2 权限问题

```java
// 检查写权限
if (!file.canWrite()) {
    System.out.println("没有写权限");
}

// 设置权限（Unix 系统）
file.setReadable(true);
file.setWritable(true);
file.setExecutable(true);
```

### 7.3 异常处理

```java
try {
    File file = new File("test.txt");
    if (!file.exists()) {
        file.createNewFile();
    }
} catch (IOException e) {
    e.printStackTrace();
}
```

## 八、小结

1. **File 对象**：表示文件或目录路径
2. **常用方法**：exists、isFile、isDirectory、createNewFile、delete
3. **目录操作**：list、listFiles、mkdir、mkdirs
4. **路径操作**：getAbsolutePath、getCanonicalPath
5. **工具类**：FileUtil、文件搜索、文件信息
