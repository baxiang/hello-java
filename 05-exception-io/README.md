# 第五部分：异常处理与 IO

本模块学习 Java 异常处理机制和 IO 流体系，包括异常的分类与处理、File 类操作、各类 IO 流的使用以及 NIO 简介，是文件操作和错误处理的基础。

## 章节列表

| 章节 | 目录 | 主题 |
|------|------|------|
| 第 23 章 | ch23-exception | 异常处理：异常体系结构、try-catch-finally、throw/throws、自定义异常、最佳实践、try-with-resources |
| 第 24 章 | ch24-file | File 类：常用方法、文件与目录操作、文件过滤器 |
| 第 25 章 | ch25-io-stream | IO 流：IO 流分类、文件字节/字符流、缓冲流、转换流、对象序列化、打印流、NIO 简介 |

## 学习目标

- 理解 Java 异常体系结构，掌握异常处理最佳实践
- 熟练使用 File 类进行文件和目录操作
- 掌握字节流、字符流、缓冲流的使用场景
- 理解对象序列化与反序列化机制
- 了解 NIO 的 Buffer、Channel、Selector 概念
