# 5.3 IO 流

## 一、IO 流概述

IO（Input/Output）流用于在程序和其他数据源之间传输数据。

**流的分类**：
- 按方向：输入流、输出流
- 按数据类型：字节流、字符流
- 按功能：节点流、处理流

```
IO 流体系
├── 字节流（处理所有类型数据）
│   ├── 输入：InputStream
│   └── 输出：OutputStream
└── 字符流（只处理文本）
    ├── 输入：Reader
    └── 输出：Writer
```

## 二、字节流

### 2.1 FileInputStream（文件输入）

```java
import java.io.*;

// 方式 1：基本使用
FileInputStream fis = null;
try {
    fis = new FileInputStream("input.txt");
    int data = fis.read();  // 读取一个字节
    while (data != -1) {
        System.out.print((char) data);
        data = fis.read();
    }
} catch (IOException e) {
    e.printStackTrace();
} finally {
    if (fis != null) {
        try { fis.close(); } catch (IOException e) {}
    }
}

// 方式 2：使用字节数组
FileInputStream fis = null;
try {
    fis = new FileInputStream("input.txt");
    byte[] buffer = new byte[1024];
    int len;
    while ((len = fis.read(buffer)) != -1) {
        System.out.print(new String(buffer, 0, len));
    }
} catch (IOException e) {
    e.printStackTrace();
} finally {
    if (fis != null) fis.close();
}

// 方式 3：try-with-resources（Java 7+，推荐）
try (FileInputStream fis = new FileInputStream("input.txt")) {
    byte[] buffer = new byte[1024];
    int len;
    while ((len = fis.read(buffer)) != -1) {
        System.out.print(new String(buffer, 0, len));
    }
} catch (IOException e) {
    e.printStackTrace();
}
```

### 2.2 FileOutputStream（文件输出）

```java
// 写入文件（覆盖）
try (FileOutputStream fos = new FileOutputStream("output.txt")) {
    fos.write("Hello".getBytes());
    fos.write(" World".getBytes());
}

// 追加写入
try (FileOutputStream fos = new FileOutputStream("output.txt", true)) {
    fos.write("\n追加内容".getBytes());
}

// 写入字节数组
byte[] data = {65, 66, 67, 68};  // ABCD
try (FileOutputStream fos = new FileOutputStream("output.txt")) {
    fos.write(data);
    fos.write(data, 1, 2);  // 从索引 1 开始，写 2 个字节
}
```

### 2.3 文件拷贝

```java
public static void copyFile(String src, String dest) throws IOException {
    try (FileInputStream fis = new FileInputStream(src);
         FileOutputStream fos = new FileOutputStream(dest)) {
        
        byte[] buffer = new byte[1024];
        int len;
        while ((len = fis.read(buffer)) != -1) {
            fos.write(buffer, 0, len);
        }
    }
}

// 使用
copyFile("source.txt", "destination.txt");
```

## 三、字符流

### 3.1 FileReader（文件输入）

```java
// 读取字符
try (FileReader fr = new FileReader("input.txt")) {
    int ch;
    while ((ch = fr.read()) != -1) {
        System.out.print((char) ch);
    }
}

// 使用字符数组
try (FileReader fr = new FileReader("input.txt")) {
    char[] buffer = new char[1024];
    int len;
    while ((len = fr.read(buffer)) != -1) {
        System.out.print(new String(buffer, 0, len));
    }
}
```

### 3.2 FileWriter（文件输出）

```java
// 写入字符
try (FileWriter fw = new FileWriter("output.txt")) {
    fw.write("Hello");
    fw.write(" World");
}

// 追加写入
try (FileWriter fw = new FileWriter("output.txt", true)) {
    fw.write("\n追加内容");
}

// 写入字符数组
char[] chars = {'H', 'e', 'l', 'l', 'o'};
try (FileWriter fw = new FileWriter("output.txt")) {
    fw.write(chars);
}

// 写入部分
try (FileWriter fw = new FileWriter("output.txt")) {
    fw.write(chars, 1, 3);  // "ell"
}
```

### 3.3 写入换行

```java
try (FileWriter fw = new FileWriter("output.txt")) {
    fw.write("第一行");
    fw.write(System.lineSeparator());  // 跨平台换行
    fw.write("第二行");
    fw.write("\r\n");  // Windows 换行
}
```

## 四、缓冲流

### 4.1 BufferedReader（缓冲输入）

```java
// 按行读取
try (BufferedReader br = new BufferedReader(new FileReader("input.txt"))) {
    String line;
    while ((line = br.readLine()) != null) {
        System.out.println(line);
    }
}

// 指定缓冲区大小
try (BufferedReader br = new BufferedReader(new FileReader("input.txt"), 8192)) {
    // ...
}
```

### 4.2 BufferedWriter（缓冲输出）

```java
try (BufferedWriter bw = new BufferedWriter(new FileWriter("output.txt"))) {
    bw.write("第一行");
    bw.newLine();  // 换行
    bw.write("第二行");
    bw.flush();  // 刷新缓冲区
}
```

### 4.3 缓冲流的好处

- 减少磁盘 IO 次数
- 提高读写效率
- 提供便捷方法（readLine、newLine）

## 五、转换流

### 5.1 InputStreamReader（字节转字符）

```java
// 指定编码读取
try (InputStreamReader isr = new InputStreamReader(
        new FileInputStream("input.txt"), "UTF-8")) {
    int ch;
    while ((ch = isr.read()) != -1) {
        System.out.print((char) ch);
    }
}

// 使用缓冲
try (BufferedReader br = new BufferedReader(
        new InputStreamReader(new FileInputStream("input.txt"), "UTF-8"))) {
    String line;
    while ((line = br.readLine()) != null) {
        System.out.println(line);
    }
}
```

### 5.2 OutputStreamWriter（字符转字节）

```java
// 指定编码写入
try (OutputStreamWriter osw = new OutputStreamWriter(
        new FileOutputStream("output.txt"), "UTF-8")) {
    osw.write("中文内容");
}

// 使用缓冲
try (BufferedWriter bw = new BufferedWriter(
        new OutputStreamWriter(new FileOutputStream("output.txt"), "UTF-8"))) {
    bw.write("中文内容");
    bw.newLine();
}
```

## 六、对象序列化

### 6.1 Serializable 接口

```java
import java.io.Serializable;

// 实现 Serializable 接口
class Person implements Serializable {
    private static final long serialVersionUID = 1L;  // 版本 ID
    
    private String name;
    private int age;
    private transient String password;  // transient 不序列化
    
    // 构造方法、getter、setter
}
```

### 6.2 ObjectOutputStream（对象输出）

```java
Person person = new Person("张三", 18, "123456");

try (ObjectOutputStream oos = new ObjectOutputStream(
        new FileOutputStream("person.dat"))) {
    oos.writeObject(person);  // 序列化对象
}
```

### 6.3 ObjectInputStream（对象输入）

```java
try (ObjectInputStream ois = new ObjectInputStream(
        new FileInputStream("person.dat"))) {
    Person person = (Person) ois.readObject();  // 反序列化
    System.out.println(person.getName());
} catch (ClassNotFoundException e) {
    e.printStackTrace();
}
```

### 6.4 序列化集合

```java
List<Person> people = Arrays.asList(
    new Person("张三", 18, "123"),
    new Person("李四", 20, "456")
);

// 序列化
try (ObjectOutputStream oos = new ObjectOutputStream(
        new FileOutputStream("people.dat"))) {
    oos.writeObject(people);
}

// 反序列化
try (ObjectInputStream ois = new ObjectInputStream(
        new FileInputStream("people.dat"))) {
    List<Person> list = (List<Person>) ois.readObject();
}
```

## 七、综合示例

### 7.1 文件工具类

```java
public class FileIOUtil {
    
    // 读取文件全部内容
    public static String readFileToString(String path) throws IOException {
        StringBuilder sb = new StringBuilder();
        try (BufferedReader br = new BufferedReader(new FileReader(path))) {
            String line;
            while ((line = br.readLine()) != null) {
                sb.append(line).append("\n");
            }
        }
        return sb.toString();
    }
    
    // 写入字符串到文件
    public static void writeStringToFile(String path, String content) throws IOException {
        try (BufferedWriter bw = new BufferedWriter(new FileWriter(path))) {
            bw.write(content);
        }
    }
    
    // 复制文件
    public static void copyFile(String src, String dest) throws IOException {
        try (FileInputStream fis = new FileInputStream(src);
             FileOutputStream fos = new FileOutputStream(dest)) {
            byte[] buffer = new byte[8192];
            int len;
            while ((len = fis.read(buffer)) != -1) {
                fos.write(buffer, 0, len);
            }
        }
    }
    
    // 读取文件为字节数组
    public static byte[] readFileToBytes(String path) throws IOException {
        try (FileInputStream fis = new FileInputStream(path);
             ByteArrayOutputStream baos = new ByteArrayOutputStream()) {
            byte[] buffer = new byte[1024];
            int len;
            while ((len = fis.read(buffer)) != -1) {
                baos.write(buffer, 0, len);
            }
            return baos.toByteArray();
        }
    }
    
    // 追加内容到文件
    public static void appendToFile(String path, String content) throws IOException {
        try (BufferedWriter bw = new BufferedWriter(new FileWriter(path, true))) {
            bw.write(content);
            bw.newLine();
        }
    }
}
```

### 7.2 配置读写

```java
public class ConfigManager {
    
    // 保存配置
    public static void saveConfig(Map<String, String> config, String path) throws IOException {
        try (ObjectOutputStream oos = new ObjectOutputStream(
                new FileOutputStream(path))) {
            oos.writeObject(config);
        }
    }
    
    // 加载配置
    @SuppressWarnings("unchecked")
    public static Map<String, String> loadConfig(String path) throws IOException, ClassNotFoundException {
        try (ObjectInputStream ois = new ObjectInputStream(
                new FileInputStream(path))) {
            return (Map<String, String>) ois.readObject();
        }
    }
}

// 使用
Map<String, String> config = new HashMap<>();
config.put("host", "localhost");
config.put("port", "8080");
ConfigManager.saveConfig(config, "config.dat");

Map<String, String> loaded = ConfigManager.loadConfig("config.dat");
```

### 7.3 日志记录器

```java
public class SimpleLogger {
    private String logFile;
    
    public SimpleLogger(String logFile) {
        this.logFile = logFile;
    }
    
    public void info(String message) {
        log("INFO", message);
    }
    
    public void error(String message) {
        log("ERROR", message);
    }
    
    public void debug(String message) {
        log("DEBUG", message);
    }
    
    private void log(String level, String message) {
        String timestamp = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss").format(new Date());
        String logLine = String.format("[%s] %s: %s", timestamp, level, message);
        
        try {
            FileIOUtil.appendToFile(logFile, logLine);
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

// 使用
SimpleLogger logger = new SimpleLogger("app.log");
logger.info("应用启动");
logger.error("发生错误");
logger.debug("调试信息");
```

## 八、小结

1. **字节流**：InputStream、OutputStream（处理所有数据）
2. **字符流**：Reader、Writer（只处理文本）
3. **缓冲流**：BufferedReader、BufferedWriter（提高效率）
4. **转换流**：InputStreamReader、OutputStreamWriter（编码转换）
5. **对象流**：ObjectInputStream、ObjectOutputStream（序列化）
6. **最佳实践**：try-with-resources 自动关闭流
