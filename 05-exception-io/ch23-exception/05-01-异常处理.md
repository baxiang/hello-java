# 5.1 异常处理

## 一、异常体系

```
Throwable
├── Error（错误，不可恢复）
│   ├── OutOfMemoryError
│   └── StackOverflowError
└── Exception（异常，可处理）
    ├── RuntimeException（运行时异常）
    │   ├── NullPointerException
    │   ├── ArrayIndexOutOfBoundsException
    │   └── IllegalArgumentException
    └── 受检异常（Checked Exception）
        ├── IOException
        ├── SQLException
        └── ClassNotFoundException
```

## 二、try-catch-finally

### 2.1 基本语法

```java
try {
    // 可能抛出异常的代码
    int result = 10 / 0;
} catch (ArithmeticException e) {
    // 处理异常
    System.out.println("除零错误：" + e.getMessage());
} finally {
    // 无论是否异常都会执行
    System.out.println("finally 块");
}
```

### 2.2 多个 catch

```java
try {
    int[] arr = {1, 2, 3};
    System.out.println(arr[5]);
} catch (ArithmeticException e) {
    System.out.println("算术异常");
} catch (ArrayIndexOutOfBoundsException e) {
    System.out.println("数组越界");
} catch (Exception e) {
    System.out.println("其他异常");
}
```

### 2.3 try-with-resources（Java 7+）

```java
// 自动关闭资源
try (FileInputStream fis = new FileInputStream("file.txt");
     BufferedReader br = new BufferedReader(new InputStreamReader(fis))) {
    String line = br.readLine();
    System.out.println(line);
} catch (IOException e) {
    e.printStackTrace();
}
```

## 三、throw 和 throws

### 3.1 throw 抛出异常

```java
public void setAge(int age) {
    if (age < 0 || age > 150) {
        throw new IllegalArgumentException("年龄不合法");
    }
    this.age = age;
}
```

### 3.2 throws 声明异常

```java
// 声明可能抛出的异常
public void readFile(String path) throws IOException {
    FileInputStream fis = new FileInputStream(path);
    fis.close();
}

// 调用处处理
try {
    readFile("test.txt");
} catch (IOException e) {
    e.printStackTrace();
}
```

## 四、自定义异常

```java
// 自定义运行时异常
public class BusinessException extends RuntimeException {
    private Integer code;
    
    public BusinessException(String message) {
        super(message);
        this.code = 500;
    }
    
    public BusinessException(Integer code, String message) {
        super(message);
        this.code = code;
    }
    
    public Integer getCode() {
        return code;
    }
}

// 使用
public void processOrder(Order order) {
    if (order.getStock() <= 0) {
        throw new BusinessException(400, "库存不足");
    }
}
```

## 五、异常最佳实践

1. **不要捕获 Exception**：尽量捕获具体异常
2. **不要吞掉异常**：至少记录日志
3. **不要滥用异常**：异常是用于异常情况，不是流程控制
4. **及时关闭资源**：使用 try-with-resources

## 六、小结

1. **异常分类**：Error、RuntimeException、受检异常
2. **处理方式**：try-catch-finally
3. **抛出异常**：throw（动作）、throws（声明）
4. **自定义异常**：继承 Exception 或 RuntimeException
