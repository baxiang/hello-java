# 74.4 实时更新：如何用 WebSocket 推送数据？

## 一、先问一个问题

**如何实现实时数据更新？**

### 生活例子：股票行情

```
传统方式
└── 手动刷新 → 慢

WebSocket 方式
└── 服务器主动推送 → 实时
```

**WebSocket 优势**：
- 双向通信
- 实时推送
- 节省资源

## 二、WebSocket 实现

### 服务端配置

```java
@Configuration
@EnableWebSocket
public class WebSocketConfig implements WebSocketConfigurer {
    
    @Override
    public void registerWebSocketHandlers(WebSocketHandlerRegistry registry) {
        registry.addHandler(new DashboardWebSocketHandler(), "/ws/dashboard")
            .setAllowedOrigins("*");
    }
}
```

### 消息推送

```java
@Component
public class DashboardWebSocketHandler extends TextWebSocketHandler {
    
    private static final CopyOnWriteArraySet<WebSocketSession> sessions = 
        new CopyOnWriteArraySet<>();
    
    @Override
    public void afterConnectionEstablished(WebSocketSession session) {
        sessions.add(session);
    }
    
    // 推送消息
    public void sendMessage(String message) throws IOException {
        for (WebSocketSession session : sessions) {
            if (session.isOpen()) {
                session.sendMessage(new TextMessage(message));
            }
        }
    }
}
```

### 定时推送

```java
@Component
public class DataPushTask {
    
    @Autowired
    private DashboardWebSocketHandler webSocketHandler;
    
    @Autowired
    private RedisTemplate<String, Object> redisTemplate;
    
    @Scheduled(fixedRate = 5000)  // 每 5 秒推送一次
    public void pushData() {
        try {
            Map<String, Object> data = new HashMap<>();
            data.put("sales", redisTemplate.opsForValue().get("dashboard:sales:today"));
            data.put("orders", redisTemplate.opsForValue().get("dashboard:orders:today"));
            
            webSocketHandler.sendMessage(JSON.toJSONString(data));
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
```

## 三、前端接收

```javascript
// 连接 WebSocket
var ws = new WebSocket('ws://localhost:8080/ws/dashboard');

ws.onmessage = function(event) {
    var data = JSON.parse(event.data);
    
    // 更新图表
    updateChart(data);
};
```

## 四、小结

| 组件 | 作用 | 说明 |
|------|------|------|
| WebSocket | 双向通信 | 实时推送 |
| @EnableWebSocket | 开启 WebSocket | 配置类 |
| WebSocketHandler | 消息处理 | 推送消息 |

**核心要点**：
- WebSocket 连接
- 定时推送数据
- 前端接收更新
