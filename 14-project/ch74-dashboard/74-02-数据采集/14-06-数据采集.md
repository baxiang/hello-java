# 74.2 数据采集：如何定时聚合数据？

## 一、先问一个问题

**数据从哪里来？**

### 生活例子：统计报表

```
每天统计
├── 销售额
├── 订单数
├── 用户数
└── 汇总到报表
```

**数据采集**：
- 定时从数据库查询
- 聚合计算
- 缓存到 Redis

## 二、定时任务

### Spring Schedule

```java
@Component
public class DataCollector {
    
    @Autowired
    private RedisTemplate<String, Object> redisTemplate;
    
    // 每 5 分钟执行一次
    @Scheduled(cron = "0 */5 * * * ?")
    public void collectSalesData() {
        // 1. 查询今日销售额
        BigDecimal todaySales = orderMapper.selectTodaySales();
        
        // 2. 查询订单量
        int todayOrders = orderMapper.selectTodayOrders();
        
        // 3. 缓存到 Redis
        redisTemplate.opsForValue().set("dashboard:sales:today", todaySales);
        redisTemplate.opsForValue().set("dashboard:orders:today", todayOrders);
    }
}
```

### 开启定时任务

```java
@SpringBootApplication
@EnableScheduling
public class DashboardApplication {
    public static void main(String[] args) {
        SpringApplication.run(DashboardApplication.class, args);
    }
}
```

## 三、小结

| 组件 | 作用 | 说明 |
|------|------|------|
| @Scheduled | 定时注解 | Cron 表达式 |
| Redis | 数据缓存 | 快速读取 |
| @EnableScheduling | 开启定时 | 启动类添加 |

**核心要点**：
- 定时采集数据
- 聚合计算
- 缓存到 Redis
