# 74.5 大屏适配：如何适配不同屏幕？

## 一、先问一个问题

**如何适配不同分辨率？**

### 生活例子：照片缩放

```
小屏幕
└── 缩小显示

大屏幕
└── 放大显示
```

**适配方案**：
- 响应式布局
- 缩放适配

## 二、响应式布局

### CSS 媒体查询

```css
/* 小屏幕 */
@media (max-width: 1366px) {
    .chart {
        width: 100%;
        height: 300px;
    }
}

/* 大屏幕 */
@media (min-width: 1920px) {
    .chart {
        width: 100%;
        height: 500px;
    }
}
```

### Flex 布局

```css
.dashboard {
    display: flex;
    flex-wrap: wrap;
}

.chart-box {
    flex: 1;
    min-width: 400px;
}
```

## 三、缩放适配

### scale 缩放

```javascript
// 根据屏幕宽度缩放
function adaptScreen() {
    const targetWidth = 1920;
    const actualWidth = document.body.clientWidth;
    const scale = actualWidth / targetWidth;
    
    document.body.style.transform = `scale(${scale})`;
}

window.addEventListener('resize', adaptScreen);
adaptScreen();
```

## 四、小结

| 方案 | 优点 | 缺点 |
|------|------|------|
| 响应式 | 灵活 | 代码多 |
| 缩放 | 简单 | 可能模糊 |

**核心要点**：
- 响应式布局
- 屏幕缩放
- 自适应图表
