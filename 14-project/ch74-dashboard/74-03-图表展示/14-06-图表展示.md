# 74.3 图表展示：如何用 ECharts 展示数据？

## 一、先问一个问题

**如何选择合适的图表？**

### 生活例子：看报表

```
趋势 → 折线图
占比 → 饼图
对比 → 柱状图
分布 → 散点图
```

**图表选择**：
- 根据数据特点
- 根据展示目的

## 二、ECharts 基础

### 引入 ECharts

```html
<script src="https://cdn.jsdelivr.net/npm/echarts@5/dist/echarts.min.js"></script>
```

### 柱状图示例

```javascript
var chart = echarts.init(document.getElementById('main'));

var option = {
    title: { text: '销售统计' },
    xAxis: {
        data: ['周一', '周二', '周三', '周四', '周五']
    },
    yAxis: {},
    series: [{
        type: 'bar',
        data: [100, 200, 150, 300, 250]
    }]
};

chart.setOption(option);
```

## 三、后端接口

### 数据接口

```java
@RestController
@RequestMapping("/api/dashboard")
public class DashboardController {
    
    @Autowired
    private RedisTemplate<String, Object> redisTemplate;
    
    @GetMapping("/sales")
    public Result<SalesVO> getSalesData() {
        SalesVO vo = new SalesVO();
        vo.setTodaySales((BigDecimal) redisTemplate.opsForValue().get("dashboard:sales:today"));
        vo.setTodayOrders((Integer) redisTemplate.opsForValue().get("dashboard:orders:today"));
        return Result.success(vo);
    }
}
```

## 四、小结

| 图表类型 | 适用场景 | 示例 |
|----------|----------|------|
| 柱状图 | 对比 | 销售对比 |
| 折线图 | 趋势 | 销售趋势 |
| 饼图 | 占比 | 品类占比 |
| 地图 | 分布 | 地域分布 |

**核心要点**：
- 选择合适的图表
- ECharts 配置
- 后端数据接口
