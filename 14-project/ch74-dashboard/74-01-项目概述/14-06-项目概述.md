# 74.1 项目概述：数据可视化大屏是什么？

## 一、先问一个问题

**什么是数据可视化大屏？**

### 生活例子：指挥中心

```
交通指挥中心
├── 实时车流量
├── 事故统计
├── 拥堵路段
└── 警力分布
```

**数据可视化大屏**：
- 关键数据一目了然
- 实时监控业务状态
- 辅助决策分析

## 二、应用场景

### 常见场景

```
数据可视化大屏
├── 销售大屏（销售额、订单量）
├── 运营大屏（用户数、活跃度）
├── 监控大屏（服务器状态、告警）
└── 政务大屏（城市数据、民生）
```

## 三、技术栈

| 技术 | 作用 | 为什么选它 |
|------|------|------------|
| Spring Boot | 后端框架 | 快速开发 |
| ECharts | 图表库 | 图表丰富 |
| WebSocket | 实时推送 | 低延迟 |
| Vue.js | 前端框架 | 响应式 |

## 四、系统架构

```
数据源
  ↓
数据采集（定时任务）
  ↓
数据聚合（Redis 缓存）
  ↓
WebSocket 推送
  ↓
前端展示（ECharts）
```

## 五、小结

| 方面 | 说明 |
|------|------|
| 项目定位 | 数据可视化展示 |
| 核心技术 | ECharts、WebSocket |
| 学习收获 | 大屏开发经验 |
