# 69.4 文章模块：如何实现文章管理？

## 一、先问一个问题

**文章管理的核心是什么？**

### 生活例子：写日记

```
写日记
├── 准备本子（创建）
├── 写内容（读取）
├── 修改内容（更新）
└── 撕掉页面（删除）
```

**文章管理就是 CRUD**：
- Create（创建）
- Read（读取）
- Update（更新）
- Delete（删除）

## 二、发布文章

### 发布流程

```
用户提交文章
  ↓
验证标题和内容
  ↓
保存到数据库
  ↓
返回文章 ID
```

### 代码实现

```java
@Service
public class ArticleService {
    
    @Autowired
    private ArticleMapper articleMapper;
    
    public Article publish(ArticleDTO dto, Long userId) {
        // 1. 验证
        if (dto.getTitle() == null || dto.getTitle().isEmpty()) {
            throw new BusinessException("标题不能为空");
        }
        
        // 2. 创建文章
        Article article = new Article();
        article.setTitle(dto.getTitle());
        article.setSummary(dto.getSummary());
        article.setContent(dto.getContent());
        article.setAuthorId(userId);
        article.setCategoryId(dto.getCategoryId());
        article.setStatus(dto.getStatus()); // 0 草稿/1 发布
        
        // 3. 保存
        articleMapper.insert(article);
        
        // 4. 保存标签
        if (dto.getTagIds() != null) {
            for (Long tagId : dto.getTagIds()) {
                articleMapper.insertTag(article.getId(), tagId);
            }
        }
        
        return article;
    }
}
```

## 三、文章列表

### 分页查询

```java
public Page<ArticleVO> listArticles(int page, int size, Long categoryId) {
    // 1. 计算偏移量
    int offset = (page - 1) * size;
    
    // 2. 查询文章
    List<Article> articles = articleMapper.selectList(categoryId, offset, size);
    
    // 3. 查询总数
    int total = articleMapper.countList(categoryId);
    
    // 4. 转换为 VO
    List<ArticleVO> voList = articles.stream()
        .map(this::convertToVO)
        .collect(Collectors.toList());
    
    return new Page<>(voList, total, page, size);
}
```

### MyBatis-Plus 分页

```java
public Page<ArticleVO> listArticles(int page, int size) {
    Page<Article> articlePage = new Page<>(page, size);
    
    IPage<Article> result = articleMapper.selectPage(
        articlePage,
        new LambdaQueryWrapper<Article>()
            .eq(Article::getStatus, 1)  // 只查询已发布
            .orderByDesc(Article::getCreateTime)  // 按时间倒序
    );
    
    return convertToVOPage(result);
}
```

## 四、文章详情

### 查询详情

```java
public ArticleVO getArticle(Long id) {
    // 1. 查询文章
    Article article = articleMapper.selectById(id);
    if (article == null) {
        throw new BusinessException("文章不存在");
    }
    
    // 2. 增加阅读量
    articleMapper.incrementViewCount(id);
    
    // 3. 查询标签
    List<Tag> tags = articleMapper.selectTagsByArticleId(id);
    
    // 4. 转换为 VO
    ArticleVO vo = new ArticleVO();
    BeanUtils.copyProperties(article, vo);
    vo.setTags(tags);
    
    return vo;
}
```

## 五、编辑和删除

### 编辑文章

```java
public void updateArticle(Long id, ArticleDTO dto, Long userId) {
    // 1. 查询文章
    Article article = articleMapper.selectById(id);
    if (article == null) {
        throw new BusinessException("文章不存在");
    }
    
    // 2. 验证权限
    if (!article.getAuthorId().equals(userId)) {
        throw new BusinessException("无权限修改");
    }
    
    // 3. 更新
    article.setTitle(dto.getTitle());
    article.setContent(dto.getContent());
    article.setCategoryId(dto.getCategoryId());
    
    articleMapper.updateById(article);
    
    // 4. 更新标签
    articleMapper.deleteTagsByArticleId(id);
    if (dto.getTagIds() != null) {
        for (Long tagId : dto.getTagIds()) {
            articleMapper.insertTag(id, tagId);
        }
    }
}
```

### 删除文章

```java
public void deleteArticle(Long id, Long userId) {
    Article article = articleMapper.selectById(id);
    if (article == null) {
        throw new BusinessException("文章不存在");
    }
    
    if (!article.getAuthorId().equals(userId)) {
        throw new BusinessException("无权限删除");
    }
    
    // 软删除（修改状态）
    article.setStatus(-1);  // -1 表示已删除
    articleMapper.updateById(article);
}
```

## 六、Markdown 支持

### 前端渲染

```javascript

<template>
  <div v-html="renderedContent"></div>
</template>

export default {
  computed: {
    renderedContent() {
      return marked(this.article.content);
    }
  }
}
</script>
```

### 后端处理

```java
// 可选：后端转换为 HTML
@Service
public class MarkdownService {
    
    public String toHtml(String markdown) {
        Parser parser = Parser.builder().build();
        Node document = parser.parse(markdown);
        HtmlRenderer renderer = HtmlRenderer.builder().build();
        return renderer.render(document);
    }
}
```

## 七、小结

| 功能 | 关键点 | 技术 |
|------|--------|------|
| 发布 | 标题验证、标签关联 | 事务 |
| 列表 | 分页、分类筛选 | MyBatis-Plus |
| 详情 | 阅读量统计 | 自增字段 |
| 编辑 | 权限验证 | 作者 ID 验证 |
| 删除 | 软删除 | 状态字段 |

**核心要点**：
- 权限验证（只能操作自己的文章）
- 软删除（不真正删除数据）
- 阅读量统计（并发更新）
