## 项目功能

```
博客系统
├── 用户模块
│   ├── 注册登录
│   ├── 个人信息
│   └── 密码修改
├── 文章模块
│   ├── 文章发布
│   ├── 文章编辑
│   ├── 文章删除
│   └── 文章列表
├── 分类模块
│   ├── 分类管理
│   └── 分类筛选
├── 标签模块
│   ├── 标签管理
│   └── 标签筛选
└── 评论模块
    ├── 发表评论
    ├── 回复评论
    └── 评论管理
```

## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | Vue.js 3 | 响应式 UI |
| 后端 | Spring Boot 3.4 | RESTful API |
| 数据库 | MySQL 8.0 | 数据存储 |
| 缓存 | Redis 7.0 | 缓存、Session |
| 认证 | JWT | Token 认证 |
| 部署 | Docker | 容器化部署 |
