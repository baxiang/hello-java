# 69.3 用户模块：如何实现注册登录？

## 一、先问一个问题

**用户登录的本质是什么？**

### 生活例子：进小区

```
进小区
├── 报房号（用户名）
├── 保安核实（验证密码）
└── 给门禁卡（Token）
```

**登录的本质**：
1. 验证身份（用户名 + 密码）
2. 发放凭证（Token）
3. 凭凭证访问（Token 验证）

## 二、注册功能

### 注册流程

```
用户提交注册信息
  ↓
验证用户名是否存在
  ↓
密码加密
  ↓
保存到数据库
  ↓
返回成功
```

### 代码实现

```java
@Service
public class UserService {
    
    @Autowired
    private UserMapper userMapper;
    
    @Autowired
    private PasswordEncoder passwordEncoder;
    
    public void register(RegisterDTO dto) {
        // 1. 验证用户名是否存在
        User exist = userMapper.selectByUsername(dto.getUsername());
        if (exist != null) {
            throw new BusinessException("用户名已存在");
        }
        
        // 2. 创建用户
        User user = new User();
        user.setUsername(dto.getUsername());
        user.setEmail(dto.getEmail());
        
        // 3. 密码加密
        user.setPassword(passwordEncoder.encode(dto.getPassword()));
        
        // 4. 保存
        userMapper.insert(user);
    }
}
```

## 三、登录功能

### 登录流程

```
用户提交登录信息
  ↓
查询用户
  ↓
验证密码
  ↓
生成 Token
  ↓
返回 Token 和用户信息
```

### JWT Token

```java
@Service
public class JwtUtil {
    
    private String secret = "my-secret-key";
    private long expiration = 7 * 24 * 60 * 60 * 1000; // 7 天
    
    // 生成 Token
    public String generateToken(User user) {
        return Jwts.builder()
            .setSubject(String.valueOf(user.getId()))
            .claim("username", user.getUsername())
            .setIssuedAt(new Date())
            .setExpiration(new Date(System.currentTimeMillis() + expiration))
            .signWith(SignatureAlgorithm.HS256, secret)
            .compact();
    }
    
    // 解析 Token
    public Claims parseToken(String token) {
        return Jwts.parser()
            .setSigningKey(secret)
            .parseClaimsJws(token)
            .getBody();
    }
    
    // 验证 Token
    public boolean validateToken(String token) {
        try {
            parseToken(token);
            return true;
        } catch (Exception e) {
            return false;
        }
    }
}
```

### 登录接口

```java
@RestController
@RequestMapping("/api/user")
public class UserController {
    
    @Autowired
    private UserService userService;
    
    @PostMapping("/login")
    public Result<LoginVO> login(@RequestBody LoginDTO dto) {
        LoginVO vo = userService.login(dto);
        return Result.success(vo);
    }
}
```

## 四、Token 验证

### 拦截器

```java
@Component
public class JwtInterceptor implements HandlerInterceptor {
    
    @Autowired
    private JwtUtil jwtUtil;
    
    @Override
    public boolean preHandle(HttpServletRequest request, 
                           HttpServletResponse response, 
                           Object handler) {
        // 获取 Token
        String token = request.getHeader("Authorization");
        
        // 验证 Token
        if (!jwtUtil.validateToken(token)) {
            response.setStatus(401);
            return false;
        }
        
        // 解析用户信息
        Claims claims = jwtUtil.parseToken(token);
        Long userId = Long.parseLong(claims.getSubject());
        request.setAttribute("userId", userId);
        
        return true;
    }
}
```

### 配置拦截器

```java
@Configuration
public class WebConfig implements WebMvcConfigurer {
    
    @Autowired
    private JwtInterceptor jwtInterceptor;
    
    @Override
    public void addInterceptors(InterceptorRegistry registry) {
        registry.addInterceptor(jwtInterceptor)
            .addPathPatterns("/api/**")  // 拦截 API 请求
            .excludePathPatterns("/api/user/login", "/api/user/register");  // 排除登录注册
    }
}
```

## 五、小结

| 功能 | 关键点 | 技术 |
|------|--------|------|
| 注册 | 用户名唯一、密码加密 | BCrypt |
| 登录 | 验证密码、生成 Token | JWT |
| 验证 | 拦截器、Token 验证 | HandlerInterceptor |

**安全要点**：
- 密码加密存储
- Token 有过期时间
- 敏感接口需要验证
