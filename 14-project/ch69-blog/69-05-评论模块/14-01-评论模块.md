# 69.5 评论模块：如何实现嵌套评论？

## 一、先问一个问题

**嵌套评论的本质是什么？**

### 生活例子：盖楼

```
1 楼（一级评论）
├── 2 楼（回复 1 楼）
│   └── 3 楼（回复 2 楼）
└── 4 楼（回复 1 楼）
```

**嵌套评论**：
- 每条评论有父评论 ID
- 通过 parent_id 建立关系
- 递归查询得到完整结构

## 二、发表评论

### 评论流程

```
用户提交评论
  ↓
验证内容
  ↓
保存到数据库
  ↓
更新文章评论数
```

### 代码实现

```java
@Service
public class CommentService {
    
    @Autowired
    private CommentMapper commentMapper;
    
    @Autowired
    private ArticleMapper articleMapper;
    
    public Comment publish(CommentDTO dto, Long userId) {
        // 1. 验证
        if (dto.getContent() == null || dto.getContent().isEmpty()) {
            throw new BusinessException("评论内容不能为空");
        }
        
        // 2. 创建评论
        Comment comment = new Comment();
        comment.setArticleId(dto.getArticleId());
        comment.setUserId(userId);
        comment.setParentId(dto.getParentId() != null ? dto.getParentId() : 0);
        comment.setContent(dto.getContent());
        
        // 3. 保存
        commentMapper.insert(comment);
        
        // 4. 更新文章评论数
        articleMapper.incrementCommentCount(dto.getArticleId());
        
        return comment;
    }
}
```

## 三、查询评论列表

### 一级评论

```java
public List<CommentVO> listComments(Long articleId) {
    // 1. 查询一级评论（parent_id = 0）
    List<Comment> comments = commentMapper.selectList(articleId, 0L);
    
    // 2. 转换为 VO
    return comments.stream()
        .map(this::convertToVO)
        .collect(Collectors.toList());
}
```

### 嵌套评论（递归查询）

```java
public List<CommentVO> listCommentsWithReplies(Long articleId) {
    // 1. 查询一级评论
    List<Comment> comments = commentMapper.selectList(articleId, 0L);
    
    // 2. 查询每条评论的回复
    for (Comment comment : comments) {
        List<Comment> replies = commentMapper.selectList(articleId, comment.getId());
        comment.setReplies(replies);
    }
    
    // 3. 转换为 VO
    return comments.stream()
        .map(this::convertToVO)
        .collect(Collectors.toList());
}
```

### 树形结构

```java
private CommentVO convertToVO(Comment comment) {
    CommentVO vo = new CommentVO();
    vo.setId(comment.getId());
    vo.setContent(comment.getContent());
    vo.setCreateTime(comment.getCreateTime());
    
    // 用户信息
    User user = userMapper.selectById(comment.getUserId());
    vo.setUsername(user.getUsername());
    vo.setAvatar(user.getAvatar());
    
    // 回复列表（递归）
    if (comment.getReplies() != null) {
        vo.setReplies(comment.getReplies().stream()
            .map(this::convertToVO)
            .collect(Collectors.toList()));
    }
    
    return vo;
}
```

## 四、回复评论

### 回复流程

```
用户提交回复
  ↓
设置 parent_id
  ↓
保存到数据库
```

### 代码实现

```java
public void reply(Long commentId, String content, Long userId) {
    // 1. 查询原评论
    Comment parent = commentMapper.selectById(commentId);
    if (parent == null) {
        throw new BusinessException("评论不存在");
    }
    
    // 2. 创建回复
    Comment reply = new Comment();
    reply.setArticleId(parent.getArticleId());
    reply.setUserId(userId);
    reply.setParentId(commentId);  // 设置父评论 ID
    reply.setContent(content);
    
    // 3. 保存
    commentMapper.insert(reply);
}
```

## 五、删除评论

### 软删除

```java
public void deleteComment(Long id, Long userId) {
    Comment comment = commentMapper.selectById(id);
    if (comment == null) {
        throw new BusinessException("评论不存在");
    }
    
    // 验证权限
    if (!comment.getUserId().equals(userId)) {
        throw new BusinessException("无权限删除");
    }
    
    // 软删除
    comment.setStatus(0);  // 0 表示已删除
    commentMapper.updateById(comment);
}
```

### 级联删除

```java
public void deleteCommentWithReplies(Long id) {
    // 1. 查询所有子评论（递归）
    List<Long> ids = new ArrayList<>();
    findChildCommentIds(id, ids);
    
    // 2. 批量删除
    commentMapper.deleteBatchIds(ids);
}

private void findChildCommentIds(Long parentId, List<Long> ids) {
    ids.add(parentId);
    
    List<Comment> children = commentMapper.selectByParentId(parentId);
    for (Comment child : children) {
        findChildCommentIds(child.getId(), ids);
    }
}
```

## 六、评论审核

### 审核流程

```
用户发表评论
  ↓
状态为待审核（status=0）
  ↓
管理员审核
  ↓
状态改为通过（status=1）
```

### 代码实现

```java
// 发表评论时设置待审核
comment.setStatus(0);  // 待审核

// 管理员审核
public void auditComment(Long id, boolean passed) {
    Comment comment = commentMapper.selectById(id);
    comment.setStatus(passed ? 1 : -1);  // 1 通过/-1 拒绝
    commentMapper.updateById(comment);
}
```

## 七、小结

| 功能 | 关键点 | 技术 |
|------|--------|------|
| 发表评论 | parent_id 设置 | 树形结构 |
| 查询评论 | 递归查询 | 递归/迭代 |
| 回复评论 | 设置父评论 ID | 自关联 |
| 删除评论 | 级联删除 | 递归查询 |
| 评论审核 | 状态字段 | 状态管理 |

**核心要点**：
- parent_id 建立树形关系
- 递归查询获取完整结构
- 软删除保留数据
