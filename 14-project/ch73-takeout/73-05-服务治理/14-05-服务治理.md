# 73.5 服务治理：Nacos 和 Sentinel 如何使用？

## 一、Nacos 服务注册

```yaml
spring:
  cloud:
    nacos:
      discovery:
        server-addr: localhost:8848
```

## 二、Sentinel 限流

```java
@SentinelResource(value = "createOrder", blockHandler = "handleBlock")
public Order createOrder(OrderDTO dto) {
    return orderService.create(dto);
}
```

## 三、小结

**核心要点**：
- Nacos 注册中心
- Sentinel 限流熔断
- Gateway 网关
