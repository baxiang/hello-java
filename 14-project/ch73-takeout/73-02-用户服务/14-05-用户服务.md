# 73.2 用户服务：如何实现登录认证？

## 一、用户服务功能

```
用户服务
├── 注册登录
├── 个人信息
└── 地址管理
```

## 二、JWT 认证

```java
@PostMapping("/login")
public Result<String> login(@RequestBody LoginDTO dto) {
    // 验证密码
    User user = userService.login(dto);
    
    // 生成 Token
    String token = jwtUtil.generateToken(user);
    
    return Result.success(token);
}
```

## 三、小结

**核心要点**：
- JWT 无状态认证
- 服务独立部署
- Feign 调用传递 Token
