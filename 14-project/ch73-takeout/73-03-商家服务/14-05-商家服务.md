# 73.3 商家服务：如何管理店铺和菜品？

## 一、商家服务功能

```
商家服务
├── 店铺管理
├── 菜品管理
└── 库存管理
```

## 二、Feign 调用

```java
@FeignClient(name = "merchant-service")
public interface MerchantClient {
    
    @GetMapping("/api/merchant/{id}")
    Result<MerchantVO> getMerchant(@PathVariable Long id);
    
    @GetMapping("/api/merchant/{id}/products")
    Result<List<ProductVO>> getProducts(@PathVariable Long id);
}
```

## 三、小结

**核心要点**：
- 店铺菜品管理
