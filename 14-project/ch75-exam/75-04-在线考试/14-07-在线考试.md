# 75.4 在线考试：如何实现考试流程？

## 一、先问一个问题

**考试流程是什么？**

### 生活例子：学校考试

```
进入考场
  ↓
发放试卷
  ↓
开始答题
  ↓
时间到交卷
  ↓
等待成绩
```

**在线考试**：
- 开始考试
- 倒计时
- 提交试卷

## 二、数据库设计

### 考试记录表

```sql
CREATE TABLE `exam_record` (
    `id` BIGINT PRIMARY KEY AUTO_INCREMENT,
    `user_id` BIGINT NOT NULL,
    `paper_id` BIGINT NOT NULL,
    `start_time` DATETIME COMMENT '开始时间',
    `end_time` DATETIME COMMENT '结束时间',
    `score` INT COMMENT '得分',
    `status` TINYINT DEFAULT 0 COMMENT '状态（0 进行中/1 已交卷/2 已阅卷）',
    `answer` JSON COMMENT '答案',
    `create_time` DATETIME DEFAULT CURRENT_TIMESTAMP
) COMMENT='考试记录表';
```

## 三、开始考试

```java
@PostMapping("/exam/start")
public Result<ExamRecord> startExam(@RequestParam Long paperId, @RequestParam Long userId) {
    // 1. 检查是否已考过
    ExamRecord exist = examRecordMapper.selectByUserIdAndPaperId(userId, paperId);
    if (exist != null) {
        throw new BusinessException("已参加过考试");
    }
    
    // 2. 创建考试记录
    ExamRecord record = new ExamRecord();
    record.setUserId(userId);
    record.setPaperId(paperId);
    record.setStartTime(LocalDateTime.now());
    record.setStatus(0);  // 进行中
    
    examRecordMapper.insert(record);
    
    return Result.success(record);
}
```

## 四、提交试卷

```java
@PostMapping("/exam/submit")
public Result<Void> submitExam(@RequestBody SubmitExamDTO dto) {
    ExamRecord record = examRecordMapper.selectById(dto.getRecordId());
    
    // 1. 保存答案
    record.setAnswer(JSON.toJSONString(dto.getAnswers()));
    record.setEndTime(LocalDateTime.now());
    record.setStatus(1);  // 已交卷
    
    examRecordMapper.updateById(record);
    
    // 2. 自动阅卷（客观题）
    gradingService.autoGrade(record.getId());
    
    return Result.success();
}
```

## 五、防作弊

### 切屏检测

```javascript
// 前端检测切屏
document.addEventListener('visibilitychange', function() {
    if (document.hidden) {
        // 发送切屏事件
        sendCheatingEvent('切屏');
    }
});

// 失去焦点检测
window.addEventListener('blur', function() {
    sendCheatingEvent('失去焦点');
});
```

### 后端记录

```java
@PostMapping("/exam/cheating")
public Result<Void> recordCheating(@RequestBody CheatingDTO dto) {
    CheatingRecord record = new CheatingRecord();
    record.setRecordId(dto.getRecordId());
    record.setType(dto.getType());
    record.setTime(LocalDateTime.now());
    
    cheatingRecordMapper.insert(record);
    
    // 超过 3 次强制交卷
    int count = cheatingRecordMapper.countByRecordId(dto.getRecordId());
    if (count >= 3) {
        examService.forceSubmit(dto.getRecordId());
    }
    
    return Result.success();
}
```

## 六、小结

| 功能 | 实现方式 | 说明 |
|------|----------|------|
| 开始考试 | 创建记录 | 记录开始时间 |
| 倒计时 | 前端计时 | 剩余时间显示 |
| 提交试卷 | 保存答案 | 触发阅卷 |
| 防作弊 | 切屏检测 | 强制交卷 |

**核心要点**：
- 考试记录管理
- 答案保存
- 防作弊机制
