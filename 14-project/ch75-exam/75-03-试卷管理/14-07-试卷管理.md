# 75.3 试卷管理：如何组卷？

## 一、先问一个问题

**如何生成试卷？**

### 生活例子：配套餐

```
手动组卷
└── 自己选菜（挑试题）

自动组卷
└── 配套餐（按规则）
```

**组卷方式**：
- 手动组卷（灵活）
- 自动组卷（高效）

## 二、数据库设计

### 试卷表

```sql
CREATE TABLE `exam_paper` (
    `id` BIGINT PRIMARY KEY AUTO_INCREMENT,
    `title` VARCHAR(200) NOT NULL COMMENT '试卷标题',
    `subject_id` BIGINT COMMENT '科目 ID',
    `total_score` INT NOT NULL COMMENT '总分',
    `pass_score` INT COMMENT '及格分',
    `duration` INT NOT NULL COMMENT '时长（分钟）',
    `type` TINYINT DEFAULT 0 COMMENT '类型（0 手动/1 自动）',
    `create_time` DATETIME DEFAULT CURRENT_TIMESTAMP
) COMMENT='试卷表';
```

### 试卷试题关联表

```sql
CREATE TABLE `exam_paper_question` (
    `paper_id` BIGINT NOT NULL,
    `question_id` BIGINT NOT NULL,
    `score` INT NOT NULL COMMENT '分值',
    `order` INT NOT NULL COMMENT '顺序',
    PRIMARY KEY (`paper_id`, `question_id`)
) COMMENT='试卷试题关联表';
```

## 三、手动组卷

```java
@PostMapping("/papers/manual")
public Result<ExamPaper> createManualPaper(@RequestBody ManualPaperDTO dto) {
    // 1. 创建试卷
    ExamPaper paper = new ExamPaper();
    paper.setTitle(dto.getTitle());
    paper.setTotalScore(dto.getTotalScore());
    paper.setType(0);  // 手动
    
    examPaperMapper.insert(paper);
    
    // 2. 添加试题
    for (PaperQuestionDTO q : dto.getQuestions()) {
        examPaperQuestionMapper.insert(paper.getId(), q.getQuestionId(), q.getScore());
    }
    
    return Result.success(paper);
}
```

## 四、自动组卷

```java
@PostMapping("/papers/auto")
public Result<ExamPaper> createAutoPaper(@RequestBody AutoPaperDTO dto) {
    // 1. 创建试卷
    ExamPaper paper = new ExamPaper();
    paper.setTitle(dto.getTitle());
    paper.setType(1);  // 自动
    
    examPaperMapper.insert(paper);
    
    // 2. 按规则抽题
    List<Question> questions = questionService.selectByRule(dto.getRule());
    
    // 3. 添加到试卷
    for (Question q : questions) {
        examPaperQuestionMapper.insert(paper.getId(), q.getId(), q.getDefaultScore());
    }
    
    return Result.success(paper);
}
```

## 五、小结

| 组卷方式 | 优点 | 缺点 |
|----------|------|------|
| 手动组卷 | 灵活可控 | 耗时 |
| 自动组卷 | 快速高效 | 需定义规则 |

**核心要点**：
- 试卷结构设计
- 试题分值分配
- 组卷规则定义
