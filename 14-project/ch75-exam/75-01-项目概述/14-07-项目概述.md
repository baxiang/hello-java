# 75.1 项目概述：在线考试系统是什么？

## 一、先问一个问题

**什么是在线考试系统？**

### 生活例子：学校考试

```
传统考试
├── 打印试卷
├── 现场答题
├── 人工阅卷
└── 统计分数

在线考试
├── 电子试卷
├── 在线答题
├── 自动阅卷
└── 实时出分
```

**在线考试系统**：
- 随时随地考试
- 自动组卷阅卷
- 实时出成绩

## 二、功能模块

```
在线考试系统
├── 题库管理
│   ├── 试题录入
│   ├── 试题分类
│   └── 试题搜索
├── 试卷管理
│   ├── 手动组卷
│   ├── 自动组卷
│   └── 试卷管理
├── 在线考试
│   ├── 参加考试
│   ├── 倒计时
│   └── 防作弊
└── 阅卷评分
    ├── 自动阅卷
    ├── 手动阅卷
    └── 成绩统计
```

## 三、技术栈

| 技术 | 作用 | 为什么选它 |
|------|------|------------|
| Spring Boot | 后端框架 | 快速开发 |
| Vue.js | 前端框架 | 响应式 |
| MySQL | 数据库 | 数据存储 |
| Redis | 缓存 | 考试数据 |

## 四、小结

| 方面 | 说明 |
|------|------|
| 项目定位 | 在线考试平台 |
| 核心功能 | 题库、组卷、考试、阅卷 |
| 学习收获 | 考试系统开发经验 |
