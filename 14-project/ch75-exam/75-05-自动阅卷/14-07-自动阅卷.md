## 二、客观题判分

### 单选题/判断题

```java
public int gradeObjectiveQuestion(Question question, String userAnswer) {
    // 完全匹配
    if (question.getAnswer().equals(userAnswer)) {
        return question.getDefaultScore();
    }
    return 0;
}
```

### 多选题

```java
public int gradeMultipleChoice(Question question, String userAnswer) {
    String standard = question.getAnswer();  // 如"ABC"
    
    // 全对
    if (standard.equals(userAnswer)) {
        return question.getDefaultScore();
    }
    
    // 部分对（漏选）
    if (standard.contains(userAnswer) && userAnswer.length() > 0) {
        return question.getDefaultScore() / 2;
    }
    
    // 错选
    return 0;
}
```

## 三、自动阅卷服务

```java
@Service
public class GradingService {
    
    @Autowired
    private ExamRecordMapper examRecordMapper;
    
    @Autowired
    private ExamPaperQuestionMapper paperQuestionMapper;
    
    // 自动阅卷
    @Transactional
    public void autoGrade(Long recordId) {
        // 1. 获取考试记录
        ExamRecord record = examRecordMapper.selectById(recordId);
        Map<String, String> userAnswers = parseAnswers(record.getAnswer());
        
        // 2. 获取试卷试题
        List<ExamPaperQuestion> paperQuestions = paperQuestionMapper.selectByPaperId(record.getPaperId());
        
        // 3. 判分
        int totalScore = 0;
        for (ExamPaperQuestion pq : paperQuestions) {
            Question question = questionMapper.selectById(pq.getQuestionId());
            String userAnswer = userAnswers.get(question.getId().toString());
            
            int score = gradeQuestion(question, userAnswer);
            totalScore += score;
        }
        
        // 4. 更新成绩
        record.setScore(totalScore);
        record.setStatus(2);  // 已阅卷
        examRecordMapper.updateById(record);
    }
    
    private int gradeQuestion(Question question, String userAnswer) {
        switch (question.getType()) {
            case 1: // 单选
                return gradeObjectiveQuestion(question, userAnswer);
            case 2: // 多选
                return gradeMultipleChoice(question, userAnswer);
            case 3: // 判断
                return gradeObjectiveQuestion(question, userAnswer);
            default:
                return 0;  // 主观题需人工阅卷
        }
    }
}
```

## 四、成绩统计

```java
@GetMapping("/exam/statistics/{paperId}")
public Result<ExamStatistics> getStatistics(@PathVariable Long paperId) {
    ExamStatistics stats = new ExamStatistics();
    
    // 参加考试人数
    stats.setTotalCount(examRecordMapper.countByPaperId(paperId));
    
    // 平均分
    stats.setAvgScore(examRecordMapper.avgScoreByPaperId(paperId));
    
    // 最高分
    stats.setMaxScore(examRecordMapper.maxScoreByPaperId(paperId));
    
    // 及格率
    int passCount = examRecordMapper.countPassByPaperId(paperId);
    stats.setPassRate((double) passCount / stats.getTotalCount() * 100);
    
    return Result.success(stats);
}
```

## 五、小结

| 题型 | 阅卷方式 | 评分规则 |
|------|----------|----------|
| 单选题 | 自动 | 对/错 |
| 多选题 | 自动 | 全对/半对/错 |
| 判断题 | 自动 | 对/错 |
| 简答题 | 人工 | 灵活评分 |

**核心要点**：
- 客观题自动判分
- 多选题部分得分
- 成绩统计分析
