# 75.2 题库管理：如何管理试题？

## 一、先问一个问题

**试题如何分类存储？**

### 生活例子：图书馆

```
图书馆
├── 按分类摆放
├── 有索引卡片
└── 方便查找
```

**题库管理**：
- 按题型分类
- 按难度分级
- 按知识点标记

## 二、数据库设计

### 试题表

```sql
CREATE TABLE `question` (
    `id` BIGINT PRIMARY KEY AUTO_INCREMENT,
    `type` TINYINT NOT NULL COMMENT '类型（1 单选/2 多选/3 判断/4 简答）',
    `content` TEXT NOT NULL COMMENT '题目内容',
    `options` JSON COMMENT '选项（JSON）',
    `answer` VARCHAR(500) NOT NULL COMMENT '答案',
    `difficulty` TINYINT DEFAULT 1 COMMENT '难度（1 易/2 中/3 难）',
    `subject_id` BIGINT COMMENT '科目 ID',
    `create_time` DATETIME DEFAULT CURRENT_TIMESTAMP
) COMMENT='试题表';
```

## 三、试题管理

### 添加试题

```java
@PostMapping("/questions")
public Result<Question> addQuestion(@RequestBody QuestionDTO dto) {
    Question question = new Question();
    question.setType(dto.getType());
    question.setContent(dto.getContent());
    question.setOptions(JSON.toJSONString(dto.getOptions()));
    question.setAnswer(dto.getAnswer());
    question.setDifficulty(dto.getDifficulty());
    
    questionMapper.insert(question);
    return Result.success(question);
}
```

### 查询试题

```java
@GetMapping("/questions")
public Result<Page<QuestionVO>> listQuestions(
    @RequestParam(required = false) Integer type,
    @RequestParam(required = false) Integer difficulty,
    @RequestParam(defaultValue = "1") int page,
    @RequestParam(defaultValue = "10") int size
) {
    Page<QuestionVO> result = questionService.list(type, difficulty, page, size);
    return Result.success(result);
}
```

## 四、小结

| 题型 | 存储方式 | 示例 |
|------|----------|------|
| 单选题 | JSON 选项 | {"A":"选项 1","B":"选项 2"} |
| 多选题 | JSON 选项 | 同上 |
| 判断题 | 无选项 | true/false |
| 简答题 | 无选项 | 文字答案 |

**核心要点**：
- 题型分类存储
- 难度分级
- 知识点标记
