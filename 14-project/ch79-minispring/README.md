# 14.8 Mini-Spring 项目

## 项目概述

Mini-Spring 是一个简化版的 Spring 框架，通过逐步实现 Spring 的核心功能，帮助深入理解 Spring 的工作原理。

## 学习建议

1. **按顺序学习**：每个章节依赖前一章的内容
2. **动手实践**：每学完一节，自己实现一遍
3. **对比 Spring**：与 Spring 源码对比理解
