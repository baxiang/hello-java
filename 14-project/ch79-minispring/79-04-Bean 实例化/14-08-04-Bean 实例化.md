# 79.4 Bean 实例化

## 一、实例化策略接口

定义 Bean 实例化的策略，支持不同的实例化方式。

```java
package com.example.minispring.beans.factory.support;

import com.example.minispring.beans.config.BeanDefinition;
import com.example.minispring.beans.exception.BeansException;

/**
 * 实例化策略接口
 */
public interface InstantiationStrategy {
    
    /**
     * 实例化 Bean
     */
    Object instantiate(BeanDefinition beanDefinition) throws BeansException;
}
```

## 二、简单实例化策略

使用无参构造方法实例化 Bean。

```java
package com.example.minispring.beans.factory.support;

import com.example.minispring.beans.config.BeanDefinition;
import com.example.minispring.beans.exception.BeansException;

/**
 * 简单实例化策略：使用无参构造方法
 */
public class SimpleInstantiationStrategy implements InstantiationStrategy {
    
    @Override
    public Object instantiate(BeanDefinition beanDefinition) throws BeansException {
        Class<?> beanClass = beanDefinition.getBeanClass();
        try {
            // 调用无参构造方法
            return beanClass.getDeclaredConstructor().newInstance();
        } catch (Exception e) {
            throw new BeansException("Failed to instantiate [" + beanClass.getName() + "]", e);
        }
    }
}
```

## 三、AbstractAutowireCapableBeanFactory

实现 Bean 的完整创建流程：实例化 → 属性填充 → 初始化。

```java
package com.example.minispring.beans.factory.support;

import com.example.minispring.beans.config.BeanDefinition;
import com.example.minispring.beans.exception.BeansException;

/**
 * 自动装配 Bean 工厂：处理 Bean 实例化和属性填充
 */
public abstract class AbstractAutowireCapableBeanFactory extends AbstractBeanFactory {
    
    /** 实例化策略 */
    private InstantiationStrategy instantiationStrategy = new SimpleInstantiationStrategy();
    
    @Override
    protected Object createBean(String beanName, BeanDefinition bd) {
        // 1. 实例化 Bean
        Object bean = instantiateBean(beanName, bd);
        
        // 2. 属性填充（后续实现）
        populateBean(beanName, bd, bean);
        
        // 3. 初始化 Bean
        initializeBean(beanName, bean, bd);
        
        return bean;
    }
    
    /**
     * 实例化 Bean
     */
    protected Object instantiateBean(String beanName, BeanDefinition bd) {
        return instantiationStrategy.instantiate(bd);
    }
    
    /**
     * 属性填充
     */
    protected void populateBean(String beanName, BeanDefinition bd, Object bean) {
        // 后续实现依赖注入
    }
    
    /**
     * 初始化 Bean
     */
    protected void initializeBean(String beanName, Object bean, BeanDefinition bd) {
        // 调用初始化方法
        invokeInitMethods(beanName, bean, bd);
    }
    
    /**
     * 调用初始化方法
     */
    protected void invokeInitMethods(String beanName, Object bean, BeanDefinition bd) {
        // 后续实现 InitializingBean 接口和 init-method
    }
}
```

## 四、DefaultListableBeanFactory 整合

整合所有功能，形成完整的 Bean 工厂。

```java
package com.example.minispring.beans.factory.support;

/**
 * 默认的 Bean 工厂实现
 */
public class DefaultListableBeanFactory extends AbstractAutowireCapableBeanFactory {
    
    // 继承自 AbstractBeanFactory
    // - beanDefinitionMap: Bean 定义注册表
    
    // 继承自 DefaultSingletonBeanRegistry
    // - singletonObjects: 一级缓存
    // - earlySingletonObjects: 二级缓存
    // - singletonFactories: 三级缓存
    
    // 继承自 AbstractAutowireCapableBeanFactory
    // - instantiationStrategy: 实例化策略
    // - createBean: 创建 Bean 的核心方法
}
```

## 五、测试代码

```java
package com.example.minispring.test;

import com.example.minispring.beans.config.BeanDefinition;
import com.example.minispring.beans.factory.support.DefaultListableBeanFactory;

public class BeanInstantiationTest {
    
    public static void main(String[] args) {
        // 1. 创建 Bean 工厂
        DefaultListableBeanFactory factory = new DefaultListableBeanFactory();
        
        // 2. 注册 Bean 定义
        BeanDefinition bd = new BeanDefinition(UserService.class);
        factory.registerBeanDefinition("userService", bd);
        
        // 3. 获取 Bean（触发实例化）
        UserService userService = (UserService) factory.getBean("userService");
        
        // 4. 使用 Bean
        userService.sayHello();
        
        // 5. 验证是单例
        UserService userService2 = (UserService) factory.getBean("userService");
        System.out.println("是否单例：" + (userService == userService2));
    }
}

class UserService {
    public UserService() {
        System.out.println("UserService 构造方法被调用");
    }
    
    public void sayHello() {
        System.out.println("Hello, Mini-Spring!");
    }
}
```

**输出**：
```
UserService 构造方法被调用
Hello, Mini-Spring!
是否单例：true
```

## 六、小结

1. **InstantiationStrategy**：实例化策略接口
2. **SimpleInstantiationStrategy**：简单实例化（无参构造）
3. **AbstractAutowireCapableBeanFactory**：实现完整创建流程
4. **DefaultListableBeanFactory**：完整的 Bean 工厂实现
