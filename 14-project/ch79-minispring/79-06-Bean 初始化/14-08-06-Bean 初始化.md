# 79.6 Bean 初始化

## 一、InitializingBean 接口

Spring 提供的初始化方法接口。

```java
package com.example.minispring.beans.factory;

/**
 * 初始化 Bean 接口
 */
public interface InitializingBean {
    
    /**
     * 在属性设置后执行
     */
    void afterPropertiesSet() throws Exception;
}
```

## 二、初始化方法调用

在 Bean 创建完成后调用初始化方法。

```java
package com.example.minispring.beans.factory.support;

import com.example.minispring.beans.config.BeanDefinition;
import com.example.minispring.beans.exception.BeansException;
import com.example.minispring.beans.factory.InitializingBean;

import java.lang.reflect.Method;

public abstract class AbstractAutowireCapableBeanFactory extends AbstractBeanFactory {
    
    // ... 前面的代码
    
    /**
     * 初始化 Bean
     */
    protected void initializeBean(String beanName, Object bean, BeanDefinition bd) {
        // 1. 调用 InitializingBean 的 afterPropertiesSet
        invokeInitMethods(beanName, bean, bd);
    }
    
    /**
     * 调用初始化方法
     */
    protected void invokeInitMethods(String beanName, Object bean, BeanDefinition bd) {
        // 1. 检查是否实现 InitializingBean 接口
        if (bean instanceof InitializingBean) {
            try {
                ((InitializingBean) bean).afterPropertiesSet();
            } catch (Exception e) {
                throw new BeansException("Failed to invoke afterPropertiesSet", e);
            }
        }
        
        // 2. 调用自定义 init-method（从 BeanDefinition 获取）
        String initMethodName = bd.getInitMethodName();
        if (initMethodName != null && !initMethodName.isEmpty()) {
            try {
                Method method = bean.getClass().getDeclaredMethod(initMethodName);
                method.setAccessible(true);
                method.invoke(bean);
            } catch (Exception e) {
                throw new BeansException("Failed to invoke init method", e);
            }
        }
    }
}
```

## 三、BeanDefinition 扩展

添加 init-method 和 destroy-method 支持。

```java
package com.example.minispring.beans.config;

public class BeanDefinition {
    
    // ... 前面的字段
    
    /** 初始化方法名 */
    private String initMethodName;
    
    /** 销毁方法名 */
    private String destroyMethodName;
    
    public String getInitMethodName() {
        return initMethodName;
    }
    
    public void setInitMethodName(String initMethodName) {
        this.initMethodName = initMethodName;
    }
    
    public String getDestroyMethodName() {
        return destroyMethodName;
    }
    
    public void setDestroyMethodName(String destroyMethodName) {
        this.destroyMethodName = destroyMethodName;
    }
}
```

## 四、DisposableBean 接口

销毁 Bean 时调用的接口。

```java
package com.example.minispring.beans.factory;

/**
 * 销毁 Bean 接口
 */
public interface DisposableBean {
    
    /**
     * 销毁 Bean 时执行
     */
    void destroy() throws Exception;
}
```

## 五、注册销毁回调

```java
package com.example.minispring.beans.factory.support;

import com.example.minispring.beans.exception.BeansException;
import com.example.minispring.beans.factory.DisposableBean;

import java.util.Map;
import java.util.concurrent.ConcurrentHashMap;

public class DefaultSingletonBeanRegistry {
    
    // ... 前面的字段
    
    /** 销毁方法缓存 */
    private final Map<String, DisposableBean> disposableBeans = new ConcurrentHashMap<>();
    
    /**
     * 注册销毁方法
     */
    public void registerDisposableBean(String beanName, DisposableBean bean) {
        disposableBeans.put(beanName, bean);
    }
    
    /**
     * 销毁所有单例 Bean
     */
    public void destroySingletons() {
        for (Map.Entry<String, DisposableBean> entry : disposableBeans.entrySet()) {
            try {
                entry.getValue().destroy();
            } catch (Exception e) {
                System.err.println("Failed to destroy bean: " + entry.getKey());
            }
        }
        disposableBeans.clear();
        singletonObjects.clear();
        earlySingletonObjects.clear();
    }
}
```

## 六、测试代码

```java
package com.example.minispring.test;

import com.example.minispring.beans.config.BeanDefinition;
import com.example.minispring.beans.factory.DisposableBean;
import com.example.minispring.beans.factory.InitializingBean;
import com.example.minispring.beans.factory.support.DefaultListableBeanFactory;

public class BeanLifecycleTest {
    
    public static void main(String[] args) {
        // 1. 创建 Bean 工厂
        DefaultListableBeanFactory factory = new DefaultListableBeanFactory();
        
        // 2. 注册 Bean 定义
        BeanDefinition bd = new BeanDefinition(LifecycleBean.class);
        bd.setInitMethodName("init");
        bd.setDestroyMethodName("destroy");
        factory.registerBeanDefinition("lifecycleBean", bd);
        
        // 3. 获取 Bean（触发初始化）
        LifecycleBean bean = (LifecycleBean) factory.getBean("lifecycleBean");
        
        // 4. 使用 Bean
        bean.doWork();
        
        // 5. 关闭工厂（触发销毁）
        factory.destroySingletons();
    }
}

class LifecycleBean implements InitializingBean, DisposableBean {
    
    public LifecycleBean() {
        System.out.println("1. 构造方法被调用");
    }
    
    @Override
    public void afterPropertiesSet() throws Exception {
        System.out.println("3. InitializingBean.afterPropertiesSet 被调用");
    }
    
    public void init() {
        System.out.println("4. 自定义 init 方法被调用");
    }
    
    public void doWork() {
        System.out.println("5. 执行业务逻辑");
    }
    
    @Override
    public void destroy() throws Exception {
        System.out.println("6. DisposableBean.destroy 被调用");
    }
    
    public void destroy() {
        System.out.println("7. 自定义 destroy 方法被调用");
    }
}
```

**输出**：
```
1. 构造方法被调用
2. 属性填充完成
3. InitializingBean.afterPropertiesSet 被调用
4. 自定义 init 方法被调用
5. 执行业务逻辑
6. DisposableBean.destroy 被调用
7. 自定义 destroy 方法被调用
```

## 七、Bean 生命周期

```
1. 实例化 Bean（构造方法）
   ↓
2. 属性填充（依赖注入）
   ↓
3. BeanPostProcessor.postProcessBeforeInitialization
   ↓
4. InitializingBean.afterPropertiesSet
   ↓
5. 自定义 init-method
   ↓
6. BeanPostProcessor.postProcessAfterInitialization
   ↓
7. Bean 就绪，可以使用
   ↓
8. 容器关闭时
   ↓
9. DisposableBean.destroy
   ↓
10. 自定义 destroy-method
```

## 八、小结

1. **InitializingBean**：初始化接口
2. **DisposableBean**：销毁接口
3. **init-method**：自定义初始化方法
4. **destroy-method**：自定义销毁方法
5. **生命周期**：实例化 → 属性填充 → 初始化 → 使用 → 销毁
