# 76.1 项目概述：CMS 系统是什么？

## 一、先问一个问题

**什么是 CMS 系统？**

### 生活例子：网站后台

```
新闻网站
├── 栏目管理（新闻分类）
├── 内容管理（发布新闻）
├── 模板管理（页面样式）
└── 用户管理（权限控制）
```

**CMS 系统**：
- 无需懂代码
- 可视化编辑
- 快速建站

## 二、功能模块

```
CMS 系统
├── 栏目管理
│   ├── 栏目创建
│   ├── 栏目排序
│   └── 栏目删除
├── 内容管理
│   ├── 文章发布
│   ├── 文章编辑
│   └── 文章审核
├── 模板管理
│   ├── 模板选择
│   └── 模板编辑
└── 系统管理
    ├── 用户管理
    └── 权限管理
```

## 三、技术栈

| 技术 | 作用 | 为什么选它 |
|------|------|------------|
| Spring Boot | 后端框架 | 快速开发 |
| Thymeleaf | 模板引擎 | 页面渲染 |
| FreeMarker | 静态化 | 页面生成 |
| MySQL | 数据库 | 数据存储 |

## 四、小结

| 方面 | 说明 |
|------|------|
| 项目定位 | 内容管理平台 |
| 核心功能 | 栏目、内容、模板 |
| 学习收获 | CMS 开发经验 |
