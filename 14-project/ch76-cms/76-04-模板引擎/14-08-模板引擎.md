# 76.4 模板引擎：如何渲染页面？

## 一、先问一个问题

**如何动态生成页面？**

### 生活例子：模板打印

```
模板
├── 固定内容（页面框架）
└── 可变内容（数据填充）
```

**模板引擎**：
- 定义模板
- 填充数据
- 生成页面

## 二、Thymeleaf 基础

### 引入依赖

```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-thymeleaf</artifactId>
</dependency>
```

### 模板示例

```html
<!DOCTYPE html>
<html xmlns:th="http://www.thymeleaf.org">
<head>
    <title th:text="${title}">标题</title>
</head>
<body>
    <h1 th:text="${article.title}">文章标题</h1>
    <div th:utext="${article.content}">文章内容</div>
    
    <!-- 遍历 -->
    <ul>
        <li th:each="item : ${items}" th:text="${item}">项</li>
    </ul>
    
    <!-- 条件 -->
    <div th:if="${user != null}">
        <span th:text="${user.name}">用户名</span>
    </div>
</body>
</html>
```

## 三、Controller 配置

```java
@Controller
@RequestMapping("/cms")
public class CmsController {
    
    @Autowired
    private ArticleService articleService;
    
    @GetMapping("/article/{id}")
    public String articleDetail(@PathVariable Long id, Model model) {
        Article article = articleService.getById(id);
        model.addAttribute("article", article);
        model.addAttribute("title", article.getTitle());
        
        return "article-detail";  // 模板名称
    }
}
```

## 四、小结

| 语法 | 作用 | 示例 |
|------|------|------|
| th:text | 文本输出 | th:text="${title}" |
| th:utext | HTML 输出 | th:utext="${content}" |
| th:each | 遍历 | th:each="item : ${items}" |
| th:if | 条件 | th:if="${user != null}" |

**核心要点**：
- 模板定义
- 数据绑定
- 页面渲染
