# 76.5 静态化：如何生成静态页面？

## 一、先问一个问题

**为什么要静态化？**

### 生活例子：印刷品

```
动态页面
└── 每次请求都生成 → 慢

静态页面
└── 提前生成好 → 快
```

**静态化优势**：
- 访问速度快
- 减轻服务器压力
- SEO 友好

## 二、FreeMarker 静态化

### 引入依赖

```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-freemarker</artifactId>
</dependency>
```

### 生成静态页面

```java
@Service
public class StaticService {
    
    @Autowired
    private Configuration freemarkerConfig;
    
    @Autowired
    private ArticleService articleService;
    
    public void generateArticlePage(Long articleId) {
        try {
            // 1. 获取文章数据
            Article article = articleService.getById(articleId);
            
            // 2. 获取模板
            Template template = freemarkerConfig.getTemplate("article.ftl");
            
            // 3. 准备数据
            Map<String, Object> data = new HashMap<>();
            data.put("article", article);
            data.put("title", article.getTitle());
            
            // 4. 生成静态文件
            File outputFile = new File("/var/www/html/article/" + articleId + ".html");
            FileWriter writer = new FileWriter(outputFile);
            
            template.process(data, writer);
            writer.close();
            
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
```

### 模板文件（article.ftl）

```html
<!DOCTYPE html>
<html>
<head>
    <title>${title}</title>
</head>
<body>
    <h1>${article.title}</h1>
    <div>${article.content}</div>
    <p>作者：${article.author}</p>
    <p>发布时间：${article.publishTime?string("yyyy-MM-dd HH:mm")}</p>
</body>
</html>
```

## 三、Nginx 配置

```nginx
server {
    listen 80;
    server_name cms.example.com;
    
    # 静态文件优先
    location / {
        root /var/www/html;
        try_files $uri $uri/ /index.html;
    }
    
    # 动态请求代理到后端
    location /admin/ {
        proxy_pass http://localhost:8080;
    }
}
```

## 四、小结

| 方案 | 优点 | 缺点 |
|------|------|------|
| 动态页面 | 实时 | 性能一般 |
| 静态页面 | 快速 | 更新麻烦 |
| 动静结合 | 平衡 | 配置复杂 |

**核心要点**：
- FreeMarker 模板
- 生成静态文件
- Nginx 配置
