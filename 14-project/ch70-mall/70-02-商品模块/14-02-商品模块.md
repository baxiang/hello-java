# 70.2 商品模块：SKU 和 SPU 是什么？

## 一、先问一个问题

**什么是 SPU 和 SKU？**

### 生活例子：买手机

```
SPU（标准化产品单元）
└── iPhone 15（这是一个 SPU）

SKU（库存量单位）
├── iPhone 15 黑色 128G
├── iPhone 15 黑色 256G
├── iPhone 15 白色 128G
└── iPhone 15 白色 256G
```

**区别**：
- SPU：一类商品（iPhone 15）
- SKU：具体规格（黑色 128G）

## 二、数据库设计

### 商品表（product）

```sql
CREATE TABLE `product` (
    `id` BIGINT PRIMARY KEY AUTO_INCREMENT,
    `name` VARCHAR(200) NOT NULL COMMENT '商品名称',
    `description` TEXT COMMENT '商品描述',
    `category_id` BIGINT COMMENT '分类 ID',
    `brand_id` BIGINT COMMENT '品牌 ID',
    `min_price` DECIMAL(10,2) COMMENT '最低价格（从 SKU 计算）',
    `status` TINYINT DEFAULT 1 COMMENT '状态（0 下架/1 上架）',
    `create_time` DATETIME DEFAULT CURRENT_TIMESTAMP,
    INDEX `idx_category` (`category_id`),
    INDEX `idx_status` (`status`)
) COMMENT='商品表（SPU）';
```

### SKU 表（sku）

```sql
CREATE TABLE `sku` (
    `id` BIGINT PRIMARY KEY AUTO_INCREMENT,
    `product_id` BIGINT NOT NULL COMMENT '商品 ID',
    `specs` VARCHAR(500) COMMENT '规格（JSON）',
    `price` DECIMAL(10,2) NOT NULL COMMENT '价格',
    `stock` INT DEFAULT 0 COMMENT '库存',
    `status` TINYINT DEFAULT 1 COMMENT '状态',
    INDEX `idx_product` (`product_id`)
) COMMENT='SKU 表';
```

### 规格示例

```json
// specs 字段内容
{
    "color": "黑色",
    "storage": "128G"
}
```

## 三、商品管理

### 发布商品

```java
@Service
public class ProductService {
    
    public Product publish(ProductDTO dto) {
        // 1. 创建 SPU
        Product product = new Product();
        product.setName(dto.getName());
        product.setDescription(dto.getDescription());
        product.setCategoryId(dto.getCategoryId());
        product.setStatus(1);
        productMapper.insert(product);
        
        // 2. 创建 SKU
        for (SkuDTO skuDto : dto.getSkus()) {
            Sku sku = new Sku();
            sku.setProductId(product.getId());
            sku.setSpecs(JSON.toJSONString(skuDto.getSpecs()));
            sku.setPrice(skuDto.getPrice());
            sku.setStock(skuDto.getStock());
            skuMapper.insert(sku);
        }
        
        // 3. 更新最低价格
        updateMinPrice(product.getId());
        
        return product;
    }
}
```

### 库存管理

```java
@Service
public class SkuService {
    
    // 扣减库存
    @Transactional
    public void decreaseStock(Long skuId, Integer quantity) {
        Sku sku = skuMapper.selectById(skuId);
        if (sku.getStock() < quantity) {
            throw new BusinessException("库存不足");
        }
        skuMapper.decreaseStock(skuId, quantity);
    }
    
    // 恢复库存
    @Transactional
    public void increaseStock(Long skuId, Integer quantity) {
        skuMapper.increaseStock(skuId, quantity);
    }
}
```

## 四、商品展示

### 商品列表

```java
@GetMapping("/products")
public Result<Page<ProductVO>> list(
    @RequestParam(required = false) Long categoryId,
    @RequestParam(defaultValue = "1") int page,
    @RequestParam(defaultValue = "10") int size
) {
    Page<ProductVO> result = productService.list(categoryId, page, size);
    return Result.success(result);
}
```

### 商品详情

```java
@GetMapping("/products/{id}")
public Result<ProductDetailVO> detail(@PathVariable Long id) {
    ProductDetailVO vo = new ProductDetailVO();
    
    // 1. 商品基本信息
    Product product = productMapper.selectById(id);
    BeanUtils.copyProperties(product, vo);
    
    // 2. 所有 SKU
    List<Sku> skus = skuMapper.selectByProductId(id);
    vo.setSkus(skus);
    
    // 3. 商品图片
    List<ProductImage> images = imageMapper.selectByProductId(id);
    vo.setImages(images);
    
    return Result.success(vo);
}
```

## 五、小结

| 概念 | 说明 | 示例 |
|------|------|------|
| SPU | 标准化产品单元 | iPhone 15 |
| SKU | 库存量单位 | iPhone 15 黑色 128G |
| 商品表 | 存储 SPU 信息 | 名称、描述 |
| SKU 表 | 存储规格库存 | 价格、库存 |

**核心要点**：
- SPU 和 SKU 分离
- 规格用 JSON 存储
- 价格从 SKU 计算
