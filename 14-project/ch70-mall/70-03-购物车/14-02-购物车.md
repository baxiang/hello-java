# 70.3 购物车：如何存储购物车数据？

## 一、先问一个问题

**购物车数据存在哪里？**

### 生活例子：超市购物篮

```
购物篮（购物车）
├── 临时存放（Redis）
├── 随时取放（高性能）
└── 结账后清空（订单生成）
```

**存储方案**：
- 未登录：LocalStorage
- 已登录：Redis

## 二、数据结构

### Redis Hash 结构

```
Key: cart:user:{userId}

Hash:
├── skuId1 → quantity1
├── skuId2 → quantity2
└── skuId3 → quantity3
```

### 示例

```
Key: cart:user:123

Hash:
├── 1001 → 2  （SKU 1001，数量 2）
├── 1002 → 1  （SKU 1002，数量 1）
└── 1003 → 3  （SKU 1003，数量 3）
```

## 三、购物车功能

### 添加商品

```java
@Service
public class CartService {
    
    @Autowired
    private RedisTemplate<String, Object> redisTemplate;
    
    public void addToCart(Long userId, Long skuId, Integer quantity) {
        String key = "cart:user:" + userId;
        
        // 1. 检查是否已存在
        Boolean hasKey = redisTemplate.opsForHash().hasKey(key, skuId.toString());
        
        if (Boolean.TRUE.equals(hasKey)) {
            // 2. 已存在，数量累加
            Integer oldQuantity = (Integer) redisTemplate.opsForHash().get(key, skuId.toString());
            redisTemplate.opsForHash().increment(key, skuId.toString(), quantity);
        } else {
            // 3. 不存在，新增
            redisTemplate.opsForHash().put(key, skuId.toString(), quantity);
        }
    }
}
```

### 查询购物车

```java
public List<CartVO> getCart(Long userId) {
    String key = "cart:user:" + userId;
    
    // 1. 从 Redis 获取
    Map<Object, Object> entries = redisTemplate.opsForHash().entries(key);
    
    // 2. 转换为 VO
    List<CartVO> list = new ArrayList<>();
    for (Map.Entry<Object, Object> entry : entries.entrySet()) {
        CartVO vo = new CartVO();
        vo.setSkuId(Long.parseLong(entry.getKey().toString()));
        vo.setQuantity((Integer) entry.getValue());
        
        // 3. 查询 SKU 信息
        Sku sku = skuMapper.selectById(vo.getSkuId());
        vo.setSku(sku);
        
        list.add(vo);
    }
    
    return list;
}
```

### 修改数量

```java
public void updateQuantity(Long userId, Long skuId, Integer quantity) {
    String key = "cart:user:" + userId;
    
    if (quantity <= 0) {
        // 数量为 0，删除
        redisTemplate.opsForHash().delete(key, skuId.toString());
    } else {
        // 更新数量
        redisTemplate.opsForHash().put(key, skuId.toString(), quantity);
    }
}
```

### 删除商品

```java
public void deleteFromCart(Long userId, Long skuId) {
    String key = "cart:user:" + userId;
    redisTemplate.opsForHash().delete(key, skuId.toString());
}
```

### 清空购物车

```java
public void clearCart(Long userId) {
    String key = "cart:user:" + userId;
    redisTemplate.delete(key);
}
```

## 四、登录合并

### 合并逻辑

```
未登录时
└── 购物车存在 LocalStorage

登录时
├── 读取 LocalStorage 购物车
├── 合并到 Redis 购物车
└── 清空 LocalStorage
```

### 代码实现

```java
public void mergeCart(Long userId, List<CartDTO> localCart) {
    for (CartDTO dto : localCart) {
        // 添加到 Redis 购物车
        addToCart(userId, dto.getSkuId(), dto.getQuantity());
    }
}
```

## 五、小结

| 操作 | Redis 命令 | 说明 |
|------|------------|------|
| 添加 | HSET/HINCRBY | 设置或累加 |
| 查询 | HGETALL | 获取全部 |
| 修改 | HSET | 更新数量 |
| 删除 | HDEL | 删除项 |
| 清空 | DEL | 删除 key |

**核心要点**：
- 使用 Hash 结构存储
- Key 设计：cart:user:{userId}
- 登录时合并购物车
