# 70.5 搜索模块：如何实现商品搜索？

## 一、先问一个问题

**为什么不用 MySQL 做搜索？**

### 生活例子：图书馆找书

```
MySQL（按目录查找）
└── 只能按固定字段查

Elasticsearch（全文检索）
├── 可以搜书名
├── 可以搜作者
├── 可以搜内容
└── 还能智能推荐
```

**Elasticsearch 优势**：
- 全文检索
- 高亮显示
- 智能分词
- 性能更好

## 二、Elasticsearch 基础

### 核心概念

| 概念 | MySQL 对应 | 说明 |
|------|------------|------|
| Index | 数据库 | 索引 |

| Field | 列 | 字段 |
| Type | 表 | 类型（7.x 后废弃） |

```json
    "description": "苹果手机，A17 芯片",
```

## 四、同步数据

### 全量同步

```java
@Service
public class ProductIndexService {
    
    @Autowired
    private RestHighLevelClient esClient;
    
    public void syncAll() {
        // 1. 查询所有商品
        List<Product> products = productMapper.selectAll();
        
        // 2. 批量写入 ES
        BulkRequest request = new BulkRequest("product");
        
        for (Product product : products) {
            ProductDoc doc = convertToDoc(product);
            request.add(new IndexRequest("product")
                .id(product.getId().toString())
                .source(JSON.toJSONString(doc), XContentType.JSON));
        }
        
        esClient.bulk(request, RequestOptions.DEFAULT);
    }
}
```

### 增量同步

```java
// 商品变更时同步
@Transactional
public void updateProduct(Product product) {
    // 1. 更新数据库
    productMapper.updateById(product);
    
    // 2. 同步到 ES
    ProductDoc doc = convertToDoc(product);
    UpdateRequest request = new UpdateRequest("product", product.getId().toString());
    request.doc(JSON.toJSONString(doc), XContentType.JSON);
    esClient.update(request, RequestOptions.DEFAULT);
}
```

## 五、搜索功能

### 基本搜索

```java
public Page<ProductDoc> search(String keyword, int page, int size) {
    SearchRequest request = new SearchRequest("product");
    
    SearchSourceBuilder sourceBuilder = new SearchSourceBuilder();
    
    // 1. 查询条件
    sourceBuilder.query(QueryBuilders.matchQuery("name", keyword));
    
    // 2. 分页
    sourceBuilder.from((page - 1) * size);
    sourceBuilder.size(size);
    
    // 3. 执行搜索
    request.source(sourceBuilder);
    SearchResponse response = esClient.search(request, RequestOptions.DEFAULT);
    
    // 4. 解析结果
    List<ProductDoc> list = parseResponse(response);
    long total = response.getHits().getTotalHits().value;
    
    return new Page<>(list, total, page, size);
}
```

### 高亮显示

```java
// 高亮配置
HighlightBuilder highlightBuilder = new HighlightBuilder();
highlightBuilder.field("name");
highlightBuilder.preTags("<em style='color:red'>");
highlightBuilder.postTags("</em>");
sourceBuilder.highlighter(highlightBuilder);

// 解析高亮结果
for (SearchHit hit : response.getHits().getHits()) {
    HighlightField field = hit.getHighlightFields().get("name");
    if (field != null) {
        product.setName(field.getFragments()[0].toString());
    }
}
```

### 筛选和排序

```java
// 筛选
sourceBuilder.postFilter(QueryBuilders.rangeQuery("price")
    .gte(minPrice)

// 排序
sourceBuilder.sort("sales", SortOrder.DESC);
```

### 聚合统计

```java
// 按品牌聚合
TermsAggregationBuilder brandAgg = AggregationBuilders.terms("brand_agg")
    .field("brand");
sourceBuilder.aggregation(brandAgg);

// 价格区间聚合
RangeAggregationBuilder priceAgg = AggregationBuilders.range("price_agg")
    .field("price")
    .addRange(0, 1000)
    .addRange(1000, 5000)
    .addRange(5000, 10000);
sourceBuilder.aggregation(priceAgg);
```

## 六、小结

| 功能 | ES Query | 说明 |
|------|----------|------|
| 全文搜索 | match_query | 分词匹配 |
| 精确匹配 | term_query | 不分词 |
| 范围查询 | range_query | 价格区间 |
| 高亮显示 | highlight | 关键词高亮 |
| 聚合统计 | aggregations | 品牌/价格统计 |

**核心要点**：
- 使用 ES 做全文搜索
- MySQL 做数据存储
- 数据同步保持同步
