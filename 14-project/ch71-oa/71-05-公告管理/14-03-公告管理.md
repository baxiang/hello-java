## 二、数据库设计

### 公告表

```sql
CREATE TABLE `notice` (
    `id` BIGINT PRIMARY KEY AUTO_INCREMENT,
    `title` VARCHAR(200) NOT NULL COMMENT '标题',
    `content` TEXT NOT NULL COMMENT '内容',
    `type` TINYINT DEFAULT 0 COMMENT '类型',
    `priority` TINYINT DEFAULT 0 COMMENT '优先级',
    `publisher_id` BIGINT NOT NULL COMMENT '发布人',
    `status` TINYINT DEFAULT 1 COMMENT '状态',
    `publish_time` DATETIME COMMENT '发布时间',
    `create_time` DATETIME DEFAULT CURRENT_TIMESTAMP,
    INDEX `idx_type` (`type`),
    INDEX `idx_status` (`status`)
) COMMENT='公告表';

-- 类型：0 普通/1 行政/2 人事/3 财务
-- 优先级：0 普通/1 重要/2 紧急
-- 状态：0 草稿/1 已发布/2 已下架
```

### 公告阅读记录

```sql
CREATE TABLE `notice_read` (
    `id` BIGINT PRIMARY KEY AUTO_INCREMENT,
    `notice_id` BIGINT NOT NULL,
    `user_id` BIGINT NOT NULL,
    `read_time` DATETIME DEFAULT CURRENT_TIMESTAMP,
    UNIQUE KEY `uk_notice_user` (`notice_id`, `user_id`)
) COMMENT='公告阅读记录';
```

## 三、发布公告

### 发布公告

```java
@Service
public class NoticeService {
    
    @Autowired
    private NoticeMapper noticeMapper;
    
    @Transactional
    public Notice publishNotice(NoticeDTO dto, Long publisherId) {
        Notice notice = new Notice();
        notice.setTitle(dto.getTitle());
        notice.setContent(dto.getContent());
        notice.setType(dto.getType());
        notice.setPriority(dto.getPriority());
        notice.setPublisherId(publisherId);
        notice.setStatus(1);  // 已发布
        notice.setPublishTime(LocalDateTime.now());
        
        noticeMapper.insert(notice);
        
        // 通知所有人
        notifyAllUsers(notice);
        
        return notice;
    }
}
```

### 草稿箱

```java
public Notice saveDraft(NoticeDTO dto, Long publisherId) {
    Notice notice = new Notice();
    notice.setTitle(dto.getTitle());
    notice.setContent(dto.getContent());
    notice.setType(dto.getType());
    notice.setPublisherId(publisherId);
    notice.setStatus(0);  // 草稿
    
    noticeMapper.insert(notice);
    return notice;
}
```

## 四、查看公告

### 公告列表

```java
public Page<NoticeVO> listNotices(Integer type, int page, int size) {
    Page<Notice> noticePage = new Page<>(page, size);
    
    LambdaQueryWrapper<Notice> wrapper = new LambdaQueryWrapper<>();
    wrapper.eq(Notice::getStatus, 1);  // 只查询已发布
    
    if (type != null) {
        wrapper.eq(Notice::getType, type);
    }
    
    wrapper.orderByDesc(Notice::getPriority)
           .orderByDesc(Notice::getPublishTime);
    
    IPage<Notice> result = noticeMapper.selectPage(noticePage, wrapper);
    
    return convertToVOPage(result);
}
```

### 公告详情

```java
@Transactional
public NoticeVO getNoticeDetail(Long id, Long userId) {
    // 1. 查询公告
    Notice notice = noticeMapper.selectById(id);
    if (notice == null) {
        throw new BusinessException("公告不存在");
    }
    
    // 2. 记录阅读
    NoticeRead read = new NoticeRead();
    read.setNoticeId(id);
    read.setUserId(userId);
    noticeReadMapper.insertIgnore(read);  // 忽略重复
    
    // 3. 转换为 VO
    NoticeVO vo = new NoticeVO();
    BeanUtils.copyProperties(notice, vo);
    
    // 4. 发布人信息
    User publisher = userMapper.selectById(notice.getPublisherId());
    vo.setPublisherName(publisher.getRealName());
    
    // 5. 阅读统计
    vo.setReadCount(noticeReadMapper.countByNoticeId(id));
    
    return vo;
}
```

## 五、阅读统计

### 统计阅读情况

```java
public NoticeStats getNoticeStats(Long noticeId) {
    // 总人数
    int totalUsers = userMapper.count();
    
    // 已读人数
    int readCount = noticeReadMapper.countByNoticeId(noticeId);
    
    // 未读人数
    int unreadCount = totalUsers - readCount;
    
    // 阅读率
    double readRate = totalUsers > 0 ? 
        (double) readCount / totalUsers * 100 : 0;
    
    NoticeStats stats = new NoticeStats();
    stats.setNoticeId(noticeId);
    stats.setTotalUsers(totalUsers);
    stats.setReadCount(readCount);
    stats.setUnreadCount(unreadCount);
    stats.setReadRate(readRate);
    
    return stats;
}
```

public List<UserVO> getUnreadUsers(Long noticeId) {

## 六、小结

| 功能 | 关键点 | 技术 |
|------|--------|------|
| 发布公告 | 状态管理 | 事务 |
| 查看公告 | 阅读记录 | 唯一索引 |
| 阅读统计 | 统计查询 | 聚合函数 |

**核心要点**：
- 公告状态管理（草稿/发布/下架）
- 阅读记录防重复
- 阅读率统计
