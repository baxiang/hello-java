# 71.3 考勤模块：如何实现打卡和请假？

## 一、先问一个问题

**考勤系统的核心是什么？**

### 生活例子：公司打卡

```
上班打卡（签到）
  ↓
下班打卡（签退）
  ↓
统计出勤（迟到/早退）
  ↓
请假扣减（请假天数）
```

**考勤核心**：
- 记录上下班时间
- 统计出勤情况
- 处理请假申请

## 三、打卡功能

### 上班打卡

```java
@Service
public class AttendanceService {
    
    public void clockIn(Long userId) {
        LocalDate today = LocalDate.now();
        LocalDateTime now = LocalDateTime.now();
        
        // 查询今日考勤记录
        Attendance attendance = attendanceMapper.selectByUserIdAndDate(userId, today);
        
        if (attendance != null) {
            throw new BusinessException("今日已打卡");
        }
        
        // 创建考勤记录
        attendance = new Attendance();
        attendance.setUserId(userId);
        attendance.setDate(today);
        attendance.setClockIn(now);
        
        // 判断是否迟到（9:30 后）
        if (now.toLocalTime().isAfter(LocalTime.of(9, 30))) {
            attendance.setStatus(1);  // 迟到
        } else {
            attendance.setStatus(0);  // 正常
        }
        
        attendanceMapper.insert(attendance);
    }
}
```

### 下班打卡

```java
public void clockOut(Long userId) {
    LocalDate today = LocalDate.now();
    LocalDateTime now = LocalDateTime.now();
    
    Attendance attendance = attendanceMapper.selectByUserIdAndDate(userId, today);
    
    if (attendance == null) {
        throw new BusinessException("请先上班打卡");
    }
    
    if (attendance.getClockOut() != null) {
        throw new BusinessException("今日已下班打卡");
    }
    
    attendance.setClockOut(now);
    
    // 判断是否早退（18:00 前）
    if (now.toLocalTime().isBefore(LocalTime.of(18, 0))) {
        if (attendance.getStatus() == 0) {
            attendance.setStatus(2);  // 早退
        }
    }
    
    attendanceMapper.updateById(attendance);
}
```

## 四、请假功能

### 提交请假

```java
@Service
public class LeaveService {
    
    @Autowired
    private AttendanceMapper attendanceMapper;
    
    @Transactional
    public void applyLeave(LeaveRequest request) {
        // 1. 计算请假天数
        BigDecimal days = calculateDays(request.getStartTime(), request.getEndTime());
        request.setDays(days);
        
        // 2. 创建请假申请
        request.setStatus(0);  // 待审批
        leaveMapper.insert(request);
        
        // 3. 通知审批人
        notifyApprover(request);
    }
    
    private BigDecimal calculateDays(LocalDateTime start, LocalDateTime end) {
        long millis = end.toInstant().toEpochMilli() - start.toInstant().toEpochMilli();
        return new BigDecimal(millis).divide(
            new BigDecimal(86400000), 2, RoundingMode.HALF_UP
        );
    }
}
```

### 审批请假

```java
@Transactional
public void approveLeave(Long requestId, boolean approved, Long approverId) {
    LeaveRequest request = leaveMapper.selectById(requestId);
    
    if (request == null) {
        throw new BusinessException("请假申请不存在");
    }
    
    if (request.getStatus() != 0) {
        throw new BusinessException("申请已处理");
    }
    
    // 更新状态
    request.setStatus(approved ? 1 : 2);
    request.setApproverId(approverId);
    request.setApproveTime(LocalDateTime.now());
    leaveMapper.updateById(request);
    
    // 如果通过，更新考勤记录
    if (approved) {
        updateAttendance(request);
    }
}

private void updateAttendance(LeaveRequest request) {
    // 在请假期间创建考勤记录（标记为请假）
    LocalDate start = request.getStartTime().toLocalDate();
    LocalDate end = request.getEndTime().toLocalDate();
    
    for (LocalDate date = start; !date.isAfter(end); date = date.plusDays(1)) {
        Attendance attendance = new Attendance();
        attendance.setUserId(request.getUserId());
        attendance.setDate(date);
        attendance.setStatus(4);  // 请假
        attendanceMapper.insert(attendance);
    }
}
```

## 五、考勤统计

### 月度统计

```java
public AttendanceStats getMonthlyStats(Long userId, YearMonth yearMonth) {
    List<Attendance> attendances = attendanceMapper.selectByUserIdAndMonth(
        userId, yearMonth
    );
    
    AttendanceStats stats = new AttendanceStats();
    stats.setUserId(userId);
    stats.setYearMonth(yearMonth);
    
    int normalDays = 0;
    int lateDays = 0;
    int earlyLeaveDays = 0;
    int absentDays = 0;
    int leaveDays = 0;
    
    for (Attendance attendance : attendances) {
        switch (attendance.getStatus()) {
            case 0: normalDays++; break;
            case 1: lateDays++; break;
            case 2: earlyLeaveDays++; break;
            case 3: absentDays++; break;
            case 4: leaveDays++; break;
        }
    }
    
    stats.setNormalDays(normalDays);
    stats.setLateDays(lateDays);
    stats.setEarlyLeaveDays(earlyLeaveDays);
    stats.setAbsentDays(absentDays);
    stats.setLeaveDays(leaveDays);
    
    return stats;
}
```

## 六、小结

| 功能 | 关键点 | 技术 |
|------|--------|------|
| 打卡 | 唯一索引防重复 | 时间比较 |
| 请假 | 计算天数、审批流程 | 事务 |
| 统计 | 按状态分类统计 | 聚合查询 |

**核心要点**：
- 每日唯一考勤记录
- 迟到早退自动判断
- 请假审批流程
