# 71.6 数据权限：如何实现数据隔离？

## 一、先问一个问题

**什么是数据权限？**

### 生活例子：公司文件柜

```
文件柜
├── 公共文件（所有人可看）
├── 部门文件（本部门可看）
└── 机密文件（仅管理层可看）
```

**数据权限**：
- 谁能看哪些数据
- 谁能改哪些数据
- 数据范围控制

## 二、数据权限模型

### 权限级别

```
数据权限级别
├── 全部数据（管理员）
├── 本部门及下级（部门经理）
├── 本部门（部门主管）
└── 仅本人（普通员工）
```

### 数据库设计

```sql
-- 部门表
CREATE TABLE `department` (
    `id` BIGINT PRIMARY KEY AUTO_INCREMENT,
    `name` VARCHAR(100) NOT NULL,
    `parent_id` BIGINT DEFAULT 0 COMMENT '父部门',
    `level` INT DEFAULT 0 COMMENT '层级',
    `ancestors` VARCHAR(500) COMMENT '祖先路径'
) COMMENT='部门表';

-- 角色数据权限
CREATE TABLE `role_data_scope` (
    `role_id` BIGINT NOT NULL,
    `scope_type` TINYINT NOT NULL COMMENT '范围类型',
    `dept_ids` VARCHAR(500) COMMENT '自定义部门'
) COMMENT='角色数据权限';

-- 范围类型：1 全部/2 本部门及下级/3 本部门/4 仅本人/5 自定义
```

## 三、实现方式

### MyBatis 拦截器

```java
@Intercepts({
    @Signature(
        type = Executor.class,
        method = "query",
        args = {MappedStatement.class, Object.class, RowBounds.class, ResultHandler.class}
    )
})
@Component
public class DataScopeInterceptor implements Interceptor {
    
    @Override
    public Object intercept(Invocation invocation) throws Throwable {
        MappedStatement ms = (MappedStatement) invocation.getArgs()[0];
        Object parameter = invocation.getArgs()[1];
        
        // 获取数据权限注解
        DataScope dataScope = getDataScopeAnnotation(ms);
        if (dataScope != null) {
            // 获取当前用户
            User user = getCurrentUser();
            
            // 生成数据权限 SQL
            String dataScopeSql = getDataScopeSql(user, dataScope);
            
            // 追加到 SQL
            BoundSql boundSql = ms.getBoundSql(parameter);
            String originalSql = boundSql.getSql();
            String newSql = originalSql + " " + dataScopeSql;
            
            // 替换 SQL
            replaceSql(ms, newSql);
        }
        
        return invocation.proceed();
    }
}
```

### 数据权限注解

```java
@Target(ElementType.METHOD)
@Retention(RetentionPolicy.RUNTIME)
public @interface DataScope {
    
    /**
     * 部门别名
     */
    String deptAlias() default "dept";
    
    /**
     * 用户别名
     */
    String userAlias() default "user";
}
```

### 使用示例

```java
@Mapper
public interface UserMapper {
    
    @DataScope(deptAlias = "d", userAlias = "u")
    List<User> selectUserList(User user);
}

// XML
/*
SELECT u.*
FROM user u
LEFT JOIN department d ON u.dept_id = d.id
WHERE u.status = 1
${dataScope}  动态追加数据权限 SQL
*/
```

## 四、数据权限 SQL

### 生成数据权限 SQL

```java
private String getDataScopeSql(User user, DataScope annotation) {
    // 获取用户角色
    List<Role> roles = roleMapper.selectByUserId(user.getId());
    
    // 获取最大权限
    int maxScope = roles.stream()
        .mapToInt(role -> role.getDataScope())
        .max()
        .orElse(4);  // 默认仅本人
    
    switch (maxScope) {
        case 1:  // 全部数据
            return "";
        
        case 2:  // 本部门及下级
            List<Long> deptIds = getDeptTree(user.getDeptId());
            return " AND d.id IN (" + String.join(",", deptIds.toString()) + ")";
        
        case 3:  // 本部门
            return " AND d.id = " + user.getDeptId();
        
        case 4:  // 仅本人
            return " AND u.id = " + user.getId();
        
        case 5:  // 自定义
            List<Long> customDeptIds = getCustomDeptIds(user);
            return " AND d.id IN (" + String.join(",", customDeptIds.toString()) + ")";
        
        default:
            return " AND 1=0";  // 无权限
    }
}
```

### 获取部门树

```java
public List<Long> getDeptTree(Long deptId) {
    List<Long> ids = new ArrayList<>();
    ids.add(deptId);
    
    // 查询子部门
    List<Department> children = deptMapper.selectByParentId(deptId);
    for (Department child : children) {
        ids.addAll(getDeptTree(child.getId()));
    }
    
    return ids;
}
```

## 五、行级权限

### 基于注解的行级权限

```java
@Target(ElementType.METHOD)
@Retention(RetentionPolicy.RUNTIME)
public @interface RowPermission {
    
    /**
     * 权限表达式
     */
    String value();
}

// 使用
@RowPermission("#user.deptId == target.deptId")
public User getUser(Long id);
```

### 实现

```java
@Component
public class RowPermissionAspect {
    
    @Around("@annotation(rowPermission)")
    public Object around(ProceedingJoinPoint pjp, RowPermission rowPermission) {
        // 执行目标方法
        Object result = pjp.proceed();
        
        // 验证权限
        String expression = rowPermission.value();
        if (!evaluateExpression(expression, result)) {
            throw new AccessDeniedException("无权限访问");
        }
        
        return result;
    }
}
```

## 六、小结

| 权限级别 | 说明 | 适用角色 |
|----------|------|----------|
| 全部数据 | 所有数据 | 管理员 |
| 本部门及下级 | 本部门 + 子部门 | 部门经理 |
| 本部门 | 仅本部门 | 部门主管 |
| 仅本人 | 自己的数据 | 普通员工 |
| 自定义 | 指定部门 | 特殊角色 |

**核心要点**：
- MyBatis 拦截器动态追加 SQL
- 角色数据权限配置
- 部门树查询
