# 71.2 权限模块：RBAC 模型如何实现？

## 一、先问一个问题

**权限管理的本质是什么？**

### 生活例子：公司门禁

```
员工卡（用户）
  ↓
门禁权限（角色）
├── 可以进办公室
├── 可以进会议室
└── 不能进财务室
```

**权限管理**：
- 谁能做什么
- 谁不能做什么
- 谁能看什么

## 二、数据库设计

### 用户表

```sql
CREATE TABLE `user` (
    `id` BIGINT PRIMARY KEY AUTO_INCREMENT,
    `username` VARCHAR(50) NOT NULL UNIQUE,
    `password` VARCHAR(100) NOT NULL,
    `real_name` VARCHAR(50),
    `dept_id` BIGINT COMMENT '部门 ID',
    `status` TINYINT DEFAULT 1,
    `create_time` DATETIME DEFAULT CURRENT_TIMESTAMP
) COMMENT='用户表';
```

### 角色表

```sql
CREATE TABLE `role` (
    `id` BIGINT PRIMARY KEY AUTO_INCREMENT,
    `name` VARCHAR(50) NOT NULL,
    `code` VARCHAR(50) NOT NULL UNIQUE COMMENT '角色编码',
    `description` VARCHAR(200),
    `status` TINYINT DEFAULT 1
) COMMENT='角色表';

-- 预置角色
INSERT INTO role (name, code) VALUES
('超级管理员', 'ADMIN'),
('部门经理', 'MANAGER'),
('普通员工', 'EMPLOYEE');
```

### 权限表

```sql
CREATE TABLE `permission` (
    `id` BIGINT PRIMARY KEY AUTO_INCREMENT,
    `name` VARCHAR(50) NOT NULL,
    `code` VARCHAR(100) NOT NULL UNIQUE COMMENT '权限编码',
    `type` TINYINT COMMENT '类型（1 菜单/2 按钮）',
    `parent_id` BIGINT DEFAULT 0,
    `path` VARCHAR(200) COMMENT '菜单路径'
) COMMENT='权限表';

-- 预置权限
INSERT INTO permission (name, code, type) VALUES
('用户管理', 'user:manage', 1),
('用户查看', 'user:view', 2),
('用户新增', 'user:add', 2),
('审批流程', 'approval:manage', 1);
```

### 关联表

```sql
-- 用户角色关联
CREATE TABLE `user_role` (
    `user_id` BIGINT NOT NULL,
    `role_id` BIGINT NOT NULL,
    PRIMARY KEY (`user_id`, `role_id`)
);

-- 角色权限关联
CREATE TABLE `role_permission` (
    `role_id` BIGINT NOT NULL,
    `permission_id` BIGINT NOT NULL,
    PRIMARY KEY (`role_id`, `permission_id`)
);
```

## 三、权限验证

### Spring Security 配置

```java
@Configuration
@EnableWebSecurity
public class SecurityConfig {
    
    @Bean
    public SecurityFilterChain filterChain(HttpSecurity http) throws Exception {
        http
            .csrf().disable()
            .authorizeHttpRequests(auth -> auth
                .requestMatchers("/api/auth/**").permitAll()
                .requestMatchers("/api/admin/**").hasRole("ADMIN")
                .anyRequest().authenticated()
            )
            .sessionManagement(session -> 
                session.sessionCreationPolicy(SessionCreationPolicy.STATELESS)
            );
        
        return http.build();
    }
}
```

### 权限注解

```java
@RestController
@RequestMapping("/api/users")
public class UserController {
    
    // 需要用户查看权限
    @PreAuthorize("hasAuthority('user:view')")
    @GetMapping
    public List<User> list() {
        return userService.list();
    }
    
    // 需要用户新增权限
    @PreAuthorize("hasAuthority('user:add')")
    @PostMapping
    public User create(@RequestBody User user) {
        return userService.create(user);
    }
}
```

## 四、JWT Token

### 生成 Token

```java
@Service
public class JwtUtil {
    
    private String secret = "my-secret-key";
    private long expiration = 7 * 24 * 60 * 60 * 1000;
    
    public String generateToken(User user) {
        return Jwts.builder()
            .setSubject(String.valueOf(user.getId()))
            .claim("username", user.getUsername())
            .claim("roles", user.getRoles())  // 角色列表
            .setIssuedAt(new Date())
            .setExpiration(new Date(System.currentTimeMillis() + expiration))
            .signWith(SignatureAlgorithm.HS256, secret)
            .compact();
    }
}
```

### 解析 Token

```java
public Claims parseToken(String token) {
    return Jwts.parser()
        .setSigningKey(secret)
        .parseClaimsJws(token)
        .getBody();
}

// 获取用户 ID
public Long getUserId(String token) {
    Claims claims = parseToken(token);
    return Long.parseLong(claims.getSubject());
}

// 获取角色列表
public List<String> getRoles(String token) {
    Claims claims = parseToken(token);
    return claims.get("roles", List.class);
}
```

## 五、动态权限

### 从数据库加载权限

```java
@Service
public class UserService {
    
    public Set<String> getPermissions(Long userId) {
        // 查询用户角色
        List<Role> roles = roleMapper.selectByUserId(userId);
        
        // 查询角色权限
        Set<String> permissions = new HashSet<>();
        for (Role role : roles) {
            List<Permission> perms = permissionMapper.selectByRoleId(role.getId());
            for (Permission perm : perms) {
                permissions.add(perm.getCode());
            }
        }
        
        return permissions;
    }
}
```

### 权限拦截器

```java
@Component
public class PermissionInterceptor implements HandlerInterceptor {
    
    @Autowired
    private PermissionService permissionService;
    
    @Override
    public boolean preHandle(HttpServletRequest request, 
                           HttpServletResponse response, 
                           Object handler) {
        // 获取当前用户
        Long userId = getCurrentUserId(request);
        
        // 获取请求路径
        String path = request.getRequestURI();
        
        // 验证权限
        if (!permissionService.hasPermission(userId, path)) {
            response.setStatus(403);
            return false;
        }
        
        return true;
    }
}
```

## 六、小结

| 表名 | 作用 | 核心字段 |
|------|------|----------|
| user | 用户 | username, password |
| role | 角色 | name, code |
| permission | 权限 | name, code, type |
| user_role | 用户角色关联 | user_id, role_id |
| role_permission | 角色权限关联 | role_id, permission_id |

**RBAC 核心**：
- 用户分配角色
- 角色分配权限
- 用户继承角色权限
