# 多线程与并发

本模块深入学习 Java 多线程与并发编程，从线程基础到线程安全、线程通信、线程池，再到并发编程进阶知识，是构建高并发应用的核心技能。

## 文件列表

| 文件 | 主题 |
|------|------|
| 01-线程基础.md | 线程基础：进程与线程概念、线程创建方式、生命周期、常用方法、线程调度 |
| 02-线程安全.md | 线程安全：线程安全问题、synchronized 关键字、同步代码块/方法、Lock 接口、volatile 关键字 |
| 03-线程通信.md | 线程通信：wait()/notify()、生产者-消费者模式、线程间数据传递 |
| 04-线程池.md | 线程池：线程池概述、Executors 工厂类、ThreadPoolExecutor 详解、参数配置、自定义线程池 |
| 05-并发工具类.md | 并发工具类：CountDownLatch、CyclicBarrier、Semaphore |
| 06-并发编程进阶.md | 并发编程进阶：Callable/Future、ThreadLocal、并发编程模式 |
| 07-原子类.md | 原子类：AtomicInteger、CAS 原理、原子引用、原子累加器 |
| 08-并发集合.md | 并发集合：ConcurrentHashMap、CopyOnWriteArrayList、BlockingQueue |
| 09-CompletableFuture.md | CompletableFuture：异步编程、组合操作、异常处理 |

## 学习目标

- 理解线程的概念与创建方式
- 掌握线程安全的解决方案（synchronized、Lock、volatile）
- 理解线程通信机制与生产者-消费者模式
- 掌握线程池的配置与使用
- 了解并发工具类、原子类、ThreadLocal 等进阶知识
