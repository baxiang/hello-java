# Java 8+ 新特性

本模块学习 Java 8 及后续版本的重要新特性，包括 Lambda 表达式、Stream API、Optional 类、各版本新特性、模块化系统与 GC 改进，以及虚拟线程与结构化并发等进阶内容，帮助你编写更简洁高效的代码。

## 章节列表

| 目录 | 主题 |
|------|------|
| 01-lambda-stream | Lambda 与 Stream：Lambda 表达式、函数式接口、方法引用、Stream API、Optional 类 |
| 02-other-features | 其他新特性：接口默认方法、重复注解、Java 21-25 新特性概览 |
| 03-java-versions | Java 版本特性：Java 11 新特性（var、HTTP Client、ZGC）、Java 17 新特性（sealed classes、record、pattern matching）、Java 21 新特性（虚拟线程、pattern matching for switch、record patterns） |
| 04-date-module-gc | 日期时间、模块化与 GC：Java 8 新日期 API 进阶、Java 9 模块系统、各版本垃圾回收器演进 |
| 05-advanced | 进阶特性：新特性最佳实践、Virtual Threads（虚拟线程原理与性能对比）、模式匹配与 Record 类、结构化并发（StructuredTaskScope） |

## 学习目标

- 掌握 Lambda 表达式的语法与函数式接口
- 熟练使用 Stream API 进行集合数据处理
- 理解 Optional 类避免空指针异常
- 了解 Java 8 至 Java 25 的重要新特性
- 理解模块化系统与 GC 演进
- 能够在新项目中选择合适的新特性提升开发效率
