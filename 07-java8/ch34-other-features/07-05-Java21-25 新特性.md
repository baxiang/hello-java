# 7.5 Java 21/25 新特性

## 一、Java 21 新特性（2023 年 LTS）

### 1.1 虚拟线程（Virtual Threads）

虚拟线程是 Java 21 最重要的特性，提供轻量级线程模型。

```java
// 创建虚拟线程
Thread.startVirtualThread(() -> {
    System.out.println("虚拟线程：" + Thread.currentThread().getName());
});

// 使用执行器
try (var executor = Executors.newVirtualThreadPerTaskExecutor()) {
    for (int i = 0; i < 10000; i++) {
        executor.submit(() -> {
            // 处理请求
            return "OK";
        });
    }
}

// 虚拟线程 vs 平台线程
Thread vThread = Thread.ofVirtual().start(() -> {
    System.out.println("虚拟线程");
});

Thread pThread = Thread.ofPlatform().start(() -> {
    System.out.println("平台线程");
});
```

**优势**：
- 内存占用小（约 1KB vs 1MB）
- 可创建百万级线程
- 适合 IO 密集型任务
- 无需线程池

### 1.2 Record 模式匹配（Pattern Matching for Record）

```java
// 定义 Record
public record Point(int x, int y) {}
public record Circle(Point center, int radius) {}
public record Rectangle(Point topLeft, Point bottomRight) {}

// 模式匹配
public String format(Shape shape) {
    return switch (shape) {
        case Circle(Point(int x, int y), int r) -> 
            String.format("圆：中心 (%d,%d), 半径 %d", x, y, r);
        case Rectangle(Point(int x1, int y1), Point(int x2, int y2)) -> 
            String.format("矩形：(%d,%d) 到 (%d,%d)", x1, y1, x2, y2);
        default -> "未知形状";
    };
}
```

### 1.3 Switch 模式匹配

```java
// Java 21 的 switch 表达式
public String getType(Object obj) {
    return switch (obj) {
        case Integer i -> "整数：" + i;
        case String s when s.length() > 5 -> "长字符串：" + s;
        case String s -> "短字符串：" + s;
        case Point(int x, int y) -> "点：" + x + "," + y;
        case null -> "空值";
        default -> "其他类型";
    };
}
```

### 1.4 无序代码块（Unnamed Variables）

```java
// 忽略 lambda 参数
list.forEach(_ -> System.out.println("Hello"));

// 忽略 catch 参数
try {
    riskyOperation();
} catch (_) {
    System.out.println("发生异常");
}

// 忽略 record 组件
record User(String name, String email, String password) {}
User user = new User("张三", "zhang@example.com", "secret");
String name = user.name();  // 只使用需要的字段
```

### 1.5 字符串模板（String Templates）

```java
// 字符串模板（Java 21 预览，Java 23 正式）
String name = "张三";
int age = 25;

// 使用 STR 处理器
String info = STR."我叫\{name}，今年\{age}岁";

// 使用多行模板
String json = STR."""
    {
        "name": "\{name}",
        "age": \{age}
    }
    """;

// 自定义处理器
String encoded = HTML."Welcome, \{name}!";  // HTML 转义
```

### 4.2 替代方案

```java
// ✅ 使用 Cleaner 替代 finalize
public class Resource implements AutoCloseable {
    private static final Cleaner cleaner = Cleaner.create();
    private final Cleaner.Cleanable cleanable;
    
    public Resource() {
        cleanable = cleaner.register(this, () -> {
            // 清理资源
        });
    }
    
    @Override
    public void close() {
        cleanable.clean();
    }
}

// ✅ 使用 try-with-resources
try (var resource = new Resource()) {
    // 使用资源
}
```

## 五、综合示例

### 5.1 虚拟线程 Web 服务器

```java
public class VirtualThreadServer {
    
    public static void main(String[] args) throws Exception {
        try (var executor = Executors.newVirtualThreadPerTaskExecutor();
             var serverSocket = new ServerSocket(8080)) {
            
            System.out.println("服务器启动在 8080 端口");
            
            while (true) {
                Socket client = serverSocket.accept();
                executor.submit(() -> handleClient(client));
            }
        }
    }
    
    private static void handleClient(Socket client) {
        try (client) {
            var in = client.getInputStream();
            var out = client.getOutputStream();
            
            // 处理请求
            String response = "HTTP/1.1 200 OK\r\n\r\nHello";
            out.write(response.getBytes());
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
```

### 5.2 Record 和模式匹配

```java
// 定义领域模型
public record Customer(String id, String name, String email) {}
public record Order(String id, Customer customer, BigDecimal amount) {}
public record ShippingOrder(Order order, String address) {}

// 处理订单
public String processOrder(OrderInfo info) {
    return switch (info) {
        case Order(String id, Customer c, BigDecimal amount) 
            when amount.compareTo(new BigDecimal("1000")) > 0 ->
            "大额订单：" + id + ", 客户：" + c.name();
            
        case ShippingOrder(Order o, String addr) ->
            "发货订单：" + o.id() + ", 地址：" + addr;
            
        case null -> "空订单";
        default -> "普通订单";
    };
}
```
