# 6.3 参数传递

## 一、参数传递概述

Java 中的参数传递只有一种方式：**值传递**。

- **基本数据类型**：传递的是值
- **引用数据类型**：传递的是地址值（引用）

## 二、基本数据类型参数传递

### 2.1 值传递示例

```java
public class PrimitivePass {
    public static void main(String[] args) {
        int x = 10;
        System.out.println("调用前：x = " + x);  // 10
        
        change(x);
        
        System.out.println("调用后：x = " + x);  // 10，不变
    }
    
    public static void change(int n) {
        n = 100;
        System.out.println("方法内：n = " + n);  // 100
    }
}
```

**执行过程**：
```
main 方法：x = 10
    ↓ 调用 change(x)
    ↓ 复制 x 的值给 n
change 方法：n = 10
    ↓ 修改 n
change 方法：n = 100
    ↓ 返回
main 方法：x = 10（不变）
```

### 2.2 多个参数

```java
public class SwapDemo {
    public static void main(String[] args) {
        int a = 10;
        int b = 20;
        
        System.out.println("交换前：a = " + a + ", b = " + b);
        
        swap(a, b);
        
        System.out.println("交换后：a = " + a + ", b = " + b);  // 不变
    }
    
    public static void swap(int x, int y) {
        int temp = x;
        x = y;
        y = temp;
        System.out.println("方法内：x = " + x + ", y = " + y);
    }
}
```

## 三、引用数据类型参数传递

### 3.1 数组参数

```java
public class ArrayPass {
    public static void main(String[] args) {
        int[] arr = {1, 2, 3};
        
        System.out.println("修改前：" + java.util.Arrays.toString(arr));
        
        changeArray(arr);
        
        System.out.println("修改后：" + java.util.Arrays.toString(arr));  // 变了
    }
    
    public static void changeArray(int[] arr) {
        arr[0] = 100;
        System.out.println("方法内：" + java.util.Arrays.toString(arr));
    }
}
```

**执行过程**：
```
main 方法：arr → [1, 2, 3]（堆内存地址 0x123）
    ↓ 调用 changeArray(arr)
    ↓ 复制地址值给参数 arr
changeArray 方法：arr → 0x123
    ↓ 修改 arr[0]
堆内存：[100, 2, 3]
    ↓ 返回
main 方法：arr → 0x123，看到修改后的数组
```

### 3.2 数组引用修改

```java
public class ArrayRefPass {
    public static void main(String[] args) {
        int[] arr = {1, 2, 3};
        
        System.out.println("原数组：" + java.util.Arrays.toString(arr));
        
        changeRef(arr);
        
        System.out.println("新数组：" + java.util.Arrays.toString(arr));  // 不变
    }
    
    public static void changeRef(int[] arr) {
        // 创建新数组，修改的是局部变量的引用
        arr = new int[]{4, 5, 6};
        System.out.println("方法内：" + java.util.Arrays.toString(arr));
    }
}
```

### 3.3 修改数组元素 vs 修改引用

```java
public class ArrayModify {
    public static void main(String[] args) {
        int[] arr = {1, 2, 3};
        
        modifyElement(arr);
        System.out.println("修改元素后：" + java.util.Arrays.toString(arr));  // [100, 2, 3]
        
        modifyReference(arr);
        System.out.println("修改引用后：" + java.util.Arrays.toString(arr));  // [100, 2, 3]
    }
    
    // 修改元素
    public static void modifyElement(int[] arr) {
        arr[0] = 100;
    }
    
    // 修改引用（不影响原引用）
    public static void modifyReference(int[] arr) {
        arr = new int[]{4, 5, 6};
    }
}
```

## 四、String 参数传递

### 4.1 String 的不可变性

```java
public class StringPass {
    public static void main(String[] args) {
        String str = "Hello";
        
        System.out.println("修改前：" + str);  // Hello
        
        changeString(str);
        
        System.out.println("修改后：" + str);  // Hello，不变
    }
    
    public static void changeString(String str) {
        str = str + " World";
        System.out.println("方法内：" + str);  // Hello World
    }
}
```

**解释**：
- String 是不可变类
- `str = str + " World"` 实际是创建了新字符串
- 原引用指向的字符串没有改变

### 4.2 StringBuilder 参数

```java
public class StringBuilderPass {
    public static void main(String[] args) {
        StringBuilder sb = new StringBuilder("Hello");
        
        System.out.println("修改前：" + sb);  // Hello
        
        changeStringBuilder(sb);
        
        System.out.println("修改后：" + sb);  // Hello World，变了
    }
    
    public static void changeStringBuilder(StringBuilder sb) {
        sb.append(" World");
        System.out.println("方法内：" + sb);
    }
}
```

## 五、对象参数传递

### 5.1 修改对象属性

```java
class Person {
    String name;
    int age;
    
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    @Override
    public String toString() {
        return "Person{name='" + name + "', age=" + age + "}";
    }
}

public class ObjectPass {
    public static void main(String[] args) {
        Person p = new Person("张三", 18);
        
        System.out.println("修改前：" + p);
        
        changePerson(p);
        
        System.out.println("修改后：" + p);  // 属性变了
    }
    
    public static void changePerson(Person p) {
        p.name = "李四";
        p.age = 20;
        System.out.println("方法内：" + p);
    }
}
```

### 5.2 修改对象引用

```java
public class ObjectRefPass {
    public static void main(String[] args) {
        Person p = new Person("张三", 18);
        
        System.out.println("修改前：" + p);
        
        changeReference(p);
        
        System.out.println("修改后：" + p);  // 不变
    }
    
    public static void changeReference(Person p) {
        // 创建新对象，修改的是局部变量的引用
        p = new Person("李四", 20);
        System.out.println("方法内：" + p);
    }
}
```

## 六、总结对比

### 6.1 传递方式对比

| 类型 | 传递内容 | 能否修改原值 | 示例 |
|------|----------|--------------|------|
| 基本类型 | 值的副本 | 否 | int, double, boolean |
| 数组 | 地址值副本 | 能修改元素 | int[], String[] |
| 对象 | 地址值副本 | 能修改属性 | Person, Student |
| String | 地址值副本 | 否（不可变） | String |
| StringBuilder | 地址值副本 | 能（可变） | StringBuilder |

### 6.2 图示

```
基本类型：
main: x = 10
      ↓ 复制值
method: n = 10

引用类型：
main: arr → [1,2,3] (0x123)
      ↓ 复制地址
method: arr → 0x123
        ↓ 通过地址修改
堆内存：[100,2,3]
```

## 七、综合示例

```java
import java.util.Arrays;

public class ParameterPassExample {
    public static void main(String[] args) {
        // 1. 基本类型
        int x = 10;
        changePrimitive(x);
        System.out.println("基本类型：" + x);  // 10
        
        // 2. 数组
        int[] arr = {1, 2, 3};
        changeArray(arr);
        System.out.println("数组：" + Arrays.toString(arr));  // [100, 2, 3]
        
        // 3. String
        String str = "Hello";
        changeString(str);
        System.out.println("String：" + str);  // Hello
        
        // 4. StringBuilder
        StringBuilder sb = new StringBuilder("Hello");
        changeStringBuilder(sb);
        System.out.println("StringBuilder：" + sb);  // Hello World
        
        // 5. 对象
        Person p = new Person("张三", 18);
        changePerson(p);
        System.out.println("对象：" + p);  // Person{name='李四', age=20}
    }
    
    public static void changePrimitive(int n) {
        n = 100;
    }
    
    public static void changeArray(int[] arr) {
        arr[0] = 100;
    }
    
    public static void changeString(String str) {
        str = str + " World";
    }
    
    public static void changeStringBuilder(StringBuilder sb) {
        sb.append(" World");
    }
    
    public static void changePerson(Person p) {
        p.name = "李四";
        p.age = 20;
    }
}

class Person {
    String name;
    int age;
    
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    @Override
    public String toString() {
        return "Person{name='" + name + "', age=" + age + "}";
    }
}
```

## 八、注意事项

### 8.1 避免副作用

```java
// ❌ 不推荐：方法修改了传入的数组
public static void process(int[] arr) {
    arr[0] = 100;  // 副作用
    // ... 其他处理
}

// ✅ 推荐：创建副本
public static void process(int[] arr) {
    int[] copy = arr.clone();
    copy[0] = 100;
    // ... 处理副本
}
```

### 8.2 文档说明

```java
/**
 * 处理用户数据
 * @param user 用户对象（会被修改）
 */
public static void processUser(User user) {
    user.setLastLogin(new Date());
}
```

## 九、小结

本节要点：
1. **Java 只有值传递**
2. **基本类型**：传递值的副本，不能修改原值
3. **引用类型**：传递地址的副本，能修改对象内容
4. **String**：特殊引用类型，不可变
5. **注意**：修改引用本身不影响原引用
