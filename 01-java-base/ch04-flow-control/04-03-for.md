# 4.3 for 循环

## 一、for 循环基本语法

```java
for (初始化表达式; 条件表达式; 更新表达式) {
    // 循环体
}
```

**执行流程**：
1. 执行初始化表达式（只执行一次）
2. 判断条件表达式
3. 条件为 true，执行循环体
4. 执行更新表达式
5. 回到步骤 2，直到条件为 false

## 二、基本示例

### 2.1 打印 1 到 10

```java
public class ForDemo {
    public static void main(String[] args) {
        for (int i = 1; i <= 10; i++) {
            System.out.println(i);
        }
    }
}
```

### 2.2 计算累加和

```java
public class SumDemo {
    public static void main(String[] args) {
        int sum = 0;
        
        for (int i = 1; i <= 100; i++) {
            sum += i;
        }
        
        System.out.println("1 到 100 的和：" + sum);  // 5050
    }
}
```

### 2.3 计算阶乘

```java
public class Factorial {
    public static void main(String[] args) {
        int n = 5;
        long factorial = 1;
        
        for (int i = 1; i <= n; i++) {
            factorial *= i;
        }
        
        System.out.println(n + "! = " + factorial);  // 120
    }
}
```

## 三、for 循环变形

### 3.1 多个变量

```java
public class MultiVar {
    public static void main(String[] args) {
        // 多个变量用逗号分隔
        for (int i = 0, j = 10; i < j; i++, j--) {
            System.out.println("i = " + i + ", j = " + j);
        }
    }
}
```

### 3.2 省略表达式

```java
public class OmitFor {
    public static void main(String[] args) {
        // 省略初始化
        int i = 0;
        for (; i < 5; i++) {
            System.out.println(i);
        }
        
        // 省略条件（死循环）
        // for (int j = 0; ; j++) { }
        
        // 省略更新
        int k = 0;
        for (; k < 5; ) {
            System.out.println(k);
            k++;
        }
        
        // 全部省略（死循环）
        // for (;;) { }
    }
}
```

### 3.3 倒序循环

```java
public class ReverseFor {
    public static void main(String[] args) {
        // 从 10 倒数到 1
        for (int i = 10; i >= 1; i--) {
            System.out.println(i);
        }
        
        // 从 100 倒数到 0，步长为 10
        for (int i = 100; i >= 0; i -= 10) {
            System.out.println(i);
        }
    }
}
```

## 五、应用场景

### 5.1 求平均值

```java
public class Average {
    public static void main(String[] args) {
        int[] scores = {85, 90, 78, 92, 88};
        int sum = 0;
        
        for (int i = 0; i < scores.length; i++) {
            sum += scores[i];
        }
        
        double average = (double) sum / scores.length;
        System.out.println("平均分：" + average);
    }
}
```

### 5.2 查找最大值

```java
public class FindMax {
    public static void main(String[] args) {
        int[] nums = {23, 45, 12, 67, 34, 89};
        int max = nums[0];
        
        for (int i = 1; i < nums.length; i++) {
            if (nums[i] > max) {
                max = nums[i];
            }
        }
        
        System.out.println("最大值：" + max);
    }
}
```

### 5.3 判断素数

```java
public class PrimeCheck {
    public static void main(String[] args) {
        int num = 17;
        boolean isPrime = true;
        
        for (int i = 2; i <= Math.sqrt(num); i++) {
            if (num % i == 0) {
                isPrime = false;
                break;
            }
        }
        
        if (isPrime) {
            System.out.println(num + " 是素数");
        } else {
            System.out.println(num + " 不是素数");
        }
    }
}
```

### 5.4 打印图案

```java
public class PrintPattern {
    public static void main(String[] args) {
        // 打印等腰三角形

        for (int i = 1; i <= rows; i++) {
            // 打印空格

            }

        }

        输出：
            *
           ***
          *****
         *******
        *********
        */
    }
}
```

## 六、注意事项

### 6.1 变量作用域

```java
// i 的作用域只在 for 循环内
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}
// System.out.println(i);  // ❌ 错误：i 已超出作用域

// 如果需要在循环外使用，在外部声明
int j;
for (j = 0; j < 5; j++) {
    System.out.println(j);
}
System.out.println(j);  // ✅ 正确
```

### 6.2 死循环

```java
// ❌ 死循环：条件永远为 true
// for (int i = 0; i >= 0; i++) { }

// ❌ 死循环：忘记更新
// for (int i = 0; i < 10; ) { }

// ✅ 正确
for (int i = 0; i < 10; i++) { }
```

### 6.3 差一错误

```java
// 打印 1 到 10
for (int i = 1; i <= 10; i++) { }  // ✅ 正确
for (int i = 1; i < 10; i++) { }   // ❌ 只到 9
for (int i = 1; i <= 11; i++) { }  // ❌ 多了一个
```

## 七、综合示例

```java
public class ForExample {
    public static void main(String[] args) {
        // 1. 计算偶数和
        int evenSum = 0;
        for (int i = 1; i <= 100; i++) {
            if (i % 2 == 0) {
                evenSum += i;
            }
        }
        System.out.println("1-100 偶数和：" + evenSum);
        
        // 2. 水仙花数（三位数，各位立方和等于本身）
        System.out.println("水仙花数：");
        for (int i = 100; i <= 999; i++) {
            int ge = i % 10;
            int shi = (i / 10) % 10;
            int bai = i / 100;
            
            if (i == ge * ge * ge + shi * shi * shi + bai * bai * bai) {
                System.out.print(i + " ");
            }
        }
        System.out.println();
        
        // 3. 打印菱形
        int n = 5;
        
        // 上半部分
        for (int i = 1; i <= n; i++) {
            for (int j = 1; j <= n - i; j++) {
                System.out.print(" ");
            }
            for (int k = 1; k <= 2 * i - 1; k++) {
                System.out.print("*");
            }
            System.out.println();
        }
        
        // 下半部分
        for (int i = n - 1; i >= 1; i--) {
            for (int j = 1; j <= n - i; j++) {
                System.out.print(" ");
            }
            for (int k = 1; k <= 2 * i - 1; k++) {
                System.out.print("*");
            }
            System.out.println();
        }
    }
}
```

## 八、小结

本节要点：
1. **基本语法**：for(初始化; 条件; 更新)
2. **执行流程**：初始化→判断→执行→更新→判断
3. **嵌套循环**：外层控制行，内层控制列
4. **应用场景**：累加、累乘、遍历、打印图案
5. **注意**：变量作用域、死循环、差一错误
