# 2.1 变量的定义与使用

### 一、什么是变量

变量是程序中存储数据的容器，其值可以在程序运行过程中改变。

```
变量 = 数据类型 + 变量名 + 值
```

### 二、变量的定义

#### 2.1 基本语法

```java
数据类型 变量名 = 值;
```

#### 2.2 示例

```java
public class VariableDemo {
    public static void main(String[] args) {
        // 定义整型变量
        int age = 18;
        
        // 定义字符串变量
        String name = "张三";
        
        // 定义小数变量
        double height = 1.75;
        
        // 输出变量值
        System.out.println("姓名：" + name);
        System.out.println("年龄：" + age);
        System.out.println("身高：" + height);
    }
}
```

### 三、变量的使用

#### 3.1 先声明后赋值

```java
int score;      // 声明变量
score = 90;     // 赋值
System.out.println(score);  // 使用变量
```

#### 3.2 声明并初始化

```java
int score = 90;  // 声明同时初始化
```

#### 3.3 变量值的修改

```java
int count = 0;
count = 1;       // 修改变量值
count = count + 1;  // 基于原值修改
```

### 四、变量命名规范

```java
// ✅ 推荐
int studentAge = 18;
String userName = "张三";
double productPrice = 99.99;

// ❌ 不推荐
int a = 18;              // 无意义
String s = "张三";       // 无意义
int student_age = 18;    // Java 习惯不用下划线
```

### 五、注意事项

#### 5.1 变量作用域

```java
public class ScopeDemo {
    public static void main(String[] args) {
        int x = 10;  // 作用域：整个 main 方法
        
        if (x > 5) {
            int y = 20;  // 作用域：if 语句块内
            System.out.println(y);  // ✅ 可以访问
        }
        
        // System.out.println(y);  // ❌ 错误，y 已超出作用域
    }
}
```

#### 5.2 变量不能重复定义

```java
int a = 10;
// int a = 20;  // ❌ 错误，变量已存在

a = 20;  // ✅ 正确，修改变量值
```

#### 5.3 变量必须先赋值后使用

```java
int x;
// System.out.println(x);  // ❌ 错误，未初始化

x = 10;
System.out.println(x);  // ✅ 正确
```

### 六、小结

本节要点：
1. 变量是存储数据的容器
2. 语法：`数据类型 变量名 = 值;`
3. 变量必须先声明后使用
4. 变量名要有意义，遵循命名规范

# 2.3 类型转换

### 一、类型转换分类

```
类型转换
├── 自动类型转换（隐式转换）
│   小范围 → 大范围
└── 强制类型转换（显式转换）
    大范围 → 小范围
```

### 二、自动类型转换

#### 2.1 转换规则

```
byte → short → int → long → float → double
                  ↑
                char
```

**条件**：
- 两种类型兼容
- 目标类型范围更大

#### 2.2 示例

```java
public class AutoConvert {
    public static void main(String[] args) {
        // byte → int
        byte b = 10;
        int i = b;  // 自动转换
        System.out.println(i);
        
        // int → long
        int x = 100;
        long y = x;  // 自动转换
        System.out.println(y);
        
        // int → double
        int m = 100;
        double n = m;  // 自动转换
        System.out.println(n);
        
        // char → int
        char c = 'A';
        int code = c;  // 自动转换，65
        System.out.println(code);
    }
}
```

#### 2.3 运算时的自动转换

```java
// 不同数据类型运算，自动转换为范围大的
int i = 10;
double d = 3.14;
double result = i + d;  // int 自动转为 double
System.out.println(result);  // 13.14

// byte 和 short 运算会转为 int
byte b1 = 10;
byte b2 = 20;
// byte b3 = b1 + b2;  // ❌ 错误，结果是 int 类型
int b3 = b1 + b2;  // ✅ 正确
```

### 三、强制类型转换

#### 3.1 转换语法

```java
目标类型 变量名 = (目标类型) 原值;
```

#### 3.2 示例

```java
public class ForcedConvert {
    public static void main(String[] args) {
        // double → int
        double d = 3.99;
        int i = (int) d;  // 强制转换，丢失小数
        System.out.println(i);  // 3
        
        // long → int
        long l = 1000L;
        int x = (int) l;
        System.out.println(x);
        
        // int → byte
        int m = 130;
        byte b = (byte) m;  // 可能溢出
        System.out.println(b);  // -126
        
        // double → int → byte
        double pi = 3.14159;
        byte b2 = (byte) (int) pi;
        System.out.println(b2);  // 3
    }
}
```

#### 3.3 精度丢失问题

```java
// 小数转整数，直接截断小数部分
double d1 = 3.99;
int i1 = (int) d1;  // 3，不是 4

// 四舍五入
double d2 = 3.99;
int i2 = (int) Math.round(d2);  // 4

// 大范围转小范围可能溢出
int big = 300;
byte small = (byte) big;  // 44（溢出）
// 300 超出 byte 范围 (-128~127)
// 计算：300 - 256 = 44
```

### 四、常见转换场景

#### 4.1 字符串转数值

```java
// String → int
String str1 = "123";
int num1 = Integer.parseInt(str1);

// String → double
String str2 = "3.14";
double num2 = Double.parseDouble(str2);

// String → long
String str3 = "1000000";
long num3 = Long.parseLong(str3);

// ❌ 错误：字符串格式不正确
// String bad = "abc";
// int x = Integer.parseInt(bad);  // NumberFormatException
```

#### 4.2 数值转字符串

```java
// 方法 1：拼接空字符串
int num = 123;
String str1 = num + "";

// 方法 2：valueOf 方法
String str2 = String.valueOf(num);

// 方法 3：toString 方法
String str3 = Integer.toString(num);
```

#### 4.3 运算中的转换

```java
// 整数除法
int a = 10;
int b = 3;
double result1 = a / b;  // 3.0，先整除再转换

// 正确的小数除法
double result2 = (double) a / b;  // 3.333...
double result3 = a * 1.0 / b;     // 3.333...
```

### 五、类型转换陷阱

#### 5.1 溢出问题

```java
int max = Integer.MAX_VALUE;  // 2147483647
int overflow = max + 1;       // -2147483648，溢出
System.out.println(overflow);

// 解决方案：使用更大的类型
long safe = (long) max + 1;   // 2147483648
```

#### 5.2 精度问题

```java
// 浮点数运算精度丢失
double a = 0.1;
double b = 0.2;
double sum = a + b;
System.out.println(sum);  // 0.30000000000000004

// 解决方案：使用 BigDecimal
import java.math.BigDecimal;
BigDecimal bd1 = new BigDecimal("0.1");
BigDecimal bd2 = new BigDecimal("0.2");
BigDecimal bd3 = bd1.add(bd2);
System.out.println(bd3);  // 0.3
```

#### 5.3 强制转换优先级

```java
int a = 10;
int b = 3;

// ❌ 错误理解
double r1 = (double) a / b;  // 3.333...，只转换 a

// ✅ 正确理解
double r2 = (double) (a / b);  // 3.0，先整除再转换
```

### 六、综合示例

```java
public class ConvertDemo {
    public static void main(String[] args) {
        // 1. 自动类型转换
        byte b = 10;
        int i = b;  // byte → int
        double d = i;  // int → double
        
        // 2. 强制类型转换
        double pi = 3.14159;
        int intPi = (int) pi;  // 3
        
        // 3. 字符串转换
        String numStr = "100";
        int num = Integer.parseInt(numStr);
        int result = num + 50;
        
        // 4. 运算转换
        int x = 10;
        int y = 3;
        double div1 = x / y;        // 3.0
        double div2 = x * 1.0 / y;  // 3.333...
        double div3 = (double) x / y;  // 3.333...
        
        System.out.println("自动转换：" + d);
        System.out.println("强制转换：" + intPi);
        System.out.println("字符串转换：" + result);
        System.out.println("除法运算：" + div1 + ", " + div2);
    }
}
```

### 七、小结

本节要点：
1. **自动转换**：小范围→大范围，无需强制
2. **强制转换**：大范围→小范围，需强制，可能丢失精度
3. **转换顺序**：byte→short→int→long→float→double
4. **字符串转换**：parseInt、parseDouble、valueOf
