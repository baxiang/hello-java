# Java 基础语法

本模块是 Java 学习的起点，涵盖 Java 语言的基础语法知识，从环境搭建到变量、运算符、流程控制、数组和方法，帮助你建立扎实的编程基础。

## 章节列表

| 目录 | 主题 |
|------|------|
| 01-intro | Java 入门：简介与 JDK、HelloWorld、开发环境与基本语法 |
| 02-variable | 变量与数据类型、类型转换与常量、Scanner 输入 |
| 03-operator | 算术与关系运算符、逻辑运算符与优先级 |
| 04-flow-control | 条件判断、循环语句、流程控制 |
| 05-array | 数组基础与遍历、数组操作与二维数组、Arrays 工具类 |
| 06-algorithm | 排序与查找算法 |
| 07-method | 方法定义与重载、参数传递与可变参数、递归方法 |

## 学习目标

- 掌握 Java 开发环境搭建与第一个程序编写
- 理解变量、数据类型与运算符的使用
- 熟练使用流程控制语句编写逻辑程序
- 掌握数组的存储与操作
- 理解方法的定义、调用与参数传递机制
