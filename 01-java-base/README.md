## 🎯 学习目标

- [ ] 理解 Java 程序的基本结构
- [ ] 掌握变量和数据类型的用法
- [ ] 熟练使用各种运算符
- [ ] 掌握流程控制语句
- [ ] 理解数组的概念和操作

## 📝 练习题

完成每个章节的练习题后，确保能够：
1. 独立编写简单的 Java 程序
2. 理解程序的执行流程
3. 使用数组存储和处理数据
4. 将功能封装成方法
