# 10.3 访问控制修饰符

## 一、访问控制概述

Java 提供了四种访问控制级别，用于控制类、方法、变量的访问范围。

## 二、四种访问级别

| 修饰符 | 同类 | 同包 | 子类 | 不同包 |
|--------|------|------|------|--------|
| private | ✅ | ❌ | ❌ | ❌ |
| (default) | ✅ | ✅ | ❌ | ❌ |
| protected | ✅ | ✅ | ✅ | ❌ |
| public | ✅ | ✅ | ✅ | ✅ |

## 三、private 私有

### 3.1 基本用法

```java
class Person {
    // 私有属性
    private String name;
    private int age;
    
    // 私有方法
    private void validate() {
        if (age < 0 || age > 150) {
            throw new IllegalArgumentException("年龄不合法");
        }
    }
    
    // 公共方法
    public void setAge(int age) {
        validate();  // 可以调用私有方法
        this.age = age;
    }
    
    public int getAge() {
        return age;
    }
}

// 使用
public class Main {
    public static void main(String[] args) {
        Person p = new Person();
        
        // ❌ 错误：不能访问私有成员
        // p.name = "张三";
        // p.validate();
        
        // ✅ 正确：通过公共方法访问
        p.setAge(18);
        System.out.println(p.getAge());
    }
}
```

### 3.2 应用场景

```java
// 1. 封装数据
class BankAccount {
    private double balance;
    
    public void deposit(double amount) {
        if (amount > 0) {
            balance += amount;
        }
    }
    
    public double getBalance() {
        return balance;
    }
}

// 2. 隐藏实现细节
class Database {
    private Connection connection;
    
    private void connect() {
        // 内部实现
    }
    
    public void query(String sql) {
        connect();
        // 执行查询
    }
}
```

## 四、default 默认（包访问权限）

### 4.1 基本用法

```java
// 不写访问修饰符，就是 default
class MyClass {
    int num = 10;  // default
    void show() {  // default
        System.out.println(num);
    }
}

// 同包可以访问
package com.example;

class OtherClass {
    void test() {
        MyClass obj = new MyClass();
        System.out.println(obj.num);  // ✅ 同包可以访问
        obj.show();
    }
}

// 不同包不能访问
package com.other;

class AnotherClass {
    void test() {
        MyClass obj = new MyClass();
        // System.out.println(obj.num);  // ❌ 不同包不能访问
    }
}
```

### 4.2 应用场景

```java
// 包内工具类
package com.example.utils;

class PackageUtils {
    // 只在本包内使用
    void helper() {
        // ...
    }
}
```

## 五、protected 受保护

### 5.1 基本用法

```java
// 父类
package com.example;

public class Parent {
    protected int value = 10;
    
    protected void show() {
        System.out.println(value);
    }
}

// 同包子类
package com.example;

public class Child1 extends Parent {
    public void test() {
        System.out.println(value);  // ✅ 可以访问
        show();
    }
}

// 不同包子类
package com.other;

public class Child2 extends Parent {
    public void test() {
        System.out.println(value);  // ✅ 可以访问（继承的）
        show();
    }
    
    public void test2() {
        Parent p = new Parent();
        // System.out.println(p.value);  // ❌ 不能访问父类实例的 protected 成员
    }
}

// 不同包非子类
package com.other;

public class Other {
    public void test() {
        Parent p = new Parent();
        // System.out.println(p.value);  // ❌ 不能访问
    }
}
```

### 5.2 应用场景

```java
// 设计给子类使用的方法
abstract class Animal {
    protected String name;
    
    protected void breathe() {
        System.out.println("呼吸");
    }
    
    public abstract void makeSound();
}

class Dog extends Animal {
    @Override
    public void makeSound() {
        breathe();  // 调用 protected 方法
        System.out.println("汪汪叫");
    }
}
```

## 六、public 公共

### 6.1 基本用法

```java
public class Person {
    public String name;
    public int age;
    
    public void introduce() {
        System.out.println("我叫" + name);
    }
}

// 任何地方都可以访问
public class Main {
    public static void main(String[] args) {
        Person p = new Person();
        p.name = "张三";
        p.age = 18;
        p.introduce();
    }
}
```

### 6.2 应用场景

```java
// 公开的 API
public class UserService {
    public User getUserById(Long id) {
        // ...
    }
    
    public void createUser(User user) {
        // ...
    }
}

// 工具类方法
public class MathUtils {
    public static int add(int a, int b) {
        return a + b;
    }
}
```

## 七、类的访问控制

### 7.1 顶层类

```java
// 只能用 public 或 default

// public 类：任何包都可以访问
public class PublicClass { }

// default 类：只有同包可以访问
class PackageClass { }
```

### 7.2 内部类

```java
public class Outer {
    // public 内部类
    public class PublicInner { }
    
    // protected 内部类
    protected class ProtectedInner { }
    
    // default 内部类
    class PackageInner { }
    
    // private 内部类
    private class PrivateInner { }
}
```

## 八、访问控制对比

### 8.1 类成员访问

```java
public class MyClass {
    // 私有：只有本类可以访问
    private int privateVar;
    private void privateMethod() { }
    
    // 默认：同包可以访问
    int defaultVar;
    void defaultMethod() { }
    
    // 受保护：同包和子类可以访问
    protected int protectedVar;
    protected void protectedMethod() { }
    
    // 公共：任何地方可以访问
    public int publicVar;
    public void publicMethod() { }
}
```

### 8.2 选择指南

| 场景 | 推荐修饰符 |
|------|------------|
| 只在本类使用 | private |
| 同包内使用 | default |
| 子类使用 | protected |
| 对外公开 | public |
| 成员变量 | 通常 private |
| 构造方法 | 通常 public 或 private（单例） |

## 九、综合示例

```java
// 包 1：com.example.model
package com.example.model;

// 公开的实体类
public class User {
    // 私有属性
    private Long id;
    private String username;
    private String password;  // 敏感信息
    private String email;
    
    // 公共构造
    public User() { }
    
    public User(String username, String email) {
        this.username = username;
        this.email = email;
    }
    
    // 公共 getter/setter
    public Long getId() { return id; }
    public void setId(Long id) { this.id = id; }
    
    public String getUsername() { return username; }
    public void setUsername(String username) { this.username = username; }
    
    public String getEmail() { return email; }
    public void setEmail(String email) { this.email = email; }
    
    // 受保护的方法：供子类使用
    protected void validate() {
        if (username == null || username.isEmpty()) {
            throw new IllegalArgumentException("用户名不能为空");
        }
    }
    
    // 默认方法：包内使用
    void internalInit() {
        // 内部初始化逻辑
    }
    
    // 私有方法：内部使用
    private String encryptPassword(String pwd) {
        // 加密逻辑
        return "***";
    }
}
```

```java
// 包 1：com.example.service
package com.example.service;

import com.example.model.User;

// 公开的服务类
public class UserService {
    // 私有属性
    private UserRepository repository;
    
    // 公共构造
    public UserService() {
        this.repository = new UserRepository();
    }
    
    // 公共方法：对外 API
    public User getUser(Long id) {
        return repository.findById(id);
    }
    
    public void createUser(User user) {
        user.validate();  // 调用 protected 方法
        repository.save(user);
    }
    
    // 受保护方法：供子类扩展
    protected void beforeSave(User user) {
        // 保存前处理
    }
    
    // 默认方法：包内使用
    User findByName(String name) {
        return repository.findByName(name);
    }
    
    // 私有方法：内部实现
    private void log(String msg) {
        System.out.println(msg);
    }
}
```

## 十、小结

本节要点：
1. **private**：只在本类访问
2. **default**：同包访问（不写修饰符）
3. **protected**：同包和子类访问
4. **public**：任何地方访问
5. **原则**：尽量使用最小的访问权限
