# 3.4 逻辑运算符

### 一、逻辑运算符概述

逻辑运算符用于连接多个布尔表达式，进行逻辑运算。

| 运算符 | 说明 | 示例 | 结果 |
|--------|------|------|------|
| `&` | 逻辑与 | `true & false` | false |
| `|` | 逻辑或 | `true \| false` | true |
| `!` | 逻辑非 | `!true` | false |
| `^` | 逻辑异或 | `true ^ false` | true |
| `&&` | 短路与 | `true && false` | false |
| `||` | 短路或 | `true \|\| false` | true |

### 二、基本逻辑运算

#### 2.1 逻辑与（&）

```java
public class AndDemo {
    public static void main(String[] args) {
        boolean a = true;
        boolean b = false;
        
        // 与运算：全真才真，一假必假
        System.out.println(true & true);   // true
        System.out.println(true & false);  // false
        System.out.println(false & true);  // false
        System.out.println(false & false); // false
        
        // 应用场景
        int age = 25;
        int score = 85;
        
        if (age >= 18 & score >= 60) {
            System.out.println("成年且及格");
        }
    }
}
```

#### 2.2 逻辑或（|）

```java
public class OrDemo {
    public static void main(String[] args) {
        // 或运算：一真必真，全假才假
        System.out.println(true | true);   // true
        System.out.println(true | false);  // true
        System.out.println(false | true);  // true
        System.out.println(false | false); // false
        
        // 应用场景
        int score = 55;
        boolean hasMakeup = true;  // 是否补考
        
        if (score >= 60 | hasMakeup) {
            System.out.println("通过");
        }
    }
}
```

#### 2.3 逻辑非（!）

```java
public class NotDemo {
    public static void main(String[] args) {
        // 非运算：取反
        System.out.println(!true);   // false
        System.out.println(!false);  // true
        
        // 应用场景
        boolean isRain = false;
        
        if (!isRain) {
            System.out.println("不下雨，可以出门");
        }
        
        // 双重否定
        System.out.println(!!true);  // true
    }
}
```

#### 2.4 逻辑异或（^）

```java
public class XorDemo {
    public static void main(String[] args) {
        // 异或运算：相同为假，不同为真
        System.out.println(true ^ true);   // false
        System.out.println(true ^ false);  // true
        System.out.println(false ^ true);  // true
        System.out.println(false ^ false); // false
        
        // 应用场景：开关切换
        boolean switchState = true;
        switchState = switchState ^ true;  // 切换状态
        System.out.println(switchState);   // false
    }
}
```

### 三、短路与 && 短路或

#### 3.1 短路与（&&）

```java
public class ShortAndDemo {
    public static void main(String[] args) {
        // 短路特性：第一个为 false，不再计算第二个
        boolean result1 = false && (5 > 3);
        System.out.println(result1);  // false
        
        // 应用场景：避免空指针
        String str = null;
        
        // ❌ 会空指针
        // if (str != null & str.equals("hello")) {}
        
        // ✅ 短路保护
        if (str != null && str.equals("hello")) {
            System.out.println("是 hello");
        } else {
            System.out.println("str 为 null 或不是 hello");
        }
    }
}
```

#### 3.2 短路或（||）

```java
public class ShortOrDemo {
    public static void main(String[] args) {
        // 短路特性：第一个为 true，不再计算第二个
        boolean result1 = true || (5 > 3);
        System.out.println(result1);  // true
        
        // 应用场景：默认值
        String input = "";
        String name = input || "默认用户";  // 伪代码
        
        // 实际写法
        if (input == null || input.isEmpty()) {
            name = "默认用户";
        }
    }
}
```

#### 3.3 & 与 && 的区别

```java
public class AndCompare {
    public static void main(String[] args) {
        int a = 10;
        
        // &：两边都执行
        boolean r1 = (a > 5) & (a++ > 10);
        System.out.println("a = " + a);  // 11，a++ 执行了
        
        a = 10;
        // &&：短路，右边不执行
        boolean r2 = (a > 5) && (a++ > 10);
        System.out.println("a = " + a);  // 10，a++ 未执行
    }
}
```

### 四、运算符优先级

```
优先级从高到低：
! > & > ^ > | > && > ||
```

```java
public class PriorityDemo {
    public static void main(String[] args) {
        int a = 5, b = 10, c = 15;
        
        // 优先级示例
        boolean r1 = a < b && b < c;  // true
        // 等价于：(a < b) && (b < c)
        
        boolean r2 = a > b || b < c && a < c;  // true
        // 等价于：(a > b) || ((b < c) && (a < c))
        
        // 推荐使用括号明确优先级
        boolean r3 = (a > b) || (b < c && a < c);
    }
}
```

### 六、德·摩根定律

```
!(A && B) = !A || !B
!(A || B) = !A && !B
```

```java
public class DeMorganDemo {
    public static void main(String[] args) {
        boolean a = true;
        boolean b = false;
        
        // !(A && B) = !A || !B
        System.out.println(!(a && b));  // true
        System.out.println(!a || !b);   // true
        
        // !(A || B) = !A && !B
        System.out.println(!(a || b));  // false
        System.out.println(!a && !b);   // false
    }
}
```

### 八、小结

本节要点：
1. **基本逻辑运算**：`&`、`|`、`!`、`^`
2. **短路运算**：`&&`、`||`，提高效率

5. **应用**：条件判断、空指针保护

# 3.5 三元运算符

### 一、三元运算符概述

三元运算符是 Java 中唯一的三目运算符，用于简化简单的 if-else 语句。

**语法**：
```java
条件表达式 ? 表达式 1 : 表达式 2
```

**执行流程**：
1. 如果条件为 true，返回表达式 1 的值
2. 如果条件为 false，返回表达式 2 的值

### 二、基本用法

#### 2.1 简单示例

```java
public class TernaryDemo {
    public static void main(String[] args) {
        int a = 10;
        int b = 20;
        
        // 求最大值
        int max = a > b ? a : b;
        System.out.println("最大值：" + max);  // 20
        
        // 等价于 if-else
        /*
        int max;
        if (a > b) {
            max = a;
        } else {
            max = b;
        }
        */
    }
}
```

#### 2.2 判断奇偶

```java
public class OddEven {
    public static void main(String[] args) {
        int num = 7;
        
        String result = num % 2 == 0 ? "偶数" : "奇数";
        System.out.println(num + " 是 " + result);
    }
}
```

#### 2.3 判断及格

```java
public class PassCheck {
```

```java
public class CompareThree {
    public static void main(String[] args) {
```

```java
    }
}
```

### 五、与 if-else 的对比

#### 5.1 三元运算符

| 多分支判断 | if-else |
| 复杂逻辑 | if-else |
| 需要执行多条语句 | if-else |

### 六、注意事项

#### 6.1 类型兼容

```java
// 两个表达式类型必须兼容
int x = 5 > 3 ? 10 : 20;        // ✅ 都是 int
double y = 5 > 3 ? 10 : 20.5;   // ✅ int 可转 double

// ❌ 类型不兼容
// String s = 5 > 3 ? 10 : "hello";
```

#### 6.2 优先级

```java
int a = 10, b = 20;

// 三元运算符优先级较低
int max = a > b ? a : b;  // ✅ 正确

// 建议加括号
int result = (a > b) ? a : b;
```

#### 6.3 可读性

```java
// ✅ 简洁清晰
String status = isLoggedIn ? "已登录" : "未登录";

// ❌ 过于复杂，降低可读性
String result = a > 0 ? (b > 0 ? (c > 0 ? "都大于 0" : "c 不大于 0") : "b 不大于 0") : "a 不大于 0";

// ✅ 使用 if-else 更清晰
String result;
if (a > 0) {
    if (b > 0) {
        if (c > 0) {
            result = "都大于 0";
        } else {
            result = "c 不大于 0";
        }
    } else {
        result = "b 不大于 0";
    }
} else {
    result = "a 不大于 0";
}
```

### 八、小结

本节要点：
1. **语法**：`条件 ? 表达式 1 : 表达式 2`
2. **执行**：条件为 true 返回表达式 1，否则返回表达式 2
3. **嵌套**：可以多层嵌套使用
4. **优势**：简化简单 if-else，代码更简洁
5. **注意**：避免过度嵌套，保持可读性

# 3.6 运算符优先级

### 一、运算符优先级表

优先级从高到低排列：

| 优先级 | 运算符 | 说明 | 结合性 |
|--------|--------|------|--------|
| 1 | `()` `[]` | 括号、数组下标 | 从左到右 |
| 2 | `++` `--` `!` `~` `+` `-` | 自增自减、逻辑非、按位非、正负号 | 从右到左 |
| 3 | `*` `/` `%` | 乘、除、取模 | 从左到右 |
| 4 | `+` `-` | 加、减 | 从左到右 |
| 5 | `<<` `>>` `>>>` | 位移运算 | 从左到右 |
| 6 | `<` `>` `<=` `>=` | 关系运算 | 从左到右 |
| 7 | `==` `!=` | 相等运算 | 从左到右 |
| 8 | `&` | 按位与 | 从左到右 |
| 9 | `^` | 按位异或 | 从左到右 |
| 10 | `\|` | 按位或 | 从左到右 |
| 11 | `&&` | 短路与 | 从左到右 |
| 12 | `\|\|` | 短路或 | 从左到右 |
| 13 | `?:` | 三元运算符 | 从右到左 |
| 14 | `=` `+=` `-=` ... | 赋值运算 | 从右到左 |

### 二、记忆口诀

```
括号老大最优先，
自增自减逻辑非，
乘除取模加减法，
关系相等按位算，
短路与或三元尾，
赋值运算优先级低。
```

### 三、常见优先级示例

#### 3.1 算术运算优先级

```java
public class ArithmeticPriority {
    public static void main(String[] args) {
        // 先乘除后加减
        int result1 = 3 + 4 * 5;  // 23，不是 35
        
        // 括号优先
        int result2 = (3 + 4) * 5;  // 35
        
        // 同级从左到右
        int result3 = 10 - 5 + 2;  // 7，不是 3
        
        System.out.println("result1 = " + result1);
        System.out.println("result2 = " + result2);
        System.out.println("result3 = " + result3);
    }
}
```

#### 3.2 关系与逻辑运算优先级

```java
public class LogicPriority {
    public static void main(String[] args) {
        int a = 5, b = 10, c = 15;
        
        // 关系运算优先级高于逻辑运算
        boolean r1 = a < b && b < c;  // true
        // 等价于：(a < b) && (b < c)
        
        // 逻辑非优先级最高
        boolean r2 = !a > b;  // ❌ 错误，! 只能用于 boolean
        boolean r3 = !(a > b);  // ✅ true
        
        // || 优先级低于 &&
        boolean r4 = a > b && b < c || a < c;  // true
        // 等价于：((a > b) && (b < c)) || (a < c)
        
        System.out.println("r1 = " + r1);
        System.out.println("r3 = " + r3);
        System.out.println("r4 = " + r4);
    }
}
```

#### 3.3 自增自减优先级

```java
public class IncrementPriority {
    public static void main(String[] args) {
        int a = 5;
        
        // 自增优先级高于算术运算
        int b = ++a * 2;  // 12，先自增 a=6，再乘 2
        System.out.println("b = " + b);
        
        a = 5;
        int c = a++ * 2;  // 10，先用 a=5 乘 2，再自增 a=6
        System.out.println("c = " + c);
        System.out.println("a = " + a);  // 6
    }
}
```

#### 3.4 赋值运算优先级

```java
public class AssignPriority {
    public static void main(String[] args) {
        int a, b, c;
        
        // 赋值优先级最低，从右向左
        a = b = c = 10;
        System.out.println("a = " + a);  // 10
        System.out.println("b = " + b);  // 10
        System.out.println("c = " + c);  // 10
        
        // 扩展赋值包含隐式转换
        int x = 10;
        x += 5.5;  // 等价于 x = (int)(x + 5.5)
        System.out.println("x = " + x);  // 15
    }
}
```

### 五、最佳实践

#### 5.1 使用括号明确优先级

```java
// ❌ 可读性差
int result = a + b * c > d && e == f || g;

// ✅ 使用括号明确
boolean condition = ((a + (b * c)) > d) && (e == f) || g;
```

#### 5.2 拆分复杂表达式

```java
// ❌ 过于复杂
boolean isValid = age >= 18 && age <= 60 && score >= 60 && score <= 100;

// ✅ 拆分
boolean isAgeValid = age >= 18 && age <= 60;
boolean isScoreValid = score >= 60 && score <= 100;
boolean isValid = isAgeValid && isScoreValid;
```

#### 5.3 避免副作用

```java
// ❌ 不推荐：表达式中有副作用
int result = ++i + i++ * i;

// ✅ 推荐：分开写
++i;
int temp = i++;
int result = i + temp * i;
```

### 六、常见错误

#### 6.1 误判优先级

```java
// ❌ 错误理解
int x = 5 > 3 ? 10 : 20 + 1;
// 实际：5 > 3 ? 10 : (20 + 1) → 10
// 不是：(5 > 3 ? 10 : 20) + 1 → 11

// ✅ 正确理解
int y = (5 > 3 ? 10 : 20) + 1;  // 11
```

#### 6.2 连等比较

```java
int a = 5, b = 10, c = 15;

// ❌ 错误：数学写法不能直接用于 Java
// if (a < b < c) {}

// ✅ 正确
if (a < b && b < c) {}
```

#### 6.3 位运算优先级

```java
// ❌ 错误：== 优先级高于 &
if (x & 1 == 0) {}  // 编译错误

// ✅ 正确
if ((x & 1) == 0) {}  // 判断偶数
```

### 七、综合示例

```java
public class PriorityExample {
    public static void main(String[] args) {
        // 1. 计算表达式
        int a = 10, b = 5, c = 2;
        int result = a + b * c - a / b;  // 10 + 10 - 2 = 18
        System.out.println("result = " + result);
        
        // 2. 复杂条件判断
        int score = 85;
        int age = 20;
        boolean isMember = true;
        
        boolean discount = (score >= 80 || age < 18) && isMember;
        System.out.println("是否优惠：" + discount);
        
        // 3. 最大值
        int x = 15, y = 25, z = 20;
        int max = x > y ? (x > z ? x : z) : (y > z ? y : z);
        System.out.println("最大值：" + max);
        
        // 4. 闰年判断
        int year = 2026;
        boolean isLeap = (year % 4 == 0 && year % 100 != 0) || (year % 400 == 0);
        System.out.println("闰年：" + isLeap);
    }
}
```

### 八、小结

本节要点：
1. **优先级最高**：括号 `()`
2. **优先级最低**：赋值运算符
3. **算术优先级**：乘除 > 加减
4. **逻辑优先级**：`!` > `&` > `^` > `|` > `&&` > `||`
5. **最佳实践**：使用括号明确优先级，拆分复杂表达式
