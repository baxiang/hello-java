# 3.1 算术运算符

### 一、算术运算符概述

算术运算符用于执行基本的数学运算。

| 运算符 | 说明 | 示例 | 结果 |
|--------|------|------|------|
| `+` | 加法 | `5 + 3` | 8 |
| `-` | 减法 | `5 - 3` | 2 |
| `*` | 乘法 | `5 * 3` | 15 |
| `/` | 除法 | `6 / 3` | 2 |
| `%` | 取模（取余） | `7 % 3` | 1 |
| `++` | 自增 | `i++` | i+1 |
| `--` | 自减 | `i--` | i-1 |

### 二、基本算术运算

#### 2.1 加减乘

```java
public class ArithmeticDemo {
    public static void main(String[] args) {
        int a = 10;
        int b = 3;
        
        // 加法
        int sum = a + b;
        System.out.println("和：" + sum);  // 13
        
        // 减法
        int diff = a - b;
        System.out.println("差：" + diff);  // 7
        
        // 乘法
        int product = a * b;
        System.out.println("积：" + product);  // 30
    }
}
```

#### 2.2 除法

```java
public class DivisionDemo {
    public static void main(String[] args) {
        // 整数除法：结果取整
        int a = 10;
        int b = 3;
        int result1 = a / b;
        System.out.println("10 / 3 = " + result1);  // 3
        
        // 浮点数除法：保留小数
        double c = 10.0;
        double d = 3.0;
        double result2 = c / d;
        System.out.println("10.0 / 3.0 = " + result2);  // 3.333...
        
        // 整数转浮点再除
        int x = 10;
        int y = 3;
        double result3 = (double) x / y;
        System.out.println("(double)10 / 3 = " + result3);  // 3.333...
        
        // 除以零
        // int z = 10 / 0;  // ❌ ArithmeticException
        double w = 10.0 / 0.0;  // ✅ Infinity
        System.out.println(w);  // Infinity
    }
}
```

#### 2.3 取模（取余）

```java
public class ModuloDemo {
    public static void main(String[] args) {
        // 基本取模
        int a = 7 % 3;
        System.out.println("7 % 3 = " + a);  // 1
        
        // 应用场景 1：判断奇偶
        int num = 10;
        if (num % 2 == 0) {
            System.out.println(num + " 是偶数");
        } else {
            System.out.println(num + " 是奇数");
        }
        
        // 应用场景 2：判断整除
        if (15 % 5 == 0) {
            System.out.println("15 能被 5 整除");
        }
        
        // 应用场景 3：循环
        for (int i = 1; i <= 10; i++) {
            if (i % 3 == 0) {
                System.out.print(i + " ");  // 3 6 9
            }
        }
        
        // 负数取模
        int b = -7 % 3;
        System.out.println("-7 % 3 = " + b);  // -1
    }
}
```

### 三、自增自减运算符

#### 3.1 基本用法

```java
public class IncrementDemo {
    public static void main(String[] args) {
        int a = 5;
        
        // 自增
        a++;  // a = a + 1
        System.out.println("a++: " + a);  // 6
        
        ++a;  // a = a + 1
        System.out.println("++a: " + a);  // 7
        
        // 自减
        a--;  // a = a - 1
        System.out.println("a--: " + a);  // 6
        
        --a;  // a = a - 1
        System.out.println("--a: " + a);  // 5
    }
}
```

#### 3.2 前缀 vs 后缀

```java
public class PrefixSuffixDemo {
    public static void main(String[] args) {
        int a = 5;
        
        // 后缀：先使用，后自增
        int b = a++;  // b = 5, 然后 a = 6
        System.out.println("b = " + b);  // 5
        System.out.println("a = " + a);  // 6
        
        // 前缀：先自增，后使用
        int c = ++a;  // a = 7, 然后 c = 7
        System.out.println("c = " + c);  // 7
        System.out.println("a = " + a);  // 7
        
        // 后缀自减
        int d = 5;
        int e = d--;  // e = 5, 然后 d = 4
        System.out.println("e = " + e);  // 5
        System.out.println("d = " + d);  // 4
        
        // 前缀自减
        int f = --d;  // d = 3, 然后 f = 3
        System.out.println("f = " + f);  // 3
        System.out.println("d = " + d);  // 3
    }
}
```

#### 3.3 记忆口诀

```
++ 在前：先加后用
++ 在后：先用后加
-- 在前：先减后用
-- 在后：先用后减
```

#### 3.4 复杂表达式

```java
public class ComplexDemo {
    public static void main(String[] args) {
        int i = 5;
        
        // 在表达式中使用
        int j = i++ + ++i;
        // 执行顺序：
        // 1. i++ → 取 i=5，然后 i=6
        // 2. ++i → i=7，取 i=7
        // 3. j = 5 + 7 = 12
        System.out.println("j = " + j);  // 12
        System.out.println("i = " + i);  // 7
        
        // 不推荐：过于复杂，可读性差
        // int k = i++ + ++i * i--;
    }
}
```

# 3.2 赋值运算符

### 一、赋值运算符概述

赋值运算符用于将值赋给变量。

| 运算符 | 说明 | 示例 | 等价形式 |
|--------|------|------|----------|
| `=` | 基本赋值 | `a = 5` | - |
| `+=` | 加后赋值 | `a += 5` | `a = a + 5` |
| `-=` | 减后赋值 | `a -= 5` | `a = a - 5` |
| `*=` | 乘后赋值 | `a *= 5` | `a = a * 5` |
| `/=` | 除后赋值 | `a /= 5` | `a = a / 5` |
| `%=` | 取模后赋值 | `a %= 5` | `a = a % 5` |

### 二、基本赋值运算符

#### 2.1 简单赋值

```java
public class AssignDemo {
    public static void main(String[] args) {
        int a = 10;
        int b = a;  // 将 a 的值赋给 b
        
        System.out.println("a = " + a);  // 10
        System.out.println("b = " + b);  // 10
        
        // 修改 a，b 不变
        a = 20;
        System.out.println("a = " + a);  // 20
        System.out.println("b = " + b);  // 10
    }
}
```

#### 2.2 连续赋值

```java
public class ChainAssign {
    public static void main(String[] args) {
        int a, b, c;
        a = b = c = 10;  // 从右向左赋值
        
        System.out.println("a = " + a);  // 10
        System.out.println("b = " + b);  // 10
        System.out.println("c = " + c);  // 10
    }
}
```

#### 3.2 自动类型转换

```java
public class TypeConvert {
    public static void main(String[] args) {
        short s = 10;
        
        // ❌ 错误：需要强制转换
        // s = s + 5;
        
        // ✅ 正确：+= 自动进行类型转换
```

# 3.3 关系运算符

### 一、关系运算符概述

关系运算符用于比较两个值的大小或相等性，结果是布尔值（true 或 false）。

| 运算符 | 说明 | 示例 | 结果 |
|--------|------|------|------|
| `==` | 等于 | `5 == 5` | true |
| `!=` | 不等于 | `5 != 3` | true |
| `>` | 大于 | `5 > 3` | true |
| `<` | 小于 | `5 < 3` | false |
| `>=` | 大于等于 | `5 >= 5` | true |
| `<=` | 小于等于 | `5 <= 3` | false |

### 二、相等运算符

#### 2.1 等于（==）

```java
public class EqualDemo {
    public static void main(String[] args) {
        int a = 10;
        int b = 10;
        int c = 20;
        
        System.out.println(a == b);  // true
        System.out.println(a == c);  // false
        
        // 字符比较
        char ch1 = 'A';
        char ch2 = 'A';
        System.out.println(ch1 == ch2);  // true
        
        // 布尔比较
        boolean flag1 = true;
        boolean flag2 = true;
        System.out.println(flag1 == flag2);  // true
    }
}
```

### 五、注意事项

#### 5.1 赋值 vs 比较

```java
int a = 5;

// a = 5   // 赋值表达式，值为 5
// a == 5  // 比较表达式，值为 true

// if (a = 5) {}   // ❌ 编译错误：int 不能转为 boolean
// if (a == 5) {}  // ✅ 正确
```

#### 5.2 浮点数比较

```java

double b = 0.2;

// ❌ 不推荐：浮点数精度问题
System.out.println(c == 0.3);  // false

// ✅ 推荐：使用误差范围
double epsilon = 0.000001;
System.out.println(Math.abs(c - 0.3) < epsilon);  // true
```

#### 5.3 引用类型比较

```java
String s1 = new String("hello");

// == 比较的是引用地址

// equals 比较的是内容

```
