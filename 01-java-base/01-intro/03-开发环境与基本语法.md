# 开发环境与基本语法

## 一、问题引入

Hello World 跑起来了，但用记事本写代码太痛苦——没有提示、没有补全、找不到拼写错误。更头疼的是：注释怎么写？变量叫什么名才规范？`class`、`public` 这些词能不能当变量名？连命名规则都不清楚，写出来的代码自己都看不懂。

## 二、核心概念

### IDE 选择

| IDE | 特点 | 适合人群 |
|-----|------|----------|
| IntelliJ IDEA | 智能提示强、重构方便 | 专业开发者（推荐） |
| Eclipse | 免费开源、插件丰富 | 传统企业 |
| VS Code | 轻量、需装 Java 扩展包 | 前端转 Java |

### 注释三种形式

| 类型 | 语法 | 用途 |
|------|------|------|
| 单行注释 | `// 内容` | 解释单行逻辑 |
| 多行注释 | `/* 内容 */` | 临时注释多行代码 |
| 文档注释 | `/** 内容 */` | 生成 API 文档 |

### 标识符命名规则

标识符是给类、方法、变量起的名字。规则如下：

| 规则 | 正确 | 错误 |
|------|------|------|
| 可用字母、数字、`_`、`$` | `name1`, `_id`, `$flag` | `student-name`（有连字符） |
| 不能以数字开头 | `name1` | `1name` |
| 不能用关键字 | `myClass` | `class` |
| 区分大小写 | `Name` ≠ `name` | — |

### 命名规范（约定俗成）

| 类型 | 规范 | 示例 |
|------|------|------|
| 类名 | 大驼峰 PascalCase | `HelloWorld`, `StudentInfo` |
| 方法名 | 小驼峰 camelCase | `getName`, `calcSum` |
| 变量名 | 小驼峰 | `age`, `studentName` |
| 常量名 | 全大写 + 下划线 | `MAX_VALUE`, `PI` |
| 包名 | 全小写 + 点号 | `com.example.demo` |

### Java 关键字

关键字是 Java 预留的具有特殊含义的单词，不能用作标识符。常用的：

| 分类 | 关键字 |
|------|--------|
| 访问控制 | `public` `protected` `private` |
| 类相关 | `class` `interface` `enum` `extends` `implements` |
| 数据类型 | `int` `long` `double` `boolean` `char` `void` |
| 流程控制 | `if` `else` `for` `while` `switch` `return` |
| 异常处理 | `try` `catch` `finally` `throw` `throws` |
| 修饰符 | `static` `final` `abstract` `synchronized` |

> `true`、`false`、`null` 不是关键字，但也不能用作标识符。`goto`、`const` 是保留字，未使用但禁止占用。

## 三、代码对比

### 注释的好坏

```java
// ❌ 重复代码含义，毫无价值
i++;  // i 加 1

// ✅ 解释为什么，而非做了什么
quickSort(arr);  // 数据量 > 10万时快排优于归并

// ✅ 标记待办
// TODO: 优化查询性能
// FIXME: 并发时可能丢失数据
```

### 命名的好坏

```java
// ❌ 类名用下划线、变量名无意义、常量不大写
public class student_info {
    private String sN;       // 什么变量？看不懂
    private int a;           // 太短
    public static final int max = 100;  // 常量应全大写
}

// ✅ 遵循命名规范
public class StudentInfo {
    private String studentName;     // 一看就懂
    private int studentAge;
    public static final int MAX_SCORE = 100;
}
```

### 标识符的常见错误

```java
int 1stPlace = 1;          // ❌ 数字开头
int class = 10;            // ❌ 使用关键字
int student-name = "张三";  // ❌ 包含连字符

int firstPlace = 1;        // ✅ 字母开头
int classCount = 10;       // ✅ 不用关键字本身
int studentName = "张三";   // ✅ 用驼峰代替连字符
```

## 四、实现方式

### IDEA 快速上手

1. **创建项目**：File → New → Project → 选 Java 21 → Create
2. **创建类**：右键 `src` → New → Java Class → 输入类名
3. **运行**：点击行号旁的绿色三角

常用快捷键：

| macOS | Windows/Linux | 功能 |
|-------|---------------|------|
| `Cmd + Shift + F10` | `Ctrl + Shift + F10` | 运行 |
| `Cmd + /` | `Ctrl + /` | 行注释 |
| `Cmd + Alt + L` | `Ctrl + Alt + L` | 格式化代码 |
| `Cmd + N` | `Alt + Insert` | 生成代码 |

### 项目结构

```
hello-java/
├── src/                  # 源代码目录
│   └── HelloWorld.java
├── out/                  # 编译输出
└── .idea/                # IDEA 配置
```

### 综合示例

```java
package com.example.demo;  // 包声明：全小写

import java.util.Scanner;  // 导入类

/**
 * 学生信息示例
 * @author 张三
 * @version 1.0
 */
public class StudentManager {  // 类名：大驼峰

    public static final int MAX_COUNT = 100;  // 常量：全大写

    public static void main(String[] args) {  // 程序入口
        Scanner scanner = new Scanner(System.in);
        System.out.print("请输入姓名：");
        String name = scanner.nextLine();     // 变量：小驼峰
        System.out.println("你好，" + name);
    }

    public void printInfo() {                 // 方法：小驼峰
        System.out.println("学生信息");
    }
}
```

## 五、Q&A

**Q：VS Code 能写 Java 吗？**
A：能，但需要安装 "Extension Pack for Java" 扩展包。初学者更推荐 IDEA Community（免费），开箱即用。

**Q：注释写多了会影响性能吗？**
A：不会。编译时注释会被完全忽略，不影响字节码和运行速度。但注释应写有价值的，别写废话。

**Q：`_` 和 `$` 能用在命名里吗？**
A：语法上可以，但 `$` 一般留给自动生成代码（如内部类），`_` 在 Java 9+ 中不能单独用作标识符。日常命名用字母和数字即可。

**Q：一个 `.java` 文件能定义多个类吗？**
A：可以，但只能有一个 `public` 类，且文件名必须与 `public` 类名一致。其他类默认包级别访问。建议一个文件一个类。

**Q：`goto` 为什么是保留字？**
A：Java 设计时认为 `goto` 破坏结构化编程，所以没实现。保留是为了编译器报错更明确——你写了 `goto`，编译器说"这是保留字"，而不是"不认识这个词"。

## 六、小结表格

| 要点 | 说明 |
|------|------|
| IDE 推荐 | IntelliJ IDEA Community（免费） |
| 注释 | 单行 `//`、多行 `/* */`、文档 `/** */` |
| 标识符规则 | 字母/数字/`_`/`$`，不能数字开头，不能用关键字 |
| 命名规范 | 类名大驼峰、方法变量小驼峰、常量全大写 |
| 关键字 | Java 保留的 50+ 个特殊单词，不可用作标识符 |
| 文件规则 | 一个 `.java` 文件最多一个 `public` 类 |
