# 7.7 对象数组

## 一、对象数组概述

### 1.1 什么是对象数组

数组的元素是对象，而不是基本数据类型。

```java
// 基本数据类型数组
int[] nums = new int[5];

// 对象数组
Student[] students = new Student[5];
```

### 1.2 内存结构

```
栈内存              堆内存
+------------+      +------------------+
| students   |----->| Student[] 数组   |
+------------+      +------------------+
                    | [0] null         |
                    | [1] null         |
                    | [2] null         |
                    | [3] null         |
                    | [4] null         |
                    +------------------+
                    
创建对象后：
                    +------------------+
                    | Student 对象 1   |
                    +------------------+
                         ↑
                    +----+----+
                    | [0]     |
                    +---------+
```

## 二、对象数组的创建

### 2.1 声明和初始化

```java
public class Main {
    public static void main(String[] args) {
        // 方式 1：声明数组
        Student[] students;
        
        // 方式 2：创建数组（元素为 null）
        students = new Student[3];
        
        // 方式 3：声明同时创建
        Student[] students2 = new Student[5];
        
        // 此时元素都是 null
        System.out.println(students[0]);  // null
    }
}
```

### 2.2 初始化数组元素

```java
public class Main {
    public static void main(String[] args) {
        // 创建数组
        Student[] students = new Student[3];
        
        // 初始化每个元素
        students[0] = new Student("张三", 18);
        students[1] = new Student("李四", 19);
        students[2] = new Student("王五", 20);
        
        // 访问元素
        System.out.println(students[0].getName());  // 张三
    }
}
```

### 2.3 静态初始化

```java
public class Main {
    public static void main(String[] args) {
        // 方式 1：完整写法
        Student[] students = new Student[]{
            new Student("张三", 18),
            new Student("李四", 19),
            new Student("王五", 20)
        };
        
        // 方式 2：简化写法
        Student[] students2 = {
            new Student("张三", 18),
            new Student("李四", 19),
            new Student("王五", 20)
        };
    }
}
```

## 三、对象数组的遍历

### 3.1 普通 for 循环

```java
public class Main {
    public static void main(String[] args) {
        Student[] students = {
            new Student("张三", 18),
            new Student("李四", 19),
            new Student("王五", 20)
        };
        
        for (int i = 0; i < students.length; i++) {
            System.out.println(students[i].getName());
        }
    }
}
```

### 3.2 增强 for 循环

```java
public class Main {
    public static void main(String[] args) {
        Student[] students = {
            new Student("张三", 18),
            new Student("李四", 19),
            new Student("王五", 20)
        };
        
        for (Student s : students) {
            System.out.println(s.getName());
        }
    }
}
```

### 3.3 注意空指针

```java
public class Main {
    public static void main(String[] args) {
        Student[] students = new Student[5];
        students[0] = new Student("张三", 18);
        students[1] = new Student("李四", 19);
        // students[2], [3], [4] 都是 null
        
        // ❌ 错误：会抛出空指针异常
        // for (Student s : students) {
        //     System.out.println(s.getName());  // NullPointerException
        // }
        
        // ✅ 正确：检查 null
        for (Student s : students) {
            if (s != null) {
                System.out.println(s.getName());
            }
        }
    }
}
```

## 四、对象数组的应用

### 4.1 学生管理系统

```java
public class Student {
    private String id;
    private String name;
    private int age;
    
    public Student(String id, String name, int age) {
        this.id = id;
        this.name = name;
        this.age = age;
    }
    
    public String getId() { return id; }
    public String getName() { return name; }
    public int getAge() { return age; }
    
    @Override
    public String toString() {
        return id + "\t" + name + "\t" + age;
    }
}

public class StudentManager {
    private Student[] students;
    private int size;
    
    public StudentManager(int capacity) {
        students = new Student[capacity];
        size = 0;
    }
    
    // 添加学生
    public boolean add(Student s) {
        if (size >= students.length) {
            System.out.println("数组已满");
            return false;
        }
        students[size++] = s;
        return true;
    }
    
    // 查找学生
    public Student find(String id) {
        for (int i = 0; i < size; i++) {
            if (students[i].getId().equals(id)) {
                return students[i];
            }
        }
        return null;
    }
    
    // 删除学生
    public boolean remove(String id) {
        int index = -1;
        for (int i = 0; i < size; i++) {
            if (students[i].getId().equals(id)) {
                index = i;
                break;
            }
        }
        
        if (index == -1) {
            return false;
        }
        
        // 移动元素
        for (int i = index; i < size - 1; i++) {
            students[i] = students[i + 1];
        }
        students[--size] = null;
        return true;
    }
    
    // 显示所有学生
    public void display() {
        System.out.println("学号\t姓名\t年龄");
        for (int i = 0; i < size; i++) {
            System.out.println(students[i]);
        }
    }
    
    // 获取学生数量
    public int getSize() {
        return size;
    }
}

public class Demo {
    public static void main(String[] args) {
        StudentManager manager = new StudentManager(10);
        
        // 添加学生
        manager.add(new Student("S001", "张三", 18));
        manager.add(new Student("S002", "李四", 19));
        manager.add(new Student("S003", "王五", 20));
        
        // 显示
        manager.display();
        
        // 查找
        Student s = manager.find("S002");
        if (s != null) {
            System.out.println("找到：" + s.getName());
        }
        
        // 删除
        manager.remove("S001");
        manager.display();
    }
}
```

### 4.2 商品库存管理

```java
class Product {
    private String id;
    private String name;
    private double price;
    private int stock;
    
    public Product(String id, String name, double price, int stock) {
        this.id = id;
        this.name = name;
        this.price = price;
        this.stock = stock;
    }
    
    public String getId() { return id; }
    public String getName() { return name; }
    public double getPrice() { return price; }
    public int getStock() { return stock; }
    
    public void setStock(int stock) { this.stock = stock; }
    
    @Override
    public String toString() {
        return id + "\t" + name + "\t" + price + "\t" + stock;
    }
}

class Inventory {
    private Product[] products;
    private int size;
    
    public Inventory(int capacity) {
        products = new Product[capacity];
        size = 0;
    }
    
    public void addProduct(Product p) {
        if (size < products.length) {
            products[size++] = p;
        }
    }
    
    public Product findProduct(String id) {
        for (int i = 0; i < size; i++) {
            if (products[i].getId().equals(id)) {
                return products[i];
            }
        }
        return null;
    }
    
    public double getTotalValue() {
        double total = 0;
        for (int i = 0; i < size; i++) {
            total += products[i].getPrice() * products[i].getStock();
        }
        return total;
    }
    
    public void display() {
        System.out.println("编号\t名称\t价格\t库存");
        for (int i = 0; i < size; i++) {
            System.out.println(products[i]);
        }
    }
}
```

### 4.3 排序对象数组

```java
import java.util.Arrays;

public class SortObjects {
    public static void main(String[] args) {
        Student[] students = {
            new Student("S003", "王五", 20, 85),
            new Student("S001", "张三", 18, 92),
            new Student("S002", "李四", 19, 78)
        };
        
        // 按成绩排序
        Arrays.sort(students, (a, b) -> Double.compare(b.getScore(), a.getScore()));
        
        for (Student s : students) {
            System.out.println(s.getName() + ": " + s.getScore());
        }
    }
}

class Student implements Comparable<Student> {
    private String id;
    private String name;
    private int age;
    private double score;
    
    public Student(String id, String name, int age, double score) {
        this.id = id;
        this.name = name;
        this.age = age;
        this.score = score;
    }
    
    public String getName() { return name; }
    public double getScore() { return score; }
    
    @Override
    public int compareTo(Student other) {
        return Double.compare(this.score, other.score);
    }
}
```

## 五、二维对象数组

```java
public class Main {
    public static void main(String[] args) {
        // 创建二维对象数组
        Student[][] classes = new Student[3][5];
        
        // 初始化
        for (int i = 0; i < classes.length; i++) {
            for (int j = 0; j < classes[i].length; j++) {
                classes[i][j] = new Student("S" + i + j, "学生" + i + j, 18);
            }
        }
        
        // 遍历
        for (int i = 0; i < classes.length; i++) {
            System.out.println("班级" + (i + 1) + ":");
            for (int j = 0; j < classes[i].length; j++) {
                System.out.println("  " + classes[i][j].getName());
            }
        }
    }
}
```

## 六、ArrayList 替代方案

```java
import java.util.ArrayList;
import java.util.List;

public class Main {
    public static void main(String[] args) {
        // 使用 ArrayList 代替对象数组
        List<Student> students = new ArrayList<>();
        
        // 添加
        students.add(new Student("S001", "张三", 18));
        students.add(new Student("S002", "李四", 19));
        
        // 访问
        Student s = students.get(0);
        
        // 删除
        students.remove(0);
        
        // 遍历
        for (Student student : students) {
            System.out.println(student.getName());
        }
        
        // 大小
        int size = students.size();
    }
}
```

## 七、综合示例

```java
public class Library {
    private Book[] books;
    private int size;
    
    public Library(int capacity) {
        books = new Book[capacity];
        size = 0;
    }
    
    public void addBook(Book book) {
        if (size < books.length) {
            books[size++] = book;
        }
    }
    
    public Book searchByTitle(String title) {
        for (int i = 0; i < size; i++) {
            if (books[i].getTitle().contains(title)) {
                return books[i];
            }
        }
        return null;
    }
    
    public void displayAll() {
        System.out.println("图书馆藏书：");
        for (int i = 0; i < size; i++) {
            System.out.println((i + 1) + ". " + books[i]);
        }
    }
    
    public int getTotalBooks() {
        return size;
    }
}

class Book {
    private String isbn;
    private String title;
    private String author;
    private boolean borrowed;
    
    public Book(String isbn, String title, String author) {
        this.isbn = isbn;
        this.title = title;
        this.author = author;
        this.borrowed = false;
    }
    
    public String getTitle() { return title; }
    
    public void borrow() { borrowed = true; }
    public void returnBook() { borrowed = false; }
    
    @Override
    public String toString() {
        return "《" + title + "》by " + author + 
               (borrowed ? " (已借出)" : " (可借阅)");
    }
}

public class LibraryDemo {
    public static void main(String[] args) {
        Library library = new Library(100);
        
        library.addBook(new Book("978-1", "Java 编程思想", "Bruce Eckel"));
        library.addBook(new Book("978-2", "Effective Java", "Joshua Bloch"));
        library.addBook(new Book("978-3", "深入理解 JVM", "周志明"));
        
        library.displayAll();
        
        Book book = library.searchByTitle("Java");
        if (book != null) {
            System.out.println("找到：" + book);
        }
    }
}
```

## 八、小结

本节要点：
1. **对象数组**：元素是对象的数组
2. **初始化**：先创建数组，再创建每个对象
3. **遍历**：普通 for 或增强 for
4. **注意**：检查 null，避免空指针
5. **应用**：学生管理、商品库存、图书馆等
