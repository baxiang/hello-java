# 7.4 this 关键字

## 一、this 概述

`this` 关键字代表当前对象的引用，指向调用该方法的对象。

**this 的用法**：
1. 区分成员变量和局部变量
2. 调用其他构造方法
3. 作为参数传递
4. 从方法中返回当前对象

## 二、区分成员变量和局部变量

### 2.1 问题场景

```java
public class Student {
    String name;  // 成员变量
    int age;
    
    public void setName(String name) {  // 参数也是 name
        name = name;  // ❌ 错误：都是局部变量
    }
}
```

### 2.2 使用 this 解决

```java
public class Student {
    String name;
    int age;
    
    public void setName(String name) {
        this.name = name;  // ✅ this.name 是成员变量
    }
    
    public void setAge(int age) {
        this.age = age;
    }
}
```

### 2.3 构造方法中

```java
public class Student {
    String name;
    int age;
    
    public Student(String name, int age) {
        this.name = name;
        this.age = age;
    }
}
```

## 三、调用其他构造方法

### 3.1 this() 语法

```java
public class Student {
    String name;
    int age;
    String school;
    
    // 无参构造
    public Student() {
        this("未知", 0);  // 调用两个参数的构造
    }
    
    // 一个参数
    public Student(String name) {
        this(name, 0);  // 调用两个参数的构造
    }
    
    // 两个参数
    public Student(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    // 三个参数
    public Student(String name, int age, String school) {
        this(name, age);  // 调用两个参数的构造
        this.school = school;
    }
}
```

### 3.2 注意事项

```java
public class Student {
    public Student() {
        // ❌ 错误：this() 必须在第一行
        System.out.println("无参构造");
        this("未知");
    }
    
    public Student(String name) {
        // ❌ 错误：不能同时调用两个构造
        // this();
        // this(name, 0);
        this(name);
    }
}
```

## 四、this 作为参数

### 4.1 传递当前对象

```java
public class Student {
    String name;
    
    public Student(String name) {
        this.name = name;
    }
    
    // 将当前对象传递给其他方法
    public void register() {
        School.register(this);
    }
}

class School {
    public static void register(Student s) {
        System.out.println(s.name + " 已注册");
    }
}
```

### 4.2 对象比较

```java
public class Person {
    String name;
    int age;
    
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    // 判断是否是同一个人
    public boolean isSame(Person other) {
        return this == other;  // 比较引用
    }
    
    // 判断年龄是否相同
    public boolean sameAge(Person other) {
        return this.age == other.age;
    }
}

public class Main {
    public static void main(String[] args) {
        Person p1 = new Person("张三", 18);
        Person p2 = new Person("李四", 18);
        Person p3 = p1;
        
        System.out.println(p1.isSame(p2));  // false
        System.out.println(p1.isSame(p3));  // true
        System.out.println(p1.sameAge(p2)); // true
    }
}
```

## 五、this 返回当前对象

### 5.1 链式调用

```java
public class Builder {
    StringBuilder sb = new StringBuilder();
    
    public Builder append(String str) {
        sb.append(str);
        return this;  // 返回当前对象
    }
    
    public Builder append(int num) {
        sb.append(num);
        return this;
    }
    
    public String build() {
        return sb.toString();
    }
}

public class Main {
    public static void main(String[] args) {
        // 链式调用
        String result = new Builder()
            .append("Hello")
            .append(" ")
            .append("World")
            .append("!")
            .build();
        
        System.out.println(result);  // Hello World!
    }
}
```

### 5.2 实际应用

```java
public class Student {
    String name;
    int age;
    String school;
    
    public Student setName(String name) {
        this.name = name;
        return this;
    }
    
    public Student setAge(int age) {
        this.age = age;
        return this;
    }
    
    public Student setSchool(String school) {
        this.school = school;
        return this;
    }
    
    public void introduce() {
        System.out.println("我叫" + name + "，今年" + age + "岁，来自" + school);
    }
}

public class Main {
    public static void main(String[] args) {
        // 链式设置属性
        new Student()
            .setName("张三")
            .setAge(18)
            .setSchool("清华大学")
            .introduce();
    }
}
```

## 六、this 在静态方法中

### 6.1 静态方法不能使用 this

```java
public class Student {
    String name;
    
    // ❌ 错误：静态方法中没有 this
    public static void printName() {
        // System.out.println(this.name);
    }
    
    // ✅ 正确：实例方法可以使用 this
    public void printName() {
        System.out.println(this.name);
    }
}
```

**原因**：静态方法属于类，不属于具体对象。

## 七、this 内存图

| s1     |-------> | Student 对象     |
+--------+         | name: "张三"     |

                         ↑

                   +--------+

                   +--------+

## 八、综合示例

```java
public class Person {
    private String name;
    private int age;
    private String gender;
    
    // 构造方法
    public Person() {
        this("未知", 0, "未知");
    }
    
    public Person(String name) {
        this(name, 0, "未知");
    }
    
    public Person(String name, int age) {
        this(name, age, "未知");
    }
    
    public Person(String name, int age, String gender) {
        this.name = name;
        this.age = age;
        this.gender = gender;
    }
    
    // Getter 和 Setter
    public String getName() {
        return name;
    }
    
    public Person setName(String name) {
        this.name = name;
        return this;
    }
    
    public int getAge() {
        return age;
    }
    
    public Person setAge(int age) {
        if (age > 0 && age < 150) {
            this.age = age;
        }
        return this;
    }
    
    public String getGender() {
        return gender;
    }
    
    public Person setGender(String gender) {
        this.gender = gender;
        return this;
    }
    
    // 比较年龄
    public int compareAge(Person other) {
        return this.age - other.age;
    }
    
    // 判断是否同龄
    public boolean isSameAge(Person other) {
        return this.age == other.age;
    }
    
    // 显示信息
    public void showInfo() {
        System.out.println("姓名：" + this.name 
                         + ", 年龄：" + this.age 
                         + ", 性别：" + this.gender);
    }
}

public class PersonDemo {
    public static void main(String[] args) {
        // 1. 使用不同构造
        Person p1 = new Person();
        Person p2 = new Person("张三");
        Person p3 = new Person("李四", 18);
        Person p4 = new Person("王五", 20, "男");
        
        // 2. 链式设置
        p1.setName("赵六").setAge(22).setGender("女");
        
        // 3. 显示信息
        p1.showInfo();
        p2.showInfo();
        p3.showInfo();
        p4.showInfo();
        
        // 4. 年龄比较
        System.out.println("年龄差：" + p3.compareAge(p4));
        System.out.println("是否同龄：" + p3.isSameAge(p4));
        
        // 5. this 作为参数
        registerPerson(p1);
    }
    
    public static void registerPerson(Person p) {
        System.out.println(p.getName() + " 注册成功");
    }
}
```

## 九、注意事项

### 9.1 this 不能在类外使用

```java
public class Student {
    String name;
}

public class Main {
    public static void main(String[] args) {
        Student s = new Student();
        // ❌ 错误：this 只能在类内部使用
        // this.name = "张三";
    }
}
```

### 9.2 静态上下文不能使用 this

```java
public class Student {
    static String school = "清华大学";
    
    public static void printSchool() {
        // ❌ 错误：静态方法不能使用 this
        // System.out.println(this.school);
        
        // ✅ 正确：使用类名
        System.out.println(Student.school);
    }
}
```

## 十、小结

本节要点：
1. **this 含义**：当前对象的引用
2. **区分变量**：`this.成员变量 = 局部变量`
3. **调用构造**：`this()` 必须在第一行
4. **作为参数**：传递当前对象
5. **返回对象**：实现链式调用
6. **限制**：静态方法不能使用 this
