# 7.2 类的定义与对象创建

## 一、类的定义

### 1.1 类的基本结构

```java
[修饰符] class 类名 {
    // 成员变量（属性）
    [修饰符] 数据类型 变量名;
    
    // 构造方法
    [修饰符] 类名 (参数列表) {
        // 初始化代码
    }
    
    // 成员方法（行为）
    [修饰符] 返回值类型 方法名 (参数列表) {
        // 方法体
    }
}
```

### 1.2 示例：学生类

```java
public class Student {
    // 成员变量（属性）
    String name;      // 姓名
    int age;          // 年龄
    String school;    // 学校
    
    // 构造方法
    public Student() {
    }
    
    public Student(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    // 成员方法（行为）
    public void study() {
        System.out.println(name + " 在学习");
    }
    
    public void introduce() {
        System.out.println("我叫" + name + "，今年" + age + "岁，来自" + school);
    }
}
```

### 1.3 类的组成部分

| 部分 | 说明 | 示例 |
|------|------|------|
| 类名 | 大驼峰命名 | `Student`, `Person` |
| 成员变量 | 描述对象的属性 | `name`, `age` |
| 构造方法 | 创建对象时调用 | `Student()` |
| 成员方法 | 对象的行为 | `study()`, `introduce()` |

## 二、对象的创建

### 2.1 使用 new 关键字

```java
public class Main {
    public static void main(String[] args) {
        // 创建对象
        Student s1 = new Student();
        
        // 设置属性
        s1.name = "张三";
        s1.age = 18;
        s1.school = "清华大学";
        
        // 调用方法
        s1.introduce();
        s1.study();
    }
}
```

### 2.2 使用构造方法

```java
public class Main {
    public static void main(String[] args) {
        // 使用有参构造创建对象
        Student s1 = new Student("张三", 18);
        s1.school = "清华大学";
        s1.introduce();
        
        Student s2 = new Student("李四", 19);
        s2.school = "北京大学";
        s2.introduce();
    }
}
```

### 2.3 对象创建过程

```
1. 分配内存空间
   ↓
2. 初始化成员变量（默认值）
   ↓
3. 调用构造方法
   ↓
4. 返回对象引用
```

## 三、成员变量 vs 局部变量

### 3.1 定义位置

```java
public class Student {
    // 成员变量（在类中，方法外）
    String name;
    int age;
    
    public void study() {
        // 局部变量（在方法内）
        int hours = 2;
        System.out.println(name + " 学习了" + hours + "小时");
    }
}
```

### 3.2 对比

| 特性 | 成员变量 | 局部变量 |
|------|----------|----------|
| 定义位置 | 类中，方法外 | 方法内或代码块内 |
| 作用域 | 整个类 | 所在方法或代码块 |
| 生命周期 | 对象存在期间 | 方法执行期间 |
| 默认值 | 有默认值 | 无默认值，必须初始化 |
| 存储位置 | 堆内存 | 栈内存 |

### 3.3 示例对比

```java
public class VariableDemo {
    // 成员变量
    String name = "张三";  // 可以不初始化
    int age;              // 有默认值 0
    
    public void method() {
        // 局部变量
        int x;
        // System.out.println(x);  // ❌ 错误：未初始化
        x = 10;  // 必须初始化才能使用
        System.out.println(x);
        
        System.out.println(name);  // ✅ 可以使用成员变量
        System.out.println(age);   // 输出 0
    }
}
```

## 五、匿名对象

### 5.1 什么是匿名对象

没有引用变量指向的对象，只能使用一次。

```java
public class AnonymousObject {
    public static void main(String[] args) {
        // 普通对象
        Student s1 = new Student("张三", 18);
        s1.introduce();
        s1.study();
        
        // 匿名对象
        new Student("李四", 19).introduce();
        new Student("王五", 20).study();
    }
}
```

### 5.2 使用场景

```java
public class UseCase {
    public static void main(String[] args) {
        // 1. 方法参数
        printStudent(new Student("张三", 18));
        
        // 2. 只调用一次
        new Calculator().add(10, 20);
    }
    
    public static void printStudent(Student s) {
        s.introduce();
    }
}
```

## 六、综合示例

```java
// 手机类
class Phone {
    // 成员变量
    String brand;
    String color;
    double price;
    
    // 构造方法
    public Phone() {
    }
    
    public Phone(String brand, String color, double price) {
        this.brand = brand;
        this.color = color;
        this.price = price;
    }
    
    // 成员方法
    public void call(String name) {
        System.out.println("给" + name + "打电话");
    }
    
    public void sendMessage(String name, String content) {
        System.out.println("给" + name + "发短信：" + content);
    }
    
    public void showInfo() {
        System.out.println("品牌：" + brand + ", 颜色：" + color + ", 价格：" + price);
    }
}

public class PhoneDemo {
    public static void main(String[] args) {
        // 创建对象
        Phone p1 = new Phone();
        p1.brand = "华为";
        p1.color = "黑色";
        p1.price = 4999;
        p1.showInfo();
        p1.call("张三");
        
        // 使用构造方法
        Phone p2 = new Phone("小米", "白色", 2999);
        p2.showInfo();
        p2.sendMessage("李四", "晚上一起吃饭");
        
        // 匿名对象
        new Phone("苹果", "金色", 8999).call("王五");
    }
}
```

## 七、注意事项

### 7.1 对象使用前必须创建

```java
Student s;
// s.name = "张三";  // ❌ 错误：s 未初始化

s = new Student();
s.name = "张三";  // ✅ 正确
```

### 7.2 多个引用指向同一对象

```java
Student s1 = new Student("张三", 18);
Student s2 = s1;  // s2 和 s1 指向同一对象

s2.name = "李四";
System.out.println(s1.name);  // 输出"李四"
```

### 7.3 对象比较

```java
Student s1 = new Student("张三", 18);
Student s2 = new Student("张三", 18);

System.out.println(s1 == s2);           // false，比较引用
System.out.println(s1.equals(s2));      // false，默认比较引用（需重写 equals）
```

## 八、小结

本节要点：
1. **类定义**：成员变量 + 构造方法 + 成员方法
2. **对象创建**：`new 类名 ()`
3. **成员变量**：有默认值，作用域整个类
4. **局部变量**：无默认值，作用域方法内
5. **匿名对象**：无引用，只能用一次
6. **内存**：栈存引用，堆存对象
