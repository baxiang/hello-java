# 2.1 变量的定义与使用

## 一、什么是变量

变量是程序中存储数据的容器，其值可以在程序运行过程中改变。

```
变量 = 数据类型 + 变量名 + 值
```

## 二、变量的定义

### 2.1 基本语法

```java
数据类型 变量名 = 值;
```

### 2.2 示例

```java
public class VariableDemo {
    public static void main(String[] args) {
        // 定义整型变量
        int age = 18;
        
        // 定义字符串变量
        String name = "张三";
        
        // 定义小数变量
        double height = 1.75;
        
        // 输出变量值
        System.out.println("姓名：" + name);
        System.out.println("年龄：" + age);
        System.out.println("身高：" + height);
    }
}
```

## 三、变量的使用

### 3.1 先声明后赋值

```java
int score;      // 声明变量
score = 90;     // 赋值
System.out.println(score);  // 使用变量
```

### 3.2 声明并初始化

```java
int score = 90;  // 声明同时初始化
```

### 3.3 变量值的修改

```java
int count = 0;
count = 1;       // 修改变量值
count = count + 1;  // 基于原值修改
```

## 四、变量命名规范

```java
// ✅ 推荐
int studentAge = 18;
String userName = "张三";
double productPrice = 99.99;

// ❌ 不推荐
int a = 18;              // 无意义
String s = "张三";       // 无意义
int student_age = 18;    // Java 习惯不用下划线
```

## 五、注意事项

### 5.1 变量作用域

```java
public class ScopeDemo {
    public static void main(String[] args) {
        int x = 10;  // 作用域：整个 main 方法
        
        if (x > 5) {
            int y = 20;  // 作用域：if 语句块内
            System.out.println(y);  // ✅ 可以访问
        }
        
        // System.out.println(y);  // ❌ 错误，y 已超出作用域
    }
}
```

### 5.2 变量不能重复定义

```java
int a = 10;
// int a = 20;  // ❌ 错误，变量已存在

a = 20;  // ✅ 正确，修改变量值
```

### 5.3 变量必须先赋值后使用

```java
int x;
// System.out.println(x);  // ❌ 错误，未初始化

x = 10;
System.out.println(x);  // ✅ 正确
```

## 六、小结

本节要点：
1. 变量是存储数据的容器
2. 语法：`数据类型 变量名 = 值;`
3. 变量必须先声明后使用
4. 变量名要有意义，遵循命名规范

## 七、练习题

2. 定义两个变量交换它们的值
