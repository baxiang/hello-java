# 8.1 继承概述

## 一、什么是继承

继承（Inheritance）是面向对象编程的三大特性之一，指一个类（子类）继承另一个类（父类）的属性和方法。

**比喻**：
```
父亲 → 儿子
  ↓      ↓
继承    被继承
(父类)  (子类)

儿子继承父亲的：
- 财产（属性）
- 技能（方法）
- 还可以有自己的东西
```

## 二、继承的好处

| 好处 | 说明 |
|------|------|
| 代码复用 | 子类自动拥有父类的非私有成员 |
| 易于扩展 | 在父类基础上添加新功能 |
| 建立关系 | 为多态奠定基础 |

## 三、继承的语法

### 3.1 基本语法

```java
[修饰符] class 子类 extends 父类 {
    // 子类特有的成员
    // 可以重写父类的方法
}
```

### 3.2 示例

```java
// 父类
class Animal {
    String name;
    int age;
    
    public void eat() {
        System.out.println("吃东西");
    }
    
    public void sleep() {
        System.out.println("睡觉");
    }
}

// 子类
class Dog extends Animal {
    // 继承的：name, age, eat(), sleep()
    // 特有的：
    public void bark() {
        System.out.println("汪汪叫");
    }
}

// 使用
public class Main {
    public static void main(String[] args) {
        Dog dog = new Dog();
        dog.name = "旺财";      // 继承的属性
        dog.age = 3;            // 继承的属性
        dog.eat();              // 继承的方法
        dog.sleep();            // 继承的方法
        dog.bark();             // 自己的方法
    }
}
```

## 四、继承的特点

### 4.1 单继承

Java 只支持单继承，一个类只能有一个直接父类。

```java
// ✅ 正确
class Dog extends Animal { }

// ❌ 错误
// class Dog extends Animal, Pet { }
```

### 4.2 多层继承

Java 支持多层继承。

```java
class Animal { }
class Mammal extends Animal { }
class Dog extends Mammal { }

// 使用
Dog dog = new Dog();
// Dog 继承 Mammal，Mammal 继承 Animal
// 所以 Dog 间接继承 Animal 的所有非私有成员
```

### 4.3 继承的内容

```java
class Parent {
    public int a;        // ✅ 子类可继承
    protected int b;     // ✅ 子类可继承
    int c;               // ✅ 同包子类可继承
    private int d;       // ❌ 子类不可继承
    
    public void method1() { }    // ✅
    protected void method2() { } // ✅
    void method3() { }           // ✅ 同包
    private void method4() { }   // ❌
}

class Child extends Parent {
    // 可以访问：a, b, c(同包), method1, method2, method3(同包)
    // 不能访问：d, method4
}
```

## 五、Object 类

### 5.1 根类

`Object` 类是所有 Java 类的根父类。

```java
// 以下两个类等价
class Person { }
class Person extends Object { }
```

### 5.2 Object 类的常用方法

```java
public class Main {
    public static void main(String[] args) {
        Person p = new Person();
        
        // toString()
        System.out.println(p.toString());
        System.out.println(p);  // 自动调用 toString()
        
        // equals()
        Person p2 = new Person();
        System.out.println(p.equals(p2));
        
        // hashCode()
        System.out.println(p.hashCode());
        
        // getClass()
        System.out.println(p.getClass());
    }
}
```

## 六、继承 vs 包含

### 6.1 继承（is-a 关系）

```java
// Dog is-a Animal
class Dog extends Animal { }
```

### 6.2 包含（has-a 关系）

```java
// Car has-a Engine
class Car {
    private Engine engine;  // 组合
}
```

### 6.3 选择原则

```
优先使用组合，而非继承

继承：当子类确实是父类的一种时
组合：当一个类包含另一个类时
```

## 七、继承的应用

### 7.1 员工系统

```java
// 父类：员工
class Employee {
    protected String name;
    protected double salary;
    
    public Employee(String name, double salary) {
        this.name = name;
        this.salary = salary;
    }
    
    public void work() {
        System.out.println(name + " 在工作");
    }
    
    public double getSalary() {
        return salary;
    }
}

// 子类：经理
class Manager extends Employee {
    private double bonus;
    
    public Manager(String name, double salary, double bonus) {
        super(name, salary);
        this.bonus = bonus;
    }
    
    // 特有方法
    public void manage() {
        System.out.println(name + " 在管理团队");
    }
    
    @Override
    public double getSalary() {
        return super.getSalary() + bonus;
    }
}

// 子类：程序员
class Programmer extends Employee {
    private String language;
    
    public Programmer(String name, double salary, String language) {
        super(name, salary);
        this.language = language;
    }
    
    public void code() {
        System.out.println(name + " 在用" + language + "写代码");
    }
}

// 使用
public class Company {
    public static void main(String[] args) {
        Manager manager = new Manager("张三", 20000, 5000);
        manager.work();
        manager.manage();
        System.out.println("经理工资：" + manager.getSalary());
        
        Programmer programmer = new Programmer("李四", 15000, "Java");
        programmer.work();
        programmer.code();
        System.out.println("程序员工资：" + programmer.getSalary());
    }
}
```

### 7.2 图形系统

```java
// 父类：图形
class Shape {
    protected String color;
    
    public Shape(String color) {
        this.color = color;
    }
    
    public double getArea() {
        return 0;
    }
    
    public double getPerimeter() {
        return 0;
    }
}

// 子类：圆形
class Circle extends Shape {
    private double radius;
    
    public Circle(String color, double radius) {
        super(color);
        this.radius = radius;
    }
    
    @Override
    public double getArea() {
        return Math.PI * radius * radius;
    }
    
    @Override
    public double getPerimeter() {
        return 2 * Math.PI * radius;
    }
}

// 子类：矩形
class Rectangle extends Shape {
    private double width;
    private double height;
    
    public Rectangle(String color, double width, double height) {
        super(color);
        this.width = width;
        this.height = height;
    }
    
    @Override
    public double getArea() {
        return width * height;
    }
    
    @Override
    public double getPerimeter() {
        return 2 * (width + height);
    }
}
```

## 八、注意事项

### 8.1 父类私有成员

```java
class Parent {
    private int money;
}

class Child extends Parent {
    // ❌ 不能访问父类私有成员
    // money = 1000;
}
```

### 8.2 构造方法不继承

```java
class Parent {
    public Parent() { }
}

class Child extends Parent {
    // ❌ 不能继承父类构造方法
    // 但会自动调用父类无参构造
}
```

### 8.3 不能循环继承

```java
// ❌ 错误：循环继承
// class A extends B { }
// class B extends A { }
```

## 九、小结

本节要点：
1. **继承**：子类继承父类的属性和方法
2. **语法**：`class 子类 extends 父类`
3. **特点**：单继承、多层继承
4. **Object 类**：所有类的根父类
5. **关系**：is-a 用继承，has-a 用组合
