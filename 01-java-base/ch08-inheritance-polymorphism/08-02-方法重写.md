# 8.2 方法重写

## 一、方法重写概述

方法重写（Override）是指子类重新定义父类中已有的方法，以提供特定的实现。

**重写 vs 重载**：
| 特性 | 重写 (Override) | 重载 (Overload) |
|------|-----------------|-----------------|
| 位置 | 父子类之间 | 同一个类中 |
| 方法名 | 相同 | 相同 |
| 参数列表 | 相同 | 不同 |
| 返回值 | 相同或是其子类 | 无关 |

## 二、重写的规则

### 2.1 两同两小一大

| 规则 | 说明 |
|------|------|
| 方法名相同 | 必须相同 |
| 参数列表相同 | 必须相同 |
| 返回值类型 | 相同或是其子类 |
| 抛出的异常 | 不能比父类更多 |
| 访问权限 | 不能比父类更严格 |

### 2.2 示例

```java
class Parent {
    public void method1() { }
    public Object method2() { return null; }
    public void method3() throws IOException { }
}

class Child extends Parent {
    // ✅ 正确：完全相同
    @Override
    public void method1() { }
    
    // ✅ 正确：返回值是 Object 的子类
    @Override
    public String method2() { return ""; }
    
    // ✅ 正确：异常是 IOException 的子类
    @Override
    public void method3() throws FileNotFoundException { }
    
    // ❌ 错误：访问权限更严格
    // @Override
    // protected void method1() { }
}
```

## 三、重写的语法

### 3.1 基本语法

```java
class Parent {
    public void show() {
        System.out.println("父类的 show 方法");
    }
}

class Child extends Parent {
    @Override
    public void show() {
        System.out.println("子类的 show 方法");
    }
}
```

### 3.2 @Override 注解

```java
class Animal {
    public void eat() {
        System.out.println("吃东西");
    }
}

class Dog extends Animal {
    // ✅ 推荐：加上 @Override 注解
    @Override
    public void eat() {
        System.out.println("狗吃骨头");
    }
    
    // ❌ 错误：方法名拼写错误，@Override 会报错
    // @Override
    // public void eatting() { }
}
```

**好处**：
- 编译器检查是否正确重写
- 提高代码可读性

## 四、super 关键字

### 4.1 调用父类方法

```java
class Parent {
    public void show() {
        System.out.println("父类");
    }
}

class Child extends Parent {
    @Override
    public void show() {
        System.out.println("子类");
        super.show();  // 调用父类的 show 方法
    }
}
```

### 4.2 扩展父类方法

```java
class Phone {
    public void call() {
        System.out.println("打电话");
    }
}

class SmartPhone extends Phone {
    @Override
    public void call() {
        // 扩展功能
        System.out.println("视频通话");
        super.call();  // 保留父类功能
    }
}
```

### 4.3 子类特有方法

```java
class SmartPhone extends Phone {
    @Override
    public void call() {
        System.out.println("视频通话");
    }
    
    // 子类特有方法
    public void videoChat() {
        System.out.println("视频聊天");
    }
    
    public void playGame() {
        System.out.println("玩游戏");
    }
}
```

## 五、重写的应用

### 5.1 toString 方法重写

```java
class Person {
    private String name;
    private int age;
    
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    // 重写 toString
    @Override
    public String toString() {
        return "Person{name='" + name + "', age=" + age + "}";
    }
}

// 使用
Person p = new Person("张三", 18);
System.out.println(p);  // 自动调用 toString()
// 输出：Person{name='张三', age=18}
```

### 5.2 equals 方法重写

```java
class Person {
    private String name;
    private int age;
    
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    // 重写 equals
    @Override
    public boolean equals(Object obj) {
        // 1. 检查是否为 null
        if (obj == null) return false;
        // 2. 检查是否为同一对象
        if (this == obj) return true;
        // 3. 检查类型
        if (!(obj instanceof Person)) return false;
        // 4. 类型转换并比较
        Person other = (Person) obj;
        return this.name.equals(other.name) && this.age == other.age;
    }
}

// 使用
Person p1 = new Person("张三", 18);
Person p2 = new Person("张三", 18);
System.out.println(p1.equals(p2));  // true（如果重写正确）
```

### 5.3 动物叫声

```java
class Animal {
    public void makeSound() {
        System.out.println("动物发出声音");
    }
}

class Dog extends Animal {
    @Override
    public void makeSound() {
        System.out.println("汪汪汪");
    }
}

class Cat extends Animal {
    @Override
    public void makeSound() {
        System.out.println("喵喵喵");
    }
}

class Duck extends Animal {
    @Override
    public void makeSound() {
        System.out.println("嘎嘎嘎");
    }
}

// 使用
public class Zoo {
    public static void main(String[] args) {
        Animal[] animals = {
            new Dog(),
            new Cat(),
            new Duck()
        };
        
        for (Animal animal : animals) {
            animal.makeSound();  // 多态
        }
    }
}
```

### 5.4 员工工资计算

```java
class Employee {
    protected String name;
    protected double baseSalary;
    
    public Employee(String name, double baseSalary) {
        this.name = name;
        this.baseSalary = baseSalary;
    }
    
    public double calculateSalary() {
        return baseSalary;
    }
}

class Manager extends Employee {
    private double bonus;
    
    public Manager(String name, double baseSalary, double bonus) {
        super(name, baseSalary);
        this.bonus = bonus;
    }
    
    @Override
    public double calculateSalary() {
        return baseSalary + bonus;
    }
}

class Salesman extends Employee {
    private double commission;
    
    public Salesman(String name, double baseSalary, double commission) {
        super(name, baseSalary);
        this.commission = commission;
    }
    
    @Override
    public double calculateSalary() {
        return baseSalary + commission;
    }
}

// 使用
public class Company {
    public static void main(String[] args) {
        Employee[] employees = {
            new Manager("张三", 10000, 5000),
            new Salesman("李四", 8000, 3000),
            new Employee("王五", 6000)
        };
        
        for (Employee emp : employees) {
            System.out.println(emp.name + " 的工资：" + emp.calculateSalary());
        }
    }
}
```

## 六、不能重写的方法

### 6.1 final 方法

```java
class Parent {
    public final void show() {
        System.out.println("不能被重写");
    }
}

class Child extends Parent {
    // ❌ 错误：不能重写 final 方法
    // @Override
    // public void show() { }
}
```

### 6.2 static 方法

```java
class Parent {
    public static void show() {
        System.out.println("父类静态方法");
    }
}

class Child extends Parent {
    // 这不是重写，是隐藏
    public static void show() {
        System.out.println("子类静态方法");
    }
}

// 使用
Parent p = new Child();
p.show();  // 输出：父类静态方法（静态方法绑定在编译时）
```

### 6.3 private 方法

```java
class Parent {
    private void show() {
        System.out.println("父类私有方法");
    }
}

class Child extends Parent {
    // 这不是重写，是定义新方法
    public void show() {
        System.out.println("子类方法");
    }
}
```

## 七、重写与重载对比

```java
class Parent {
    public void method(int a) { }
}

class Child extends Parent {
    // ✅ 重写：父子类，方法名相同，参数相同
    @Override
    public void method(int a) { }
    
    // ✅ 重载：同一个类中，方法名相同，参数不同
    public void method(int a, int b) { }
    public void method(String s) { }
}
```

## 八、综合示例

```java
// 父类：图形
class Shape {
    protected String color;
    
    public Shape(String color) {
        this.color = color;
    }
    
    public double getArea() {
        return 0;
    }
    
    @Override
    public String toString() {
        return "Shape{color='" + color + "'}";
    }
    
    @Override
    public boolean equals(Object obj) {
        if (this == obj) return true;
        if (obj == null || getClass() != obj.getClass()) return false;
        Shape shape = (Shape) obj;
        return color != null && color.equals(shape.color);
    }
}

// 子类：圆形
class Circle extends Shape {
    private double radius;
    
    public Circle(String color, double radius) {
        super(color);
        this.radius = radius;
    }
    
    @Override
    public double getArea() {
        return Math.PI * radius * radius;
    }
    
    @Override
    public String toString() {
        return "Circle{color='" + color + "', radius=" + radius + "}";
    }
}

// 子类：矩形
class Rectangle extends Shape {
    private double width;
    private double height;
    
    public Rectangle(String color, double width, double height) {
        super(color);
        this.width = width;
        this.height = height;
    }
    
    @Override
    public double getArea() {
        return width * height;
    }
    
    @Override
    public String toString() {
        return "Rectangle{color='" + color + "', width=" + width + ", height=" + height + "}";
    }
}

// 使用
public class ShapeDemo {
    public static void main(String[] args) {
        Shape[] shapes = {
            new Circle("红色", 5),
            new Rectangle("蓝色", 4, 6)
        };
        
        for (Shape shape : shapes) {
            System.out.println(shape);  // 调用重写的 toString
            System.out.println("面积：" + shape.getArea());  // 调用重写的 getArea
        }
    }
}
```

## 九、小结

本节要点：
1. **重写**：子类重新定义父类方法
2. **规则**：两同两小一大
3. **@Override**：注解检查是否正确重写
4. **super**：调用父类方法
5. **不能重写**：final、static、private 方法
