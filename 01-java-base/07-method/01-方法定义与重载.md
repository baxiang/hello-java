# 6.1 方法的定义与调用

### 一、方法概述

方法（Method）是完成特定功能的代码块，可以被重复调用。

**好处**：
- 提高代码复用性
- 提高代码可维护性
- 使程序结构更清晰

### 二、方法的定义

#### 2.1 基本语法

```java
修饰符 返回值类型 方法名 (参数列表) {
    // 方法体
    return 返回值;  // 如果返回值类型不是 void
}
```

#### 2.2 方法组成部分

| 部分 | 说明 | 示例 |
|------|------|------|
| 修饰符 | 控制访问权限等 | `public`, `static` |
| 返回值类型 | 方法返回的数据类型 | `int`, `String`, `void` |
| 方法名 | 方法的名称 | `sum`, `printInfo` |
| 参数列表 | 传入的数据 | `int a, int b` |
| 方法体 | 执行的具体代码 | `{ ... }` |
| 返回值 | 返回给调用者的值 | `return result;` |

#### 2.3 示例

```java
public class MethodDemo {
    public static void main(String[] args) {
        // 调用方法
        int result = sum(10, 20);
        System.out.println("结果：" + result);
    }
    
    // 定义方法
    public static int sum(int a, int b) {
        int result = a + b;
        return result;
    }
}
```

### 三、方法的调用

#### 3.1 调用方式

```java
public class MethodCall {
    public static void main(String[] args) {
        // 方式 1：单独调用（void 方法）
        printStar(5);
        
        // 方式 2：打印调用（有返回值）
        System.out.println(sum(10, 20));
        
        // 方式 3：赋值调用（有返回值）
        int result = sum(10, 20);
        System.out.println("结果：" + result);
    }
    
    public static void printStar(int n) {
        for (int i = 0; i < n; i++) {
            System.out.print("* ");
        }
        System.out.println();
    }
    
    public static int sum(int a, int b) {
        return a + b;
    }
}
```

#### 3.2 调用过程

```
调用 sum(10, 20)
    ↓
参数传递：a=10, b=20
    ↓
执行方法体：result = 10 + 20 = 30
    ↓
返回结果：return 30
    ↓
调用处接收返回值
```

### 四、方法的分类

#### 4.1 有返回值的方法

```java
public class ReturnMethod {
    public static void main(String[] args) {

        System.out.println("最大值：" + max);
    }
    
    public static int getMax(int a, int b) {
        return a > b ? a : b;
    }
}
```

#### 4.2 无返回值的方法（void）

```java
public class VoidMethod {

    }
    
    public static void printInfo(String name, int age) {
        System.out.println("姓名：" + name);
        System.out.println("年龄：" + age);
    }

```

#### 4.3 有参数的方法

```java
public class ParameterMethod {

        System.out.println("和：" + result);
    }

    }

```

#### 4.4 无参数的方法

```java

        printHello();
    }

        System.out.println("Hello, World!");
    }

```

### 五、方法重载

#### 5.1 重载概念

方法重载：在同一个类中，方法名相同，参数列表不同。

#### 5.2 重载示例

```java
public class OverloadDemo {
    public static void main(String[] args) {
        System.out.println(sum(10, 20));        // 30
        System.out.println(sum(10, 20, 30));    // 60
        System.out.println(sum(10.5, 20.5));    // 31.0
    }
    
    // 两个整数相加
    public static int sum(int a, int b) {
        return a + b;
    }
    
    // 三个整数相加
    public static int sum(int a, int b, int c) {
        return a + b + c;
    }
    
    // 两个小数相加
    public static double sum(double a, double b) {
        return a + b;
    }
}
```

#### 5.3 重载判断标准

```java
public class OverloadCheck {
    // ✅ 重载：参数个数不同
    public void method(int a) { }
    public void method(int a, int b) { }
    
    // ✅ 重载：参数类型不同
    public void method(int a) { }
    public void method(double a) { }
    
    // ✅ 重载：参数顺序不同
    public void method(int a, double b) { }
    public void method(double a, int b) { }
    
    // ❌ 不是重载：仅返回值不同
    // public int method(int a) { }
    // public double method(int a) { }
    
    // ❌ 不是重载：仅修饰符不同
    // public void method(int a) { }
    // static void method(int a) { }
}
```

### 六、注意事项

#### 6.1 return 语句

```java
public class ReturnDemo {
    public static void main(String[] args) {
        System.out.println(getSign(5));   // 正数
        System.out.println(getSign(-5));  // 负数
        System.out.println(getSign(0));   // 零
    }
    
    public static String getSign(int num) {
        if (num > 0) {
            return "正数";
        } else if (num < 0) {
            return "负数";
        } else {
            return "零";
        }
    }
}
```

#### 6.2 void 方法中的 return

```java
public class VoidReturn {
    public static void main(String[] args) {
        printNumber(5);
    }
    
    public static void printNumber(int n) {
        if (n < 0) {
            return;  // void 方法可以省略 return
        }
        System.out.println(n);
    }
}
```

#### 6.3 参数传递

```java
public class ParameterPass {
    public static void main(String[] args) {
        int x = 10;
        change(x);
        System.out.println(x);  // 10，基本类型值传递
        
        int[] arr = {1, 2, 3};
        changeArray(arr);
        System.out.println(arr[0]);  // 100，引用类型地址传递
    }
    
    public static void change(int n) {
        n = 100;
    }
    
    public static void changeArray(int[] arr) {
        arr[0] = 100;
    }
}
```

### 七、综合示例

```java
public class MethodExample {
    public static void main(String[] args) {
        // 1. 基本调用
        int sum = add(10, 20);
        System.out.println("和：" + sum);
        
        // 2. 重载调用
        System.out.println("三数和：" + add(10, 20, 30));
        System.out.println("小数和：" + add(1.5, 2.5));
        
        // 3. void 方法
        printLine(20);
        
        // 4. 组合调用
        int max = getMax(10, 20);
        int min = getMin(10, 20);
        System.out.println("最大值：" + max + ", 最小值：" + min);
    }
    
    // 两个整数相加
    public static int add(int a, int b) {
        return a + b;
    }
    
    // 三个整数相加
    public static int add(int a, int b, int c) {
        return a + b + c;
    }
    
    // 两个小数相加
    public static double add(double a, double b) {
        return a + b;
    }
    
    // 打印分隔线
    public static void printLine(int n) {
        for (int i = 0; i < n; i++) {
            System.out.print("-");
        }
        System.out.println();
    }
    
    // 获取最大值
    public static int getMax(int a, int b) {
        return a > b ? a : b;
    }
    
    // 获取最小值
    public static int getMin(int a, int b) {
        return a < b ? a : b;
    }
}
```

### 八、小结

本节要点：
1. **方法定义**：修饰符 + 返回值类型 + 方法名 + 参数列表 + 方法体
2. **方法调用**：单独调用、打印调用、赋值调用
3. **方法分类**：有/无返回值，有/无参数
4. **方法重载**：方法名相同，参数列表不同
5. **参数传递**：基本类型值传递，引用类型地址传递

# 6.2 方法的重载

### 一、重载概述

方法重载（Overload）是指在同一个类中，方法名相同但参数列表不同的多个方法。

**特点**：
- 方法名必须相同
- 参数列表必须不同（个数、类型、顺序）
- 与返回值类型无关
- 与访问修饰符无关

### 二、重载的判定

#### 2.1 参数个数不同

```java
public class OverloadByCount {
    public static void main(String[] args) {
        System.out.println(sum(10, 20));        // 30
        System.out.println(sum(10, 20, 30));    // 60
        System.out.println(sum(10, 20, 30, 40)); // 100
    }
    
    // 两个数相加
    public static int sum(int a, int b) {
        return a + b;
    }
    
    // 三个数相加
    public static int sum(int a, int b, int c) {
        return a + b + c;
    }
    
    // 四个数相加
    public static int sum(int a, int b, int c, int d) {
        return a + b + c + d;
    }
}
```

#### 2.2 参数类型不同

```java
public class OverloadByType {
    public static void main(String[] args) {
        System.out.println(add(10, 20));        // 30
        System.out.println(add(1.5, 2.5));      // 4.0
        System.out.println(add("Hello", "World")); // HelloWorld
    }
    
    // 整数相加
    public static int add(int a, int b) {
        return a + b;
    }
    
    // 小数相加
    public static double add(double a, double b) {
        return a + b;
    }
    
    // 字符串拼接
    public static String add(String a, String b) {
        return a + b;
    }
}
```

#### 2.3 参数顺序不同

```java
public class OverloadByOrder {
    public static void main(String[] args) {
        printInfo("张三", 18);      // 调用 (String, int)
        printInfo(18, "张三");      // 调用 (int, String)
    }
    
    // 先姓名后年龄
    public static void printInfo(String name, int age) {
        System.out.println("姓名：" + name + ", 年龄：" + age);
    }
    
    // 先年龄后姓名
    public static void printInfo(int age, String name) {
        System.out.println("年龄：" + age + ", 姓名：" + name);
    }
}
```

### 三、不是重载的情况

#### 3.1 仅返回值不同

```java
public class NotOverload1 {
    // ❌ 编译错误：仅返回值不同，不是重载
    // public int method(int a) { return 1; }
    // public double method(int a) { return 1.0; }
}
```

#### 3.2 仅修饰符不同

```java
public class NotOverload2 {
    // ❌ 编译错误：仅修饰符不同，不是重载
    // public void method(int a) { }
    // static void method(int a) { }
    // private void method(int a) { }
}
```

#### 3.3 仅参数名不同

```java
public class NotOverload3 {
    // ❌ 编译错误：仅参数名不同，不是重载
    // public void method(int a) { }
    // public void method(int b) { }
}
```

### 四、重载的应用场景

#### 4.1 构造函数重载

```java
public class Person {
    String name;
    int age;
    
    // 无参构造
    public Person() {
        this.name = "未知";
        this.age = 0;
    }
    
    // 一个参数
    public Person(String name) {
        this.name = name;
        this.age = 0;
    }
    
    // 两个参数
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
}
```

#### 4.2 工具方法重载

```java
public class MathUtils {
    // 两个整数最大值
    public static int max(int a, int b) {
        return a > b ? a : b;
    }
    
    // 三个整数最大值
    public static int max(int a, int b, int c) {
        return max(max(a, b), c);
    }
    
    // 两个小数最大值
    public static double max(double a, double b) {
        return a > b ? a : b;
    }
}
```

#### 4.3 打印方法重载

```java
public class Printer {
    // 打印整数
    public static void print(int value) {
        System.out.println("整数：" + value);
    }
    
    // 打印小数
    public static void print(double value) {
        System.out.println("小数：" + value);
    }
    
    // 打印字符串
    public static void print(String value) {
        System.out.println("字符串：" + value);
    }
    
    // 打印布尔值
    public static void print(boolean value) {
        System.out.println("布尔：" + value);
    }

```

#### 4.4 数组操作重载

```java
import java.util.Arrays;

public class ArrayUtils {

}
```

### 五、重载的匹配规则

#### 5.1 精确匹配优先

```java
public class MatchRule {
    public static void main(String[] args) {
        method(10);        // 调用 method(int)
        method(10L);       // 调用 method(long)
        method(10.5);      // 调用 method(double)
    }
    
    public static void method(int a) {
        System.out.println("int");
    }
    
    public static void method(long a) {
        System.out.println("long");
    }
    
    public static void method(double a) {
        System.out.println("double");
    }
}
```

#### 5.2 自动类型提升

```java
public class TypePromotion {
    public static void main(String[] args) {
        byte b = 10;
        method(b);  // 调用 method(int)，byte 自动提升为 int
    }
    
    public static void method(int a) {
        System.out.println("int");
    }
    
    public static void method(long a) {
        System.out.println("long");
    }
}
```

#### 5.3 可变参数优先级最低

```java
public class VarargsPriority {
    public static void main(String[] args) {
        method(10);        // 调用 method(int)
        method(10, 20);    // 调用 method(int...)
    }
    
    public static void method(int a) {
        System.out.println("固定参数");
    }
    
    public static void method(int... a) {
        System.out.println("可变参数");
    }
}
```

### 六、重载与重写的区别

| 特性 | 重载 (Overload) | 重写 (Override) |
|------|-----------------|-----------------|
| 发生位置 | 同一个类中 | 父子类之间 |
| 方法名 | 相同 | 相同 |
| 参数列表 | 不同 | 相同 |
| 返回值 | 无关 | 相同或是其子类 |
| 访问修饰符 | 无关 | 不能更严格 |

### 七、综合示例

```java
public class OverloadExample {
    public static void main(String[] args) {
        // 1. 基本重载
        System.out.println("和：" + calculate(10, 20));
        System.out.println("和：" + calculate(10, 20, 30));
        System.out.println("和：" + calculate(1.5, 2.5));
        
        // 2. 不同类型
        System.out.println("拼接：" + calculate("Hello", "World"));
        
        // 3. 打印不同格式
        print(100);
        print(3.14);
        print(true);
        print("Java");
        
        // 4. 数组操作
        int[] nums = {1, 2, 3};
        printArray(nums);
        
        String[] strs = {"A", "B", "C"};
        printArray(strs);
    }
    
    // 两个整数相加
    public static int calculate(int a, int b) {
        return a + b;
    }
    
    // 三个整数相加
    public static int calculate(int a, int b, int c) {
        return a + b + c;
    }
    
    // 两个小数相加
    public static double calculate(double a, double b) {
        return a + b;
    }
    
    // 字符串拼接
    public static String calculate(String a, String b) {
        return a + b;
    }
    
    // 打印整数
    public static void print(int value) {
        System.out.println("整数：" + value);
    }
    
    // 打印小数
    public static void print(double value) {
        System.out.println("小数：" + value);
    }
    
    // 打印布尔值
    public static void print(boolean value) {
        System.out.println("布尔：" + value);
    }
    
    // 打印字符串
    public static void print(String value) {
        System.out.println("字符串：" + value);
    }
    
    // 打印整数数组
    public static void printArray(int[] arr) {
        System.out.print("整数数组：[");
        for (int i = 0; i < arr.length; i++) {
            System.out.print(arr[i]);
            if (i < arr.length - 1) System.out.print(", ");
        }
        System.out.println("]");
    }
    
    // 打印字符串数组
    public static void printArray(String[] arr) {
        System.out.print("字符串数组：[");
        for (int i = 0; i < arr.length; i++) {
            System.out.print(arr[i]);
            if (i < arr.length - 1) System.out.print(", ");
        }
        System.out.println("]");
    }
}
```

### 八、小结

本节要点：
1. **重载定义**：方法名相同，参数列表不同
2. **判断标准**：参数个数、类型、顺序不同
3. **不是重载**：仅返回值、修饰符、参数名不同
4. **匹配规则**：精确匹配优先，自动类型提升
5. **应用场景**：构造函数、工具方法、打印方法
