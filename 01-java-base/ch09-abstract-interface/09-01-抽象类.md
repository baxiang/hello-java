# 9.1 抽象类概述

## 一、抽象类的概念

抽象类（Abstract Class）是用 `abstract` 修饰的类，不能被实例化，只能被继承。

**特点**：
- 用 `abstract` 修饰
- 不能创建对象
- 可以包含抽象方法和普通方法
- 子类必须实现所有抽象方法（除非子类也是抽象类）

## 二、抽象类的语法

### 2.1 定义抽象类

```java
// 抽象类
abstract class Animal {
    String name;
    
    // 普通方法
    public void sleep() {
        System.out.println("睡觉");
    }
    
    // 抽象方法（没有方法体）
    public abstract void makeSound();
}
```

### 2.2 继承抽象类

```java
// 子类必须实现抽象方法
class Dog extends Animal {
    @Override
    public void makeSound() {
        System.out.println("汪汪叫");
    }
}

class Cat extends Animal {
    @Override
    public void makeSound() {
        System.out.println("喵喵叫");
    }
}

// 使用
public class Main {
    public static void main(String[] args) {
        // ❌ 错误：抽象类不能实例化
        // Animal a = new Animal();
        
        // ✅ 正确：创建子类对象
        Animal a1 = new Dog();
        Animal a2 = new Cat();
        
        a1.makeSound();  // 汪汪叫
        a2.makeSound();  // 喵喵叫
    }
}
```

## 三、抽象方法

### 3.1 抽象方法的特点

```java
abstract class Animal {
    // ✅ 抽象方法：没有方法体
    public abstract void makeSound();
    
    // ❌ 错误：抽象方法不能有方法体
    // public abstract void makeSound() { }
    
    // ❌ 错误：抽象方法不能是 private
    // private abstract void method();
    
    // ❌ 错误：抽象方法不能是 static
    // static abstract void method();
    
    // ❌ 错误：抽象方法不能是 final
    // final abstract void method();
}
```

### 3.2 抽象方法的访问修饰符

```java
abstract class Animal {
    // ✅ 可以是 public
    public abstract void method1();
    
    // ✅ 可以是 protected
    protected abstract void method2();
    
    // ✅ 可以是 default（包访问权限）
    abstract void method3();
    
    // ❌ 不能是 private
    // private abstract void method4();
}
```

## 四、抽象类的应用

### 4.1 图形系统

```java
// 抽象类：图形
abstract class Shape {
    protected String color;
    
    public Shape(String color) {
        this.color = color;
    }
    
    // 普通方法
    public String getColor() {
        return color;
    }
    
    // 抽象方法
    public abstract double getArea();
    
    public abstract double getPerimeter();
}

// 子类：圆形
class Circle extends Shape {
    private double radius;
    
    public Circle(String color, double radius) {
        super(color);
        this.radius = radius;
    }
    
    @Override
    public double getArea() {
        return Math.PI * radius * radius;
    }
    
    @Override
    public double getPerimeter() {
        return 2 * Math.PI * radius;
    }
}

// 子类：矩形
class Rectangle extends Shape {
    private double width;
    private double height;
    
    public Rectangle(String color, double width, double height) {
        super(color);
        this.width = width;
        this.height = height;
    }
    
    @Override
    public double getArea() {
        return width * height;
    }
    
    @Override
    public double getPerimeter() {
        return 2 * (width + height);
    }
}

// 使用
public class ShapeDemo {
    public static void main(String[] args) {
        Shape[] shapes = {
            new Circle("红色", 5),
            new Rectangle("蓝色", 4, 6)
        };
        
        for (Shape shape : shapes) {
            System.out.println("颜色：" + shape.getColor());
            System.out.println("面积：" + shape.getArea());
            System.out.println("周长：" + shape.getPerimeter());
            System.out.println();
        }
    }
}
```

### 4.2 员工系统

```java
// 抽象类：员工
abstract class Employee {
    protected String name;
    protected double baseSalary;
    
    public Employee(String name, double baseSalary) {
        this.name = name;
        this.baseSalary = baseSalary;
    }
    
    // 普通方法
    public void work() {
        System.out.println(name + " 在工作");
    }
    
    // 抽象方法：计算工资
    public abstract double calculateSalary();
    
    // 抽象方法：获取职位
    public abstract String getPosition();
}

// 子类：经理
class Manager extends Employee {
    private double bonus;
    
    public Manager(String name, double baseSalary, double bonus) {
        super(name, baseSalary);
        this.bonus = bonus;
    }
    
    @Override
    public double calculateSalary() {
        return baseSalary + bonus;
    }
    
    @Override
    public String getPosition() {
        return "经理";
    }
    
    @Override
    public void work() {
        System.out.println(name + " 在管理团队");
    }
}

// 子类：程序员
class Programmer extends Employee {
    private String language;
    
    public Programmer(String name, double baseSalary, String language) {
        super(name, baseSalary);
        this.language = language;
    }
    
    @Override
    public double calculateSalary() {
        return baseSalary;
    }
    
    @Override
    public String getPosition() {
        return "程序员";
    }
    
    @Override
    public void work() {
        System.out.println(name + " 在用" + language + "写代码");
    }
}

// 使用
public class Company {
    public static void main(String[] args) {
        Employee[] employees = {
            new Manager("张三", 20000, 10000),
            new Programmer("李四", 15000, "Java"),
            new Programmer("王五", 15000, "Python")
        };
        
        double totalSalary = 0;
        for (Employee emp : employees) {
            emp.work();
            System.out.println(emp.getPosition() + " " + emp.name + 
                             " 工资：" + emp.calculateSalary());
            totalSalary += emp.calculateSalary();
        }
        
        System.out.println("总工资：" + totalSalary);
    }
}
```

### 4.3 支付系统

```java
// 抽象类：支付方式
abstract class Payment {
    protected String accountId;
    
    public Payment(String accountId) {
        this.accountId = accountId;
    }
    
    // 模板方法
    public final void pay(double amount) {
        if (validate()) {
            deduct(amount);
            System.out.println("支付成功：" + amount);
        } else {
            System.out.println("支付失败");
        }
    }
    
    // 抽象方法
    protected abstract boolean validate();
    protected abstract void deduct(double amount);
}

// 子类：支付宝
class Alipay extends Payment {
    public Alipay(String accountId) {
        super(accountId);
    }
    
    @Override
    protected boolean validate() {
        System.out.println("验证支付宝账户：" + accountId);
        return true;
    }
    
    @Override
    protected void deduct(double amount) {
        System.out.println("从支付宝扣除：" + amount);
    }
}

// 子类：微信
class WechatPay extends Payment {
    public WechatPay(String accountId) {
        super(accountId);
    }
    
    @Override
    protected boolean validate() {
        System.out.println("验证微信账户：" + accountId);
        return true;
    }
    
    @Override
    protected void deduct(double amount) {
        System.out.println("从微信扣除：" + amount);
    }
}

// 使用
public class PaymentDemo {
    public static void main(String[] args) {
        Payment p1 = new Alipay("alipay123");
        Payment p2 = new WechatPay("wechat456");
        
        p1.pay(100);
        p2.pay(200);
    }
}
```

## 五、抽象类 vs 普通类

| 特性 | 抽象类 | 普通类 |
|------|--------|--------|
| 实例化 | 不能 | 能 |
| 抽象方法 | 可以有 | 不能有 |
| 普通方法 | 可以有 | 可以有 |
| 构造方法 | 可以有 | 可以有 |
| 成员变量 | 可以有 | 可以有 |

## 六、注意事项

### 6.1 抽象类必须有子类

```java
// 定义了抽象类但没有子类，没有意义
abstract class UselessAbstractClass {
    public abstract void method();
}
```

### 6.2 子类必须实现所有抽象方法

```java
abstract class Animal {
    public abstract void eat();
    public abstract void sleep();
}

// ❌ 错误：没有实现所有抽象方法
// class Dog extends Animal {
//     @Override
//     public void eat() { }
//     // 缺少 sleep() 实现
// }

// ✅ 正确
class Dog extends Animal {
    @Override
    public void eat() {
        System.out.println("狗吃骨头");
    }
    
    @Override
    public void sleep() {
        System.out.println("狗睡觉");
    }
}
```

### 6.3 抽象类可以有构造方法

```java
abstract class Animal {
    String name;
    
    // 构造方法供子类调用
    public Animal(String name) {
        this.name = name;
    }
}

class Dog extends Animal {
    public Dog(String name) {
        super(name);  // 调用父类构造
    }
}
```

## 七、小结

本节要点：
1. **抽象类**：用 abstract 修饰，不能实例化
2. **抽象方法**：没有方法体，子类必须实现
3. **应用**：定义模板、规范子类行为
4. **注意**：抽象方法不能是 private、static、final
