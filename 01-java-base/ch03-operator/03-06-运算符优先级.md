# 3.6 运算符优先级

## 一、运算符优先级表

优先级从高到低排列：

| 优先级 | 运算符 | 说明 | 结合性 |
|--------|--------|------|--------|
| 1 | `()` `[]` | 括号、数组下标 | 从左到右 |
| 2 | `++` `--` `!` `~` `+` `-` | 自增自减、逻辑非、按位非、正负号 | 从右到左 |
| 3 | `*` `/` `%` | 乘、除、取模 | 从左到右 |
| 4 | `+` `-` | 加、减 | 从左到右 |
| 5 | `<<` `>>` `>>>` | 位移运算 | 从左到右 |
| 6 | `<` `>` `<=` `>=` | 关系运算 | 从左到右 |
| 7 | `==` `!=` | 相等运算 | 从左到右 |
| 8 | `&` | 按位与 | 从左到右 |
| 9 | `^` | 按位异或 | 从左到右 |
| 10 | `\|` | 按位或 | 从左到右 |
| 11 | `&&` | 短路与 | 从左到右 |
| 12 | `\|\|` | 短路或 | 从左到右 |
| 13 | `?:` | 三元运算符 | 从右到左 |
| 14 | `=` `+=` `-=` ... | 赋值运算 | 从右到左 |

## 二、记忆口诀

```
括号老大最优先，
自增自减逻辑非，
乘除取模加减法，
关系相等按位算，
短路与或三元尾，
赋值运算优先级低。
```

## 三、常见优先级示例

### 3.1 算术运算优先级

```java
public class ArithmeticPriority {
    public static void main(String[] args) {
        // 先乘除后加减
        int result1 = 3 + 4 * 5;  // 23，不是 35
        
        // 括号优先
        int result2 = (3 + 4) * 5;  // 35
        
        // 同级从左到右
        int result3 = 10 - 5 + 2;  // 7，不是 3
        
        System.out.println("result1 = " + result1);
        System.out.println("result2 = " + result2);
        System.out.println("result3 = " + result3);
    }
}
```

### 3.2 关系与逻辑运算优先级

```java
public class LogicPriority {
    public static void main(String[] args) {
        int a = 5, b = 10, c = 15;
        
        // 关系运算优先级高于逻辑运算
        boolean r1 = a < b && b < c;  // true
        // 等价于：(a < b) && (b < c)
        
        // 逻辑非优先级最高
        boolean r2 = !a > b;  // ❌ 错误，! 只能用于 boolean
        boolean r3 = !(a > b);  // ✅ true
        
        // || 优先级低于 &&
        boolean r4 = a > b && b < c || a < c;  // true
        // 等价于：((a > b) && (b < c)) || (a < c)
        
        System.out.println("r1 = " + r1);
        System.out.println("r3 = " + r3);
        System.out.println("r4 = " + r4);
    }
}
```

### 3.3 自增自减优先级

```java
public class IncrementPriority {
    public static void main(String[] args) {
        int a = 5;
        
        // 自增优先级高于算术运算
        int b = ++a * 2;  // 12，先自增 a=6，再乘 2
        System.out.println("b = " + b);
        
        a = 5;
        int c = a++ * 2;  // 10，先用 a=5 乘 2，再自增 a=6
        System.out.println("c = " + c);
        System.out.println("a = " + a);  // 6
    }
}
```

### 3.4 赋值运算优先级

```java
public class AssignPriority {
    public static void main(String[] args) {
        int a, b, c;
        
        // 赋值优先级最低，从右向左
        a = b = c = 10;
        System.out.println("a = " + a);  // 10
        System.out.println("b = " + b);  // 10
        System.out.println("c = " + c);  // 10
        
        // 扩展赋值包含隐式转换
        int x = 10;
        x += 5.5;  // 等价于 x = (int)(x + 5.5)
        System.out.println("x = " + x);  // 15
    }
}
```

## 四、复杂表达式分析

### 4.1 示例 1

```java
public class ComplexExpr1 {
    public static void main(String[] args) {
        int a = 5, b = 10, c = 15;
        
        // 分析步骤：
        // 1. a + b * c → a + (b * c) → 5 + 150 → 155
        // 2. (a + b) * c → 15 * 15 → 225
        int result1 = a + b * c;
        int result2 = (a + b) * c;
        
        System.out.println("result1 = " + result1);  // 155
        System.out.println("result2 = " + result2);  // 225
    }
}
```

### 4.2 示例 2

```java
public class ComplexExpr2 {
    public static void main(String[] args) {
        int x = 10, y = 5, z = 3;
        
        // 分析：
        // 1. x > y → true
        // 2. y > z → true
        // 3. true && true → true
        boolean result = x > y && y > z;
        
        System.out.println("result = " + result);  // true
    }
}
```

### 4.3 示例 3

```java
public class ComplexExpr3 {
    public static void main(String[] args) {
        int a = 5, b = 10;
        
        // 三元运算符优先级低于关系运算
        int max = a > b ? a : b;
        // 等价于：(a > b) ? a : b
        
        System.out.println("max = " + max);  // 10
    }
}
```

## 五、最佳实践

### 5.1 使用括号明确优先级

```java
// ❌ 可读性差
int result = a + b * c > d && e == f || g;

// ✅ 使用括号明确
boolean condition = ((a + (b * c)) > d) && (e == f) || g;
```

### 5.2 拆分复杂表达式

```java
// ❌ 过于复杂
boolean isValid = age >= 18 && age <= 60 && score >= 60 && score <= 100;

// ✅ 拆分
boolean isAgeValid = age >= 18 && age <= 60;
boolean isScoreValid = score >= 60 && score <= 100;
boolean isValid = isAgeValid && isScoreValid;
```

### 5.3 避免副作用

```java
// ❌ 不推荐：表达式中有副作用
int result = ++i + i++ * i;

// ✅ 推荐：分开写
++i;
int temp = i++;
int result = i + temp * i;
```

## 六、常见错误

### 6.1 误判优先级

```java
// ❌ 错误理解
int x = 5 > 3 ? 10 : 20 + 1;
// 实际：5 > 3 ? 10 : (20 + 1) → 10
// 不是：(5 > 3 ? 10 : 20) + 1 → 11

// ✅ 正确理解
int y = (5 > 3 ? 10 : 20) + 1;  // 11
```

### 6.2 连等比较

```java
int a = 5, b = 10, c = 15;

// ❌ 错误：数学写法不能直接用于 Java
// if (a < b < c) {}

// ✅ 正确
if (a < b && b < c) {}
```

### 6.3 位运算优先级

```java
// ❌ 错误：== 优先级高于 &
if (x & 1 == 0) {}  // 编译错误

// ✅ 正确
if ((x & 1) == 0) {}  // 判断偶数
```

## 七、综合示例

```java
public class PriorityExample {
    public static void main(String[] args) {
        // 1. 计算表达式
        int a = 10, b = 5, c = 2;
        int result = a + b * c - a / b;  // 10 + 10 - 2 = 18
        System.out.println("result = " + result);
        
        // 2. 复杂条件判断
        int score = 85;
        int age = 20;
        boolean isMember = true;
        
        boolean discount = (score >= 80 || age < 18) && isMember;
        System.out.println("是否优惠：" + discount);
        
        // 3. 最大值
        int x = 15, y = 25, z = 20;
        int max = x > y ? (x > z ? x : z) : (y > z ? y : z);
        System.out.println("最大值：" + max);
        
        // 4. 闰年判断
        int year = 2026;
        boolean isLeap = (year % 4 == 0 && year % 100 != 0) || (year % 400 == 0);
        System.out.println("闰年：" + isLeap);
    }
}
```

## 八、小结

本节要点：
1. **优先级最高**：括号 `()`
2. **优先级最低**：赋值运算符
3. **算术优先级**：乘除 > 加减
4. **逻辑优先级**：`!` > `&` > `^` > `|` > `&&` > `||`
5. **最佳实践**：使用括号明确优先级，拆分复杂表达式
