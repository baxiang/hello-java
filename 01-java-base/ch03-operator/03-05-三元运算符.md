# 3.5 三元运算符

## 一、三元运算符概述

三元运算符是 Java 中唯一的三目运算符，用于简化简单的 if-else 语句。

**语法**：
```java
条件表达式 ? 表达式 1 : 表达式 2
```

**执行流程**：
1. 如果条件为 true，返回表达式 1 的值
2. 如果条件为 false，返回表达式 2 的值

## 二、基本用法

### 2.1 简单示例

```java
public class TernaryDemo {
    public static void main(String[] args) {
        int a = 10;
        int b = 20;
        
        // 求最大值
        int max = a > b ? a : b;
        System.out.println("最大值：" + max);  // 20
        
        // 等价于 if-else
        /*
        int max;
        if (a > b) {
            max = a;
        } else {
            max = b;
        }
        */
    }
}
```

### 2.2 判断奇偶

```java
public class OddEven {
    public static void main(String[] args) {
        int num = 7;
        
        String result = num % 2 == 0 ? "偶数" : "奇数";
        System.out.println(num + " 是 " + result);
    }
}
```

### 2.3 判断及格

```java
public class PassCheck {
    public static void main(String[] args) {
        int score = 75;
        
        String level = score >= 60 ? "及格" : "不及格";
        System.out.println(level);
    }
}
```

## 三、嵌套三元运算符

### 3.1 多层判断

```java
public class NestedTernary {
    public static void main(String[] args) {
        int score = 85;
        
        // 嵌套三元运算
        String grade = score >= 90 ? "优秀" 
                     : score >= 80 ? "良好"
                     : score >= 60 ? "及格"
                     : "不及格";
        
        System.out.println("等级：" + grade);  // 良好
    }
}
```

### 3.2 比较三个数

```java
public class CompareThree {
    public static void main(String[] args) {
        int a = 10, b = 25, c = 15;
        
        // 求最大值
        int max = a > b ? (a > c ? a : c) : (b > c ? b : c);
        System.out.println("最大值：" + max);  // 25
        
        // 求最小值
        int min = a < b ? (a < c ? a : c) : (b < c ? b : c);
        System.out.println("最小值：" + min);  // 10
    }
}
```

## 四、应用场景

### 4.1 绝对值

```java
public class AbsoluteValue {
    public static void main(String[] args) {
        int num = -10;
        
        int abs = num >= 0 ? num : -num;
        System.out.println("绝对值：" + abs);  // 10
    }
}
```

### 4.2 最大值最小值

```java
public class MaxMin {
    public static void main(String[] args) {
        int x = 100;
        int y = 200;
        
        int max = x > y ? x : y;
        int min = x < y ? x : y;
        
        System.out.println("最大值：" + max);
        System.out.println("最小值：" + min);
    }
}
```

### 4.3 性别判断

```java
public class GenderCheck {
    public static void main(String[] args) {
        char gender = 'M';
        
        String desc = gender == 'M' ? "男" : "女";
        System.out.println("性别：" + desc);
    }
}
```

### 4.4 折扣计算

```java
public class Discount {
    public static void main(String[] args) {
        double price = 100.0;
        boolean isMember = true;
        
        // 会员 8 折，非会员 9 折
        double finalPrice = isMember ? price * 0.8 : price * 0.9;
        System.out.println("最终价格：" + finalPrice);
    }
}
```

## 五、与 if-else 的对比

### 5.1 三元运算符

```java
// 简洁
int max = a > b ? a : b;

// 可直接用于赋值和输出
String status = age >= 18 ? "成年" : "未成年";
System.out.println(score >= 60 ? "及格" : "不及格");
```

### 5.2 if-else 语句

```java
// 适合复杂逻辑
int max;
if (a > b) {
    max = a;
} else {
    max = b;
}

// 适合多分支
if (score >= 90) {
    grade = "优秀";
} else if (score >= 80) {
    grade = "良好";
} else {
    grade = "不及格";
}
```

### 5.3 选择建议

| 场景 | 推荐方式 |
|------|----------|
| 简单二选一 | 三元运算符 |
| 需要赋值 | 三元运算符 |
| 多分支判断 | if-else |
| 复杂逻辑 | if-else |
| 需要执行多条语句 | if-else |

## 六、注意事项

### 6.1 类型兼容

```java
// 两个表达式类型必须兼容
int x = 5 > 3 ? 10 : 20;        // ✅ 都是 int
double y = 5 > 3 ? 10 : 20.5;   // ✅ int 可转 double

// ❌ 类型不兼容
// String s = 5 > 3 ? 10 : "hello";
```

### 6.2 优先级

```java
int a = 10, b = 20;

// 三元运算符优先级较低
int max = a > b ? a : b;  // ✅ 正确

// 建议加括号
int result = (a > b) ? a : b;
```

### 6.3 可读性

```java
// ✅ 简洁清晰
String status = isLoggedIn ? "已登录" : "未登录";

// ❌ 过于复杂，降低可读性
String result = a > 0 ? (b > 0 ? (c > 0 ? "都大于 0" : "c 不大于 0") : "b 不大于 0") : "a 不大于 0";

// ✅ 使用 if-else 更清晰
String result;
if (a > 0) {
    if (b > 0) {
        if (c > 0) {
            result = "都大于 0";
        } else {
            result = "c 不大于 0";
        }
    } else {
        result = "b 不大于 0";
    }
} else {
    result = "a 不大于 0";
}
```

## 七、综合示例

```java
public class TernaryExample {
    public static void main(String[] args) {
        // 1. 判断正负
        int num = -5;
        String sign = num > 0 ? "正数" : num < 0 ? "负数" : "零";
        System.out.println(num + " 是 " + sign);
        
        // 2. 闰年判断
        int year = 2026;
        String leapYear = (year % 4 == 0 && year % 100 != 0) || (year % 400 == 0) 
                        ? "闰年" : "平年";
        System.out.println(year + " 是 " + leapYear);
        
        // 3. 票价计算
        int age = 25;
        double price = 100.0;
        double ticketPrice = age < 12 ? price * 0.5 
                          : age >= 65 ? price * 0.7 
                          : price;
        System.out.println("票价：" + ticketPrice);
        
        // 4. 字符串拼接
        int score = 85;
        String info = "成绩：" + score + "，" + (score >= 60 ? "及格" : "不及格");
        System.out.println(info);
    }
}
```

## 八、小结

本节要点：
1. **语法**：`条件 ? 表达式 1 : 表达式 2`
2. **执行**：条件为 true 返回表达式 1，否则返回表达式 2
3. **嵌套**：可以多层嵌套使用
4. **优势**：简化简单 if-else，代码更简洁
5. **注意**：避免过度嵌套，保持可读性
