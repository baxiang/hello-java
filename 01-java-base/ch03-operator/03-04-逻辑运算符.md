# 3.4 逻辑运算符

## 一、逻辑运算符概述

逻辑运算符用于连接多个布尔表达式，进行逻辑运算。

| 运算符 | 说明 | 示例 | 结果 |
|--------|------|------|------|
| `&` | 逻辑与 | `true & false` | false |
| `|` | 逻辑或 | `true \| false` | true |
| `!` | 逻辑非 | `!true` | false |
| `^` | 逻辑异或 | `true ^ false` | true |
| `&&` | 短路与 | `true && false` | false |
| `||` | 短路或 | `true \|\| false` | true |

## 二、基本逻辑运算

### 2.1 逻辑与（&）

```java
public class AndDemo {
    public static void main(String[] args) {
        boolean a = true;
        boolean b = false;
        
        // 与运算：全真才真，一假必假
        System.out.println(true & true);   // true
        System.out.println(true & false);  // false
        System.out.println(false & true);  // false
        System.out.println(false & false); // false
        
        // 应用场景
        int age = 25;
        int score = 85;
        
        if (age >= 18 & score >= 60) {
            System.out.println("成年且及格");
        }
    }
}
```

### 2.2 逻辑或（|）

```java
public class OrDemo {
    public static void main(String[] args) {
        // 或运算：一真必真，全假才假
        System.out.println(true | true);   // true
        System.out.println(true | false);  // true
        System.out.println(false | true);  // true
        System.out.println(false | false); // false
        
        // 应用场景
        int score = 55;
        boolean hasMakeup = true;  // 是否补考
        
        if (score >= 60 | hasMakeup) {
            System.out.println("通过");
        }
    }
}
```

### 2.3 逻辑非（!）

```java
public class NotDemo {
    public static void main(String[] args) {
        // 非运算：取反
        System.out.println(!true);   // false
        System.out.println(!false);  // true
        
        // 应用场景
        boolean isRain = false;
        
        if (!isRain) {
            System.out.println("不下雨，可以出门");
        }
        
        // 双重否定
        System.out.println(!!true);  // true
    }
}
```

### 2.4 逻辑异或（^）

```java
public class XorDemo {
    public static void main(String[] args) {
        // 异或运算：相同为假，不同为真
        System.out.println(true ^ true);   // false
        System.out.println(true ^ false);  // true
        System.out.println(false ^ true);  // true
        System.out.println(false ^ false); // false
        
        // 应用场景：开关切换
        boolean switchState = true;
        switchState = switchState ^ true;  // 切换状态
        System.out.println(switchState);   // false
    }
}
```

## 三、短路与 && 短路或

### 3.1 短路与（&&）

```java
public class ShortAndDemo {
    public static void main(String[] args) {
        // 短路特性：第一个为 false，不再计算第二个
        boolean result1 = false && (5 > 3);
        System.out.println(result1);  // false
        
        // 应用场景：避免空指针
        String str = null;
        
        // ❌ 会空指针
        // if (str != null & str.equals("hello")) {}
        
        // ✅ 短路保护
        if (str != null && str.equals("hello")) {
            System.out.println("是 hello");
        } else {
            System.out.println("str 为 null 或不是 hello");
        }
    }
}
```

### 3.2 短路或（||）

```java
public class ShortOrDemo {
    public static void main(String[] args) {
        // 短路特性：第一个为 true，不再计算第二个
        boolean result1 = true || (5 > 3);
        System.out.println(result1);  // true
        
        // 应用场景：默认值
        String input = "";
        String name = input || "默认用户";  // 伪代码
        
        // 实际写法
        if (input == null || input.isEmpty()) {
            name = "默认用户";
        }
    }
}
```

### 3.3 & 与 && 的区别

```java
public class AndCompare {
    public static void main(String[] args) {
        int a = 10;
        
        // &：两边都执行
        boolean r1 = (a > 5) & (a++ > 10);
        System.out.println("a = " + a);  // 11，a++ 执行了
        
        a = 10;
        // &&：短路，右边不执行
        boolean r2 = (a > 5) && (a++ > 10);
        System.out.println("a = " + a);  // 10，a++ 未执行
    }
}
```

## 四、运算符优先级

```
优先级从高到低：
! > & > ^ > | > && > ||
```

```java
public class PriorityDemo {
    public static void main(String[] args) {
        int a = 5, b = 10, c = 15;
        
        // 优先级示例
        boolean r1 = a < b && b < c;  // true
        // 等价于：(a < b) && (b < c)
        
        boolean r2 = a > b || b < c && a < c;  // true
        // 等价于：(a > b) || ((b < c) && (a < c))
        
        // 推荐使用括号明确优先级
        boolean r3 = (a > b) || (b < c && a < c);
    }
}
```

## 五、应用场景

### 5.1 用户登录验证

```java
public class LoginCheck {
    public static void main(String[] args) {
        String username = "admin";
        String password = "123456";
        
        // 验证用户名和密码
        boolean isUserCorrect = username.equals("admin");

            System.out.println("登录成功");

```

### 5.2 成绩等级判断

```java
public class GradeCheck {
    public static void main(String[] args) {
        int score = 85;
        
        // 优秀：90-100
        boolean isExcellent = score >= 90 && score <= 100;
        
        // 良好：80-89
        boolean isGood = score >= 80 && score < 90;
        
        // 及格：60-79
        boolean isPass = score >= 60 && score < 80;
        
        // 不及格：0-59
        boolean isFail = score >= 0 && score < 60;
        
        System.out.println("优秀：" + isExcellent);
        System.out.println("良好：" + isGood);
        System.out.println("及格：" + isPass);
        System.out.println("不及格：" + isFail);
    }
}
```

### 5.3 闰年判断

public class LeapYearCheck {

        int year = 2026;

## 六、德·摩根定律

```
!(A && B) = !A || !B
!(A || B) = !A && !B
```

```java
public class DeMorganDemo {
    public static void main(String[] args) {
        boolean a = true;
        boolean b = false;
        
        // !(A && B) = !A || !B
        System.out.println(!(a && b));  // true
        System.out.println(!a || !b);   // true
        
        // !(A || B) = !A && !B
        System.out.println(!(a || b));  // false
        System.out.println(!a && !b);   // false
    }
}
```

## 七、综合示例

```java
public class LogicExample {
    public static void main(String[] args) {
        // 判断是否适合户外运动
        boolean isRain = false;
        boolean isTooHot = true;
        boolean isWeekend = true;
        
        boolean canOutdoor = !isRain && !isTooHot && isWeekend;
        System.out.println("适合户外运动：" + canOutdoor);
        
        // 判断是否可以申请贷款
        int age = 25;
        int income = 8000;
        int creditScore = 650;
        
        boolean canLoan = age >= 18 && age <= 60 
                       && income >= 5000 
                       && creditScore >= 600;
        System.out.println("可以贷款：" + canLoan);
    }
}
```

## 八、小结

本节要点：
1. **基本逻辑运算**：`&`、`|`、`!`、`^`
2. **短路运算**：`&&`、`||`，提高效率
3. **优先级**：`!` > `&` > `^` > `|` > `&&` > `||`
4. **短路特性**：&& 遇 false 停止，|| 遇 true 停止
5. **应用**：条件判断、空指针保护
