# 3.1 算术运算符

## 一、算术运算符概述

算术运算符用于执行基本的数学运算。

| 运算符 | 说明 | 示例 | 结果 |
|--------|------|------|------|
| `+` | 加法 | `5 + 3` | 8 |
| `-` | 减法 | `5 - 3` | 2 |
| `*` | 乘法 | `5 * 3` | 15 |
| `/` | 除法 | `6 / 3` | 2 |
| `%` | 取模（取余） | `7 % 3` | 1 |
| `++` | 自增 | `i++` | i+1 |
| `--` | 自减 | `i--` | i-1 |

## 二、基本算术运算

### 2.1 加减乘

```java
public class ArithmeticDemo {
    public static void main(String[] args) {
        int a = 10;
        int b = 3;
        
        // 加法
        int sum = a + b;
        System.out.println("和：" + sum);  // 13
        
        // 减法
        int diff = a - b;
        System.out.println("差：" + diff);  // 7
        
        // 乘法
        int product = a * b;
        System.out.println("积：" + product);  // 30
    }
}
```

### 2.2 除法

```java
public class DivisionDemo {
    public static void main(String[] args) {
        // 整数除法：结果取整
        int a = 10;
        int b = 3;
        int result1 = a / b;
        System.out.println("10 / 3 = " + result1);  // 3
        
        // 浮点数除法：保留小数
        double c = 10.0;
        double d = 3.0;
        double result2 = c / d;
        System.out.println("10.0 / 3.0 = " + result2);  // 3.333...
        
        // 整数转浮点再除
        int x = 10;
        int y = 3;
        double result3 = (double) x / y;
        System.out.println("(double)10 / 3 = " + result3);  // 3.333...
        
        // 除以零
        // int z = 10 / 0;  // ❌ ArithmeticException
        double w = 10.0 / 0.0;  // ✅ Infinity
        System.out.println(w);  // Infinity
    }
}
```

### 2.3 取模（取余）

```java
public class ModuloDemo {
    public static void main(String[] args) {
        // 基本取模
        int a = 7 % 3;
        System.out.println("7 % 3 = " + a);  // 1
        
        // 应用场景 1：判断奇偶
        int num = 10;
        if (num % 2 == 0) {
            System.out.println(num + " 是偶数");
        } else {
            System.out.println(num + " 是奇数");
        }
        
        // 应用场景 2：判断整除
        if (15 % 5 == 0) {
            System.out.println("15 能被 5 整除");
        }
        
        // 应用场景 3：循环
        for (int i = 1; i <= 10; i++) {
            if (i % 3 == 0) {
                System.out.print(i + " ");  // 3 6 9
            }
        }
        
        // 负数取模
        int b = -7 % 3;
        System.out.println("-7 % 3 = " + b);  // -1
    }
}
```

## 三、自增自减运算符

### 3.1 基本用法

```java
public class IncrementDemo {
    public static void main(String[] args) {
        int a = 5;
        
        // 自增
        a++;  // a = a + 1
        System.out.println("a++: " + a);  // 6
        
        ++a;  // a = a + 1
        System.out.println("++a: " + a);  // 7
        
        // 自减
        a--;  // a = a - 1
        System.out.println("a--: " + a);  // 6
        
        --a;  // a = a - 1
        System.out.println("--a: " + a);  // 5
    }
}
```

### 3.2 前缀 vs 后缀

```java
public class PrefixSuffixDemo {
    public static void main(String[] args) {
        int a = 5;
        
        // 后缀：先使用，后自增
        int b = a++;  // b = 5, 然后 a = 6
        System.out.println("b = " + b);  // 5
        System.out.println("a = " + a);  // 6
        
        // 前缀：先自增，后使用
        int c = ++a;  // a = 7, 然后 c = 7
        System.out.println("c = " + c);  // 7
        System.out.println("a = " + a);  // 7
        
        // 后缀自减
        int d = 5;
        int e = d--;  // e = 5, 然后 d = 4
        System.out.println("e = " + e);  // 5
        System.out.println("d = " + d);  // 4
        
        // 前缀自减
        int f = --d;  // d = 3, 然后 f = 3
        System.out.println("f = " + f);  // 3
        System.out.println("d = " + d);  // 3
    }
}
```

### 3.3 记忆口诀

```
++ 在前：先加后用
++ 在后：先用后加
-- 在前：先减后用
-- 在后：先用后减
```

### 3.4 复杂表达式

```java
public class ComplexDemo {
    public static void main(String[] args) {
        int i = 5;
        
        // 在表达式中使用
        int j = i++ + ++i;
        // 执行顺序：
        // 1. i++ → 取 i=5，然后 i=6
        // 2. ++i → i=7，取 i=7
        // 3. j = 5 + 7 = 12
        System.out.println("j = " + j);  // 12
        System.out.println("i = " + i);  // 7
        
        // 不推荐：过于复杂，可读性差
        // int k = i++ + ++i * i--;
    }
}
```

## 四、字符的算术运算

```java
public class CharArithmetic {
    public static void main(String[] args) {
        char c = 'A';
        
        // 字符参与运算，使用 ASCII 码
        int code = c + 1;
        System.out.println(code);  // 66
        
        // 字符转换
        char next = (char) (c + 1);
        System.out.println(next);  // 'B'
        
        // 大小写转换
        char lower = 'a';
        char upper = (char) (lower - 32);
        System.out.println(upper);  // 'A'
    }
}
```

## 五、综合示例

### 5.1 数字拆分

```java
public class NumberSplit {
    public static void main(String[] args) {
        int num = 1234;
        
        // 提取个位
        int ge = num % 10;  // 4
        
        // 提取十位
        int shi = (num / 10) % 10;  // 3
        
        // 提取百位
        int bai = (num / 100) % 10;  // 2
        
        // 提取千位
        int qian = num / 1000;  // 1
        
        System.out.println("千位：" + qian);
        System.out.println("百位：" + bai);
        System.out.println("十位：" + shi);
        System.out.println("个位：" + ge);
    }
}
```

### 5.2 时间转换

```java
public class TimeConvert {
    public static void main(String[] args) {
        int totalSeconds = 3665;
        
        // 计算小时
        int hours = totalSeconds / 3600;  // 1
        
        // 计算分钟
        int minutes = (totalSeconds % 3600) / 60;  // 1
        
        // 计算秒
        int seconds = totalSeconds % 60;  // 5
        
        System.out.println(hours + "小时 " + minutes + "分 " + seconds + "秒");
        // 输出：1 小时 1 分 5 秒
    }
}
```

## 六、注意事项

### 6.1 整数溢出

```java
int max = Integer.MAX_VALUE;  // 2147483647
int overflow = max + 1;       // -2147483648，溢出
System.out.println(overflow);

// 解决方案：使用 long
long safe = (long) max + 1;   // 2147483648
```

### 6.2 浮点数精度

```java
double a = 0.1;
double b = 0.2;
double sum = a + b;
System.out.println(sum);  // 0.30000000000000004

// 精确计算使用 BigDecimal
```

### 6.3 运算符优先级

```java
int result = 3 + 4 * 5;  // 23，不是 35
// 优先级：++ -- > * / % > + -
```

## 七、小结

本节要点：
1. **基本运算符**：+、-、*、/、%
2. **整数除法**：结果取整，注意类型转换
3. **取模运算**：判断奇偶、整除
4. **自增自减**：前缀先变后用，后缀先用后变
5. **优先级**：++ -- > * / % > + -
