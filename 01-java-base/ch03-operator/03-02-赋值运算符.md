# 3.2 赋值运算符

## 一、赋值运算符概述

赋值运算符用于将值赋给变量。

| 运算符 | 说明 | 示例 | 等价形式 |
|--------|------|------|----------|
| `=` | 基本赋值 | `a = 5` | - |
| `+=` | 加后赋值 | `a += 5` | `a = a + 5` |
| `-=` | 减后赋值 | `a -= 5` | `a = a - 5` |
| `*=` | 乘后赋值 | `a *= 5` | `a = a * 5` |
| `/=` | 除后赋值 | `a /= 5` | `a = a / 5` |
| `%=` | 取模后赋值 | `a %= 5` | `a = a % 5` |

## 二、基本赋值运算符

### 2.1 简单赋值

```java
public class AssignDemo {
    public static void main(String[] args) {
        int a = 10;
        int b = a;  // 将 a 的值赋给 b
        
        System.out.println("a = " + a);  // 10
        System.out.println("b = " + b);  // 10
        
        // 修改 a，b 不变
        a = 20;
        System.out.println("a = " + a);  // 20
        System.out.println("b = " + b);  // 10
    }
}
```

### 2.2 连续赋值

```java
public class ChainAssign {
    public static void main(String[] args) {
        int a, b, c;
        a = b = c = 10;  // 从右向左赋值
        
        System.out.println("a = " + a);  // 10
        System.out.println("b = " + b);  // 10
        System.out.println("c = " + c);  // 10
    }
}
```

## 三、扩展赋值运算符

### 3.1 加减乘除模赋值

```java
public class CompoundAssign {
    public static void main(String[] args) {
        int a = 10;
        
        // 加赋值
        a += 5;  // a = a + 5
        System.out.println("a += 5: " + a);  // 15
        
        // 减赋值
        a -= 3;  // a = a - 3
        System.out.println("a -= 3: " + a);  // 12
        
        // 乘赋值
        a *= 2;  // a = a * 2
        System.out.println("a *= 2: " + a);  // 24
        
        // 除赋值
        a /= 4;  // a = a / 4
        System.out.println("a /= 4: " + a);  // 6
        
        // 模赋值
        a %= 4;  // a = a % 4
        System.out.println("a %= 4: " + a);  // 2
    }
}
```

### 3.2 自动类型转换

```java
public class TypeConvert {
    public static void main(String[] args) {
        short s = 10;
        
        // ❌ 错误：需要强制转换
        // s = s + 5;
        
        // ✅ 正确：+= 自动进行类型转换
        s += 5;  // 等价于 s = (short)(s + 5)
        System.out.println("s = " + s);  // 15
        
        // 同理
        byte b = 10;
        b += 127;  // 等价于 b = (byte)(b + 127)
        System.out.println("b = " + b);  // -119（溢出）
    }
}
```

## 四、使用场景

### 4.1 累加器

```java
public class Accumulator {
    public static void main(String[] args) {
        int sum = 0;
        
        // 累加 1 到 100
        for (int i = 1; i <= 100; i++) {
            sum += i;  // sum = sum + i
        }
        
        System.out.println("1 到 100 的和：" + sum);  // 5050
    }
}
```

### 4.2 计数器

```java
public class Counter {
    public static void main(String[] args) {
        int count = 0;
        
        // 统计满足条件的数量
        for (int i = 1; i <= 10; i++) {
            if (i % 2 == 0) {
                count++;  // count = count + 1
            }
        }
        
        System.out.println("偶数个数：" + count);  // 5
    }
}
```

### 4.3 数值更新

```java
public class UpdateDemo {
    public static void main(String[] args) {
        // 价格打折
        double price = 100.0;
        price *= 0.8;  // 打 8 折
        System.out.println("打折后：" + price);  // 80.0
        
        // 增长百分比
        int population = 1000;
        population += population * 0.1;  // 增长 10%
        System.out.println("增长后：" + population);  // 1100
    }
}
```

## 五、注意事项

### 5.1 赋值 vs 相等

```java
int a = 5;
// a == 5  // 比较，返回 true
// a = 5   // 赋值，将 5 赋给 a

// if (a = 5) {}  // ❌ 错误：赋值语句没有 boolean 返回值
// if (a == 5) {} // ✅ 正确：比较
```

### 5.2 左边必须是变量

```java
int a = 5;
int b = 10;

// 5 = a;      // ❌ 错误：字面量不能被赋值
// a + b = 10; // ❌ 错误：表达式不能被赋值
```

### 5.3 类型兼容

```java
int i = 10;
// double d = i;    // ✅ 自动转换
// i = (int) 3.14;  // ✅ 强制转换

// String s = i;    // ❌ 错误：类型不兼容
```

## 六、综合示例

```java
public class AssignmentExample {
    public static void main(String[] args) {
        // 初始值
        int score = 100;
        System.out.println("原始分数：" + score);
        
        // 加分
        score += 10;
        System.out.println("加 10 分后：" + score);
        
        // 扣分
        score -= 5;
        System.out.println("扣 5 分后：" + score);
        
        // 翻倍
        score *= 2;
        System.out.println("翻倍后：" + score);
        
        // 平均分配
        score /= 3;
        System.out.println("除以 3 后：" + score);
        
        // 取余
        score %= 7;
        System.out.println("对 7 取余后：" + score);
    }
}
```

## 七、小结

本节要点：
1. **基本赋值**：`=` 从右向左赋值
2. **扩展赋值**：`+=`、`-=`、`*=`、`/=`、`%=`
3. **自动转换**：扩展赋值包含隐式类型转换
4. **常见用途**：累加器、计数器、数值更新
