# 3.3 关系运算符

## 一、关系运算符概述

关系运算符用于比较两个值的大小或相等性，结果是布尔值（true 或 false）。

| 运算符 | 说明 | 示例 | 结果 |
|--------|------|------|------|
| `==` | 等于 | `5 == 5` | true |
| `!=` | 不等于 | `5 != 3` | true |
| `>` | 大于 | `5 > 3` | true |
| `<` | 小于 | `5 < 3` | false |
| `>=` | 大于等于 | `5 >= 5` | true |
| `<=` | 小于等于 | `5 <= 3` | false |

## 二、相等运算符

### 2.1 等于（==）

```java
public class EqualDemo {
    public static void main(String[] args) {
        int a = 10;
        int b = 10;
        int c = 20;
        
        System.out.println(a == b);  // true
        System.out.println(a == c);  // false
        
        // 字符比较
        char ch1 = 'A';
        char ch2 = 'A';
        System.out.println(ch1 == ch2);  // true
        
        // 布尔比较
        boolean flag1 = true;
        boolean flag2 = true;
        System.out.println(flag1 == flag2);  // true
    }
}
```

### 2.2 不等于（!=）

```java
public class NotEqualDemo {
    public static void main(String[] args) {
        int a = 10;
        int b = 20;
        
        System.out.println(a != b);  // true
        System.out.println(a != 10); // false
        
        // 应用场景：判断不为零
        int count = 5;
        if (count != 0) {
            System.out.println("数量不为零");
        }
    }
}
```

### 2.3 数值比较

```java
public class NumericCompare {
    public static void main(String[] args) {
        int x = 10;
        int y = 20;
        
        // 大于
        System.out.println(x > y);   // false
        System.out.println(y > x);   // true
        
        // 小于
        System.out.println(x < y);   // true
        System.out.println(y < x);   // false
        
        // 大于等于
        System.out.println(x >= 10); // true
        System.out.println(x >= 11); // false
        
        // 小于等于
        System.out.println(y <= 20); // true
        System.out.println(y <= 19); // false
    }
}
```

## 三、应用场景

public class AdultCheck {

public class PassCheck {

            System.out.println("及格");

### 3.3 判断范围

```java
public class RangeCheck {
    public static void main(String[] args) {
        int score = 85;
        
        // 判断是否在 0-100 范围内
        if (score >= 0 && score <= 100) {
            System.out.println("分数有效");
        } else {
            System.out.println("分数无效");
        }
        
        // 判断分数段
        if (score >= 90) {
            System.out.println("优秀");
        } else if (score >= 80) {
            System.out.println("良好");
        } else if (score >= 60) {
            System.out.println("及格");
        } else {
            System.out.println("不及格");
        }
    }
}
```

### 3.4 判断闰年

```java
public class LeapYear {
    public static void main(String[] args) {
        int year = 2026;
        
        // 闰年条件：
        // 1. 能被 4 整除但不能被 100 整除
        // 2. 或者能被 400 整除
        boolean isLeap = (year % 4 == 0 && year % 100 != 0) || (year % 400 == 0);
        
        if (isLeap) {
            System.out.println(year + " 是闰年");
        } else {
            System.out.println(year + " 不是闰年");
        }
    }
}
```

## 四、字符比较

```java
public class CharCompare {
    public static void main(String[] args) {
        char c1 = 'A';
        char c2 = 'B';
        char c3 = 'a';
        
        // 字符比较基于 Unicode 值
        System.out.println(c1 < c2);  // true，65 < 66
        System.out.println(c1 < c3);  // true，65 < 97
        
        // 大小写字母
        System.out.println('A' < 'a');  // true
        
        // 数字字符
        System.out.println('0' < '9');  // true
        System.out.println('5' - '0');  // 5，字符转数字
    }
}
```

## 五、注意事项

### 5.1 赋值 vs 比较

```java
int a = 5;

// a = 5   // 赋值表达式，值为 5
// a == 5  // 比较表达式，值为 true

// if (a = 5) {}   // ❌ 编译错误：int 不能转为 boolean
// if (a == 5) {}  // ✅ 正确
```

### 5.2 浮点数比较

```java
double a = 0.1;
double b = 0.2;
double c = a + b;

// ❌ 不推荐：浮点数精度问题
System.out.println(c == 0.3);  // false

// ✅ 推荐：使用误差范围
double epsilon = 0.000001;
System.out.println(Math.abs(c - 0.3) < epsilon);  // true
```

### 5.3 引用类型比较

```java
String s1 = new String("hello");
String s2 = new String("hello");

// == 比较的是引用地址
System.out.println(s1 == s2);  // false

// equals 比较的是内容
System.out.println(s1.equals(s2));  // true
```

## 六、综合示例

```java
public class CompareExample {
    public static void main(String[] args) {
        // 比较三个数的最大值
        int a = 10, b = 25, c = 15;
        
        int max = a;
        if (b > max) {
            max = b;
        }
        if (c > max) {
            max = c;
        }
        System.out.println("最大值：" + max);
        
        // 判断三角形
        int x = 3, y = 4, z = 5;
        boolean isTriangle = (x + y > z) && (x + z > y) && (y + z > x);
        
        if (isTriangle) {
            System.out.println("可以构成三角形");
        } else {
            System.out.println("不能构成三角形");
        }
    }
}
```

## 七、小结

本节要点：
1. **关系运算符**：`==`、`!=`、`>`、`<`、`>=`、`<=`
2. **结果类型**：布尔值（true 或 false）
3. **数值比较**：直接比较大小
4. **字符比较**：基于 Unicode 值
5. **注意**：`==` 是比较，`=` 是赋值
