# 数组操作与二维数组

## 一、问题引入

一个班 3 名学生各考了 4 门课，如何存储所有成绩？用 12 个变量？还是 3 个一维数组？

```java
// ❌ 多个一维数组，关联关系丢失
int[] stu1 = {85, 90, 78, 92};
int[] stu2 = {76, 88, 91, 85};
int[] stu3 = {92, 87, 89, 90};
```

需要二维数组——"数组的数组"，一个变量管理所有成绩。同时，数组的常见操作（拷贝、删除等）也有标准套路。

> 生活类比：二维数组像电影院座位表，行号+列号定位一个座位。

## 二、核心概念

**二维数组**：每个元素本身是一维数组。行数由 `matrix.length` 决定，列数由 `matrix[i].length` 决定。

**内存结构**：

```
栈                堆
matrix ──────→ [引用0, 引用1, 引用2]
                  ↓       ↓       ↓
                [1,2,3] [4,5,6] [7,8,9]
```

二维数组不必是"矩形"——每行长度可以不同（不规则数组）。

## 三、代码对比

### 数组拷贝

```java
int[] source = {1, 2, 3, 4, 5};

// ❌ 直接赋值只是复制引用，两个变量指向同一数组
int[] copy = source;
copy[0] = 99;  // source[0] 也变成 99

// ✅ 真正拷贝，创建新数组
int[] copy = Arrays.copyOf(source, source.length);
copy[0] = 99;  // source 不受影响
```

### 数组比较

```java
int[] a = {1, 2, 3};
int[] b = {1, 2, 3};

// ❌ == 比较引用地址
a == b  // false

// ✅ Arrays.equals 比较内容
Arrays.equals(a, b)  // true
```

### 二维数组访问

```java
int[][] matrix = {{1, 2, 3}, {4, 5, 6}};

// ❌ 列数不能用 matrix.length
int cols = matrix.length;  // 这是行数 2，不是列数

// ✅ 用 matrix[i].length 获取列数
int cols = matrix[0].length;  // 3
```

## 四、实现方式

### 二维数组创建与遍历

```java
// 动态初始化
int[][] matrix = new int[3][4];  // 3行4列

// 静态初始化
int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

// 遍历
for (int i = 0; i < matrix.length; i++) {       // 行
    for (int j = 0; j < matrix[i].length; j++) { // 列
        System.out.print(matrix[i][j] + " ");
    }
    System.out.println();
}
// 输出:
// 1 2 3
// 4 5 6
// 7 8 9
```

### 不规则数组

```java
// 每行长度不同
int[][] arr = {
    {1, 2},
    {3, 4, 5},
    {6, 7, 8, 9}
};

System.out.println(arr[0].length);  // 2
System.out.println(arr[1].length);  // 3
System.out.println(arr[2].length);  // 4
```

### 删除元素

```java
int[] nums = {1, 2, 3, 4, 5};
int target = 3;

// 找到目标索引
int index = -1;
for (int i = 0; i < nums.length; i++) {
    if (nums[i] == target) { index = i; break; }
}

// 创建新数组，跳过目标元素
if (index != -1) {
    int[] newNums = new int[nums.length - 1];
    for (int i = 0, j = 0; i < nums.length; i++) {
        if (i != index) newNums[j++] = nums[i];
    }
    nums = newNums;
}
// 输出: [1, 2, 4, 5]
```

### 打印数组

```java
int[] nums = {1, 2, 3};

// ❌ 直接打印得到内存地址
System.out.println(nums);  // [I@15db9742

// ✅ 用 Arrays.toString
System.out.println(Arrays.toString(nums));  // [1, 2, 3]
```

### 矩阵转置

```java
int[][] matrix = {{1, 2, 3}, {4, 5, 6}};  // 2行3列
int[][] transpose = new int[matrix[0].length][matrix.length];  // 3行2列

for (int i = 0; i < matrix.length; i++) {
    for (int j = 0; j < matrix[i].length; j++) {
        transpose[j][i] = matrix[i][j];  // 行列互换
    }
}
// 输出: [[1, 4], [2, 5], [3, 6]]
```

## 五、Q&A

**Q：二维数组必须每行长度一样吗？**
A：不必。Java 二维数组是"数组的数组"，每行可以独立指定长度，即不规则数组。

**Q：数组删除元素为什么这么麻烦？**
A：数组长度固定，"删除"本质是创建更短的新数组再拷贝。频繁增删应使用 `ArrayList`。

**Q：`Arrays.copyOf` 和 `System.arraycopy` 怎么选？**
A：`Arrays.copyOf` 更简洁，适合整体拷贝或扩容；`System.arraycopy` 更灵活，可指定范围和位置。

## 六、小结表格

| 要点 | 说明 |
|------|------|
| 二维数组定义 | `int[][] m = new int[3][4]` |
| 行数/列数 | `m.length` / `m[i].length` |
| 遍历 | 嵌套循环，外层行内层列 |
| 不规则数组 | 每行长度可不同 |
| 拷贝 | `Arrays.copyOf` 创建新数组 |
| 比较 | `Arrays.equals` 比较内容 |
| 删除 | 创建新数组，跳过目标元素 |
| 打印 | `Arrays.toString`，勿直接 `println` |
