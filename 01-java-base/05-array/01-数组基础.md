# 5.1 数组的定义与初始化

### 一、数组概述

数组是用于存储**相同类型**数据的容器，具有以下特点：
- 长度固定，创建后不能改变
- 存储相同类型的元素
- 元素按顺序排列，通过索引访问
- 索引从 0 开始

### 二、数组的定义

#### 2.1 基本语法

```java
// 方式 1：先声明后创建
数据类型[] 数组名;
数组名 = new 数据类型[长度];

// 方式 2：声明同时创建
数据类型[] 数组名 = new 数据类型[长度];

// 方式 3：C 语言风格（不推荐）
数据类型 数组名[];
```

#### 2.2 示例

```java
public class ArrayDefine {
    public static void main(String[] args) {
        // 方式 1：先声明后创建
        int[] nums1;
        nums1 = new int[5];
        
        // 方式 2：声明同时创建
        int[] nums2 = new int[5];
        
        // 字符串数组
        String[] names = new String[3];
        
        // 布尔数组
        boolean[] flags = new boolean[10];
    }
}
```

### 三、数组的初始化

#### 3.1 动态初始化

指定数组长度，系统赋予默认值。

```java
public class DynamicInit {
    public static void main(String[] args) {
        // 创建长度为 5 的整型数组
        int[] nums = new int[5];
        
        // 默认值：0
        System.out.println(nums[0]);  // 0
        
        // 手动赋值
        nums[0] = 10;
        nums[1] = 20;
        nums[2] = 30;
        nums[3] = 40;
        nums[4] = 50;
        
        System.out.println(nums[0]);  // 10
    }
}
```

#### 3.2 静态初始化

直接指定数组元素，长度自动确定。

```java
public class StaticInit {
    public static void main(String[] args) {
        // 方式 1：完整写法
        int[] nums1 = new int[]{1, 2, 3, 4, 5};
        
        // 方式 2：简化写法（推荐）
        int[] nums2 = {1, 2, 3, 4, 5};
        
        // 字符串数组
        String[] names = {"张三", "李四", "王五"};
        
        // 布尔数组
        boolean[] flags = {true, false, true};
    }
}
```

### 四、数组的访问

#### 4.2 遍历数组

```java
public class ArrayTraverse {
```

### 五、注意事项

#### 5.1 数组越界

```java
int[] nums = {1, 2, 3};

// ❌ 错误：索引超出范围
// System.out.println(nums[3]);  // ArrayIndexOutOfBoundsException

// ✅ 正确
System.out.println(nums[nums.length - 1]);  // 3
```

#### 5.2 空指针异常

```java
int[] nums = null;

// ❌ 错误：空指针
// System.out.println(nums.length);  // NullPointerException
// System.out.println(nums[0]);      // NullPointerException

// ✅ 正确
if (nums != null) {
    System.out.println(nums.length);
}
```

#### 5.3 数组长度不可变

```java
int[] nums = {1, 2, 3};

// ❌ 错误：不能修改长度
// nums.length = 5;  // 编译错误

// ✅ 需要扩容时创建新数组
int[] newNums = new int[5];
System.arraycopy(nums, 0, newNums, 0, nums.length);
```

#### 5.4 声明与创建分开

```java
// ❌ 错误：静态初始化不能分开
// int[] nums;
// nums = {1, 2, 3};  // 编译错误

// ✅ 正确
int[] nums;
nums = new int[]{1, 2, 3};

// 或者
int[] nums2 = {1, 2, 3};
```

### 六、综合示例

```java
public class ArrayExample {
    public static void main(String[] args) {
        // 1. 创建并初始化数组
        int[] scores = {85, 90, 78, 92, 88};
        
        // 2. 计算总分
        int sum = 0;
        for (int score : scores) {
            sum += score;
        }
        System.out.println("总分：" + sum);
        
        // 3. 计算平均分
        double average = (double) sum / scores.length;
        System.out.println("平均分：" + average);
        
        // 4. 查找最高分
        int max = scores[0];
        for (int i = 1; i < scores.length; i++) {
            if (scores[i] > max) {
                max = scores[i];
            }
        }
        System.out.println("最高分：" + max);
        
        // 5. 修改数组元素
        scores[0] = 95;
        System.out.println("修改后第一个分数：" + scores[0]);
    }
}
```

### 七、数组的内存结构

```
栈内存 (stack)          堆内存 (heap)
+----------------+      +------------------+
| nums (引用)    |----->| [10, 20, 30, 40] |
+----------------+      +------------------+
                         索引：0   1   2   3
```

### 八、小结

本节要点：
1. **定义**：`数据类型 [] 数组名 = new 数据类型 [长度];`
2. **初始化**：动态初始化（指定长度）、静态初始化（指定元素）
3. **访问**：通过索引，从 0 开始
4. **长度**：`数组名.length`，不可修改
5. **注意**：数组越界、空指针、长度不可变

# 5.2 数组的遍历

### 一、遍历方式概述

数组遍历是指依次访问数组中的每个元素。Java 提供了多种遍历方式。

### 二、普通 for 循环

#### 2.1 基本用法

```java
public class NormalFor {
    public static void main(String[] args) {
        int[] nums = {10, 20, 30, 40, 50};
        
        for (int i = 0; i < nums.length; i++) {
            System.out.println("nums[" + i + "] = " + nums[i]);
        }
    }
}
```

#### 2.2 需要索引时使用

```java
public class ForWithIndex {
    public static void main(String[] args) {
        int[] nums = {10, 20, 30, 40, 50};
        
        // 需要索引的场景
        for (int i = 0; i < nums.length; i++) {
            // 访问当前元素和下一个元素
            if (i < nums.length - 1) {
                System.out.println(nums[i] + " + " + nums[i + 1] + " = " + (nums[i] + nums[i + 1]));
            }
        }
    }
}
```

### 三、增强 for 循环（foreach）

#### 3.1 基本用法

```java
public class ForeachLoop {
```

#### 3.3 注意事项


### 四、遍历方式对比

| 方式 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| 普通 for | 可获取索引，可修改元素 | 代码稍复杂 | 需要索引或修改元素 |
| 增强 for | 代码简洁，不易越界 | 无法获取索引 | 只读访问 |

### 七、综合示例

```java
public class ArrayTraversalExample {
    public static void main(String[] args) {
        int[] scores = {85, 92, 78, 90, 88, 76, 95, 82, 89, 91};
        
        // 1. 计算总分
        int sum = 0;
        for (int score : scores) {
            sum += score;
        }
        System.out.println("总分：" + sum);
        
        // 2. 计算平均分
        double avg = (double) sum / scores.length;
        System.out.println("平均分：" + avg);
        
        // 3. 找出最高分和最低分
        int max = scores[0], min = scores[0];
        for (int score : scores) {
            if (score > max) max = score;
            if (score < min) min = score;
        }
        System.out.println("最高分：" + max + ", 最低分：" + min);
        
        // 4. 统计各分数段人数
        int excellent = 0, good = 0, pass = 0, fail = 0;
        for (int score : scores) {
            if (score >= 90) excellent++;
            else if (score >= 80) good++;
            else if (score >= 60) pass++;
            else fail++;
        }
        System.out.println("优秀：" + excellent + ", 良好：" + good + ", 及格：" + pass + ", 不及格：" + fail);
        
        // 5. 查找特定分数的索引
        int target = 90;
        for (int i = 0; i < scores.length; i++) {
            if (scores[i] == target) {
                System.out.println("找到 " + target + "，索引：" + i);
            }
        }
    }
}
```

### 八、小结

本节要点：
1. **普通 for 循环**：可获取索引，可修改元素
2. **增强 for 循环**：代码简洁，只读访问
3. **选择**：需要索引用普通 for，否则用增强 for
4. **应用**：求和、查找、统计、拷贝、反转

### 八、手动实现常见操作

#### 8.1 冒泡排序

```java
public class BubbleSort {
    public static void main(String[] args) {
        int[] nums = {5, 3, 8, 1, 9};
        
        // 外层控制轮数，内层进行相邻比较
        for (int i = 0; i < nums.length - 1; i++) {
            for (int j = 0; j < nums.length - 1 - i; j++) {
                if (nums[j] > nums[j + 1]) {
                    // 交换
                    int temp = nums[j];
                    nums[j] = nums[j + 1];
                    nums[j + 1] = temp;
                }
            }
        }
        
        System.out.println(java.util.Arrays.toString(nums));  // [1, 3, 5, 8, 9]
    }
}
```

#### 8.2 选择排序

```java
public class SelectionSort {
    public static void main(String[] args) {
        int[] nums = {5, 3, 8, 1, 9};
        
        for (int i = 0; i < nums.length - 1; i++) {
            int minIndex = i;
            
            // 找到最小值的索引
            for (int j = i + 1; j < nums.length; j++) {
                if (nums[j] < nums[minIndex]) {
                    minIndex = j;
                }
            }
            
            // 交换
            if (minIndex != i) {
                int temp = nums[i];
                nums[i] = nums[minIndex];
                nums[minIndex] = temp;
            }
        }
        
        System.out.println(java.util.Arrays.toString(nums));  // [1, 3, 5, 8, 9]
    }
}
```

```java
public class InsertionSort {
    public static void main(String[] args) {
        int[] nums = {5, 3, 8, 1, 9};
        
        for (int i = 1; i < nums.length; i++) {
            int key = nums[i];
            int j = i - 1;
            
            // 将比 key 大的元素向后移动
            while (j >= 0 && nums[j] > key) {
                nums[j + 1] = nums[j];
                j--;
            }
            
            nums[j + 1] = key;
        }
        
        System.out.println(java.util.Arrays.toString(nums));  // [1, 3, 5, 8, 9]
    }
}
```

```java
public class RemoveElement {
    public static void main(String[] args) {
        int[] nums = {1, 2, 3, 4, 5};
        int target = 3;
        
        // 找到目标索引
        int index = -1;
        for (int i = 0; i < nums.length; i++) {
            if (nums[i] == target) {
                index = i;
                break;
            }
        }
        
        if (index != -1) {
            // 创建新数组
            int[] newNums = new int[nums.length - 1];
            
            // 拷贝目标之前的元素
            for (int i = 0; i < index; i++) {
                newNums[i] = nums[i];
            }
            
            // 拷贝目标之后的元素
            for (int i = index; i < nums.length - 1; i++) {
                newNums[i] = nums[i + 1];
            }
            
            nums = newNums;
        }

    }
```

### 九、综合示例

```java
import java.util.Arrays;

public class ArrayOperationsExample {
    public static void main(String[] args) {
        // 1. 创建数组
        int[] nums = {5, 3, 8, 1, 9, 2};
        
        // 2. 排序
        Arrays.sort(nums);
        System.out.println("排序后：" + Arrays.toString(nums));
        
        // 3. 查找
        int index = Arrays.binarySearch(nums, 8);
        System.out.println("8 的索引：" + index);
        
        // 4. 拷贝并扩容
        int[] copy = Arrays.copyOf(nums, 10);
        System.out.println("扩容后：" + Arrays.toString(copy));
        
        // 5. 填充
        Arrays.fill(copy, nums.length, copy.length, 0);
        System.out.println("填充后：" + Arrays.toString(copy));
        
        // 6. 比较
        int[] nums2 = {1, 2, 3, 5, 8, 9};
        System.out.println("数组相等：" + Arrays.equals(nums, nums2));
        
        // 7. 统计
        int sum = 0;
        for (int num : nums) {
            sum += num;
        }
        System.out.println("总和：" + sum);
        System.out.println("平均值：" + (double) sum / nums.length);
    }
}
```
