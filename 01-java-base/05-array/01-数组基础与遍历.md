# 数组基础与遍历

## 一、问题引入

如何存储全班 50 人的成绩？定义 50 个变量？

```java
// ❌ 50 个变量？不可维护
int score1 = 85;
int score2 = 90;
int score3 = 78;
// ... 重复 50 次，崩溃
```

需要一个能**批量存储同类型数据**的容器——数组。

> 生活类比：数组像一排储物柜，每个柜子有编号（下标），从 0 开始，通过编号快速存取物品。

## 二、核心概念

数组是存储**相同类型**数据的容器，特点：

- 长度固定，创建后不可变
- 元素按顺序排列，通过索引（从 0 开始）访问
- 内存中连续存储

**定义语法**：

```java
int[] nums = new int[5];       // 声明并创建，长度为 5
int[] nums = {1, 2, 3, 4, 5};  // 静态初始化，长度自动为 5
```

**默认值**（动态初始化时系统自动赋予）：

| 类型 | 默认值 |
|------|--------|
| byte/short/int/long | 0 |
| float/double | 0.0 |
| char | '\u0000' |
| boolean | false |
| 引用类型 | null |

## 三、代码对比

### 初始化方式

```java
// ❌ 静态初始化不能拆成两行
int[] nums;
nums = {1, 2, 3};  // 编译错误

// ✅ 正确写法
int[] nums;
nums = new int[]{1, 2, 3};  // 完整写法可以拆行

// ✅ 更推荐：声明时直接初始化
int[] nums = {1, 2, 3};
```

### 访问元素

```java
int[] nums = {10, 20, 30};

// ❌ 越界访问
nums[3]  // ArrayIndexOutOfBoundsException

// ✅ 安全访问最后一个元素
nums[nums.length - 1]  // 30

// ❌ 空指针
int[] arr = null;
arr.length  // NullPointerException

// ✅ 先判空
if (arr != null) {
    System.out.println(arr.length);
}
```

### 遍历时修改

```java
int[] nums = {1, 2, 3};

// ❌ foreach 不能修改原数组
for (int num : nums) {
    num = num * 2;  // 只改了临时变量，原数组不变
}

// ✅ 用普通 for 修改
for (int i = 0; i < nums.length; i++) {
    nums[i] = nums[i] * 2;
}
// 输出: [2, 4, 6]
```

## 四、实现方式

### 动态初始化

```java
int[] nums = new int[5];  // 长度为 5，默认值全为 0
nums[0] = 10;
nums[1] = 20;
```

### 静态初始化

```java
int[] nums = {1, 2, 3, 4, 5};         // 简化写法（推荐）
int[] nums = new int[]{1, 2, 3, 4, 5}; // 完整写法
```

### 两种遍历方式

```java
int[] scores = {85, 92, 78, 90, 88};

// 普通 for — 需要索引或修改元素时使用
for (int i = 0; i < scores.length; i++) {
    System.out.println("索引" + i + "：" + scores[i]);
}

// 增强 for — 只读访问时使用，更简洁
for (int score : scores) {
    System.out.println(score);
}
```

### 常见遍历应用

```java
int[] nums = {23, 45, 12, 67, 34, 89};

// 求和
int sum = 0;
for (int n : nums) { sum += n; }
// 输出: 270

// 查找最大值
int max = nums[0];
for (int i = 1; i < nums.length; i++) {
    if (nums[i] > max) max = nums[i];
}
// 输出: 89

// 反转（双指针法）
for (int i = 0, j = nums.length - 1; i < j; i++, j--) {
    int temp = nums[i];
    nums[i] = nums[j];
    nums[j] = temp;
}
// 输出: [89, 34, 67, 12, 45, 23]
```

## 五、Q&A

**Q：`nums.length` 和 `nums.length()` 有什么区别？**
A：`length` 是数组属性，不加括号；`length()` 是 String 方法，加括号。数组用 `nums.length`，字符串用 `str.length()`。

**Q：数组长度可以修改吗？**
A：不能。`nums.length` 是 `final` 的。需要扩容只能创建新数组再用 `System.arraycopy` 或 `Arrays.copyOf` 拷贝。

**Q：foreach 能获取当前索引吗？**
A：不能。如果需要索引，用普通 for 循环。

**Q：`int nums[]` 和 `int[] nums` 有什么区别？**
A：功能相同，但 `int[] nums` 是 Java 推荐风格，`int nums[]` 是 C 风格，不推荐。

## 六、小结表格

| 要点 | 说明 |
|------|------|
| 定义 | `int[] nums = new int[5]` |
| 静态初始化 | `int[] nums = {1, 2, 3}` |
| 访问 | `nums[0]`，索引从 0 开始 |
| 长度 | `nums.length`，不可修改 |
| 普通 for | 需要索引或修改元素时 |
| 增强 for | 只读访问，代码简洁 |
| 常见异常 | `ArrayIndexOutOfBoundsException`、`NullPointerException` |
