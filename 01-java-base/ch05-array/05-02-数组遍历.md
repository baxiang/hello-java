# 5.2 数组的遍历

## 一、遍历方式概述

数组遍历是指依次访问数组中的每个元素。Java 提供了多种遍历方式。

## 二、普通 for 循环

### 2.1 基本用法

```java
public class NormalFor {
    public static void main(String[] args) {
        int[] nums = {10, 20, 30, 40, 50};
        
        for (int i = 0; i < nums.length; i++) {
            System.out.println("nums[" + i + "] = " + nums[i]);
        }
    }
}
```

### 2.2 需要索引时使用

```java
public class ForWithIndex {
    public static void main(String[] args) {
        int[] nums = {10, 20, 30, 40, 50};
        
        // 需要索引的场景
        for (int i = 0; i < nums.length; i++) {
            // 访问当前元素和下一个元素
            if (i < nums.length - 1) {
                System.out.println(nums[i] + " + " + nums[i + 1] + " = " + (nums[i] + nums[i + 1]));
            }
        }
    }
}
```

## 三、增强 for 循环（foreach）

### 3.1 基本用法

```java
public class ForeachLoop {
    public static void main(String[] args) {
        int[] nums = {10, 20, 30, 40, 50};
        
        // 语法：for (元素类型 变量名：数组名)
        for (int num : nums) {
            System.out.println(num);
        }
    }
}
```

### 3.2 字符串数组遍历

```java
public class StringArray {
    public static void main(String[] args) {
        String[] names = {"张三", "李四", "王五", "赵六"};
        
        for (String name : names) {
            System.out.println("姓名：" + name);
        }
    }
}
```

### 3.3 注意事项

```java
int[] nums = {10, 20, 30, 40, 50};

// ❌ 错误：无法获取索引
// for (int num : nums) {
//     System.out.println("索引：" + ???);  // 无法获取索引
// }

// ❌ 错误：不能修改原数组
// for (int num : nums) {
//     num = num * 2;  // 不会改变原数组
// }

// ✅ 正确：只读访问
for (int num : nums) {
    System.out.println(num);
}
```

## 四、遍历方式对比

| 方式 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| 普通 for | 可获取索引，可修改元素 | 代码稍复杂 | 需要索引或修改元素 |
| 增强 for | 代码简洁，不易越界 | 无法获取索引 | 只读访问 |

## 五、应用场景

### 5.1 求和

```java
public class ArraySum {
    public static void main(String[] args) {
        int[] nums = {1, 2, 3, 4, 5};
        int sum = 0;
        
        for (int num : nums) {
            sum += num;
        }
        
        System.out.println("总和：" + sum);  // 15
    }
}
```

### 5.2 查找最大值

```java
public class FindMax {
    public static void main(String[] args) {
        int[] nums = {23, 45, 12, 67, 34, 89};
        int max = nums[0];
        
        for (int i = 1; i < nums.length; i++) {
            if (nums[i] > max) {
                max = nums[i];
            }
        }
        
        System.out.println("最大值：" + max);  // 89
    }
}
```

### 5.3 查找元素

```java
public class FindElement {
    public static void main(String[] args) {
        int[] nums = {12, 45, 67, 23, 89, 34};
        int target = 23;
        int index = -1;
        
        for (int i = 0; i < nums.length; i++) {
            if (nums[i] == target) {
                index = i;
                break;
            }
        }
        
        if (index != -1) {
            System.out.println("找到，索引：" + index);
        } else {
            System.out.println("未找到");
        }
    }
}
```

### 5.4 统计满足条件的元素

```java
public class CountElements {
    public static void main(String[] args) {
        int[] scores = {85, 92, 78, 90, 88, 76, 95};
        int count = 0;
        
        for (int score : scores) {
            if (score >= 90) {
                count++;
            }
        }
        
        System.out.println("90 分以上的人数：" + count);  // 3
    }
}
```

### 5.5 数组拷贝

```java
public class ArrayCopy {
    public static void main(String[] args) {
        int[] source = {1, 2, 3, 4, 5};
        int[] target = new int[source.length];
        
        // 方式 1：手动拷贝
        for (int i = 0; i < source.length; i++) {
            target[i] = source[i];
        }
        
        // 方式 2：使用 System.arraycopy
        System.arraycopy(source, 0, target, 0, source.length);
        
        // 方式 3：使用 Arrays.copyOf（Java 6+）
        int[] copy = java.util.Arrays.copyOf(source, source.length);
    }
}
```

### 5.6 反转数组

```java
public class ReverseArray {
    public static void main(String[] args) {
        int[] nums = {1, 2, 3, 4, 5};
        
        // 双指针法
        for (int i = 0, j = nums.length - 1; i < j; i++, j--) {
            int temp = nums[i];
            nums[i] = nums[j];
            nums[j] = temp;
        }
        
        // 输出反转后的数组
        for (int num : nums) {
            System.out.print(num + " ");  // 5 4 3 2 1
        }
    }
}
```

### 5.7 打印数组

```java
public class PrintArray {
    public static void main(String[] args) {
        int[] nums = {1, 2, 3, 4, 5};
        
        // 方式 1：普通遍历
        System.out.print("[");
        for (int i = 0; i < nums.length; i++) {
            System.out.print(nums[i]);
            if (i < nums.length - 1) {
                System.out.print(", ");
            }
        }
        System.out.println("]");
        
        // 方式 2：使用 Arrays.toString（推荐）
        System.out.println(java.util.Arrays.toString(nums));
    }
}
```

## 六、二维数组遍历

```java
public class TwoDArray {
    public static void main(String[] args) {
        int[][] matrix = {
            {1, 2, 3},
            {4, 5, 6},
            {7, 8, 9}
        };
        
        // 普通 for 循环
        for (int i = 0; i < matrix.length; i++) {
            for (int j = 0; j < matrix[i].length; j++) {
                System.out.print(matrix[i][j] + " ");
            }
            System.out.println();
        }
        
        // 增强 for 循环
        for (int[] row : matrix) {
            for (int num : row) {
                System.out.print(num + " ");
            }
            System.out.println();
        }
    }
}
```

## 七、综合示例

```java
public class ArrayTraversalExample {
    public static void main(String[] args) {
        int[] scores = {85, 92, 78, 90, 88, 76, 95, 82, 89, 91};
        
        // 1. 计算总分
        int sum = 0;
        for (int score : scores) {
            sum += score;
        }
        System.out.println("总分：" + sum);
        
        // 2. 计算平均分
        double avg = (double) sum / scores.length;
        System.out.println("平均分：" + avg);
        
        // 3. 找出最高分和最低分
        int max = scores[0], min = scores[0];
        for (int score : scores) {
            if (score > max) max = score;
            if (score < min) min = score;
        }
        System.out.println("最高分：" + max + ", 最低分：" + min);
        
        // 4. 统计各分数段人数
        int excellent = 0, good = 0, pass = 0, fail = 0;
        for (int score : scores) {
            if (score >= 90) excellent++;
            else if (score >= 80) good++;
            else if (score >= 60) pass++;
            else fail++;
        }
        System.out.println("优秀：" + excellent + ", 良好：" + good + ", 及格：" + pass + ", 不及格：" + fail);
        
        // 5. 查找特定分数的索引
        int target = 90;
        for (int i = 0; i < scores.length; i++) {
            if (scores[i] == target) {
                System.out.println("找到 " + target + "，索引：" + i);
            }
        }
    }
}
```

## 八、小结

本节要点：
1. **普通 for 循环**：可获取索引，可修改元素
2. **增强 for 循环**：代码简洁，只读访问
3. **选择**：需要索引用普通 for，否则用增强 for
4. **应用**：求和、查找、统计、拷贝、反转
