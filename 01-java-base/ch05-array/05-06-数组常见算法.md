# 5.6 数组常见算法

## 一、排序算法

### 1.1 冒泡排序

```java
public class BubbleSort {
    public static void main(String[] args) {
        int[] nums = {5, 3, 8, 1, 9};
        
        // 外层控制轮数，内层进行相邻比较
        for (int i = 0; i < nums.length - 1; i++) {
            for (int j = 0; j < nums.length - 1 - i; j++) {
                if (nums[j] > nums[j + 1]) {
                    // 交换
                    int temp = nums[j];
                    nums[j] = nums[j + 1];
                    nums[j + 1] = temp;
                }
            }
        }
        
        System.out.println(java.util.Arrays.toString(nums));  // [1, 3, 5, 8, 9]
    }
}
```

**特点**：
- 时间复杂度：O(n²)
- 空间复杂度：O(1)
- 稳定排序

### 1.2 选择排序

```java
public class SelectionSort {
    public static void main(String[] args) {
        int[] nums = {5, 3, 8, 1, 9};
        
        for (int i = 0; i < nums.length - 1; i++) {
            int minIndex = i;
            
            // 找到最小值的索引
            for (int j = i + 1; j < nums.length; j++) {
                if (nums[j] < nums[minIndex]) {
                    minIndex = j;
                }
            }
            
            // 交换
            if (minIndex != i) {
                int temp = nums[i];
                nums[i] = nums[minIndex];
                nums[minIndex] = temp;
            }
        }
        
        System.out.println(java.util.Arrays.toString(nums));  // [1, 3, 5, 8, 9]
    }
}
```

**特点**：
- 时间复杂度：O(n²)
- 空间复杂度：O(1)
- 不稳定排序

### 1.3 插入排序

```java
public class InsertionSort {
    public static void main(String[] args) {
        int[] nums = {5, 3, 8, 1, 9};
        
        for (int i = 1; i < nums.length; i++) {
            int key = nums[i];
            int j = i - 1;
            
            // 将比 key 大的元素向后移动
            while (j >= 0 && nums[j] > key) {
                nums[j + 1] = nums[j];
                j--;
            }
            
            nums[j + 1] = key;
        }
        
        System.out.println(java.util.Arrays.toString(nums));  // [1, 3, 5, 8, 9]
    }
}
```

**特点**：
- 时间复杂度：O(n²)
- 空间复杂度：O(1)
- 稳定排序
- 适合基本有序的数据

### 1.4 快速排序

```java
public class QuickSort {
    public static void main(String[] args) {
        int[] nums = {5, 3, 8, 1, 9, 2};
        quickSort(nums, 0, nums.length - 1);
        System.out.println(java.util.Arrays.toString(nums));  // [1, 2, 3, 5, 8, 9]
    }
    
    public static void quickSort(int[] arr, int left, int right) {
        if (left >= right) {
            return;
        }
        
        // 分区
        int pivotIndex = partition(arr, left, right);
        
        // 递归排序左右两部分
        quickSort(arr, left, pivotIndex - 1);
        quickSort(arr, pivotIndex + 1, right);
    }
    
    public static int partition(int[] arr, int left, int right) {
        int pivot = arr[left];  // 选择第一个元素为基准
        int i = left + 1;
        int j = right;
        
        while (true) {
            // 从左找到第一个大于 pivot 的元素
            while (i <= j && arr[i] <= pivot) {
                i++;
            }
            // 从右找到第一个小于 pivot 的元素
            while (i <= j && arr[j] >= pivot) {
                j--;
            }
            // 交换
            if (i < j) {
                swap(arr, i, j);
            } else {
                break;
            }
        }
        
        // 将 pivot 放到正确位置
        swap(arr, left, j);
        return j;
    }
    
    public static void swap(int[] arr, int i, int j) {
        int temp = arr[i];
        arr[i] = arr[j];
        arr[j] = temp;
    }
}
```

**特点**：
- 时间复杂度：平均 O(n log n)，最坏 O(n²)
- 空间复杂度：O(log n)
- 不稳定排序
- 实际应用中最快的排序算法之一

## 二、查找算法

### 2.1 线性查找

```java
public class LinearSearch {
    public static void main(String[] args) {
        int[] nums = {12, 45, 67, 23, 89, 34};
        int target = 23;
        
        int index = linearSearch(nums, target);
        
        if (index != -1) {
            System.out.println("找到，索引：" + index);
        } else {
            System.out.println("未找到");
        }
    }
    
    public static int linearSearch(int[] arr, int target) {
        for (int i = 0; i < arr.length; i++) {
            if (arr[i] == target) {
                return i;
            }
        }
        return -1;
    }
}
```

**特点**：
- 时间复杂度：O(n)
- 适用于无序数组

### 2.2 二分查找

```java
public class BinarySearch {
    public static void main(String[] args) {
        int[] nums = {1, 3, 5, 7, 9, 11, 13};
        int target = 7;
        
        int index = binarySearch(nums, target);
        System.out.println("索引：" + index);  // 3
    }
    
    public static int binarySearch(int[] arr, int target) {
        int left = 0;
        int right = arr.length - 1;
        
        while (left <= right) {
            int mid = left + (right - left) / 2;  // 防止溢出
            
            if (arr[mid] == target) {
                return mid;
            } else if (arr[mid] < target) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }
        
        return -1;
    }
}
```

**特点**：
- 时间复杂度：O(log n)
- 适用于有序数组
- 效率高

### 2.3 递归二分查找

```java
public class RecursiveBinarySearch {
    public static void main(String[] args) {
        int[] nums = {1, 3, 5, 7, 9, 11, 13};
        int target = 7;
        
        int index = binarySearch(nums, 0, nums.length - 1, target);
        System.out.println("索引：" + index);  // 3
    }
    
    public static int binarySearch(int[] arr, int left, int right, int target) {
        if (left > right) {
            return -1;
        }
        
        int mid = left + (right - left) / 2;
        
        if (arr[mid] == target) {
            return mid;
        } else if (arr[mid] < target) {
            return binarySearch(arr, mid + 1, right, target);
        } else {
            return binarySearch(arr, left, mid - 1, target);
        }
    }
}
```

```java
public class ArraySum {

    }
}
```

### 3.2 求平均值

```java
public class ArrayAverage {

        int sum = 0;
        for (int num : nums) {

        }
```

### 3.3 求最大值和最小值

```java
public class MaxMin {
    public static void main(String[] args) {
        int[] nums = {23, 45, 12, 67, 34, 89};
        
        int max = nums[0];
        int min = nums[0];
        
        for (int i = 1; i < nums.length; i++) {
            if (nums[i] > max) {
                max = nums[i];
            }
            if (nums[i] < min) {
                min = nums[i];
            }
        }
        
        System.out.println("最大值：" + max);  // 89
        System.out.println("最小值：" + min);  // 12
    }
}
```

## 四、数组操作算法

### 4.1 反转数组

```java
public class ReverseArray {
    public static void main(String[] args) {
        int[] nums = {1, 2, 3, 4, 5};
        
        // 双指针法
        for (int i = 0, j = nums.length - 1; i < j; i++, j--) {
            int temp = nums[i];
            nums[i] = nums[j];
            nums[j] = temp;
        }
        
        System.out.println(java.util.Arrays.toString(nums));  // [5, 4, 3, 2, 1]
    }
}
```

### 4.2 数组去重

```java
public class RemoveDuplicates {
    public static void main(String[] args) {
        int[] nums = {1, 2, 2, 3, 3, 3, 4, 5};
        
        // 假设数组已排序
        int[] temp = new int[nums.length];
        int count = 0;
        
        temp[count++] = nums[0];
        
        for (int i = 1; i < nums.length; i++) {
            if (nums[i] != nums[i - 1]) {
                temp[count++] = nums[i];
            }
        }
        
        // 创建结果数组
        int[] result = java.util.Arrays.copyOf(temp, count);
        
        System.out.println(java.util.Arrays.toString(result));  // [1, 2, 3, 4, 5]
    }
}
```

### 4.3 数组合并

```java
public class MergeArrays {
    public static void main(String[] args) {
        int[] arr1 = {1, 3, 5};
        int[] arr2 = {2, 4, 6};
```

### 4.4 有序数组合并

```java
public class MergeSortedArrays {
    public static void main(String[] args) {
        int[] arr1 = {1, 3, 5, 7};
        int[] arr2 = {2, 4, 6, 8};
        
        int[] result = new int[arr1.length + arr2.length];
        int i = 0, j = 0, k = 0;
        
        // 合并两个有序数组
        while (i < arr1.length && j < arr2.length) {
            if (arr1[i] <= arr2[j]) {
                result[k++] = arr1[i++];
            } else {
                result[k++] = arr2[j++];
            }
        }
        
        // 处理剩余元素
        while (i < arr1.length) {
            result[k++] = arr1[i++];
        }
        while (j < arr2.length) {
            result[k++] = arr2[j++];
        }
        
        System.out.println(java.util.Arrays.toString(result));  // [1, 2, 3, 4, 5, 6, 7, 8]
    }
}
```

## 五、综合示例

```java
import java.util.Arrays;

public class ArrayAlgorithmsExample {
    public static void main(String[] args) {
        int[] nums = {5, 3, 8, 1, 9, 2, 7, 4, 6};
        
        // 1. 排序
        System.out.println("原始：" + Arrays.toString(nums));
        quickSort(nums, 0, nums.length - 1);
        System.out.println("快速排序：" + Arrays.toString(nums));
        
        // 2. 查找
        int index = binarySearch(nums, 7);
        System.out.println("7 的索引：" + index);
        
        // 3. 统计
        int sum = 0;
        int max = nums[0];
        int min = nums[0];
        
        for (int num : nums) {
            sum += num;
            if (num > max) max = num;
            if (num < min) min = num;
        }
        
        System.out.println("总和：" + sum);
        System.out.println("最大值：" + max);
        System.out.println("最小值：" + min);
        System.out.println("平均值：" + (double) sum / nums.length);
        
        // 4. 反转
        reverse(nums);
        System.out.println("反转后：" + Arrays.toString(nums));
    }
    
    // 快速排序
    public static void quickSort(int[] arr, int left, int right) {
        if (left >= right) return;
        
        int pivotIndex = partition(arr, left, right);
        quickSort(arr, left, pivotIndex - 1);
        quickSort(arr, pivotIndex + 1, right);
    }
    
    public static int partition(int[] arr, int left, int right) {
        int pivot = arr[left];
        int i = left + 1, j = right;
        
        while (true) {
            while (i <= j && arr[i] <= pivot) i++;
            while (i <= j && arr[j] >= pivot) j--;
            if (i < j) swap(arr, i, j);
            else break;
        }
        swap(arr, left, j);
        return j;
    }
    
    // 二分查找
    public static int binarySearch(int[] arr, int target) {
        int left = 0, right = arr.length - 1;
        while (left <= right) {
            int mid = left + (right - left) / 2;
            if (arr[mid] == target) return mid;
            else if (arr[mid] < target) left = mid + 1;
            else right = mid - 1;
        }
        return -1;
    }
    
    // 反转数组
    public static void reverse(int[] arr) {
        for (int i = 0, j = arr.length - 1; i < j; i++, j--) {
            swap(arr, i, j);
        }
    }
    
    // 交换
    public static void swap(int[] arr, int i, int j) {
        int temp = arr[i];
        arr[i] = arr[j];
        arr[j] = temp;
    }
}
```
