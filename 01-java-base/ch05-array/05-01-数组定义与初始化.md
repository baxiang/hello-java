# 5.1 数组的定义与初始化

## 一、数组概述

数组是用于存储**相同类型**数据的容器，具有以下特点：
- 长度固定，创建后不能改变
- 存储相同类型的元素
- 元素按顺序排列，通过索引访问
- 索引从 0 开始

## 二、数组的定义

### 2.1 基本语法

```java
// 方式 1：先声明后创建
数据类型[] 数组名;
数组名 = new 数据类型[长度];

// 方式 2：声明同时创建
数据类型[] 数组名 = new 数据类型[长度];

// 方式 3：C 语言风格（不推荐）
数据类型 数组名[];
```

### 2.2 示例

```java
public class ArrayDefine {
    public static void main(String[] args) {
        // 方式 1：先声明后创建
        int[] nums1;
        nums1 = new int[5];
        
        // 方式 2：声明同时创建
        int[] nums2 = new int[5];
        
        // 字符串数组
        String[] names = new String[3];
        
        // 布尔数组
        boolean[] flags = new boolean[10];
    }
}
```

## 三、数组的初始化

### 3.1 动态初始化

指定数组长度，系统赋予默认值。

```java
public class DynamicInit {
    public static void main(String[] args) {
        // 创建长度为 5 的整型数组
        int[] nums = new int[5];
        
        // 默认值：0
        System.out.println(nums[0]);  // 0
        
        // 手动赋值
        nums[0] = 10;
        nums[1] = 20;
        nums[2] = 30;
        nums[3] = 40;
        nums[4] = 50;
        
        System.out.println(nums[0]);  // 10
    }
}
```

### 3.2 静态初始化

直接指定数组元素，长度自动确定。

```java
public class StaticInit {
    public static void main(String[] args) {
        // 方式 1：完整写法
        int[] nums1 = new int[]{1, 2, 3, 4, 5};
        
        // 方式 2：简化写法（推荐）
        int[] nums2 = {1, 2, 3, 4, 5};
        
        // 字符串数组
        String[] names = {"张三", "李四", "王五"};
        
        // 布尔数组
        boolean[] flags = {true, false, true};
    }
}
```

### 3.3 默认值

| 数据类型 | 默认值 |
|----------|--------|
| byte, short, int, long | 0 |
| float, double | 0.0 |
| char | '\u0000' |
| boolean | false |
| 引用类型 | null |

```java
public class DefaultValue {
    public static void main(String[] args) {
        int[] nums = new int[3];
        System.out.println(nums[0]);  // 0
        
        double[] prices = new double[3];
        System.out.println(prices[0]);  // 0.0
        
        char[] chars = new char[3];
        System.out.println(chars[0]);  // 空字符
        
        boolean[] bools = new boolean[3];
        System.out.println(bools[0]);  // false
        
        String[] strs = new String[3];
        System.out.println(strs[0]);  // null
    }
}
```

## 四、数组的访问

### 4.1 通过索引访问

```java
public class ArrayAccess {
    public static void main(String[] args) {
        int[] nums = {10, 20, 30, 40, 50};
        
        // 访问元素
        System.out.println(nums[0]);  // 10
        System.out.println(nums[2]);  // 30
        
        // 修改元素
        nums[0] = 100;
        System.out.println(nums[0]);  // 100
        
        // 数组长度
        int length = nums.length;
        System.out.println("长度：" + length);  // 5
    }
}
```

### 4.2 遍历数组

```java
public class ArrayTraverse {
    public static void main(String[] args) {
        int[] nums = {10, 20, 30, 40, 50};
        
        // 方式 1：普通 for 循环
        for (int i = 0; i < nums.length; i++) {
            System.out.println("nums[" + i + "] = " + nums[i]);
        }
        
        // 方式 2：增强 for 循环（foreach）
        for (int num : nums) {
            System.out.println(num);
        }
    }
}
```

## 五、注意事项

### 5.1 数组越界

```java
int[] nums = {1, 2, 3};

// ❌ 错误：索引超出范围
// System.out.println(nums[3]);  // ArrayIndexOutOfBoundsException

// ✅ 正确
System.out.println(nums[nums.length - 1]);  // 3
```

### 5.2 空指针异常

```java
int[] nums = null;

// ❌ 错误：空指针
// System.out.println(nums.length);  // NullPointerException
// System.out.println(nums[0]);      // NullPointerException

// ✅ 正确
if (nums != null) {
    System.out.println(nums.length);
}
```

### 5.3 数组长度不可变

```java
int[] nums = {1, 2, 3};

// ❌ 错误：不能修改长度
// nums.length = 5;  // 编译错误

// ✅ 需要扩容时创建新数组
int[] newNums = new int[5];
System.arraycopy(nums, 0, newNums, 0, nums.length);
```

### 5.4 声明与创建分开

```java
// ❌ 错误：静态初始化不能分开
// int[] nums;
// nums = {1, 2, 3};  // 编译错误

// ✅ 正确
int[] nums;
nums = new int[]{1, 2, 3};

// 或者
int[] nums2 = {1, 2, 3};
```

## 六、综合示例

```java
public class ArrayExample {
    public static void main(String[] args) {
        // 1. 创建并初始化数组
        int[] scores = {85, 90, 78, 92, 88};
        
        // 2. 计算总分
        int sum = 0;
        for (int score : scores) {
            sum += score;
        }
        System.out.println("总分：" + sum);
        
        // 3. 计算平均分
        double average = (double) sum / scores.length;
        System.out.println("平均分：" + average);
        
        // 4. 查找最高分
        int max = scores[0];
        for (int i = 1; i < scores.length; i++) {
            if (scores[i] > max) {
                max = scores[i];
            }
        }
        System.out.println("最高分：" + max);
        
        // 5. 修改数组元素
        scores[0] = 95;
        System.out.println("修改后第一个分数：" + scores[0]);
    }
}
```


## 八、小结

本节要点：
1. **定义**：`数据类型 [] 数组名 = new 数据类型 [长度];`
2. **初始化**：动态初始化（指定长度）、静态初始化（指定元素）
3. **访问**：通过索引，从 0 开始
4. **长度**：`数组名.length`，不可修改
5. **注意**：数组越界、空指针、长度不可变
