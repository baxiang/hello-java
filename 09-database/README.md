# 第九部分：数据库与 JDBC

本模块学习关系型数据库 MySQL 的基础知识和 JDBC 编程技术，包括 SQL 语法、查询优化、事务处理、数据库连接池等，是后端开发必备的数据访问技能。

## 章节列表

| 章节 | 目录 | 主题 |
|------|------|------|
| 第 38 章 | ch38-mysql | MySQL 基础：数据库概念、SQL 语法（DDL/DML/DQL/DCL）、单表与多表查询、聚合函数、索引与优化、事务基础 |
| 第 39 章 | ch39-jdbc | JDBC：JDBC 编程步骤、PreparedStatement、事务处理、数据库连接池（Druid/HikariCP）、DBUtils 工具类 |

## 学习目标

- 掌握 MySQL 数据库的基本操作与 SQL 语法
- 理解多表连接查询与聚合函数
- 了解索引原理与基础优化方法
- 掌握 JDBC 编程步骤与 PreparedStatement 的使用
- 理解数据库连接池的原理与配置
