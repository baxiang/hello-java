# 9.2 JDBC

## 一、JDBC 概述

JDBC（Java Database Connectivity）是 Java 连接数据库的标准 API。

**核心接口**：
- Driver：数据库驱动
- Connection：数据库连接
- Statement：执行 SQL 语句
- PreparedStatement：预编译 SQL
- ResultSet：结果集

## 二、JDBC 编程步骤

### 2.1 基本步骤

```java
// 1. 加载驱动
Class.forName("com.mysql.cj.jdbc.Driver");

// 2. 获取连接
Connection conn = DriverManager.getConnection(
    "jdbc:mysql://localhost:3306/demo",
    "root",
    "123456"
);

// 3. 创建 Statement
Statement stmt = conn.createStatement();

// 4. 执行 SQL
ResultSet rs = stmt.executeQuery("SELECT * FROM user");

// 5. 处理结果
while (rs.next()) {
    System.out.println(rs.getString("username"));
}

// 6. 关闭资源
rs.close();
stmt.close();
conn.close();
```

### 2.2 try-with-resources

```java
String url = "jdbc:mysql://localhost:3306/demo";
String user = "root";
String password = "123456";

try (Connection conn = DriverManager.getConnection(url, user, password);
     Statement stmt = conn.createStatement();
     ResultSet rs = stmt.executeQuery("SELECT * FROM user")) {
    
    while (rs.next()) {
        System.out.println(rs.getString("username"));
    }
} catch (SQLException e) {
    e.printStackTrace();
}
```

## 三、Statement

### 3.1 执行查询

```java
Statement stmt = conn.createStatement();
ResultSet rs = stmt.executeQuery("SELECT * FROM user");

while (rs.next()) {
    Long id = rs.getLong("id");
    String username = rs.getString("username");
    String email = rs.getString("email");
    int age = rs.getInt("age");
    
    System.out.println(id + ", " + username + ", " + email + ", " + age);
}
```

### 3.2 执行更新

```java
Statement stmt = conn.createStatement();

// 插入
int rows = stmt.executeUpdate(
    "INSERT INTO user (username, email, age) VALUES ('张三', 'zhang@example.com', 18)"
);
System.out.println("插入 " + rows + " 行");

// 更新
rows = stmt.executeUpdate(
    "UPDATE user SET age = 19 WHERE username = '张三'"
);
System.out.println("更新 " + rows + " 行");

// 删除
rows = stmt.executeUpdate(
    "DELETE FROM user WHERE username = '张三'"
);
System.out.println("删除 " + rows + " 行");
```

## 四、PreparedStatement

### 4.1 基本使用

```java
// 预编译 SQL
String sql = "INSERT INTO user (username, email, age) VALUES (?, ?, ?)";
PreparedStatement pstmt = conn.prepareStatement(sql);

// 设置参数
pstmt.setString(1, "张三");
pstmt.setString(2, "zhang@example.com");
pstmt.setInt(3, 18);

// 执行
int rows = pstmt.executeUpdate();
```

### 4.2 查询

```java
String sql = "SELECT * FROM user WHERE id = ?";
PreparedStatement pstmt = conn.prepareStatement(sql);
pstmt.setLong(1, 1L);

ResultSet rs = pstmt.executeQuery();
if (rs.next()) {
    System.out.println(rs.getString("username"));
}
```

### 4.3 批量操作

```java
String sql = "INSERT INTO user (username, email, age) VALUES (?, ?, ?)";
PreparedStatement pstmt = conn.prepareStatement(sql);

// 添加批量
for (int i = 0; i < 10; i++) {
    pstmt.setString(1, "用户" + i);
    pstmt.setString(2, "user" + i + "@example.com");
    pstmt.setInt(3, 20 + i);
    pstmt.addBatch();
}

// 执行批量
int[] results = pstmt.executeBatch();
System.out.println("插入 " + results.length + " 行");
```

### 4.4 防止 SQL 注入

```java
// ❌ 不安全：SQL 注入
String username = "' OR '1'='1";
Statement stmt = conn.createStatement();
ResultSet rs = stmt.executeQuery(
    "SELECT * FROM user WHERE username = '" + username + "'"
);

// ✅ 安全：使用 PreparedStatement
String sql = "SELECT * FROM user WHERE username = ?";
PreparedStatement pstmt = conn.prepareStatement(sql);
pstmt.setString(1, username);
ResultSet rs = pstmt.executeQuery();
```

## 五、事务处理

### 5.1 基本事务

```java
Connection conn = null;
try {
    conn = DriverManager.getConnection(url, user, password);
    
    // 关闭自动提交
    conn.setAutoCommit(false);
    
    // 执行多个操作
    Statement stmt = conn.createStatement();
    stmt.executeUpdate("UPDATE account SET balance = balance - 100 WHERE id = 1");
    stmt.executeUpdate("UPDATE account SET balance = balance + 100 WHERE id = 2");
    
    // 提交事务
    conn.commit();
    
} catch (SQLException e) {
    // 回滚事务
    if (conn != null) {
        try {
            conn.rollback();
        } catch (SQLException ex) {
            ex.printStackTrace();
        }
    }
    e.printStackTrace();
} finally {
    // 恢复自动提交
    if (conn != null) {
        try {
            conn.setAutoCommit(true);
            conn.close();
        } catch (SQLException e) {}
    }
}
```

### 5.2 保存点

```java
conn.setAutoCommit(false);

Statement stmt = conn.createStatement();
stmt.executeUpdate("INSERT INTO user (username) VALUES ('A')");

// 设置保存点
Savepoint sp = conn.setSavepoint("sp1");

stmt.executeUpdate("INSERT INTO user (username) VALUES ('B')");

// 回滚到保存点
conn.rollback(sp);

// 提交（只插入 A）
conn.commit();
```

### 5.3 事务隔离级别

```java
// 查看隔离级别
int level = conn.getTransactionIsolation();

// 设置隔离级别
conn.setTransactionIsolation(Connection.TRANSACTION_READ_COMMITTED);

// 隔离级别常量
Connection.TRANSACTION_READ_UNCOMMITTED    // 读未提交
Connection.TRANSACTION_READ_COMMITTED      // 读已提交
Connection.TRANSACTION_REPEATABLE_READ     // 可重复读
Connection.TRANSACTION_SERIALIZABLE        // 串行化
```

## 七、DBUtils 工具类

### 7.1 Apache Commons DbUtils

```xml
<!-- 添加依赖 -->
<dependency>
    <groupId>commons-dbutils</groupId>
    <artifactId>commons-dbutils</artifactId>
    <version>1.7</version>
</dependency>
```

### 7.2 基本使用

```java
import org.apache.commons.dbutils.QueryRunner;
import org.apache.commons.dbutils.handlers.*;

// 创建 QueryRunner（使用连接池）
QueryRunner runner = new QueryRunner(dataSource);

// 查询单个对象
User user = runner.query(
    "SELECT * FROM user WHERE id = ?",
    new BeanHandler<>(User.class),
    1L
);

// 查询列表
List<User> users = runner.query(
    "SELECT * FROM user",
    new BeanListHandler<>(User.class)
);

// 查询单个值
Long count = runner.query(
    "SELECT COUNT(*) FROM user",
    new ScalarHandler<Long>()
);

// 更新
int rows = runner.update(
    "INSERT INTO user (username, email) VALUES (?, ?)",
    "张三",
    "zhang@example.com"
);
```

### 7.3 自定义 ResultSetHandler

```java
// 自定义处理器
ResultSetHandler<Map<String, Object>> handler = rs -> {
    Map<String, Object> result = new HashMap<>();
    if (rs.next()) {
        ResultSetMetaData meta = rs.getMetaData();
        for (int i = 1; i <= meta.getColumnCount(); i++) {
            result.put(meta.getColumnName(i), rs.getObject(i));
        }
    }
    return result;
};

Map<String, Object> user = runner.query(
    "SELECT * FROM user WHERE id = ?",
    handler,
    1L
);
```

## 八、综合示例

### 8.1 数据库工具类

```java
public class DBUtil {
    
    private static DataSource dataSource;
    
    static {
        // 初始化连接池
        HikariConfig config = new HikariConfig();
        config.setJdbcUrl("jdbc:mysql://localhost:3306/demo");
        config.setUsername("root");
        config.setPassword("123456");
        config.setDriverClassName("com.mysql.cj.jdbc.Driver");
        config.setMaximumPoolSize(10);
        
        dataSource = new HikariDataSource(config);
    }
    
    // 获取连接
    public static Connection getConnection() throws SQLException {
        return dataSource.getConnection();
    }
    
    // 关闭资源
    public static void close(Connection conn, Statement stmt, ResultSet rs) {
        try {
            if (rs != null) rs.close();
            if (stmt != null) stmt.close();
            if (conn != null) conn.close();
        } catch (SQLException e) {
            e.printStackTrace();
        }
    }
}
```

### 8.2 用户 DAO

```java
public class UserDao {
    
    private QueryRunner runner = new QueryRunner(DBUtil.getDataSource());
    
    // 根据 ID 查询
    public User findById(Long id) throws SQLException {
        String sql = "SELECT * FROM user WHERE id = ?";
        return runner.query(sql, new BeanHandler<>(User.class), id);
    }
    
    // 查询所有
    public List<User> findAll() throws SQLException {
        String sql = "SELECT * FROM user";
        return runner.query(sql, new BeanListHandler<>(User.class));
    }
    
    // 插入
    public Long insert(User user) throws SQLException {
        String sql = "INSERT INTO user (username, email, age) VALUES (?, ?, ?)";
        runner.update(sql, user.getUsername(), user.getEmail(), user.getAge());
        
        // 获取自增 ID
        Long id = runner.query("SELECT LAST_INSERT_ID()", new ScalarHandler<>());
        return id;
    }
    
    // 更新
    public int update(User user) throws SQLException {
        String sql = "UPDATE user SET username = ?, email = ?, age = ? WHERE id = ?";
        return runner.update(sql, user.getUsername(), user.getEmail(), user.getAge(), user.getId());
    }
    
    // 删除
    public int delete(Long id) throws SQLException {
        String sql = "DELETE FROM user WHERE id = ?";
        return runner.update(sql, id);
    }
}
```

### 8.3 事务示例

```java
public class UserService {
    
    private UserDao userDao = new UserDao();
    
    // 转账（事务）
    public void transfer(Long fromId, Long toId, BigDecimal amount) {
        Connection conn = null;
        try {
            conn = DBUtil.getConnection();
            conn.setAutoCommit(false);
            
            // 扣减余额
            String sql1 = "UPDATE account SET balance = balance - ? WHERE id = ?";
            new QueryRunner().update(conn, sql1, amount, fromId);
            
            // 增加余额
            String sql2 = "UPDATE account SET balance = balance + ? WHERE id = ?";
            new QueryRunner().update(conn, sql2, amount, toId);
            
            // 提交事务
            conn.commit();
            
        } catch (SQLException e) {
            // 回滚事务
            if (conn != null) {
                try {
                    conn.rollback();
                } catch (SQLException ex) {
                    ex.printStackTrace();
                }
            }
            throw new RuntimeException("转账失败", e);
        } finally {
            DBUtil.close(conn, null, null);
        }
    }
}
```

## 九、小结

1. **JDBC 步骤**：加载驱动、获取连接、创建 Statement、执行 SQL、处理结果、关闭资源
2. **PreparedStatement**：预编译 SQL、防止 SQL 注入、支持批量
3. **事务处理**：setAutoCommit、commit、rollback
4. **连接池**：Druid、HikariCP
5. **DBUtils**：简化 JDBC 操作
