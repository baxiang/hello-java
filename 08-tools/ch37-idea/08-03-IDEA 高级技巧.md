## 二、代码模板

### 2.1 实时模板（Live Templates）

```
psvm → public static void main(String[] args) {}
sout → System.out.println();
soutv → System.out.println("var = " + var);
soutp → System.out.println("param = " + param);
soutm → System.out.println("method = " + method);
fori → for (int i = 0; i < ; i++) {}
iter → for (Type item : collection) {}
ifn → if (obj == null) {}
inn → if (obj != null) {}
nn → obj != null
null → obj == null
```

### 2.2 自定义模板

```
1. Settings → Editor → Live Templates
2. 点击 + 创建新模板
3. 设置缩写和描述
4. 编写模板内容
5. 设置应用范围
```

**示例：创建 Builder 模板**

```java
// 缩写：builder
// 模板内容：
public class $CLASS$Builder {
    private $CLASS$ $FIELD$;
    
    public $CLASS$Builder $METHOD$($TYPE$ $FIELD$) {
        this.$FIELD$ = $FIELD$;
        return this;
    }
    
    public $CLASS$ build() {
        return new $CLASS$($FIELD$);
    }
}
```

## 三、代码生成

### 3.1 Generate 菜单（Cmd + N / Alt + Insert）

```
Constructor - 生成构造方法
Getter and Setter - 生成 getter/setter
equals() and hashCode() - 生成 equals/hashCode
toString() - 生成 toString
Delegate - 生成委托方法
Override Methods - 重写父类方法
Implement Methods - 实现接口方法
```

### 3.2 使用 Lombok

```java
// 添加依赖
<dependency>
    <groupId>org.projectlombok</groupId>
    <artifactId>lombok</artifactId>
    <version>1.18.28</version>
    <scope>provided</scope>
</dependency>

// 常用注解
@Data              // getter + setter + toString + equals + hashCode
@Getter            // getter
@Setter            // setter
@NoArgsConstructor // 无参构造
@AllArgsConstructor // 全参构造
@Builder          // Builder 模式
@ToString         // toString
```

### 4.3 调试技巧

```
1. 条件断点：i > 10
2. 日志断点：不中断，只打印日志
3. 异常断点：捕获未捕获的异常
4. 监视变量：右键变量 → Add to Watches
5. 计算表达式：Alt + F8
```
