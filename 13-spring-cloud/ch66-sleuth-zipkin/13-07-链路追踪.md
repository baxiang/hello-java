# 13.7 链路追踪

## 一、链路追踪简介

### 1.1 为什么需要链路追踪

**微服务问题**：
- 服务调用链复杂
- 问题定位困难
- 性能瓶颈难找

**链路追踪作用**：
- 追踪请求调用链
- 分析服务依赖
- 定位性能瓶颈
- 发现异常问题

### 1.2 核心概念

| 概念 | 说明 |
|------|------|
| Trace | 一次请求的完整链路 |
| Span | 链路中的一个节点（服务调用） |
| TraceId | 链路唯一标识 |
| SpanId | 节点唯一标识 |

## 三、整合 SkyWalking

### 3.1 添加依赖

```xml
<dependency>
    <groupId>org.apache.skywalking</groupId>
    <artifactId>apm-toolkit-trace</artifactId>
    <version>9.0.0</version>
</dependency>
```

### 3.2 配置应用

```yaml
spring:
  application:
    name: service-user
  sleuth:
    sampler:
      probability: 1.0  # 采样率 100%
```

### 3.3 启动参数

```bash
# 下载 SkyWalking agent
# 配置 agent.config
vim agent/config/agent.config

# 启动应用
java -javaagent:/path/to/skywalking/agent/skywalking-agent.jar \
     -Dskywalking.agent.service_name=service-user \
     -Dskywalking.collector.backend_service=localhost:11800 \
     -jar service-user.jar
```

## 八、综合示例

### 8.1 完整链路配置

```yaml
# 所有服务统一配置
spring:
  zipkin:
    base-url: http://zipkin:9411
    sender:
      type: web
  sleuth:
    enabled: true
    sampler:
      probability: 1.0
    propagation:
      type: B3
```

### 8.2 自定义 Span

```java
@Service
public class OrderService {
    
    @Autowired
    private Tracer tracer;
    
    public Order createOrder(OrderDTO dto) {
        // 创建自定义 Span
        Span newSpan = tracer.nextSpan().name("createOrder").start();
        
        try (Tracer.SpanInScope ws = tracer.withSpan(newSpan)) {
            // 添加标签
            newSpan.tag("orderId", dto.getId().toString());
            newSpan.tag("userId", dto.getUserId().toString());
            newSpan.tag("amount", dto.getAmount().toString());
            
            // 业务逻辑
            Order order = orderMapper.insert(dto);
            
            return order;
        } catch (Exception e) {
            // 记录异常
            newSpan.error(e);
            throw e;
        } finally {
            newSpan.end();
        }
    }
}
```

### 8.3 异步链路追踪

```java
@Service
public class AsyncService {
    
    @Autowired
    private Tracer tracer;
    
    @Async
    public void asyncTask() {
        // 传递 Trace 上下文
        try (Tracer.SpanInScope ws = tracer.withSpan(tracer.currentSpan())) {
            // 异步业务逻辑
            System.out.println("异步任务执行");
        }
    }
}
```
